केवळ पूर्ण सेटमध्ये

ऑड्यासिटी विकास माहितीपुस्तिकेवरून
येथे जा: निर्देशक, शोध
जेव्हा आपण आड़मार्ग पहाल तेव्हा Ctrl + J   Extra i.e. with the word 'Extra' as a superscript, it means that the shortcut is only available if you choose the full set of shortcuts. The standard default set of shortcuts does not include it
  • पुर्वनिर्धारित: ऑड्यासिटी मध्ये पुर्वनिर्धारित आड़मार्गचे दोन संच उपलब्ध आहेत. पुर्वनिर्धारित सेट आहे "मानक". आड़मार्गचा संच काहीसा सुलभ करण्यासाठी आणि स्वत: चा आड़मार्ग तयार करू इच्छिणाया वापरकर्त्यांसाठी अधिक फ्लेक्सिबिल्ट प्रदान करण्यासाठी हा सेट कमी केलेला संच आहे.
आड़मार्गचा इतर उपलब्ध संच आहे"पूर्ण". हा सेट आहे, जसे नावाने ऑड्यासिटी २.१..3 आणि पूर्वीच्या काळात उपलब्ध आड़मार्गचा पूर्ण आणि विस्तृत सेट सूचित केला आहे. आपण निवडून आड़मार्गच्या पूर्ण संचावर परत जाणे निवडू शकता "Full" वरून प्रवेश केलेल्या ड्रॉपडाउन मेनूमधून पुर्वनिर्धारित मध्ये बटण कळफलक प्राधान्ये संवाद.

 


आपण वापरू शकता पुर्वनिर्धारित कोणत्याही वेळी आड़मार्गच्या दोन प्रदान केलेल्या पुर्वनिर्धारित सेट दरम्यान स्विच करण्यासाठी बटण.
आपण सानुकूल आड़मार्ग बनवत असल्यास आणि आपले बदल योग्य न झाल्याचे आणि पुन्हा प्रारंभ करू इच्छित असल्यास हे बटण सुलभ आहे.
