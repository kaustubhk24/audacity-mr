एलओएफ(LOF) धारीका

ओड्यासिटी विकास महितीपुस्तिका पासून
वर जा: निर्देशक, शोध
एक एलओएफ(LOF) धारीका एक साधा मजकूर फाईल आहे जी ".लोफ" विस्तारासह समाप्त होते ज्यामध्ये ओड्यासिटी प्रकल्प विंडोमध्ये आयात करण्यासाठी आवाज धारीकाची सूची असते. यात नवीन प्रकल्प विंडोज तयार करण्यासाठी आज्ञा वैकल्पिकरित्या असू शकतात.

मांडणी

तेथे फक्त दोन आज्ञा आहेत "धारीका" आणि "विंडो", प्रत्येक संबंधित पर्यायांसहः

  • धारीका: (आवश्यक) त्या संपूर्ण मार्ग निर्दिष्ट करते जे सध्याच्या विंडोमध्ये नवीन गितपट्टा म्हणून आयात करते (पथ नेहमीच दुहेरी अवतरणात असावा)
    • ऑफसेट : फाईल आदेशासाठी पर्यायी मापदंड, सेकंदात ध्वनी धारीकाचा प्रारंभ वेळ निर्दिष्ट
  • विंडो : (पर्यायी) सूचीमधील त्यानंतरच्या धारीकासाठी नवीन ओड्यासिटी प्रकल्प विंडो तयार करते (विद्यमान प्रकल्प रिक्त नसल्यास प्रथम फाइल नेहमीच नवीन विंडो तयार करते) )
    • ऑफसेट : विंडो आदेशाचे वैकल्पिक मापदंड, विंडोमध्ये डावीकडील सर्वात जास्त वेळ दर्शविणारा निर्दिष्ट
    • कालावधी : विंडो आदेशाचे वैकल्पिक मापदंड, विंडोमध्ये किती वेळ दर्शविला जावा हे निर्दिष्ट करते
  • #: (पर्यायी) एका टिप्पणीच्या आधी एका ओळीवर कोठेही वापरले जाऊ शकते - त्या ओळीवरील # नंतर सर्वकाही आयातकाकडे दुर्लक्ष केले जाते.

टिपा:

  • प्रत्येक आज्ञा वेगळ्या ओळीवर ठेवणे आवश्यक आहे.
  • एलओएफ धारीकामध्ये दर्शविलेल्या क्रमाने धारीका आयात केल्या जातात.
  • जर आयात करायची धारीका एलओएफ धारीकाच्या समान धारीकामध्ये असेल तर आपल्याला दुहेरी अवतरणातील धारीकाचे नाव देणे आवश्यक आहे.

उदाहरण

रिकामी प्रोजेक्ट विंडो दिल्यास, सी येथे असलेली खालील एलओएफ धारीका डेस्कटॉप त्या विंडोमध्ये डेस्कटॉपवरून "नमूना 1.wav" ध्वनी धारीका आयात करते आणि त्याखालील खाली "नमून 2.wav" आणि "sample.wav" ध्वनी धारीका आयात करते. मग एक नवीन प्रोजेक्ट विंडो तयार होईल आणि त्यामध्ये डेस्कटॉप वरून "sample.wav" ध्वनी धारीका आयात केली जाईल. 

# कॅरेक्टरच्या खालील सर्व गोष्टींकडे दुर्लक्ष केले जाते
विंडो # इनिशियल विंडो आज्ञा अंतर्निहित आणि पर्यायी आहे
file "sample1.wav"                   # sample1.wav is displayed
file "C:\Ken\sample2.wav" offset 5   # sample2.wav is displayed with a 5s offset
file "C:\Dave\sample3.wav"           # sample3.wav is displayed with no offset
window offset 5 duration 10          # open a new window, then zoom to display
5 सेकंद ते 15 सेकंद ते नमुना 4.wav चे 10 सेकंद, ते 2.5 सेकंदाने ऑफसेट करत आहे
धारीका "sample4.wav" ऑफसेट 2.5

दर्शविलेल्या धारीकाची नावे विंडोज सिस्टमसाठी आहेत, परंतु समान सिंटॅक्स योग्य धारीका पथातील स्पष्ट बदलासह मॅकिंटोश आणि लिनक्सवर लागू होते.
