23 An AUP file can only be saved and used in Audacity. It is just a small text file that tells Audacity how to work with the audio you see in the project window. It lets you resume working on that audio in Audacity at a later time if you need to. एक एयूपी धारीका केवळ जतन केली जाऊ शकते आणि ओड्यासिटीमध्ये वापरली जाऊ शकते. ही केवळ एक लहान मजकूर धारीका आहे जी आपल्याला प्रकल्प विंडोमध्ये दिसत असलेल्या ध्वनीसह कार्य कसे करावे हे ओड्यासिटीला सांगते.आपल्याला आपल्यास आवश्यक असल्यास त्या ओड्यासिटीमध्ये त्या ध्वनीवर नंतर पुन्हा कार्य करण्यास अनुमती देते.

34 Why does Audacity create a folder full of .au files when I save a project? मी प्रकल्प जतन करतेवेळी ओड्यासिटी .au धारीकांनी भरलेले निर्देशिका का तयार करते?
35 Audacity breaks long tracks into small pieces so it can edit them more efficiently. When you save an AUP project file, Audacity stores all the pieces in a _data folder that has the same name as the AUP file. For example, saving "song.aup" saves the required data pieces as AU files inside the "song_data" folder. ओड्यासिटी लांब गितपट्टा लहान तुकडे करते जेणेकरून ते अधिक कार्यक्षमतेने संपादित करू शकेल.जेव्हा आपण एक एओपी प्रकल्प धारीका जतन करता तेव्हा ओड्यासिटी सर्व तुकडे _ डेटा डेटामध्ये ठेवते ज्याचे नाव एयूपी धारीका आहे.उदाहरणार्थ, "सॉन्ग.ऑप" जतन करणे "गाणे_डाटा" निर्देशिकामध्ये एयू धारीका म्हणून आवश्यक डेटा तुकडे जतन करते.
36 To open the project, open the AUP file then Audacity will load the AU files in the correct sequence automatically. A project you saved recently can be opened from File > Recent Files. प्रकल्प उघडण्यासाठी, AUP धारीका उघडा नंतर ओड्यासिटी योग्य पद्धतीने एयू धारीका स्वयंचलितपणे लोड करेल.आपण अलीकडे जतन केलेला प्रकल्प धारीकाअलीकडील धारीका वरून उघडला जाऊ शकतो .
37 Long projects contain a lot of data. Be sure the project is fully saved before exiting Audacity or shutting down the computer. If there is a progress dialog for saving the project, wait for it to complete before shutting down. The Status Bar at the bottom shows a message when the project has been saved. लांब प्रकल्पांमध्ये भरपूर डेटा असतो. ओड्यासिटीतून बाहेर पडण्यापूर्वी किंवा संगणक बंद करण्यापूर्वी प्रकल्प पूर्णपणे जतन झाला आहे याची खात्री करा. प्रकल्प जतन करण्यासाठी प्रगती संवाद असल्यास, बंद होण्यापूर्वी ते पूर्ण होण्याची नक्कल करा.प्रकल्प जतन झाल्यावर तळाशीअसलेली सटीक बार एक संदेश दर्शवितो.
38 You should not move, delete or rename the _data folder or any of the AU files inside it. आपण त्यामध्ये _डेटा निर्देशिका किंवा कोणत्याही एयू धारीका हलवू, हटवू किंवा पुनर्नामित करू नये.
39 To learn more about working with Audacity projects, please see Audacity Projects. ओड्यासिटी प्रकल्प्ससह कार्य करण्याबद्दल अधिक जाणून घेण्यासाठी, कृपया ओड्यासिटी प्रकल्प पहा.

However it is a wise precaution to backup the project periodically to separately named AUP files in case Audacity is unable to recover correctly.
