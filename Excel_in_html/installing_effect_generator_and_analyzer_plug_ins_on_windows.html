५९ ५९ Installing Vamp plug-ins व्हँप प्लग-इन स्थापित करीत आहे
६० ६० Vamp plug-ins are usually for analyzing audio so will appear under Audacity's Analyze Menu. You can do interesting things like attempt to track beats, note pitches, chords or frequencies. Any Vamp plug-ins whose output is suitable for a label track should work in Audacity on Windows. To add a new Vamp analysis tool, add the plug-in's DLL, DYLIB or SO file and any supplied category or RDF files to any of the Vamp search locations then enable the tool in the Plug-in Manager: Effects, Generators and Analyzers dialog. व्हँप प्लग-इन सहसा ध्वनी विश्लेषणासाठी असतात जेणेकरून ओड्यासिटीच्या विश्लेषण मेनू अंतर्गत दिसतील. आपण बीट्सचा मागोवा घेण्याचा प्रयत्न करणे, पिच नोट्स, जीवा किंवा वारंवारितांचा मागोवा घेणे यासारख्या मनोरंजक गोष्टी करू शकता. लेबल गीतपट्टासाठी योग्य असे कोणतेही व्हँप प्लगइन विंडोजवरील ओड्यासिटीमध्ये कार्य केले पाहिजेत. नवीन व्हँप एनालिसिस साधन जोडण्यासाठी, प्लग-इनची डीएलएल, डीवायआयएलबी किंवा एसओ धारिका आणि कोणत्याही पुरविलेल्या श्रेणी किंवा आरडीएफ धारिका व्हॅम्प शोध स्थानांवर जोडा त्यानंतर प्लग-इन व्यवस्थापकामधील साधन सक्षम करा: प्रभाव, जनरेटर आणि विश्लेषक संवाद .
