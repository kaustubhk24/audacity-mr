1६ 1६ Advanced Mixing Options प्रगत मिश्रण पर्याय
1७ 1७ By default, all exports are mixed down to mono (one channel) or stereo (two channels). If you want to export to more channels, enable "Use custom mix" in Import / Export Preferences. After pressing "Save" in the export dialog, the Advanced Mixing Options dialog appears where you can map the Audacity tracks to channels. Metadata Editor (if enabled) appears after OK'ing the Advanced Mixing Options. पूर्वनिर्धारितनुसार, सर्व निर्यात खाली मोनो (एक वाहिनी) किंवा स्टीरिओ (दोन वाहिनी) मध्ये मिसळली जातात. आपण अधिक वाहिनीवर निर्यात करू इच्छित असल्यास, आयात / निर्यात प्राधान्यांमध्ये "सानुकूल मिश्रित वापरा" सक्षम करा. निर्यात संवादात "जतन करा" दाबल्यानंतर, प्रगत मिक्सिंग पर्याय संवाद दिसून येईल जिथे आपण वाहिनीवर ओड्यासिटी गीतपट्टाचा नकाशा बनवू शकता. प्रगत मिश्रण पर्याय ओके केल्यावर मेटाडेटा संपादक (सक्षम असल्यास) दिसून येईल.
1८ 1८ Custom FFmpeg export options सानुकूल FFmpeg निर्यात पर्याय
1९ 1९ In the Export dialog when you select Custom FFmpeg Export in the Save as type dropdown, then the Options pane will contain the Open custom FFmpeg format options button. निर्यात संवादात जेव्हा आपण जतन म्हणून टाइप ड्रॉपडाऊनमध्ये कस्टम एफएफएमपीएग निर्यातनिवडता तेव्हा ऑप्शन्स पॅनमध्ये ओपन कस्टम एफएफम्पेग फॉर्मेट ऑप्शन बटण असेल.
20 20 Clicking on the button will bring up the custom options dialog. See Custom FFmpeg Export Options for more details. बटणावर क्लिक करणे सानुकूल पर्याय संवाद आणेल. अधिक तपशीलांसाठी सानुकूल एफएफम्पेग निर्यात पर्याय पहा.
21 21 Note that when using custom FFmpeg export it is necessary to supply the full file name and the extension in the File name: field as Audacity does not automatically add the extension in this case. लक्षात ठेवा सानुकूल एफएफम्पेप निर्यात वापरताना संपूर्ण धारिका नाव आणि धारिकाच्या नावामध्ये विस्तार प्रदान करणे आवश्यक आहे: ओड्यासिटी म्हणून फील्ड या प्रकरणात आपोआप विस्तार जोडत नाही.
22 22 Otherwise you will need to manually add the extension to the file name after export. अन्यथा निर्यातानंतर धारिकाच्या नावामध्ये आपल्याला व्यक्तिचलितरित्या विस्तार जोडण्याची आवश्यकता असेल.
