७ ७ MIDI Devices Preferences. एमआयडीआय उपकरण प्राधान्ये.
८ ८ Interface संवादपटल
९ ९ The interface that Audacity should use through PortMidi. ओड्यासिटीचा संवादपटल पोर्टमिडीद्वारे वापरावा.
१० १० The available options depend on the OS, but there is only one option for each OS: उपलब्ध पर्याय ओएसवर अवलंबून आहेत, परंतु प्रत्येक ओएससाठी एकच पर्याय आहेः
११ ११ On Windows, it is MMSystem. विंडोजवर, ते एमएमएससिस्टम आहे.
१2 १2 On Mac, it is CoreMIDI. मॅक वर, ते कोअरमिडी आहे.
१३ १३ On Linux, it is ALSA. लिनक्स वर ते ALSA आहे.
१४ १४ Playback प्लेबॅक
१५ १५ Device: The MIDI device to play to. उपकरण: प्ले करण्यासाठी एमआयडीआय उपकरण.
१६ १६ MIDI Synthesizer Latency (ms): The latency field enables compensation for a MIDI device that has latency between receiving events and actually playing them - it provides a way to synchronize audio. मिडी सिंथेसाइझर लेटन्सी (एमएस): लेटेंसी फील्ड मिडी उपकरणसाठी भरपाई सक्षम करते ज्यामध्ये इव्हेंट प्राप्त करणे आणि प्रत्यक्षात ते प्ले करणे यामध्ये विलंब होते - हे ध्वनी समक्रमित करण्याचा एक मार्ग प्रदान करते.
