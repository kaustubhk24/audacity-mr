4 4 This set of commands let you define selection regions and store and retrieve the position of the editing cursor or selection region. कमांडचा हा सेट आपल्याला निवड क्षेत्रे परिभाषित करू देतो आणि संपादन कर्सर किंवा निवड प्रदेशाची स्थिती संचयित आणि पुनर्प्राप्त करू देतो.

6 6 Left at Playback Position  [ प्लेबॅक स्थानावर डावीकडे [
7 7 When playing or recording: Sets the left boundary of a potential selection by moving the cursor to the current position of the green playback cursor (or red recording cursor). The selection can then be drawn by using "Right at Playback Position" (below). प्ले करताना किंवा रेकॉर्डिंग करताना: कर्सरला ग्रीन प्लेबॅक कर्सरच्या (किंवा लाल रेकॉर्डिंग कर्सर) सद्य स्थितीत हलवून संभाव्य निवडीची डावे सीमा सेट करते. त्यानंतर "राइट Playट प्लेबॅक पोझिशन" (खाली) वापरून निवड काढता येते.
8 8 When there is no audio playing or recording: Opens the "Set Left Selection Boundary" dialog. This is useful for keyboard users or for time-exact manipulation of the selection instead of using Selection Toolbar. Move the selection boundary dialog's time digits backwards to create a selection leftwards from the cursor position or to extend the current selection leftwards. Move the time digits forwards to move the cursor forwards to an exact point or to contract the current selection from the left. See "Creating selections at an exact time" for more details. जेव्हा कोणतेही ऑडिओ प्लेइंग किंवा रेकॉर्डिंग नसते: "डावे निवड सीमा सेट करा" संवाद उघडते. हे कीबोर्ड वापरकर्त्यांसाठी किंवा निवड टूलबार वापरण्याऐवजी वेळेच्या अचूक हाताळणीसाठी उपयुक्त आहे. कर्सर स्थानावरून डावीकडील निवड तयार करण्यासाठी किंवा सद्य निवडी डावीकडील बाजूपर्यंत वाढविण्यासाठी निवड सीमा संवादाचे वेळ अंक मागे हलवा. कर्सर पुढे अचूक बिंदूकडे नेण्यासाठी किंवा डावीकडील सद्य निवडीस संकुचित करण्यासाठी वेळ अंक पुढे करा. अधिक तपशीलांसाठी "अचूक वेळी निवडी तयार करणे" पहा.
9 9 Right at Playback Position  ] उजवीकडे प्लेबॅक स्थितीत]
10 10 When playing or recording: Sets the right boundary of the selection, thus drawing the selection from the cursor position to the current position of the green playback cursor (or red recording cursor). This is a quick way to mark a selection for future editing while playing or recording. Selections can be labeled while playing or recording using Edit > Labels > Add Label at Selection or shortcut Ctrl + B. प्ले करताना किंवा रेकॉर्डिंग करताना: निवडीची योग्य सीमा सेट करते, अशा प्रकारे निवड कर्सर स्थानावरून ग्रीन प्लेबॅक कर्सरच्या (किंवा लाल रेकॉर्डिंग कर्सर) सद्य स्थितीकडे आणते. खेळताना किंवा रेकॉर्डिंग करताना भविष्यातील संपादनासाठी निवड चिन्हांकित करण्याचा हा एक द्रुत मार्ग आहे. संपादन> लेबले> निवड येथे लेबल जोडा किंवा शॉर्टकट Ctrl + B वापरुन निवडताना प्ले किंवा रेकॉर्डिंग करताना लेबल दिले जाऊ शकतात.
11 11 It will usually be more convenient while playing or recording to use the equivalent keyboard shortcuts ( by default,  [  and  ]   ) to set the left and right boundary respectively. अनुक्रमे डावी आणि उजवी सीमा सेट करण्यासाठी समकक्ष कीबोर्ड शॉर्टकट (डीफॉल्टनुसार, [आणि]) वापरण्यासाठी प्ले करताना किंवा रेकॉर्डिंग करताना हे अधिक सोयीचे असेल.
12 12 When there is no audio playing or recording: Opens the "Set Right Selection Boundary" dialog. Move the dialog's time digits forwards to create a selection rightwards from the cursor position or to extend the current selection rightwards. Move the time digits backwards to move the cursor backwards to an exact point or to contract the current selection from the right. जेव्हा कोणतेही ऑडिओ प्लेइंग किंवा रेकॉर्डिंग नसते: "उजवे निवड सीमा सेट करा" संवाद उघडते. कर्सर स्थानावरून उजवीकडील निवड करण्यासाठी किंवा सद्यस्थिती निवड उजवीकडे वाढविण्यासाठी संवादातील वेळ अंक पुढे हलवा. अचूक बिंदूकडे कर्सर मागे सरकण्यासाठी किंवा उजवीकडील सद्य निवडीस संकुचित करण्यासाठी वेळ अंक मागे वळा.
13 13 Track Start to Cursor  Shift + J ट्रॅक स्टार्ट टू कर्सर शिफ्ट + जे
14 14 Selects a region in the selected track(s) from the start of the track to the cursor position (or extends the current selection so that it starts from the start of the track). ट्रॅकच्या सुरूवातीपासून कर्सर स्थानापर्यंत (किंवा) निवडलेल्या ट्रॅकमधील एक प्रदेश निवडते (किंवा सद्य निवडी वाढवते जेणेकरून ती ट्रॅकच्या सुरूवातीपासून सुरू होते).
15 15 Cursor to Track End  Shift + K एंड शिफ्ट + के ट्रॅक करण्यासाठी कर्सर
16 16 Selects a region in the selected track(s) from the cursor position to the end of the track (or extends the current selection so that it ends at the end of the track). निवडलेल्या ट्रॅकमधील कर्सर स्थानापासून ट्रॅकच्या शेवटी (किंवा सद्य निवडी वाढवते जेणेकरून ती ट्रॅकच्या शेवटी समाप्त होते) निवडते.
17 17 Store Selection स्टोअर निवड
18 18 Stores the current position of the selection region or cursor, as observed on the Timeline. The position of this region or cursor can then be recalled by using Retrieve Selection (see below). Using this command again will overwrite the previously stored position information. टाइमलाइनवर पाहिल्यानुसार निवड प्रदेश किंवा कर्सरची सद्य स्थिती संचयित करते. नंतर या प्रदेशाचे स्थान किंवा कर्सर पुन्हा निवड पुनर्प्राप्त (पुन्हा पहा) वापरून परत लक्षात येऊ शकते. हा आदेश पुन्हा वापरल्याने पूर्वी संग्रहित स्थिती माहिती अधिलिखित होईल.
19 19 The position is saved in memory for each project. Therefore using Retrieve Region will not have any effect in another project unless you have already used Store Region in that project. When you close a project, its stored region is discarded. प्रत्येक प्रकल्पासाठी स्थिती स्मृतीत जतन केली जाते. म्हणूनच आपण या प्रकल्पात आधीपासून स्टोअर प्रदेश वापरल्याशिवाय रीट्रीव्ह रीजनचा वापर दुसर्‍या प्रोजेक्टमध्ये होणार नाही. आपण एखादा प्रकल्प बंद करता तेव्हा त्याचा संचयित क्षेत्र टाकून दिले जाते.
20 20 The Store Region command is grayed-out if there are no audio tracks. ऑडिओ ट्रॅक नसल्यास स्टोअर रीजन कमांड ग्रे आहे.
21 21 Retrieve Selection निवड पुनर्प्राप्त करा
22 22 Moves the selection region or cursor to the position stored by the last Store Selection command (see above). If a region is retrieved, this is only explicitly placed inside those tracks which are currently selected. शेवटच्या स्टोअर सिलेक्शन कमांडद्वारे संग्रहित स्थानावर निवड क्षेत्र किंवा कर्सर हलवते (वरील पहा). जर एखादा प्रदेश पुनर्प्राप्त केला असेल तर हे केवळ निवडलेल्या त्या ट्रॅकमध्ये स्पष्टपणे ठेवले आहे.
23 23 Only the position of the region or cursor is retrieved. No audio content is retrieved. Labels or split lines that were associated with the region or cursor are not retrieved. केवळ प्रदेश किंवा कर्सरचे स्थान प्राप्त केले आहे. कोणतीही ऑडिओ सामग्री पुनर्प्राप्त केलेली नाही. प्रदेश किंवा कर्सरशी संबंधित लेबल किंवा स्प्लिट लाइन पुनर्प्राप्त केल्या नाहीत.
24 24 The screen focus does not return to the retrieved position, so you may need to zoom out or scroll the project to see the retrieved position. स्क्रीन फोकस पुनर्प्राप्त स्थितीवर परत येत नाही, म्हणून पुनर्प्राप्त स्थिती पाहण्यासाठी आपल्याला प्रकल्प झूम कमी करणे किंवा स्क्रोल करणे आवश्यक असू शकते.
25 25 If the region was stored when in Spectrogram view, the spectral selection within the region is also retrieved, even if the region is retrieved when in another view. If the cursor position was stored when in Spectrogram view, the vertical position (and hence center frequency) of the click that placed the cursor is retrieved. स्पेक्ट्रोग्राम दृश्यात असताना हा प्रदेश संग्रहित केला गेला असल्यास, त्या प्रदेशातील वर्णक्रमीय निवड देखील पुनर्प्राप्त केली गेली, जरी दुसर्‍या दृश्यात असताना हा प्रदेश पुनर्प्राप्त केला गेला. स्पेक्ट्रोग्राम व्ह्यूमध्ये असताना कर्सर स्थान संचयित केले असल्यास, कर्सर ठेवलेल्या क्लिकची अनुलंब स्थिती (आणि म्हणूनच केंद्र वारंवारता) पुनर्प्राप्त होईल.
