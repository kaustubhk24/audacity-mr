४ ४ This page explains how to align multiple tracks so that their audio starts at the same point on the timeline above the waveform. For example you would often want to do this when you have recorded one track against another and need to ensure the recorded track is properly synchronized with the other track(s). हे पृष्ठ एकाधिक गीतपट्टा संरेखित कसे करावे हे स्पष्ट करते जेणेकरून त्यांचा ध्वनी त्याच वेगापासून तरंगरूपच्या वरील वेळलाइनवर प्रारंभ होईल. उदाहरणार्थ जेव्हा आपण एखादा गीतपट्टा दुसर्‍या विरूद्ध ध्वनीमुद्रण केला असेल तेव्हा आपणास असे करणे आवडेल आणि ध्वनीमुद्रण केलेला गीतपट्टा इतर गीतपट्टा (टे) सह योग्यरितीने संकालित केला गेला पाहिजे याची खात्री करा.

१० १० Using the Tracks Menu गीतपट्टा मेनू वापरणे
११ ११ The menu commands in the Tracks > Align Tracks submenu provide an automatic way of aligning selected tracks with the cursor, the selection, or with the start of the project: गीतपट्टा मध्ये संरेखित मेनू आज्ञा> गीतपट्टा संरेखित करा सबमेनू कर्सर, निवड किंवा प्रकल्पाच्या सुरूवातीस निवडलेले गीतपट्टा संरेखित करण्याचा स्वयंचलित मार्ग प्रदान करते:
१२ १२ Align End to End: Aligns the selected tracks one after the other, based on their top-to-bottom order in the project window. The first selected track remains where it is, the start of the second selected track aligns with the end of the first selected track and so on. समाप्ती ते समाप्ती संरेखित करा: प्रकल्प विंडोमधील त्यांच्या वरच्या-खालच्या क्रमावर आधारित निवडलेले गीतपट्टा एकामागून एक संरेखित करते. पहिला निवडलेला गीतपट्टा जिथे आहे तिथेच राहतो, दुस selected्या निवडलेल्या गीतपट्टाची सुरूवात पहिल्या निवडलेल्या गीतपट्टाच्या शेवटी संरेखित होते आणि याप्रमाणे.
१३ १३ Align Together: एकत्र संरेखित करा:
१४ १४ If you select multiple tracks, this command moves the tracks so that they all start at the same time, so audio within them changes its synchronization. आपण एकाधिक गीतपट्टा निवडल्यास, ही आज्ञा गीतपट्टा हलवते जेणेकरून ते सर्व एकाच वेळी प्रारंभ होतील, म्हणून त्यांच्यामधील ध्वनी त्याचे संकालन बदलू शकेल.
१५ १५ Start to Zero: Aligns the start of selected tracks with the start of the project. झिरोपासून प्रारंभ करा: प्रकल्प सुरू झाल्याने निवडलेल्या गीतपट्टाची सुरूवात संरेखित करते.
१६ १६ Start to Cursor/Selection Start: Aligns the start of selected tracks with the current cursor position or with the start of the current selection. कर्सर / निवड प्रारंभ करा प्रारंभः वर्तमान कर्सर स्थानासह किंवा वर्तमान निवडीच्या प्रारंभासह निवडलेल्या गीतपट्टाचा प्रारंभ संरेखित करते.
१७ १७ Start to Selection End: Aligns the start of selected tracks with the end of the current selection. निवडीची समाप्ती सुरू करा: सध्याच्या निवडीच्या समाप्तीसह निवडलेल्या गीतपट्टाची सुरूवात संरेखित करते.
१८ १८ End to Cursor/Selection Start: Aligns the end of selected tracks with the current cursor position or with the start of the current selection. कर्सर / निवडीची समाप्ती समाप्त: निवडलेल्या गीतपट्टाचा शेवटच्या कर्सर स्थानासह किंवा वर्तमान निवडीच्या प्रारंभासह संरेखित करते.
१९ १९ End to Selection End: Aligns the end of selected tracks with the end of the current selection. निवडीची समाप्ती शेवट: वर्तमान निवडीच्या समाप्तीसह निवडलेल्या गीतपट्टाचा शेवट संरेखित करते.
२० २० If Sync-Lock Tracks is enabled, unselected tracks in a Sync-Locked Track Group will also move so that all the tracks in that group remain synchronized, except for the Align "End to End" and "Together" commands where only the selected tracks will move. Aligning end to end or together overrides Sync-Lock Tracks if necessary, so the selected tracks in that group may become desynchronized. संकालन-लॉक गीतपट्टा सक्षम केलेले असल्यास, संकालित-लॉक केलेला गीतपट्टा गटातील निवड रद्द केलेले गीतपट्टा देखील हलतील जेणेकरून त्या गटातील सर्व गीतपट्टा संकालित राहतील, "एंड टू एंड" संरेखित करा आणि फक्त निवडलेले गीतपट्टा हलवेल समाप्तीची समाप्ती संरेखित करणे किंवा एकत्रितपणे आवश्यक असल्यास समक्रमण-लॉक गीतपट्टा अधिलिखित करते, जेणेकरून त्या गटामधील निवडलेले गीतपट्टा विभाजित होऊ शकतात.
२१ २१ Time Shift Tool वेळ शिफ्ट साधन
२२ २२ An easy way for sighted users to align tracks with each other दृष्टी असलेल्या वापरकर्त्यांचा एकमेकांशी गीतपट्टा संरेखित करण्याचा सोपा मार्ग
२३ २३ is to use the Time Shift Tool म्हणजे वेळ शिफ्ट साधन वापरणे
२४ २४ which is located at the top of the Audacity screen in the Tools Toolbar. जी साधने साधनपट्टीमधील ओड्यासिटी स्क्रीनच्या शीर्षस्थानी आहे.
२५ २५ To use the Time Shift Tool, click on the वेळ शिफ्ट साधन वापरण्यासाठी, वर क्लिक करा
२६ २६ button so it appears to be depressed, or press its shortcut F५. If you wish to change its shortcut to something other than F५, you can do so on the Keyboard tab of Preferences. Then zoom in with Ctrl + १ (or ⌘ + १ on a Mac) so you can make a more precise alignment. Click in the track you want to move with the left button of the mouse, hold the button down while dragging the track to your required position, then release the mouse button. बटण जेणेकरून ते निराश झाल्यासारखे दिसते किंवा त्याचे शॉर्टकट F५ दाबा. आपण त्याचे शॉर्टकट एफ ५ व्यतिरिक्त अन्य कशासाठी बदलू इच्छित असल्यास आपण हे प्राधान्यांच्या कीबोर्ड टॅबवर करू शकता. त्यानंतर Ctrl + १ (किंवा मॅकवरील ⌘ + १) सह झूम वाढवा जेणेकरून आपण अधिक अचूक संरेखन करू शकता. माउसच्या डाव्या बटणासह आपण हलवू इच्छित गीतपट्टावर क्लिक करा, गीतपट्टा आपल्या आवश्यक स्थानावर ड्रॅग करताना बटण दाबून ठेवा, त्यानंतर माउस बटण सोडा.
२७ २७ Aligning tracks without a mouse - for visually impaired users दृष्टिहीन वापरकर्त्यांसाठी - माउसशिवाय गीतपट्टा संरेखित करीत आहे
२८ २८ Visually impaired users can use this Time Shifter plug-in to time-shift tracks without a mouse. This is one of a number of दृष्टिहीन वापरकर्ते हे वेळ शिफ्टर प्लग-इन माउसशिवाय वेळ-शिफ्ट गीतपट्टामध्ये वापरू शकतात. हे असंख्यांपैकी एक आहे
२९ २९ Nyquist plug-ins available for use in Audacity and works on both mono and stereo tracks. Instructions are given in Timeshif.TXT inside the zip folder. To install new plug-ins, place them in the "Plug-Ins" folder inside the Audacity installation folder. On Windows computers, your installation folder is usually under "Program Files". On Mac, it is usually under "अनुप्रयोग". Time shifter will appear in the Effect Menu underneath the separator after you restart Audacity. ओड्यासिटीमध्ये वापरासाठी Nyquist प्लग-इन उपलब्ध आहेत आणि दोन्ही मोनो आणि स्टिरीओ गीतपट्टावर कार्य करतात. पिन निर्देशिकामध्ये वेळशीफ.टीएक्सटी मध्ये सूचना दिल्या आहेत. नवीन प्लग-इन स्थापित करण्यासाठी, त्यांना ओड्यासिटी स्थापना निर्देशिकामध्ये "प्लग-इन" निर्देशिकामध्ये ठेवा. विंडोज संगणकांवर, आपले इन्स्टॉलेशन निर्देशिका सामान्यत: "प्रोग्राम धारिका्स" अंतर्गत असते. मॅक वर, हे सहसा ""अनुप्रयोग्स" अंतर्गत असते. आपण ओड्यासिटी रीस्टार्ट केल्यानंतर वेळ शिफ्टर विभाजक खाली मेनूमध्ये परिणाममेनूमध्ये दिसून येईल.
३० ३० Marking align points संरेखित बिंदू चिन्हांकित करीत आहे
३१ ३१ Some users find it easier to align tracks together when there is a vertical reference point extending down through all the tracks. जेव्हा सर्व गीतपट्टामधून एक अनुलंब संदर्भ बिंदू वाढविला जातो तेव्हा काही वापरकर्त्यांना गीतपट्टा एकत्र संरेखित करणे सोपे होते.
३२ ३२ You can make such a reference point by dragging the cursor down कर्सर खाली ड्रॅग करून आपण असा संदर्भ बिंदू बनवू शकता
३३ ३३ into the other tracks, and optionally by marking cursor positions with labels. इतर गीतपट्टामध्ये आणि वैकल्पिकरित्या लेबलसह कर्सर स्थान चिन्हांकित करून.
३४ ३४ To drag the cursor into the other tracks, make sure you are using the Selection Tool इतर गीतपट्टामध्ये कर्सर ड्रॅग करण्यासाठी, आपण निवड साधन वापरत असल्याचे सुनिश्चित करा
३५ ३५ (shortcut F१), and place your cursor at a point where you want to line up the tracks. Then hold down Shift on your keyboard and click in the Track Control Panels (where the mute/solo buttons are) of all your other tracks. Now you'll see the cursor displayed in all the tracks, and you can switch to the Time Shift Tool and line up all the tracks against the cursor line. (शॉर्टकट एफ १) वर क्लिक करा आणि आपला कर्सर अशा ठिकाणी ठेवा जेथे आपण गीतपट्टा लावायचे इच्छिता. नंतर आपल्या कीबोर्डवरील शिफ्ट दाबून ठेवा आणि आपल्या इतर सर्व गीतपट्टाच्या गीतपट्टा नियंत्रण पॅनेलमध्ये (जिथे निःशब्द / एकल बटणे आहेत) क्लिक करा. आता आपणास सर्व गीतपट्टामध्ये प्रदर्शित केलेला कर्सर दिसेल आणि आपण वेळ शिफ्ट साधनवर स्विच करू शकता आणि सर्व गीतपट्टा कर्सर लाइनच्या विरूद्ध तयार करू शकता.
३६ ३६ To add a label at a cursor point, use Edit > Labels > Add Label at Selection. You can use the shortcut Ctrl + B (or ⌘ + B on a Mac) instead. The label is created in a new Label Track underneath the audio tracks. When you click on the label, all tracks will be selected and the cursor will be displayed through all the tracks, just like when you selected all the tracks by clicking on their Track Control Panels. However the label provides a permanent reference point you can come back to, and because you can have multiple labels in a Label Track, there can be multiple points you can line up with. कर्सर बिंदूवर लेबल जोडण्यासाठी, संपादन> लेबल> निवडीवर लेबल जोडा. त्याऐवजी आपण Ctrl + B (किंवा मॅक वर ⌘ + बी) शॉर्टकट वापरू शकता. ध्वनी गीतपट्टाच्या खाली नवीन लेबल गीतपट्टामध्ये लेबल तयार केले गेले आहे. जेव्हा आपण लेबलवर क्लिक कराल, तेव्हा सर्व गीतपट्टा निवडले जातील आणि कर्सर सर्व गीतपट्टामधून दिसून येईल जसे की आपण त्यांच्या गीतपट्टा नियंत्रण पॅनेलवर क्लिक करून सर्व गीतपट्टा निवडले आहेत. तथापि हे लेबल कायम संदर्भ बिंदू प्रदान करते जिथे आपण परत येऊ शकता आणि आपल्याकडे एका लेबल गीतपट्टामध्ये एकाधिक लेबले असू शकतात, तेथे अनेक पॉइंट्स असू शकतात ज्या आपण एकत्र करू शकता.

३८ ३८ If you use the current version of Audacity, there is no visual guesswork involved in aligning tracks: you can Snap-To labels when you time shift a track towards them, and if you drag a selection region, the start point of the area on the time line is displayed dynamically in the Selection Toolbar as you drag. जर आपण ओड्यासिटीची सद्य आवृत्ती वापरत असाल तर गीतपट्टा संरेखित करण्यात कोणतेही दृश्य अंदाज कार्य करीत नाहीत: जेव्हा आपण त्यांच्याकडे एखादा गीतपट्टा स्थानांतरित करता तेव्हा आपण स्नॅप-टू लेबले वापरू शकता आणि आपण निवड प्रदेश ड्रॅग केल्यास त्यावरील क्षेत्राचा प्रारंभ बिंदू आपण ड्रॅग करतानानिवडसाधनपट्टीमध्ये वेळ लाइन गतिकरित्या दर्शविली जाते.
