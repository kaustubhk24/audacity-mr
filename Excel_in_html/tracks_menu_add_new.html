४ ४ This submenu adds new tracks in your project. हे सबमेनू आपल्या प्रकल्पात नवीन गीतपट्टा जोडते.
५ ५ Click on the image to learn more अधिक जाणून घेण्यासाठी प्रतिमेवर क्लिक करा
६ ६ There are five kinds of track: पाच प्रकारचे गीतपट्टा आहेत:
७ ७ Mono Tracks मोनो गीतपट्टा
८ ८ Stereo Tracks स्टिरिओ गीतपट्टा
९ ९ These two can be in waveform or spectrogram view. हे दोन वेव्हफॉर्म किंवा स्पेक्ट्रोग्राम दृश्यात असू शकतात.
१० १० Label Tracks - for annotating audio लेबल गीतपट्टा - भाष्य ध्वनीसाठी
११ ११ Time Tracks - for time warping the audio वेळ गीतपट्टा - वेळ ध्वनी warped साठी
१२ १२ Note Tracks - created by importing MIDI, and not from this menu. टीप गीतपट्टा - एमआयडीआय आयात करुन तयार केले आणि या मेनूमधून नाही.
१३ १३ Mono Track  Ctrl + Shift + N मोनो गीतपट्टा सीटीआरएल + शिफ्ट + एन
१४ १४ Extra अतिरिक्त
१५ १५ Creates a new, empty mono audio track. नवीन, रिक्त मोनो ध्वनी गीतपट्टा तयार करते.
१६ १६ This command is rarely needed, since importing, recording and mixing automatically create new tracks as needed. But you can use this to cut or copy data from an existing track and paste it into an empty track. ही आज्ञा क्वचितच आवश्यक आहे, आयात करणे, ध्वनीमुद्रण करणे आणि एकत्र करणे आवश्यकतेनुसार आपोआप नवीन गीतपट्टा तयार करते. परंतु आपण याचा वापर विद्यमान गीतपट्टावरून डेटा कट करण्यासाठी किंवा नक्कल करण्यासाठी आणि रिक्त गीतपट्टामध्ये पेस्ट करू शकता.
१७ १७ Stereo Track स्टिरिओ गीतपट्टा
१८ १८ Creates a new, empty stereo Audio Track. You may need this if you start with a set of mono tracks and want to export a stereo mix. नवीन, रिक्त स्टिरीओ ध्वनी गीतपट्टा तयार करते. आपण मोनो गीतपट्टाच्या संचासह प्रारंभ केल्यास आणि स्टिरिओ मिक्स निर्यात करू इच्छित असल्यास आपल्याला याची आवश्यकता असू शकेल.
१९ १९ Label Track लेबल गीतपट्टा
२० २० Creates a new, empty Label Track. When a label track has the yellow focus border you can create a label just by typing. Labels can be used for textual annotation and are commonly used when exporting multiple files from a single track. नवीन, रिक्त लेबल गीतपट्टा तयार करते. जेव्हा लेबल गीतपट्टामध्ये पिवळ्या रंगाची फोकस सीमा असते तेव्हा आपण फक्त टाइप करुन लेबल तयार करू शकता. लेबल मजकूर भाष्य करण्यासाठी वापरले जाऊ शकतात आणि सामान्यपणे एकाच गीतपट्टावरून एकाधिक धारिका निर्यात करताना वापरले जातात.
२१ २१ Time Track वेळ गीतपट्टा
२२ २२ Creates a Time Track that can progressively increase or decrease playback speed (and pitch) of all audio tracks in the project. The range of possible speed warp is set in the Set Range (upper and lower) option in the Time Track dropdown menu to left of the track. The amount of and direction of warping within that range is controlled using Envelope Tool. वेळ गीतपट्टा तयार करतो जो प्रकल्पातील सर्व ध्वनी गीतपट्टाचा प्लेबॅक गती (आणि खेळपट्टी) क्रमिकपणे वाढवू किंवा कमी करू शकतो. संभाव्य स्पीड वार्पची श्रेणी गीतपट्टाच्या डावीकडील वेळ गीतपट्टा ड्रॉपडाउन मेनूमध्ये स्थापित रेंज (अप्पर आणि लोअर) पर्यायात स्थापित केली जाते. लिफाफा साधन वापरुन त्या श्रेणीमधील वार्पिंगची मात्रा आणि दिशा नियंत्रित केली जाते.
२३ २३ Note Tracks नोट्स गीतपट्टा
२४ २४ Note Tracks are created by importing MIDI, for example, and not from this menu. टीप गीतपट्टा एमआयडीआय आयात करून तयार केले आहेत, उदाहरणार्थ, या मेनूमधून नाही.
