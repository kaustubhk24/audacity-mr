४ ४ The Sort Tracks submenu provides commands to sort your tracks by Track Names or the tracks' start times. गीतपट्टाची क्रमवारी सबमेनू आपल्या मागोवा गीतपट्टा नावे किंवा गीतपट्टाच्या प्रारंभ वेळांनुसार क्रमवारी लावण्यासाठी आदेश प्रदान करते.

६ ६ By Start time प्रारंभ वेळेत
७ ७ Sorts all tracks in the project from top to bottom in the project window. After the sort the top track will have the earliest start time and the bottom track will have the latest start time. प्रकल्प विंडोमध्ये वरपासून खालपर्यंत सर्व गीतपट्टाची क्रमवारी लावते. क्रमवारी लावल्यानंतर शीर्ष गीतपट्टावर लवकरात लवकर प्रारंभ वेळ असेल आणि तळाशी असलेल्या गीतपट्टावर नवीनतम प्रारंभ वेळ असेल.
८ ८ By Name नावाने
९ ९ Sort all tracks in the project alphanumerically by track name. Numbers precede letters, and upper-case letters precede lower case letters. It does not use natural number sort, so "१०" comes before "२" (but after "०२"). प्रकल्पातील सर्व गीतपट्टा अक्षरेद्वारे गीतपट्टा नावाने क्रमवारी लावा. संख्या आधीची अक्षरे आणि अपर-केस अक्षरे लोअर केस अक्षरे आधी. हे नैसर्गिक संख्या क्रमवारी वापरत नाही, म्हणून "२" "२" च्या आधी येईल (परंतु "०२" नंतर).
