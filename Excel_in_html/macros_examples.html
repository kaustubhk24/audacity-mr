2६ 2६ If the Macro is applied to the current project (as above), the current selection is used to create the Noise Profile. Therefore, other effect commands in the Macro will also only apply to that selection. If an export command is added, the entire file will be exported. सध्याच्या प्रकल्पावर मॅक्रो लागू केला असल्यास (वरील प्रमाणे), सद्य निवडीचा उपयोग नॉइस प्रोधारिका तयार करण्यासाठी केला जातो. म्हणूनच, मॅक्रो मधील इतर प्रभाव आदेश केवळ त्या निवडीस लागू होतील. निर्यात आदेश जोडल्यास, संपूर्ण धारिका निर्यात केली जाईल.
2७ 2७ If the Macro is applied to files, the first file (all of it) is used to create the Noise Profile. It may be useful to prepare a file containing a suitable Noise Profile and name it so that it is alphabetically the first file of those to be run in the Macro. जर धारिकांवर मॅक्रो लागू केला गेला असेल तर प्रथम धारिका (त्या सर्व) नॉइस प्रोधारिका तयार करण्यासाठी वापरली जाते. योग्य ध्वनी प्रोधारिका असलेली धारिका तयार करणे आणि त्यास नावे ठेवणे उपयुक्त ठरेल जेणेकरून मॅक्रोमध्ये चालविल्या जाणार्‍या त्यातील वर्णमाला वर्णक्रमानुसार असेल.
