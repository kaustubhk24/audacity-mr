४ ४ A LOF file is a plain text file ending with the extension ".lof" that contains a list of audio files to import into an Audacity project window. It can optionally contain commands to create new project windows. एक एलओएफ धारिका एक साधा मजकूर धारिका आहे जी ".लोफ" विस्तारासह समाप्त होते ज्यामध्ये ओड्यासिटी प्रकल्प विंडोमध्ये आयात करण्यासाठी ध्वनी धारिकांची सूची असते. यात नवीन प्रकल्प विंडोज तयार करण्यासाठी आज्ञा्स वैकल्पिकरित्या असू शकतात.
५ ५ Syntax मांडणी
६ ६ There are only two commands "file" and "window", each with associated options: तेथे फक्त दोन आज्ञास आहेत "धारिका" आणि "विंडो", प्रत्येक संबंधित पर्यायांसहः
७ ७ file: (required) specifies the full path to the file which then imports as a new track in the current window (the path must always be in double quotes) धारिका: (आवश्यक) धारिकाचा संपूर्ण मार्ग निर्दिष्ट करते जी सध्याच्या विंडोमध्ये नवीन गीतपट्टा म्हणून आयात करते (पथ नेहमी डबल कोट्समध्ये असणे आवश्यक आहे)
८ ८ offset: an optional parameter to the file command, specifying the start time of the audio file in seconds ऑफसेट: धारिका आज्ञासाठी पर्यायी पॅरामीटर, सेकंदात ध्वनी धारिकाचा प्रारंभ वेळ निर्दिष्ट करते
९ ९ window: (optional) creates a new Audacity project window for a subsequent file in the list (the first file always creates a new window unless the existing project is empty) विंडो: (पर्यायी) सूचीतील त्यानंतरच्या धारिकासाठी नवीन ओड्यासिटी प्रकल्प विंडो तयार करते (विद्यमान प्रकल्प रिक्त असल्याशिवाय प्रथम धारिका नेहमीच नवीन विंडो तयार करते)
१० १० offset: an optional parameter to the window command, specifying the left-most time displayed in the window ऑफसेट: विंडो आज्ञाचा एक वैकल्पिक पॅरामीटर, विंडोमध्ये डावीकडील सर्वात जास्त वेळ दर्शविणारा निर्दिष्ट करा
११ ११ duration: an optional parameter to the window command, specifying how much time should be displayed in the window कालावधी: विंडो आदेशाचे वैकल्पिक मापदंड, जे विंडोमध्ये किती वेळ दाखवायचे ते निर्दिष्ट करते
१2 १2 #: (optional) can be used anywhere on a line to precede a comment - everything after # on that line is ignored by the importer. #: (पर्यायी) टिप्पणीच्या आधी ओळीवर कोठेही वापरले जाऊ शकते - त्या ओळीवरील # नंतरचे सर्वकाही आयातकाकडे दुर्लक्ष केले जाते.
१३ १३ Tips: टिपा:
१४ १४ Each command must be placed on a separate line. प्रत्येक आज्ञा वेगळ्या ओळीवर ठेवला पाहिजे.
१५ १५ Files are imported in the order shown in the LOF file. एलओएफ धारिकामध्ये दर्शविलेल्या क्रमाने धारिका आयात केल्या जातात.
१६ १६ If the file to be imported is in the same folder as the LOF file, you need only give the file name inside double quotes. जर आयात करायची धारिका एलओएफ धारिकाच्या समान फोल्डरमध्ये असेल तर आपल्याला दुहेरी अवतरणातील धारिकाचे नाव देणे आवश्यक आहे.
१७ १७ Example उदाहरण
१८ १८ Given an empty project window, the following LOF file located at C:\Desktop imports the audio file "sample१.wav" from the Desktop into that window, followed by "sample2.wav" and "sample३.wav" from other locations underneath. Then a new project window is created and the audio file "sample४.wav" from the Desktop is imported into it. रिक्त प्रकल्प विंडो दिल्यास, सी येथे असलेली खालील एलओएफ धारिका डेस्कटॉप त्या विंडोमध्ये डेस्कटॉपवरून "नमूना १.wav" ध्वनी धारिका आयात करते आणि त्याखालील खाली "नमून 2.wav" आणि "नमून३.wav" ध्वनी धारिका आयात करते. मग एक नवीन प्रकल्प विंडो तयार होईल आणि त्यामध्ये डेस्कटॉप वरून "નમૂના४.wav" ध्वनी धारिका आयात केली जाईल.
