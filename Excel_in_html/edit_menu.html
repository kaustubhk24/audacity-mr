11 11 Redo  Ctrl + Y पुन्हा करा Ctrl + Y
1२ 1२ Redoes any editing operations that were just undone. नुकतीच पूर्ववत केलेली कोणतीही संपादन ऑपरेशन्स पुन्हा करा.
13 13 After you perform a new editing operation, you can no longer redo the operations that were undone. आपण नवीन संपादन ऑपरेशन केल्यानंतर आपण यापुढे पूर्ववत केलेल्या ऑपरेशन्स पुन्हा करू शकत नाही.
14 14 Cut  Ctrl + X Ctrl + X कट करा
1५ 1५ Removes the selected audio data and/or labels and places these on the Audacity clipboard. Audio or labels after the selection move leftwards. निवडलेला ध्वनी डेटा आणि / किंवा नावपट्टीे काढून ओड्यासिटी फीतबोर्डवर ठेवतो. निवडीनंतर ध्वनी किंवा नावपट्टी डावीकडे हलवा.
1६ 1६ Delete  Ctrl + K Ctrl + K हटवा
1७ 1७ Similar to Cut, but removes the audio data and/or labels that are currently selected without copying them to the Audacity clipboard. कट प्रमाणेच, परंतु ध्वनी डेटा आणि / किंवा नावपट्टीे काढून टाकतात जे ओड्यासिटी फीतबोर्डवर प्रतकेल्याशिवाय सध्या निवडलेले आहेत.
1८ 1८ Copy  Ctrl + C Ctrl + C प्रत करा
1९ 1९ Copies the selected audio data to the Audacity clipboard without removing it from the project. निवडलेला ध्वनी डेटा प्रकल्पातून न काढता ओड्यासिटी फीतबोर्डवर प्रत करतो.

२९ २९ Clip Boundaries फीत सीमा
30 30 The Clip Boundaries submenu has commands to create or remove separate clips in the audio track. A clip inside an audio track is a separate section of that track which has been split so that it can be manipulated somewhat independently of the other clips in the track. फीत बाउंडरीज सबमेनूमध्ये ध्वनी गीतपट्टामध्ये स्वतंत्र फीत तयार करण्यासाठी किंवा काढण्यासाठी आज्ञा आहेत. ध्वनी गीतपट्टाच्या आत असलेली फीत त्या गीतपट्टाचा वेगळा विभाग आहे जो विभाजित झाला आहे जेणेकरून गीतपट्टामधील इतर फीतच्या तुलनेत हे काहीसे स्वतंत्रपणे हाताळू शकते.

33 33 This command invokes the Labels Editor which lets you add or remove Label Tracks and edit their labels entirely using the keyboard, so is particularly useful for visually impaired users. ही आज्ञा नावपट्टी एडिटरची विनंती करते जी आपल्याला कीबोर्ड वापरुन नावपट्टी गीतपट्टा जोडू किंवा काढू देते आणि त्यांची नावपट्टीे पूर्णपणे संपादित करू देते, जेणेकरून दृष्टिहीन वापरकर्त्यांसाठी उपयुक्त आहे.

3६ 3६ Labeled Audio नावपट्टी केलेला ध्वनी
3७ 3७ The Labeled Audio submenu has commands that offer a time saving way of performing Edit Menu operations on the audio of multiple fully selected range labels. नावपट्टी केलेले ध्वनी सबमेनूमध्ये असे आदेश आहेत जे एकाधिक पूर्णपणे निवडलेल्या श्रेणी नावपट्टीांच्या ध्वनीवर संपादन मेनू ऑपरेशन्सची वेळ बचत मार्ग प्रदान करतात.
3८ 3८ The commands apply to all labeled audio regions that are fully inside a selection drawn in a label track. The selection may extend beyond the label boundaries, but audio that is not labeled and audio whose region label is only partly within the selection will not be acted on. नावपट्टी गीतपट्टामध्ये काढलेल्या निवडीच्या आत असलेल्या सर्व नावपट्टी ध्वनी प्रदेशांवर आदेश लागू होतात. निवड नावपट्टीच्या सीमांच्या पलीकडे वाढू शकते परंतु नावपट्टी न केलेले ध्वनी आणि ज्याचे प्रदेश नावपट्टी केवळ अंशतः निवडीमध्ये असेल त्या ध्वनीवर कारवाई केली जाणार नाही.
3९ 3९ If none of the audio tracks are included in the selection, the Labeled Audio commands apply to all audio tracks in the project. However if you include only certain audio tracks in the selection, the Labeled Audio commands will only affect those selected audio tracks. कोणत्याही ध्वनी गीतपट्टा निवडीमध्ये समाविष्ट न केल्यास, नावपट्टी केलेले ध्वनी आदेश प्रकल्पातील सर्व ध्वनी गीतपट्टावर लागू होतात. तथापि आपण निवडीमध्ये काही विशिष्ट ध्वनी गीतपट्टा समाविष्ट करत असल्यास, नावपट्टी केलेले ध्वनी आदेश केवळ त्या निवडलेल्या ध्वनी गीतपट्टावर परिणाम करतील.
