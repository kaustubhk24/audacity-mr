14 14 Tracks गीतपट्टा
15 15 The Tracks submenu has commands to extend the current selection up and/or down into all tracks, or into all Sync-Locked tracks in the project. गीतपट्टा्स सबमेनूला सध्याची निवड सर्व आणि खालील गीतपट्टामध्ये किंवा प्रकल्पातील सर्व समक्रमित-लॉक गीतपट्टामध्ये वाढविण्याचे आदेश आहेत.
16 16 Region प्रदेश
17 17 The Region submenu has commands for defining selection regions and lets you store and retrieve the position of the editing cursor or selection region. प्रदेश सबमेनू मध्ये निवड प्रदेश परिभाषित करण्यासाठी आज्ञा आहेत आणि आपल्याला संपादन कर्सर किंवा निवड प्रदेशाची स्थिती संचयित आणि पुनर्प्राप्त करू देते.
18 18 Spectral वर्णक्रमीय
19 19 The Spectral submenu has contains to enable you to make Spectral selection which can be used to make selections that include a frequency range as well as a time range on tracks in Spectrogram view. Spectral selection is used with special spectral edit effects to make changes to the frequency content of the selected audio. स्पेक्ट्रल सबमेनूमध्ये आपल्याला स्पेक्ट्रल निवड करण्यास सक्षम करते जे निवडी करण्यासाठी वापरले जाऊ शकते ज्यामध्ये वारंवारता श्रेणी तसेच स्पेक्ट्रोग्राम दृश्यातील गीतपट्टावर वेळ श्रेणीचा समावेश आहे. निवडलेल्या ध्वनीच्या वारंवारता सामग्रीमध्ये बदल करण्यासाठी स्पेक्ट्रल निवड विशेष वर्णक्रमीय संपादन प्रभावासह वापरली जाते.
20 20 Clip Boundaries क्लिप सीमा
21 21 The Clip Boundaries submenu lets you select between the editing cursor and clip boundaries or to navigate though clips, selecting all of the current clip. क्लिप सीमा उपमेनू आपल्याला संपादन कर्सर आणि क्लिप सीमा दरम्यान निवडण्याची परवानगी देते किंवा वर्तमान क्लिप निवडून क्लिप्स नॅव्हिगेट करण्यासाठी.
22 22 Cursor to Stored Cursor Position संग्रहित कर्सर स्थिती कर्सर
23 23 When playing or recording (or paused):, selection is made from the the position of the Playback cursor प्ले करताना किंवा रेकॉर्डिंग करताना (किंवा विराम दिलेला): प्लेबॅक कर्सरच्या स्थानावरून निवड केली जाते
24 24 or Recording cursor किंवा रेकॉर्डिंग कर्सर
25 25 to the position previously saved by the Select > Store Cursor Position command. For easier use without having to Pause, you may want to add a shortcut for "Cursor to Stored Cursor Position". यापूर्वी सेलेक्ट> स्टोअर कर्सर पोजिशन आज्ञाद्वारे जतन केलेल्या स्थितीत जा. विराम न देता सुलभ वापरासाठी आपल्याला "कर्सर टू स्टोअर कर्सर पोझिशन" साठी शॉर्टकट जोडायचा आहे.
26 26 When there is no active audio: selects from the current position of the editing cursor (or from the left edge of the current selection region) to the stored position. जेव्हा कोणताही सक्रिय ध्वनी नसतो: संपादन कर्सरच्या सद्य स्थितीपासून (किंवा वर्तमान निवड प्रदेशाच्या डाव्या काठावरुन) संग्रहित स्थितीवर निवडतो.
27 27 Example: Suppose you had paused playback (or stopped playback and set cursor) at the starting point of an intended selection and had stored that position. You could then start playback further along from the stored position in order to find the ending point of the intended selection. When you hear that ending, use "Cursor to Stored Cursor Position" to select from the stored position to the position of the playback cursor. You can then stop playback and the selection region will be retained. उदाहरणः समजा एखाद्या हेतू निवडीच्या सुरूवातीच्या वेळी आपण प्लेबॅक (किंवा प्लेबॅक थांबविला आहे आणि कर्सर सेट केला आहे) आणि त्या स्थितीस संग्रहित केले आहे. त्यानंतर आपण इच्छित निवडीचा शेवटचा बिंदू शोधण्यासाठी संग्रहित स्थितीसह प्लेबॅक सुरू करू शकता. जेव्हा आपण हे शेवट ऐकता तेव्हा संग्रहित स्थानावरून प्लेबॅक कर्सरच्या स्थानावर निवडण्यासाठी "कर्सर ते संग्रहित कर्सर स्थिती" वापरा. त्यानंतर आपण प्लेबॅक थांबवू शकता आणि निवड प्रदेश कायम ठेवला जाईल.
28 28 An alternative procedure if you want to store many positions is to add a label when required at the editing cursor or playback/recording cursor. आपण बर्‍याच पोझिशन्स संचयित करू इच्छित असल्यास पर्यायी प्रक्रिया म्हणजे संपादन कर्सर किंवा प्लेबॅक / रेकॉर्डिंग कर्सर आवश्यक असल्यास लेबल जोडणे.
29 29 Store Cursor Position स्टोअर कर्सर स्थिती
30 30 Stores a cursor position as defined below, which can then be used with the Select > Cursor to Stored Cursor Position command to create or modify a selection region. खाली परिभाषित केल्यानुसार कर्सर स्थिती संचयित करते, जे नंतर निवड प्रदेश तयार करण्यासाठी किंवा सुधारित करण्यासाठी सिलेक्ट> कर्सर टू स्टोर्ड कर्सर पोजिशन आज्ञासह वापरली जाऊ शकते.
31 31 If audio is not actively playing, recording or paused: the stored cursor position is the position of the editing cursor or of the left edge of the selection region. ध्वनी सक्रियपणे प्ले होत नसल्यास, रेकॉर्डिंग करत नाही किंवा विराम दिला असल्यास: संचयित कर्सर स्थिती संपादन कर्सर किंवा निवड प्रदेशाच्या डाव्या काठाची स्थिती आहे.
32 32 If audio is active: the stored cursor position is the position of the Playback cursor ध्वनी सक्रिय असल्यास: संग्रहित कर्सर स्थिती प्लेबॅक कर्सरची स्थिती आहे
33 33 or Recording cursor किंवा रेकॉर्डिंग कर्सर
34 34 at the time you issue the command to store. For easier use without having to Pause, you may want to add a shortcut for "Store Cursor Position". आपण संचयित करण्यासाठी आज्ञा जारी करता तेव्हा. विराम न देता सुलभ वापरासाठी आपल्याला "स्टोअर कर्सर पोझिशन" साठी शॉर्टकट जोडू शकता.
35 35 If you want to store the position of the editing cursor (or both boundaries of a selection) while audio is active, use the Select > Region > Store Selection command. आपण ध्वनी कार्यरत असताना संपादन कर्सर (किंवा निवडीच्या दोन्ही सीमा) ची स्थिती संचयित करू इच्छित असल्यास, निवडा> क्षेत्र> स्टोअर निवड आदेश वापरा.
36 36 There is no visual cue marker to remind you what position you saved, you will need to remember it. आपण कोणते स्थान जतन केले याची आठवण करून देण्यासाठी व्हिज्युअल क्यू चिन्हक नाही, आपल्याला ते लक्षात ठेवण्याची आवश्यकता असेल.
37 37 At Zero Crossings  Z झिरो क्रॉसिंग झेड येथे
38 38 At Zero Crossings helps to avoid clicks at edit points when making cuts and joins by moving the edges of a selection region (or the cursor position) very slightly so they are at a rising zero crossing point. झीरो क्रॉसिंग्ज जेव्हा निवड क्षेत्राच्या किना (्यावर (किंवा कर्सर स्थितीत) किंचित हलवून कट बनवतात आणि सामील होतात तेव्हा संपादन बिंदूवरील क्लिक टाळण्यास मदत करतात जेणेकरून ते वाढत्या शून्य क्रॉसिंग पॉईंटवर असतात.
39 39 This feature does not necessarily find the nearest zero crossing to the current position. It aims to find the crossing where the average amplitude of samples in the vicinity is lowest. हे वैशिष्ट्य सद्य स्थितीत सर्वात जवळचे शून्य ओलांडणे शोधू शकत नाही. हे ओलांडणे शोधण्याचे उद्दीष्ट आहे जेथे आसपासच्या नमुन्यांची सरासरी मोठेपणा सर्वात कमी आहे.
