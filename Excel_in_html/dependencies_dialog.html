4 4 Uncompressed WAV and AIFF files that are imported into a project with the "Read uncompressed audio files directly from the original (faster)" option in Import / Export Preferences are not copied into the project. Instead, Audacity keeps a reference to these files, and the project is dependent on them. Dependencies Dialog lists these dependencies and allows you to optionally remove them. You may select some, none or all the files to be copied in. आयात / निर्यात प्राधान्यांमधील "मूळ (वेगवान) कडून कॉम्प्रेसप्रेस ध्वनी धारिका थेट वाचा" पर्यायांसह प्रकल्पात आयात केलेल्या संकुचित WAV आणि AIFF धारिका प्रकल्पात कॉपी केल्या जात नाहीत. त्याऐवजी ओड्यासिटी या धारिकांचा संदर्भ ठेवते आणि प्रकल्प त्यांच्यावर अवलंबून असतो. अवलंबित्व संवाद या अवलंबित्वांची यादी करतो आणि आपल्याला त्यांना वैकल्पिकरित्या काढण्याची परवानगी देतो. आपण कॉपी करू शकता काही, काहीही नाही किंवा सर्व धारिका कॉपी केल्या पाहिजेत.
५ ५ Accessed by Help > Diagnostics > Check Dependencies... or when saving a project that depends on other audio files मदत> डायग्नोस्टिक्स> अवलंबित्व तपासा ... किंवा इतर ध्वनी धारिकांवर अवलंबून असलेला एखादा प्रकल्प जतन करताना प्रवेश केला
६ ६ If you select "Make a copy of uncompressed audio files before editing (safer)" in Import / Export Preferences, your projects will never have dependencies, but more disk space may be used to store the project, and importing WAV or AIFF files will be slower. आपण आयात / निर्यात प्राधान्यांमध्ये संपादन करण्यापूर्वी "सुरक्षित न केलेल्या ध्वनी धारिकांची एक प्रत बनवा" निवडल्यास आपल्या प्रकल्पांवर कधीही अवलंबून नसते, परंतु प्रकल्प संचयित करण्यासाठी अधिक डिस्क स्पेस वापरली जाऊ शकते आणि डब्ल्यूएव्ही किंवा एआयएफएफ धारिका आयात केल्या जातील हळू
७ ७ Project Dependencies प्रकल्प अवलंबित्व
८ ८ This dialog appears when you click Help > Diagnostics > Check Dependencies... and the project depends on other files. जेव्हा आपण मदत> निदान> चेक अवलंबिता ... क्लिक करता तेव्हा हा संवाद दिसून येतो आणि प्रकल्प अन्य धारिकांवर अवलंबून असतो.
९ ९ File list: When the dialog opens, you can move into the list using the keyboard by pressing <up arrow> (on Mac, focus is already in the list). You can resize the list columns by hovering the mouse over the boundary between them and then drag when the pointer changes to arrows. On Windows, hovering over a file name shows a tooltip containing the complete path to the file. धारिका सूची: जेव्हा संवाद उघडतो, तेव्हा आपण <अप एरो> दाबून कीबोर्डचा वापर करुन सूचीमध्ये जाऊ शकता (मॅक वर, आधीपासूनच सूचीत फोकस आहे). आपण सूचीच्या स्तंभांचे आकार बदलू शकता आणि त्या दरम्यानच्या सीमेवर माऊस फिरवून नंतर बाण बदलला की मग ड्रॅग करा. विंडोजवर, धारिकाच्या नावावर फिरताना धारिकाचा संपूर्ण मार्ग असलेले साधनटिप दर्शविले जाते.
१0 १0 Audio File: Lists the uncompressed files that were imported using the "Read uncompressed audio files directly from the original (faster)" option in Import / Export Preferences. ध्वनी धारिकाः आयात / निर्यात प्राधान्यांमधील "मूळ (वेगवान) वरुन थेट कॉम्प्रेसप्रेस ध्वनी धारिका थेट वाचा" पर्याय वापरुन आयात केलेल्या संकुचित धारिका सूचीबद्ध करतात.
११ ११ Disk space: Shows the amount of additional disk space that will be used by the project if the file(s) are copied in. The disk space shown will depend on the default sample format of the project as set in Quality Preferences. डिस्क स्पेस: धारिका कॉपी केल्या असल्यास प्रकल्पद्वारे वापरल्या जाणार्‍या अतिरिक्त डिस्क स्पेसचे प्रमाण दर्शविते. दर्शविलेले डिस्क स्पेस गुणवत्ता प्राधान्यांमध्ये सेट केल्यानुसार प्रकल्पाच्या पूर्वनिर्धारित नमुना स्वरूपवर अवलंबून असेल.
१2 १2 Copy Selected Files: Copies the files selected in the list into the project. The project no longer depends on the selected files but still depends on the unselected files. निवडलेल्या धारिका कॉपी करा: प्रकल्पमध्ये यादीमध्ये निवडलेल्या धारिका्स कॉपी करा. प्रकल्प यापुढे निवडलेल्या धारिकांवर अवलंबून नाही परंतु अद्याप न निवडलेल्या धारिकांवर अवलंबून आहे.
१3 १3 Do Not Copy: Dismisses the dialog (but continues to save the project if you were saving it) without copying any of the listed files into the project. The project will continue to depend on the listed files, so you must not move, rename or delete those files, or the folder they are in. कॉपी करू नका: प्रकल्पमध्ये सूचीबद्ध केलेल्या कोणत्याही धारिका कॉपी न करता संवाद डिसमिस करतो (परंतु आपण ते जतन करीत असल्यास प्रकल्प जतन करणे सुरू ठेवत आहे). प्रकल्प सूचीबद्ध धारिकांवर अवलंबून राहिल, म्हणून आपण त्या धारिका किंवा त्यामध्ये असलेले फोल्डर हलविणे, नाव बदलणे किंवा हटविणे आवश्यक नाही.
१4 १4 Copy All Files (Safer): All the listed files will be copied into the project, removing all the dependencies. You can then safely move, rename or delete those files if you wish. सर्व धारिका कॉपी करा (अधिक सुरक्षित): सर्व सूचीबद्ध धारिका प्रकल्पांवर कॉपी केल्या जातील, सर्व अवलंबन काढून टाकतील. त्यानंतर आपली इच्छा असल्यास आपण त्या धारिका सुरक्षितपणे हलवू, नाव बदलू किंवा हटवू शकता.
१५ १५ When saving a project that depends on other files इतर धारिकांवर अवलंबून असलेला एखादा प्रकल्प जतन करताना
१६ १६ This similar dialog appears when you save a project that has dependencies, and have chosen in Projects Preferences to be asked whether dependent files should be copied in or not. जेव्हा आपण एक प्रकल्प जतन करते जेव्हा हे अवलंबन असते आणि आपण अवलंबून असलेल्या धारिकांमध्ये कॉपी करावी की नाही हे विचारले जाण्यासाठी प्रकल्प प्रिफरेन्समध्ये निवडल्यास हा समान संवाद दिसून येतो.
१७ १७ As with the above version, you may select some, none or all the files for copying in. This dialog gives you the option of directly setting your preference for copying in behavior when saving a project. वरील आवृत्ती प्रमाणे, आपण कॉपी करण्यासाठी काही, कोणतीही किंवा सर्व धारिका निवडू शकत नाही. हा संवाद आपल्याला प्रकल्प जतन करताना वर्तनमध्ये कॉपी करण्यासाठी आपले प्राधान्य थेट सेट करण्याचा पर्याय देतो.
१८ १८ Copy Selected Files Saves the project while copying only the files selected in the list into the project. The project no longer depends on the selected files but still depends on the unselected files. निवडलेल्या धारिका कॉपी करा प्रकल्पमध्ये यादीमध्ये निवडलेल्या फक्त धारिका्सची कॉपी करताना प्रकल्प वाचतो. प्रकल्प यापुढे निवडलेल्या धारिकांवर अवलंबून नाही परंतु अद्याप न निवडलेल्या धारिकांवर अवलंबून आहे.
१९ १९ Cancel Save: Allows you to cancel saving of the project, without copying any of the listed files into it. You will be asked to say "Yes" if this is your intention. The project will continue to depend on the listed files. जतन करा रद्द करा: त्यात सूचीबद्ध केलेल्या कोणत्याही धारिका कॉपी केल्याशिवाय आपल्याला या प्रकल्पाची बचत रद्द करण्याची परवानगी देते. हा आपला हेतू असेल तर आपल्याला "होय" म्हणायला सांगितले जाईल. प्रकल्प सूचीबद्ध धारिकांवर अवलंबून राहिल.
20 20 Save without Copying: Save the project without copying in the dependent files. The project will continue to depend on the listed files. कॉपी केल्याशिवाय जतन करा: अवलंबून असणार्‍या धारिकांमध्ये कॉपी न करता प्रकल्प जतन करा. प्रकल्प सूचीबद्ध धारिकांवर अवलंबून राहिल.
2१ 2१ Copy All Files (Safer): Saves the project while copying all the listed files into the project, removing all the dependencies. You can then safely move, rename or delete those files if you wish. सर्व धारिका कॉपी करा (अधिक सुरक्षित): प्रकल्पातील सर्व सूचीबद्ध धारिका कॉपी करताना, सर्व अवलंबन काढून टाकून प्रकल्प वाचवितो. त्यानंतर आपली इच्छा असल्यास आपण त्या धारिका सुरक्षितपणे हलवू, नाव बदलू किंवा हटवू शकता.
22 22 Whenever a project depends on other files: जेव्हा एखादा प्रकल्प इतर धारिकांवर अवलंबून असतो:
23 23 Ask me The default option; the Dependencies Dialog will always appear when saving a project that has dependencies. मला विचारा पूर्वनिर्धारित पर्याय; अवलंबित्वांमधील संवाद हा नेहमीच दिसून येईल ज्यावर अवलंबन असणार्‍या प्रकल्पची बचत करते.
24 24 Always copy all audio (safest): On all future occasions when saving a project that has dependencies, Dependencies Dialog will not appear, and all dependent files will be copied into the project. सर्व ध्वनीची नेहमी कॉपी करा (सर्वात सुरक्षित): भविष्यातील सर्व प्रसंगी ज्यात प्रकल्प जतन करते तेव्हा डिपेंडेंडेस डायलॉग दिसणार नाही आणि सर्व धारिकाी प्रकल्पमध्ये कॉपी केल्या जातील.
2५ 2५ Never copy any audio: On all future occasions when saving a project that has dependencies, Dependencies Dialog will not appear, and no files will be copied into the project. The project will continue to depend on the listed files. कोणत्याही ध्वनीची कधीही कॉपी करु नकाः भविष्यातील सर्व प्रसंगी ज्या प्रकल्पवर अवलंबन असेल ते जतन करतेवेळी डिपेंडेंसीज डायलॉग दिसणार नाही आणि प्रकल्पमध्ये कोणत्याही धारिका कॉपी केल्या जाणार नाहीत. प्रकल्प सूचीबद्ध धारिकांवर अवलंबून राहिल.
2६ 2६ You can change your decision in this menu at any time by going to Projects Preferences. आपण या मेनूमधील प्रकल्प कोणत्याही वेळी प्राधान्यक्रमात जाऊन आपला निर्णय बदलू शकता.
