४ ४ Here are a few tips to make it easier to access the effect you want from the Effect menu without using a mouse. माउसचा वापर न करता प्रभाव मेनूमधून आपल्यास इच्छित परिणामावर प्रवेश करणे सुलभ करण्यासाठी येथे काही टिप्स आहेत.
५ ५ Accessing effects with shortcuts आडमार्ग सह प्रभाव प्रवेश
६ ६ Shortcuts can be added, or changed, in Keyboard Preferences to items in the Generate, Effect or Analyze menus, including plug-ins that you added yourself. The shortcut activates the effect if the effect has no dialog, or opens the effect dialog. आपण स्वत: ला जोडलेल्या प्लग-इनसह मेनू व्युत्पन्न, प्रभाव किंवा विश्लेषण मेनूमधील आयटमच्या कळफलक प्राधान्यांमध्ये आडमार्ग जोडले किंवा बदलले जाऊ शकतात. प्रभावात संवाद नसल्यास आडमार्ग प्रभाव सक्रिय करतो, किंवा प्रभाव संवाद उघडतो.
७ ७ Repeat Last Effect Ctrl + R (or ⌘ + R on Mac) can be used to repeat the last used effect at its last used settings. If you obtain a noise profile with Noise Reduction and then use "Repeat Last Effect", noise will be reduced at the current settings rather than obtaining a new noise profile. पुनरावृत्ती शेवटचा प्रभाव Ctrl + R (किंवा मॅक वर ⌘ + आर) त्याच्या शेवटल्या वापरल्या जाणार्‍या समायोजनमध्ये वापरलेल्या शेवटच्या प्रभावाची पुनरावृत्ती करण्यासाठी वापरला जाऊ शकतो. आपण नॉइस रिडक्शनसह आवाजाचे प्रोधारिका प्राप्त केले आणि नंतर "पुनरावृत्ती शेवटचा प्रभाव" वापरल्यास नवीन ध्वनी प्रोधारिका प्राप्त करण्याऐवजी वर्तमान समायोजनमध्ये आवाज कमी होईल.
८ ८ Other keyboard methods इतर कळफलक पद्धती
९ ९ On Windows and Linux, pressing Alt then C opens the Effect menu and takes you to the top of the list without having to use a mouse विंडोज आणि लिनक्सवर Alt नंतर C दाबल्याने प्रभाव मेनू उघडतो आणि तुम्हाला माऊसचा वापर न करता सूचीच्या शीर्षस्थानी नेतो.
१० १० On Linux you may have to hold down Alt. लिनक्स वर आपल्याला Alt दाबून ठेवावे लागू शकते.
११ ११ You can scroll up and down the menu one item at a time with the up and down arrow keys. If you are already at the top of the menu, the up arrow will skip to the bottom item of the menu. आपण एका वेळी वर आणि खाली बाण कीसह मेनूमधील एक आयटम वर आणि खाली स्क्रोल करू शकता. जर आपण आधीच मेनूच्या वरच्या बाजूला असाल तर अप एरो मेनूच्या खालच्या बाबीकडे जाईल.
१2 १2 On Linux, effects underneath the divider are grouped as "Plugins १ - १५", "Plugins १६ - ३०" and so on. Use Right Arrow to move into the list of plug-ins in the group. लिनक्स वर, विभाजक च्या खाली असलेल्या प्रभावांना "प्लगइन्स १ - १" "," प्लगइन्स १ - - "० "वगैरे म्हणून गटबद्ध केले जाते. गटातील प्लगइनच्या सूचीमध्ये जाण्यासाठी उजवा बाण वापरा.
१३ १३ Press Enter on the keyboard to open the highlighted effect. हायलाइट केलेला प्रभाव उघडण्यासाठी कळफलकवरील एंटर दाबा.
१४ १४ Once in the Effect, you can use Tab to navigate forwards through the text boxes, sliders and buttons. Hold Shift while pressing Tab to skip backwards. You can use the arrow keys, Home, End. Page Up or Page Down to control a slider. एकदा प्रभावानंतर, आपण मजकूर बॉक्स, स्लाइडर आणि बटणेद्वारे पुढे नेव्हिगेट करण्यासाठी टॅब वापरू शकता. मागे जाण्यासाठी टॅब दाबताना शिफ्ट दाबून ठेवा. आपण होम, एंड या बाण की वापरू शकता. स्लाइडर नियंत्रित करण्यासाठी पृष्ठ वर किंवा पृष्ठ खाली.
१५ १५ On Mac, use Page Up or Page Down to move to the start or end of the slider respectively. Home and End do not work with sliders. मॅकवर, स्लाइडरच्या सुरूवातीस किंवा शेवटी जाण्यासाठी क्रमशः पृष्ठ वर किंवा पृष्ठ खाली वापरा. होम आणि एंड स्लाइडर्ससह कार्य करत नाही.
१६ १६ Once you have used an effect, you can use "Repeat Last Effect" at the top of the menu to repeat that effect with those last used parameters. This also has a shortcut Ctrl + R (or ⌘ + R on a Mac). You can change this shortcut to something else in the Keyboard Preferences. एकदा आपण प्रभाव वापरल्यानंतर, आपण त्या अंतिम वापरलेल्या पॅरामीटर्ससह पुनरावृत्ती करण्यासाठी मेनूच्या शीर्षस्थानी "अंतिम परिणाम पुन्हा करा" वापरू शकता. यात Ctrl + R (किंवा मॅकवर ⌘ + आर) आडमार्ग देखील आहे. आपण हा आडमार्ग कळफलक प्राधान्यांमधून दुसर्‍या कशासाठी बदलू शकता.
