४ ४ This displays runs of clipped samples in a Label Track, as a screen-reader accessible alternative to View > Show Clipping. A run must include at least one clipped sample, but may include unclipped samples too. हे लेबल गीतपट्टामध्ये क्लिप केलेल्या नमुन्यांची धाव दाखवते, दर्शवा क्लिपिंग दर्शविण्यासाठी स्क्रीन-रीडर प्रवेशयोग्य पर्याय म्हणून. धाव मध्ये कमीतकमी एक क्लिप केलेला नमुना असणे आवश्यक आहे, परंतु त्यात न पडलेले नमुनेदेखील असू शकतात.
५ ५ Accessed by: Analyze > Find Clipping... याद्वारे प्रवेश केलेले: विश्लेषण> क्लिपिंग शोधा ...
६ ६ Start Threshold थ्रेशोल्ड प्रारंभ करा
७ ७ This is the number of contiguous clipped samples (default 3 samples) that must be present before they will be labeled as an individual run. ही वैयक्तिक क्लिप केलेल्या नमुन्यांची संख्या (पूर्वनिर्धारित 3 नमुने) आहेत जी त्यांना वैयक्तिक धाव म्हणून लेबल लावण्यापूर्वी उपस्थित असणे आवश्यक आहे.
८ ८ The higher the threshold, the less likely a clipped region will be recognized and the fewer labels that will be generated. उंबरठा जितका जास्त असेल तितका कमी क्लिप केलेला प्रदेश ओळखला जाईल आणि जे कमी जनतेची निर्मिती केली जाईल.
९ ९ Stop Threshold थ्रेशोल्ड थांबवा
१0 १0 This is the number of contiguous unclipped samples (default 3 samples) that must be present before the run of clipped samples will be terminated. The higher the threshold, the longer a clipped region will be and the fewer labels that will be generated. ही क्लिप्ड नमुने चालविण्यापूर्वी अस्तित्वात असणे आवश्यक असलेल्या अव्यवस्थित नमुने (पूर्वनिर्धारित 3 नमुने) ची संख्या आहे. उंबरठा जितका जास्त असेल तितका जास्त क्लिप केलेला प्रदेश असेल आणि जेणेकरून कमी लेबले तयार होतील.
११ ११ Each individual run of clipped samples is marked in the format "number of number". For example, "११ of ११" means there are ११ samples in this run of clipped samples and all of them are clipped, while "९ of ११" means only nine samples in the run of ११ are clipped. In the case of a stereo track each channel will have its own label; if at one point both channels are clipped two labels will appear one above the other. क्लिप केलेल्या नमुन्यांची प्रत्येक वैयक्तिक धाव संख्या "संख्या" या स्वरूपात चिन्हांकित केली जाते. उदाहरणार्थ, "११ पैकी ११" चा अर्थ असा की क्लिप केलेल्या नमुन्यांच्या धावण्यांमध्ये ११ नमुने आहेत आणि ते सर्व क्लिप केलेले आहेत, तर "११ पैकी ९" म्हणजे ११ च्या धावण्यातील केवळ नऊ नमुने क्लिप केलेले आहेत. स्टिरिओ गीतपट्टाच्या बाबतीत प्रत्येक वाहिनीचे स्वतःचे लेबल असेल; जर एका क्षणी दोन्ही वाहिनी क्लिप केल्या गेल्या तर एका वरील दोन लेबले दिसतील.
१2 १2 Unlike View > Show Clipping and Meter Toolbars "Find Clipping" is not automatically updated. व्यू> शो क्लिपिंग आणि मीटर साधनपट्टीच्या विपरीत "क्लिपिंग शोधा" स्वयंचलितपणे अद्यतनित केले जात नाही.
१3 १3 After any editing that affects volume, or after import or recording of a new track, Find Clipping must be run again. Labels from another run of Find Clipping will be added to the existing label track - close the label track using the [X] on the left of the track if you want to view only labels for current clipping. व्हॉल्यूमवर परिणाम करणार्‍या कोणत्याही संपादनानंतर किंवा नवीन गीतपट्टाच्या आयात किंवा ध्वनीमुद्रणिंगनंतर, फाइंडर क्लिपिंग पुन्हा चालवणे आवश्यक आहे. विद्यमान लेबल गीतपट्टावर फाइन्ड क्लीपिंगच्या दुसर्‍या धावतील लेबले जोडली जातील - आपल्याला सध्याच्या क्लिपिंगसाठी फक्त लेबले पहायची असतील तर गीतपट्टाच्या डावीकडे [एक्स] वापरून लेबल गीतपट्टा बंद करा.
