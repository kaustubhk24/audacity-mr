४ ४ Timer Record operates in a manner similar to setting your digital video recorder or cassette recorder to make a scheduled recording.The Timer Record setup dialog enables you to schedule when a recording begins and ends and optionally: वेळर ध्वनीमुद्रण अनुसूचित ध्वनीमुद्रणिंग करण्यासाठी आपला डिजिटल व्हिडिओ ध्वनीमुद्रणर किंवा कॅसेट ध्वनीमुद्रणर सेट करण्याइतकेच कार्य करते. वेळर ध्वनीमुद्रण सेटअप संवाद जेव्हा ध्वनीमुद्रणिंग सुरू होते आणि समाप्त होते आणि वैकल्पिकरित्या आपल्याला वेळापत्रक तयार करण्यास सक्षम करते:
५ ५ effect an Automatic Save Project or an automatic Export to any supported audio format, स्वयंचलित बचत प्रकल्प किंवा कोणत्याही समर्थित ध्वनी स्वरूपात स्वयंचलित निर्यात परिणाम द्या,
६ ६ specify that Audacity closes or the computer shuts down on completion of the Timer Record. वेळर ध्वनीमुद्रण पूर्ण झाल्यावर ओड्यासिटी बंद होईल किंवा संगणक बंद होईल हे निर्दिष्ट करा.
७ ७ Accessed by: Transport > Recording > Timer Record... याद्वारे प्रवेश केलेले: परिवहन> ध्वनीमुद्रणिंग> वेळर ध्वनीमुद्रण ...
८ ८ Before you start a Timer Record session it is very important that you make all necessary settings in Audacity as while a Timer Record session is in progress, including in the period while Audacity is waiting to start the recording, the Audacity controls and menus will be inoperable until the recording finishes, or is Stopped/Cancelled. वेळर ध्वनीमुद्रण सत्र सुरू करण्यापूर्वी तुम्ही ऑडिटीमध्ये सर्व आवश्यक समायोजन तयार करणे खूप महत्वाचे आहे कारण वेळर ध्वनीमुद्रण सत्र चालू असताना ऑडिटी ध्वनीमुद्रणिंग सुरू होण्याच्या प्रतीक्षेत असताना ओड्यासिटी नियंत्रणे आणि मेनू अक्षम होतील ध्वनीमुद्रणिंग पूर्ण होईपर्यंत किंवा थांबविले / रद्द केले जाईपर्यंत.
९ ९ In particular you need to pay close attention to setting the input signal level as Audacity's sliders cannot be used during a Timer Record session (you may though have a hardware gain control on your input device which would allow you to adjust the signal level during a Timer record session). विशेषत: आपल्याला इनपुट सिग्नल पातळी निश्चित करण्याकडे बारीक लक्ष देणे आवश्यक आहे कारण वेळर ध्वनीमुद्रण सत्रामध्ये ऑडिटीच्या स्लाइडरचा वापर केला जाऊ शकत नाही (कदाचित आपल्या इनपुट उपकरणवर हार्डवेअर गेन कंट्रोल असू शकेल जे आपल्याला वेळर दरम्यान सिग्नल पातळी समायोजित करण्यास परवानगी देईल. ध्वनीमुद्रण सत्र).
१० १० Note that you cannot initiate a Timer Record of you have any unsaved data in your project, nor if you have more that one Audacity project open. लक्षात घ्या की आपण आपल्या प्रकल्पात कोणताही जतन न केलेला डेटा आपल्याकडे वेळर ध्वनीमुद्रण आरंभ करू शकत नाही किंवा आपल्याकडे एखादा ओड्यासिटी प्रकल्प अधिक असल्यास.
११ ११ Setting the timer controls टायमर नियंत्रणे सेट करत आहे
१२ १२ Recording start time ध्वनीमुद्रणिंग प्रारंभ वेळ
१३ १३ "Start Date and Time" shows the current live time on opening the dialog, meaning that the recording you schedule will start immediately unless you change this control to a time and date in the future. "प्रारंभ तारीख आणि वेळ" हा संवाद उघडण्यासाठी सध्याचा थेट वेळ दर्शवितो, याचा अर्थ असा की आपण भविष्यात हे नियंत्रण वेळ आणि तारखेपर्यंत बदलत नाही तोपर्यंत आपण शेड्यूल केलेले ध्वनीमुद्रणिंग त्वरित सुरू होईल.
१४ १४ Recording end time or duration ध्वनीमुद्रणिंग समाप्ती वेळ किंवा कालावधी
१५ १५ You can set the end of the scheduled recording either by specifying this at "End Date and Time", or by choosing the length of the recording at "Duration". The digits in the time controls are interlinked, so that typing on a digit or incrementing it forwards or backwards with keyboard up arrow or down arrow updates the preceding digits. The time controls themselves are also interlinked, so that "Start Date and Time" updates the "End Date and Time", and changing the "End Date and Time" modifies the "Duration" (and vice versa). आपण एकतर "समाप्ती तारीख आणि वेळ" येथे निर्दिष्ट करुन किंवा "कालावधी" येथे ध्वनीमुद्रणिंगची लांबी निवडून अनुसूचित ध्वनीमुद्रणिंगचा शेवट सेट करू शकता. वेळेच्या नियंत्रणामधील अंक एकमेकांशी जोडलेले असतात, जेणेकरून एखाद्या अंकावर टाइप करणे किंवा कीबोर्ड अप बाण किंवा खाली बाणाने पुढे किंवा मागील बाजूस वाढविणे मागील अंक अद्यतनित करते. वेळ स्वत: चे नियंत्रण देखील एकमेकांशी जोडलेले असते, जेणेकरून "प्रारंभ तारीख आणि वेळ" "समाप्ती तारीख आणि वेळ" अद्यतनित करते आणि "समाप्ती तारीख आणि वेळ" बदलल्याने "कालावधी" (आणि उलट) सुधारित होते.
१६ १६ There is no default "Duration". When you open Timer Record next time, Audacity remembers the duration that was used for the previous scheduled recording. पूर्वनिर्धारित "कालावधी" नाही. जेव्हा आपण पुढील वेळी वेळर ध्वनीमुद्रण उघडता तेव्हा ओड्यासिटी आधीच्या अनुसूचित ध्वनीमुद्रणिंगसाठी वापरलेला कालावधी आठवते.
१७ १७ If you need to change a date forwards, incrementing either hour digit in the time controls with keyboard up arrow will always increment the date automatically. आपणास तारीख अग्रेषित बदलण्याची आवश्यकता असल्यास, कीबोर्ड अप एरोसह वेळ नियंत्रणामध्ये एक तासातील अंक वाढविणे नेहमी स्वयंचलितपणे तारीख वाढवते.
१८ १८ If you want to start an immediate recording, changing only the Duration is often the simplest method because this changes the End Date if necessary as well as the End Time. आपण त्वरित ध्वनीमुद्रणिंग सुरू करू इच्छित असल्यास केवळ कालावधी बदलणे ही सर्वात सोपी पद्धत आहे कारण आवश्यक असल्यास समाप्ती तारीख तसेच अंत वेळ बदलते.
१९ १९ Completion actions पूर्ण क्रिया
२० २० There are a number of things you can request Audacity to do on succesful completion of a Timer Recording. वेळर ध्वनीमुद्रणिंग यशस्वीपणे पूर्ण करण्यासाठी आपण पुष्कळ गोष्टी ओड्यासिटीला विनंती करु शकता.
२१ २१ Automatic Save स्वयंचलित जतन करा
२२ २२ Checking the checkbox  Enable Automatic Save enables you to then use the Select button to select a location and file name for an automatic save of the project when the recording finishes. चेकबॉक्स तपासणे स्वयंचलित जतन सक्षम करा नंतर ध्वनीमुद्रणिंग समाप्त होईल तेव्हा प्रकल्पाच्या स्वयंचलित जतनसाठी स्थान आणि धारिका नाव निवडण्यासाठी सिलेक्ट बटणाचा वापर करण्यास आपल्याला सक्षम करते.
२३ २३ If you have already Saved the project then that field will be populated with "Current Project". आपण यापूर्वीच प्रकल्प जतन केला असेल तर ते फील्ड "चालू प्रकल्प" सह प्रसिध्द होईल.
२४ २४ Automatic Export स्वयंचलित निर्यात
२५ २५ Checking the checkbox  Enable Automatic Export enables you to then use the Select button to select an audio file format for the export and a location and file name for the automatic export of the project when the recording finishes. चेकबॉक्सची तपासणी करत असल्यास ऑटोमॅटिक निर्यात सक्षम करा निर्यातसाठी ध्वनी धारिका फॉरमॅट निवडण्यासाठी सिलेक्ट बटन वापरा आणि ध्वनीमुद्रणिंग संपल्यावर प्रकल्पच्या ऑटोमॅटिक निर्यातसाठी धारिका नेम व धारिकाचे नाव निवडणे तुम्हाला सक्षम करते.
२६ २६ If you have "Use custom mix" set to be "on" in Import/Export preferences, and if you use the automatic export feature of Timer Record you will find that after the recording completes the automatic export will be interrupted with the "Advanced Mixing Options" mapping dialog, requiring manual input from you. आपल्याकडे आयात / निर्यात प्राधान्यांमध्ये "चालू" असल्याचे "सानुकूल मिक्स वापरा" असल्यास आणि आपण वेळर ध्वनीमुद्रणचे स्वयंचलित निर्यात वैशिष्ट्य वापरल्यास आपणास आढळेल की ध्वनीमुद्रणिंग पूर्ण झाल्यावर "प्रगत मिश्रणासह व्यत्यय आणला जाईल" आपल्याकडून व्यक्तिचलित इनपुट आवश्यक असलेले "मॅपिंग संवाद" पर्याय.
२७ २७ You can select both an Automatic Save and an Automatic Export if required. आवश्यक असल्यास आपण स्वयंचलित जतन आणि स्वयंचलित निर्यात दोन्ही निवडू शकता.
२८ २८ Options - After recording completes पर्याय - ध्वनीमुद्रणिंग पूर्ण झाल्यानंतर
२९ २९ If you have selected either an Automatic Save or Export the you can choose an action for Audacity to perform once the recording has completed and the automatic save or export has been made. आपण एकतर स्वयंचलित जतन किंवा निर्यात निवडलेली असल्यास आपण ध्वनीमुद्रणिंग पूर्ण झाल्यावर आणि स्वयंचलित जतन किंवा निर्यात केले की ओड्यासिटी करण्यासाठी कृती निवडू शकता.
३० ३० The "Options" "पर्याय"
३१ ३१ dropdown menu will activate and you can choose: ड्रॉपडाउन मेनू सक्रिय होईल आणि आपण हे निवडू शकता:
३२ ३२ Do Nothing काही करू नको
३३ ३३ Exit Audacity निर्गमन ओड्यासिटी
३४ ३४ Restart System - only available on Windows रीस्टार्ट सिस्टम - केवळ विंडोजवर उपलब्ध
३५ ३५ Shutdown - only available on Windows बंद - फक्त विंडोज वर उपलब्ध
३६ ३६ The latter three options will give you a one minute period of grace to abort the exit, shutdown or restart if you change your mind later. नंतरचे तीन पर्याय आपल्याला बाहेर पडणे, बंद करणे किंवा नंतर आपण आपला विचार बदलल्यास रीस्टार्ट करण्यासाठी एक मिनिटांची ग्रेस देतील.
३७ ३७ A dialog will be shown for that minute enabling you to abort the previously selected action by using the Cancel button, see below. त्या मिनिटासाठी एक संवाद दर्शविला जाईल ज्यामुळे आपल्याला रद्द करा बटण वापरुन पूर्वी निवडलेल्या क्रियेचा त्याग करणे शक्य होईल, खाली पहा.
३८ ३८ Audacity will remember your completion action settings and restore them the next time you use Timer Record. धृष्टता आपल्या पूर्ण क्रिया समायोजन लक्षात ठेवेल आणि पुढच्या वेळी आपण टायमर ध्वनीमुद्रण वापरता तेव्हा त्या पुनर्संचयित करतील.
३९ ३९ Starting the Timer Record वेळर ध्वनीमुद्रण प्रारंभ करीत आहे
४० ४० When you have made all your required settings simply press the OK button to start the Timer Record which will then either wait to start to record later or start immediately depending on your settings in the timer contols, the "Start Date and Time". Alternatively you can abort the Timer Record by pressing the Cancel button. जेव्हा आपण आपल्या सर्व आवश्यक समायोजन केल्या असतील तर फक्त वेळर ध्वनीमुद्रण प्रारंभ करण्यासाठी ठीक बटण दाबा जे नंतर एकतर ध्वनीमुद्रण करण्यास प्रारंभ होण्याची प्रतीक्षा करेल किंवा वेळर कॉन्ट्रॉल्समधील समायोजनच्या आधारावर त्वरित प्रारंभ होईल, "प्रारंभ तारीख आणि वेळ". वैकल्पिकरित्या आपण रद्द करा बटण दाबून टायमर ध्वनीमुद्रण थांबवू शकता.
४१ ४१ It is not currently possible to undertake any other activities in Audacity while recordings are running or waiting to start. To work in other applications while Timer Record is running, use the Task Switcher utility on your system, usually Alt + Tab. ध्वनीमुद्रणिंग चालू असताना किंवा प्रारंभ होण्याची वाट पहात असताना ओड्यासिटीमध्ये इतर कोणतेही क्रियाकलाप करणे सध्या शक्य नाही. टायमर ध्वनीमुद्रण चालू असताना इतर अनुप्रयोगांमध्ये कार्य करण्यासाठी, आपल्या सिस्टमवरील टास्क स्विचर उपयुक्तता, सामान्यत: Alt + Tab वापरा.
४२ ४२ Waiting for Start dialog प्रारंभ संवादची प्रतीक्षा करत आहे
४३ ४३ If you had set the recording's Start Date and Time in advance of the current time, pressing OK in the setup dialog will cause the Waiting for Start dialog to appear: जर आपण ध्वनीमुद्रणिंगची प्रारंभ वेळ आणि वेळ यापूर्वीच्या वेळेस सेट केली असेल तर सेटअप संवादात ओके दाबण्यामुळे प्रारंभ होण्याची प्रतीक्षा प्रारंभ संवाद दिसेल:
४४ ४४ This shows the timing details of your requested Timer Record along with a summary of the requested completion options. हे आपल्या विनंती केलेल्या वेळर ध्वनीमुद्रणची वेळ तपशील तसेच विनंती पूर्ण झालेल्या पर्यायांच्या सारांश दाखवते.
४५ ४५ If you want to abort this recording before it starts, press the Cancel button. आपण हे ध्वनीमुद्रणिंग सुरू होण्यापूर्वी खंडित करू इच्छित असल्यास रद्द करा बटण दाबा.
४६ ४६ You will then be presented with a confirmation dialog त्यानंतर आपणास पुष्टीकरण संवाद सादर केला जाईल
४७ ४७ asking you to confirm the cancellation (similar to the Stop cancellation dialog shown below in the progress section). आपल्याला रद्द करण्याची पुष्टी करण्यास सांगत आहे (प्रगती विभागात खाली दर्शविलेले रद्दीकरण थांबवा संवाद सारखे आहे).
४८ ४८ Timer Record Progress dialog वेळर ध्वनीमुद्रण प्रगती संवाद
४९ ४९ When the recording starts, the Timer Record Progress dialog appears: जेव्हा ध्वनीमुद्रणिंग सुरू होते, तेव्हा वेळर ध्वनीमुद्रण प्रगती संवाद आढळतो:
