2६ 2६ The Label Track's Dropdown Menu can be used to name the track, move it up or down or set the font of all label text. लेबल गीतपट्टाच्या ड्रॉपडाउन मेनूचा उपयोग गीतपट्टाचे नाव ठेवण्यासाठी किंवा खाली हलविण्यासाठी किंवा सर्व लेबल मजकूराचा फॉन्ट सेट करण्यासाठी केला जाऊ शकतो.
2७ 2७ Label tracks are included when saving an Audacity Project. ओड्यासिटी प्रकल्प जतन करताना लेबल गीतपट्टा समाविष्ट केले जातात.
2८ 2८ Label tracks can be used to define sets of Sync-Locked Track Groups. लेबल गीतपट्टा सिंक्रोनाइझेशन-लॉक केलेले गीतपट्टा ग्रुप्सचे सेट परिभाषित करण्यासाठी वापरले जाऊ शकतात.
2९ 2९ Standard Edit Menu commands can be applied to the audio of multiple labeled regions (excluding audio outside those regions). ३० ३० Limitations of Labels लेबलांची मर्यादा
३१ ३१ The text in a label should be less than 2६० characters. Although more than 2६० characters may be entered, if the project is saved, the label will not be loaded when the project is reopened. लेबलमधील मजकूर 2६० वर्णांपेक्षा कमी असावा. 2६० वर्णांपेक्षा जास्त वर्ण प्रविष्ट केले जाऊ शकतात, प्रकल्प जतन केल्यास प्रकल्प पुन्हा उघडल्यावर लेबल लोड केले जाणार नाही.
३2 ३2 If there a very large number of labels in a project (hundreds or thousands), Audacity may become noticeably sluggish. प्रकल्पमध्ये (शेकडो किंवा हजारो) लेबलांची संख्या खूप असल्यास, ओड्यासिटी सुस्त होऊ शकेल.
३३ ३३ There is no simple way to convert a "region label" into a "point label". The easiest way is to delete the region label and add a new point label. "प्रांत लेबल" ला "पॉइंट लेबल" मध्ये रूपांतरित करण्याचा कोणताही सोपा मार्ग नाही. प्रदेश लेबल हटविणे आणि नवीन पॉईंट लेबल जोडणे हा सर्वात सोपा मार्ग आहे.
३४ ३४ (Microsoft Windows) Labels do not support IME input. To enter Chinese / Japanese characters into a label, use copy and paste. (मायक्रोसॉफ्ट विंडोज) लेबल आयएमई इनपुटला समर्थन देत नाहीत. लेबलमध्ये चीनी / जपानी वर्ण प्रविष्ट करण्यासाठी, कॉपी आणि पेस्ट वापरा.
३५ ३५ Point labels and region labels बिंदू लेबले आणि प्रदेश लेबले
३६ ३६ Audacity supports labeling points, a single moment in the audio, and labeling regions, a span with a start and end time. ओड्यासिटी लेबलिंग पॉइंट्स, ध्वनीमधील एक क्षण आणि लेबलिंग क्षेत्रांना प्रारंभ आणि समाप्ती कालावधीसह समर्थन देते.
३७ ३७ A point label is just a region label with the same start and end. पॉईंट लेबल हे समान प्रारंभ आणि अंत असलेले फक्त एक प्रदेश लेबल आहे.
३८ ३८ If you have several region labels one after the other, and each one starts where the previous one ends, they join together. आपल्याकडे एकापाठोपाठ एक प्रांताची अनेक लेबले असल्यास आणि मागील एक जिथे समाप्त होईल तेथे प्रत्येकजण प्रारंभ झाला तर ते एकत्र सामील होतील.
३९ ३९ You can then adjust the end of one label and the start of the next at the same time. त्यानंतर आपण एकाच वेळी एका लेबलचा शेवट आणि पुढील प्रारंभ समायोजित करू शकता.

५७ ५७ Selecting a Label एक लेबल निवडत आहे
