४ ४ Time shifting operations on Sync-Locked Track Groups समक्रमित-लॉक केलेल्या गीतपट्टा गटांवर वेळ बदलण्याची क्रिया
५ ५ Tracks > Align Tracks: All of the "Align Tracks" commands (with the exception of the "Align Together" command) will move all tracks in a group by the same amount, even if this means pushing the start(s) of some track(s) before zero on the timeline. By its nature the Align Tracks Together command cannot respect sync-lock since its job is to align the selected tracks so they all start at the same point on the timeline. गीतपट्टा> गीतपट्टा संरेखित करा: सर्व "गीतपट्टा संरेखित करा" आज्ञा्स ("अलाइन टुगेदर" आज्ञा वगळता) सर्व गीतपट्टा एकाच प्रमाणात गटामध्ये हलवतील, जरी याचा अर्थ काही गीतपट्टाची सुरूवात (ती) ढकलणे असेल. वेळलाइनवर शून्य होण्यापूर्वी त्याच्या स्वभावाने अलाइन गीतपट्टा टुगेदर आज्ञा सिंक-लॉकचा आदर करू शकत नाही कारण त्याचे काम निवडलेल्या गीतपट्टाची संरेखित करणे आहे जेणेकरून ते सर्व एकाच वेळेस वेळलाइनवर सुरू होतील.
६ ६ Time Shift Tool: Using the Time Shift tool on any audio track in a Sync-Locked Track Group will move all clips in all audio tracks, and will move any Label Track defining the Group, even if none of the tracks are selected. However a clip will not move unless it overlaps with at least one other clip above or below. If you want non-overlapping clips to move too, select over all clips you want to move then drag from inside the selection. वेळ शिफ्ट साधन: समक्रमित-लॉक केलेला गीतपट्टा समूहातील कोणत्याही ध्वनी गीतपट्टावर वेळ शिफ्ट साधन वापरणे सर्व क्लिप सर्व ध्वनी गीतपट्टावर हलवेल आणि कोणताही गीतपट्टा निवडला नसला तरीही, समूह परिभाषित करणारे कोणतेही लेबल गीतपट्टा हलवेल. तथापि क्लिप वर किंवा खाली कमीतकमी एका अन्य क्लिपने ओव्हरलॅप केल्याशिवाय हलणार नाही. आपल्याला नॉन-आच्छादित क्लिप्स देखील हलवू इच्छित असल्यास, आपण हलवू इच्छित असलेल्या सर्व क्लिप्सवर निवडा आणि निवडीच्या आतून ड्रॅग करा.
७ ७ Example-१: Start to Cursor/Selection Start उदाहरण -१: कर्सरला प्रारंभ / निवड प्रारंभ
८ ८ Before Tracks > Align Tracks > Start to Cursor/Selection Start गीतपट्टा करण्यापूर्वी> गीतपट्टा संरेखित करा> कर्सर प्रारंभ करा / निवड प्रारंभ
९ ९ After Tracks > Align Tracks > Start to Cursor/Selection Start गीतपट्टा नंतर> गीतपट्टा संरेखित करा> कर्सर प्रारंभ / निवड प्रारंभ
१० १० Result: The selected track is moved so the start of the track aligns with the cursor. निकाल: निवडलेला गीतपट्टा हलविला गेला आहे म्हणून गीतपट्टाची सुरूवात कर्सरशी संरेखित होते.
११ ११ The Sync-Locked track moves by the same amount. समक्रमित-लॉक केलेला गीतपट्टा समान प्रमाणात हलविला जातो.
