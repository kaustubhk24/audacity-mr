५५ ५५ Where the Preferences are stored जेथे प्राधान्ये संग्रहित आहेत
५६ ५६ Audacity Preferences are stored in a configuration file called audacity.cfg. It is a text file which saves all your settings when you exit Audacity normally. The audacity.cfg file can be edited with any text editor and changes to it can be saved if Audacity is already closed. You can find audacity.cfg inside Audacity's folder for application data as follows. ओड्यासिटी प्राधान्ये ओड्यासिटी.एफजी नावाच्या कॉन्फिगरेशन धारिकामध्ये संग्रहित केल्या जातात. ही एक मजकूर धारिका आहे जी आपण सामान्यपणे ओड्यासिटीमधून बाहेर पडता तेव्हा आपल्या सर्व समायोजन जतन करते. ओड्यासिटी.एफ.जी. धारिका कोणत्याही मजकूर संपादकासह संपादित केली जाऊ शकते आणि ओड्यासिटी आधीच बंद असल्यास त्यामधील बदल जतन केले जाऊ शकतात. अनुप्रयोग डेटासाठी खालीलप्रमाणे ऑडिटीच्या फोल्डरमध्ये आपणास ओड्यासिटी.सीएफजी सापडेल.
५७ ५७ Windows: Users\<username>\AppData\Roaming\Audacity\ विंडोजः वापरकर्ते u <वापरकर्तानाव> \ अ‍ॅपडेटा \ रोमिंग \ ओड्यासिटी \
५८ ५८ Mac: ~/Library/Application Support/audacity/ मॅक: ~ / लायब्ररी / अनुप्रयोग समर्थन / धृष्टता /
५९ ५९ Linux: ~/.audacity-data/ लिनक्स: ~ / .audacity-डेटा /
६० ६० In order to see audacity.cfg on Windows, OS X १०.७ or later or GNU/Linux, you must show hidden files and folders or type the folder location into your file manager's address bar. विंडोज, ओएस एक्स १०.७ किंवा त्यानंतरच्या किंवा जीएनयू / लिनक्सवर ऑडीसिटी.सीएफजी पाहण्यासाठी, आपण लपलेल्या धारिका आणि फोल्डर्स दर्शविणे आवश्यक आहे किंवा आपल्या धारिका व्यवस्थापकाच्या अ‍ॅड्रेस बारमध्ये फोल्डरचे स्थान टाइप करणे आवश्यक आहे.

If necessary, show hidden files and folders on Windows or type %appdata%\audacity or shell:appdata\audacity into the Explorer address bar then press Enter on your keyboard. विंडोजः एक्सप्लोररच्या डावीकडील झाडावर, "वापरकर्ते" वर डबल क्लिक करा आणि नंतर आपले वापरकर्तानाव डबल क्लिक करा, नंतर उजवीकडे, अ‍ॅपटाटा किंवा अनुप्रयोग डेटा फोल्डरवर डबल क्लिक करा आणि त्यामधून नॅव्हिगेट करा. आवश्यक असल्यास, विंडोजवर लपविलेल्या धारिका आणि फोल्डर्स दर्शवा किंवा एक्सप्लोरर अ‍ॅड्रेस बारमध्ये% appdata% \ ऑडीसिटी किंवा शेल: dपडेटा \ ऑडीसिटी टाइप करा नंतर आपल्या कीबोर्डवरील एंटर दाबा.
६२ ६२ Mac OS X १०.७ or later: Open Finder, use the "Go" menu, choose "Go to Folder" and type ~/Library/Application Support/audacity/, or set Finder to show your User Library folder. मॅक ओएस एक्स १०.७ किंवा नंतरचे: फाइंडर उघडा, "गो" मेनू वापरा, "फोल्डर वर जा" निवडा आणि User / लायब्ररी / Supportप्लिकेशन सपोर्ट / ओड्यासिटी / टाइप करा किंवा आपला वापरकर्ता लायब्ररी फोल्डर दर्शविण्यासाठी फाइंडर सेट करा.
६३ ६३ Portable Settings पोर्टेबल समायोजन
६४ ६४ You can create a folder called Portable Settings in a special location relative to the directory you installed or extracted Audacity to, so that audacity.cfg and Audacity's other settings will be stored in that "Portable Settings" folder instead. आपण स्थापित केलेल्या डिरेक्टरीशी संबंधित विशिष्ट स्थानात पोर्टेबल समायोजन नावाचे एक फोल्डर तयार करू शकता किंवा ओड्यासिटी काढली आहे, जेणेकरून त्याऐवजी ओड्यासिटी.एफएफ आणि ओड्यासिटीच्या इतर समायोजन त्या "पोर्टेबल समायोजन" फोल्डरमध्ये संग्रहित केल्या जातील.
६५ ६५ The permitted locations are: परवानगी स्थाने अशीः
६६ ६६ Windows: in the folder Audacity is launched from (for example, in Program Files (x८६)\Audacity on Windows ६४-bit) विंडोजः फोल्डरमध्ये ओड्यासिटी लॉन्च केले गेले आहे (उदाहरणार्थ, प्रोग्राम धारिका्समध्ये (x८६) Windows विंडोज ६४-बिटवरील ओड्यासिटी)
६७ ६७ macOS: in the "Contents" folder inside Audacity.app (for example in /Applications/Audacity.app/Contents/ (to access the "Contents" folder, right-click or Ctrl-click over Audacity.app and choose "Show Package Contents") मॅकओएसः ओड्यासिटी.अॅप मधील "सामग्री" फोल्डरमध्ये (उदाहरणार्थ / अनुप्रयोग / ओड्यासिटी.अॅप / कॉन्टेन्ट्स मध्ये अनुक्रमणिका ")
६८ ६८ GNU/Linux: in the folder where the "audacity" binary is (for example, for Audacity installed by the package manager for your distribution, in usr/bin/). जीएनयू / लिनक्स: फोल्डरमध्ये जिथे "ऑडीसिटी" बायनरी आहे (उदाहरणार्थ, आपल्या वितरणासाठी पॅकेज मॅनेजरने ओड्यासिटी स्थापित केले आहे, usr / bin / मध्ये).
६९ ६९ The "Portable Settings" folder becomes active after restarting Audacity. ओड्यासिटी रीस्टार्ट केल्यानंतर "पोर्टेबल समायोजन" फोल्डर सक्रिय होते.
७० ७० You can then use the "Portable Settings" folder to store a different group of settings for a specific purpose, then delete or rename the folder and restart Audacity when you want to use the previous settings as stored in Audacity's folder for application data. त्यानंतर आपण विशिष्ट हेतूसाठी समायोजनचा भिन्न गट संचयित करण्यासाठी "पोर्टेबल समायोजन" फोल्डर वापरू शकता, नंतर अनुप्रयोग हटविण्यासाठी ओड्यासिटीच्या फोल्डरमध्ये पूर्वीच्या समायोजन वापरू इच्छित असल्यास फोल्डर हटवा किंवा नाव बदलून ऑडिटी रीस्टार्ट करा.
७१ ७१ If you copy the "Portable Settings" folder to a USB stick or CD this lets you take your current Audacity settings with you if you use Audacity on another computer. आपण "पोर्टेबल समायोजन" फोल्डर एखाद्या यूएसबी स्टिक किंवा सीडी वर कॉपी केल्यास हे आपण दुसर्‍या संगणकावर ओड्यासिटी वापरत असल्यास आपल्यास आपल्या वर्तमान ओड्यासिटी समायोजन सोबत ठेवू देते.
७२ ७२ The Portable Settings folder must be given full permissions for everyone who wants to use it. पोर्टेबल सेटिंग्‍ज फोल्‍डरला ज्यांना वापरायचे आहे त्या प्रत्येकासाठी पूर्ण परवानग्या दिल्या पाहिजेत.
७३ ७३ On Windows, sufficient permission is normally granted if the administrator creates the folder. If necessary you can use the Icacls command in a terminal, or set access for the required users in the folder's properties. विंडोजवर, प्रशासकाने फोल्डर तयार केल्यास सामान्यत: पुरेशी परवानगी दिली जाते. आवश्यक असल्यास आपण टर्मिनलमध्ये Icacls आदेश वापरू शकता किंवा फोल्डरच्या गुणधर्मांमधील आवश्यक वापरकर्त्यांसाठी प्रवेश सेट करू शकता.

७६ ७६ Upgrading from previous versions or reinstalling मागील आवृत्त्यांमधून श्रेणीसुधारित करणे किंवा पुन्हा स्थापित करणे
७७ ७७ By default, Audacity settings are not changed when you upgrade from a previous release of Audacity or explicitly uninstall or reinstall Audacity. This allows you to upgrade from a previous Audacity version (legacy १.३.२ and later) and retain your previous preferences wherever current Audacity still has an equivalent setting. पूर्वनिर्धारितनुसार, जेव्हा आपण ओड्यासिटीच्या मागील रीलिझवरुन अपग्रेड कराल किंवा स्पष्टपणे ओड्यासिटी विस्थापित कराल किंवा ओड्यासिटी पुन्हा स्थापित कराल तेव्हा ओड्यासिटी समायोजन बदलली जात नाहीत. हे आपल्याला मागील ओड्यासिटी आवृत्ती (लेगसी १.३.२ आणि नंतरच्या) वरून अपग्रेड करण्यास आणि आपल्या मागील प्राधान्ये टिकवून ठेवू देते जिथे सध्याच्या ओड्यासिटीमध्ये समतुल्य सेटिंग आहे.
७८ ७८ Therefore it is not necessary to reinstall the optional LAME or FFmpeg libraries after upgrading Audacity unless the Release Notes for the current Audacity version indicate there is a new recommended version of LAME or FFmpeg. म्हणून ओड्यासिटी अपग्रेड केल्यानंतर वैकल्पिक एलएएमई किंवा एफएफएमपीजी लायब्ररी पुन्हा स्थापित करणे आवश्यक नाही जोपर्यंत चालू ओड्यासिटी आवृत्तीच्या रिलीझ नोट्समध्ये Lame किंवा FFmpeg ची नवीन शिफारस केलेली आवृत्ती असल्याशिवाय सूचित केले जात नाही.
७९ ७९ Resetting Preferences प्राधान्ये रीसेट करत आहे
८० ८० Resetting Preferences can sometimes fix freezes, crashes or unexplained Audacity behavior. प्राधान्ये रीसेट करणे कधीकधी गोठवलेले, क्रॅश किंवा अस्पृश्य ओड्यासिटी वर्तन निश्चित करू शकते.
८१ ८१ You can reset Preferences to factory defaults by exiting Audacity, deleting the audacity.cfg settings file in Audacity's folder for application data then restarting Audacity. On Windows, you can also reinstall Audacity, making sure you select the "Reset Preferences" box during the installation. अनुप्रयोग डेटासाठी ओड्यासिटीच्या फोल्डरमध्ये ओड्यासिटी.एफएफ समायोजन समायोजन हटवून ओड्यासिटी रीस्टार्ट करून आपण फॅक्टरी पूर्वनिर्धारितवर प्राधान्ये रीसेट करू शकता. Windows वर, आपण प्रतिष्ठापन दरम्यान "प्राधान्ये रीसेट करा" बॉक्स निवडल्याची खात्री करुन ओड्यासिटी पुन्हा स्थापित देखील करू शकता.
८२ ८२ Reset Preferences using Windows installer विंडोज इंस्टॉलरचा वापर करून प्राधान्ये पूर्वनिर्धारित करा

८४ ८४ Navigate to the "Select Additional Tasks" screen then put a checkmark in the "Reset Preferences" box and complete installation. "अतिरिक्त कार्ये निवडा" पटलावर नेव्हिगेट करा नंतर "प्राधान्ये पूर्वनिर्धारित करा" बॉक्समध्ये चेकमार्क लावा आणि स्थापना पूर्ण करा.
८५ ८५ Launch Audacity then in the "Reset Audacity Preferences" dialog, choose Yes. ओड्यासिटी उघडा नंतर "पूर्वनिर्धारित करा ओड्यासिटी प्राधान्ये" संवादात, निवडा
