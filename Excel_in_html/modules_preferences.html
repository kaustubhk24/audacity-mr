९ ९ No modules are yet shipped with Audacity. ओड्यासिटीसह अद्याप कोणतेही मॉड्यूल पाठविलेले नाहीत.
१० १० Nyquist Workbench provides the ability to create and run arbitrary Nyquist code in Audacity from a graphical editor. The Workbench can be compiled from our source code. Nyquist Workbench is recommended (though not essential) for anyone interested in writing Nyquist plug-ins for Audacity - good functionality to test Nyquist plug-ins is also available in Nyquist Prompt. Nyquist वर्कबेंच ग्राफिकल एडिटरकडून ओड्यासिटीमध्ये अनियंत्रित Nyquist कोड तयार करण्याची आणि चालविण्याची क्षमता प्रदान करते. वर्कबेंच आमच्या स्त्रोत कोडमधून संकलित केले जाऊ शकते. ऑडीसिटीसाठी Nyquist प्लगइन लिहिण्यास इच्छुक असलेल्या कोणालाही Nyquist वर्कबेंचची शिफारस (आवश्यक नसली तरी) केली जाते - Nyquist प्लगइनची चाचणी घेण्यासाठी चांगली कार्यक्षमता Nyquist प्रॉमप्ट वर देखील उपलब्ध आहे.

१६ १६ Mod-script pipe can only be compiled from our source code. This is a GUI plug-in that allows Audacity to be driven from an external Perl script, or using any other scripting language that supports named pipes. Please be aware of the security implications of allowing other applications or processes to control Audacity. For more information, see Scripting. मोड-स्क्रिप्ट पाईप केवळ आमच्या स्त्रोत कोडवरून संकलित केली जाऊ शकते. हे जीयूआय प्लग-इन आहे जे ऑडिटीला बाह्य पर्ल स्क्रिप्टवरून चालविण्याची परवानगी देते किंवा नामित पाईप्सचे समर्थन करणारी इतर स्क्रिप्टिंग भाषा वापरते. कृपया इतर अनुप्रयोगांना किंवा प्रक्रियांना ओड्यासिटी नियंत्रित करण्याची परवानगी देण्याच्या सुरक्षिततेच्या परिणामाबद्दल जागरूक रहा. अधिक माहितीसाठी स्क्रिप्टिंग पहा.
१७ १७ Mod-track-panel: This is the start of an experiment that will ultimately provide a more flexible panel for the audio, label, MIDI and note tracks. It is currently only of value to developers and must be compiled from the latest Audacity source code. मोड-गीतपट्टा-पॅनेलः ही एका प्रयोगाची सुरूवात आहे जी शेवटी ध्वनी, लेबल, एमआयडीआय आणि टीप गीतपट्टासाठी अधिक लवचिक पॅनेल प्रदान करेल. हे सध्या विकसकांसाठी केवळ मोलाचे आहे आणि नवीनतम ओड्यासिटी स्रोत कोडमधून संकलित केले जाणे आवश्यक आहे.
