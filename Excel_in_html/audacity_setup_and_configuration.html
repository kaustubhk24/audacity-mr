४ ४ For many users, Audacity will just work "out of the box", and you can immediately begin playing or recording using the system-default playback and recording devices. बर्‍याच वापरकर्त्यांसाठी ओड्यासिटी फक्त "बॉक्सच्या बाहेर" कार्य करेल आणि आपण सिस्टम-पूर्वनिर्धारित प्लेबॅक आणि रेकॉर्डिंग उपकरण वापरुन ताबडतोब प्ले करणे किंवा रेकॉर्डिंग सुरू करू शकता.
५ ५ However, because there are so many different types of computers and ways to configure them, some users may need to set up Audacity the first time they use it. तथापि, तेथे बरेच प्रकार आहेत संगणक आणि त्यांचे कॉन्फिगर करण्याचे मार्ग, काही वापरकर्त्यांनी प्रथमच ओड्यासिटी वापरण्याची आवश्यकता असू शकते.
६ ६ Setting Your Audacity Preferences आपली धृष्टता प्राधान्ये सेट करत आहे
७ ७ The main place to set up Audacity and customize it to work the way that you require is to use Audacity's Preferences dialog screens. ओड्यासिटी सेट अप करण्यासाठी आणि त्यास आवश्यकतेनुसार कार्य करण्यासाठी सानुकूलित करण्यासाठी मुख्य स्थान म्हणजे ओड्यासिटीची प्राधान्ये संवाद पडदे वापरणे.
८ ८ Preferences let you change most of the default behaviors of Audacity. प्राधान्ये आपल्याला ओड्यासिटीचे बहुतेक पूर्वनिर्धारित आचरण बदलू देतात.
९ ९ The Preferences dialog can be accessed using the Edit Menu (or by using the shortcut Ctrl + P). On a Mac, Preferences can be found under the Audacity Menu or its shortcut ⌘ + ,. संपादन मेनूचा वापर करून (किंवा आडमार्ग Ctrl + P वापरुन) पसंती संवादात प्रवेश करता येतो. मॅक वर, प्राधान्ये ओड्यासिटी मेनू किंवा त्याच्या आडमार्ग ⌘ +, अंतर्गत आढळू शकतात.
१० १० The Preferences dialog is split into a number of sections each with its own dialog page. There are many settings that can be made in your Preferences, so do take some time to explore and understand these. पसंती संवाद त्याच्या स्वतःच्या संवाद पृष्ठासह प्रत्येकास अनेक विभागात विभागले गेले आहे. आपल्या समायोजनमध्ये बनवल्या जाऊ शकणार्‍या बर्‍याच समायोजन आहेत, म्हणून या एक्सप्लोर करण्यात आणि समजून घेण्यासाठी थोडा वेळ घ्या.

१६ १६ Changing settings with Audacity's toolbars and menus ओड्यासिटीच्या साधनपट्टी आणि मेनूसह समायोजन बदलत आहे
१७ १७ Some of the more commonly used settings can be changed and reset from the main Audacity Window: सामान्यतः वापरल्या जाणार्‍या काही समायोजन बदलल्या जाऊ शकतात आणि मुख्य ओड्यासिटी विंडोमधून रीसेट केल्या जाऊ शकतात:
१८ १८ Device Toolbar can be used to change the recording and playback devices, the recording host and the number of channels उपकरण साधनपट्टी रेकॉर्डिंग आणि प्लेबॅक साधने, रेकॉर्डिंग होस्ट आणि वाहिनीची संख्या बदलण्यासाठी वापरला जाऊ शकतो
१९ १९ Selection Toolbar can be used to change the Project Rate and Selection Format units प्रकल्प रेट आणि सिलेक्शन फॉर्मेट युनिट बदलण्यासाठी सिलेक्शन साधनपट्टीचा वापर केला जाऊ शकतो
२० २० Transport Menu can be used to control Overdub, Software Playthrough and Sound Activated Recording. ओव्हरडब, अनुप्रयोग प्लेथ्रू आणि साऊंड अ‍ॅक्टिवेटेड रेकॉर्डिंग नियंत्रित करण्यासाठी ट्रान्सपोर्ट मेनूचा वापर केला जाऊ शकतो.
२१ २१ Using these can be quicker than accessing the preferences dialog to change these settings. या समायोजन बदलण्यासाठी प्राधान्ये संवादमध्ये प्रवेश करणे यापेक्षा हे वापरणे जलद असू शकते.
२२ २२ You can also reset Audacity settings to their factory-shipped state. This may occasionally fix unexpected or unwanted behavior. आपण ओड्यासिटी समायोजन त्यांच्या फॅक्टरी-पाठवलेल्या स्थितीमध्ये रीसेट देखील करू शकता. हे कधीकधी अनपेक्षित किंवा अवांछित वर्तन निराकरण करू शकते.
