१० १० Spectrograms Preferences. स्पेक्ट्रोग्राम प्राधान्ये.
११ ११ The options in this window apply to the Spectrogram View in the Audio Track Dropdown Menu. Some options have no effect, or a different effect, when the Pitch (EAC) algorithm is selected - these will be noted in the descriptions. या विंडोमधील पर्याय ध्वनी गीतपट्टा ड्रॉपडाउन मेनूमधील स्पेक्ट्रोग्राम दृश्यावर लागू होतात. जेव्हा पिच (ईएसी) अल्गोरिदम निवडला जातो तेव्हा काही पर्यायांवर कोणताही प्रभाव किंवा भिन्न प्रभाव पडत नाही - हे वर्णनात नमूद केले जाईल.
१२ १२ Be aware that it is possible to make settings for individual Spectrogram view tracks that can over-ride the global settings that you make here in the Spectrograms Preferences dialog. स्पेक्ट्रोग्राम प्राधान्ये संवादात आपण बनवलेल्या जागतिक समायोजनमध्ये जास्त सवारी करू शकणार्‍या वैयक्तिक स्पेक्ट्रोग्राम दृश्य गीतपट्टासाठी समायोजन बनविणे शक्य आहे याची जाणीव ठेवा.
१३ १३ See Spectrogram Settings on the Spectrogram View page. स्पेक्ट्रोग्राम दृश्य पृष्ठावरील स्पेक्ट्रोग्राम समायोजन पहा.
१४ १४ Scale स्केल
१५ १५ Scale (in Spectrogram views): स्केल (स्पेक्ट्रोग्राम दृश्यांमध्ये):
१६ १६ Linear The linear vertical scale goes linearly from ० kHz to ८ kHz frequency by default. रेखीय रेखीय अनुलंब स्केल पूर्वनिर्धारितनुसार ० केएचझेड ते ८ केएचझेड वारंवारतापासून रेषात्मकपणे जातो.
१७ १७ Logarithmic: This view is the same as the linear view except that the vertical scale is logarithmic. See Spectrogram View for a contrasting example of linear versus logarithmic spectrogram view. लोगारिथमिकः हे दृश्य रेखीय दृश्यासारखेच आहे परंतु त्याशिवाय अनुलंब स्केल लॉगरिथमिक आहे. रेखीय विरूद्ध लॉगॅरिथमिक स्पेक्ट्रोग्राम दृश्यासाठी विरोधाभासी उदाहरणासाठी स्पेक्ट्रोग्राम दृश्य पहा.
१८ १८ Mel: The name Mel comes from the word मेल: मेल नावाच्या शब्दावरून आला आहे
१९ १९ melody to indicate that the scale is based on pitch comparisons. See this Wikipedia page. स्केल खेळपट्टीच्या तुलनेत आधारित आहे हे दर्शविण्यासाठी चाल. हे विकिपीडिया पृष्ठ पहा.
२० २० Bark: This is a psychoacoustical scale based on subjective measurements of loudness. झाडाची साल: मोठ्याने जोर देण्याच्या व्यक्तिनिष्ठ मोजमापांवर आधारित हा एक मनोवैज्ञानिक स्तर आहे.
२१ २१ It is related to, but somewhat less popular than, the Mel scale. हे मेल स्केलशी संबंधित आहे, परंतु काहीसे कमी लोकप्रिय आहे.
२२ २२ See this Wikipedia page. हे विकिपीडिया पृष्ठ पहा.
२३ २३ ERB: The Equivalent Rectangular Bandwidth scale or ERB is a measure used in psychoacoustics, which gives an approximation to the bandwidths of the filters in human hearing. ईआरबीः समतुल्य आयताकृती बँडविड्थ स्केल किंवा ईआरबी मानसशास्त्रात वापरला जाणारा एक उपाय आहे, जो मानवी सुनावणीतील गाळणीच्या बँडविड्थांना अंदाजे स्थान देतो.
२४ २४ It is implemented as a function ERBS(f) which returns the number of equivalent rectangular bandwidths below the given frequency "f". हे फंक्शन ईआरबीएस (एफ) म्हणून लागू केले गेले आहे जे दिलेली वारंवारता "एफ" च्या खाली समांतर आयताकृती बँडविड्थची संख्या परत करते.
२५ २५ See this Wikipedia page. हे विकिपीडिया पृष्ठ पहा.
२६ २६ Period: is the previously undocumented scale used by Pitch (EAC) view. पीरियडः पिच (ईएसी) दृश्याने वापरलेले पूर्वीचे Undocumented स्केल आहे.
२७ २७ It is for making the same displays of Pitch possible as in earlier versions of Audacity. हे ओड्यासिटीच्या पूर्वीच्या आवृत्तींप्रमाणेच पिचचे समान प्रदर्शन शक्य करण्यासाठी आहे.
२८ २८ Minimum Frequency: This value corresponds to the bottom of the vertical scale in the spectrogram. Frequencies below this value will not be visible. The default value of "०" here will be treated as "१" when using "Spectrogram Logarithmic" view mode because a logarithmic scale cannot start at zero. किमान वारंवारता: हे मूल्य स्पेक्ट्रोग्राममधील अनुलंब स्केलच्या तळाशी संबंधित आहे. या मूल्याच्या खाली वारंवारता दृश्यमान होणार नाहीत. "स्पेक्ट्रोग्राम लोगारिथमिक" व्ह्यू मोडचा वापर करताना येथे "०" चे पूर्वनिर्धारित मूल्य "१" म्हणून समजले जाईल कारण लॉगेरिथमिक स्केल शून्यावर प्रारंभ होऊ शकत नाही.
२९ २९ Maximum Frequency: This value corresponds to the top of the vertical scale. The value can be set to १०० Hz or any higher value. Irrespective of the entered value, the top of the scale will never exceed half the current sample rate of the track (for example, २२०५० Hz if the track rate is ४४१०० Hz) because any given sample rate can only carry frequencies up to half that rate. A good use of this setting is in speech recognition or pitch extraction, where you can hide the visually unimportant highest frequencies and focus on the lower frequencies. जास्तीत जास्त वारंवारता: हे मूल्य अनुलंब स्केलच्या शीर्षाशी संबंधित आहे. मूल्य १०० हर्ट्ज किंवा कोणत्याही उच्च मूल्यावर सेट केले जाऊ शकते. प्रविष्‍ट केलेल्या मूल्याची पर्वा न करता, प्रमाणातील सर्वात वरचा भाग गीतपट्टाच्या सद्य नमुना दरापेक्षा अर्धा कधीही ओलांडू शकणार नाही (उदाहरणार्थ, गीतपट्टाचा दर ४४१०० हर्ट्ज असल्यास २२०५० हर्ट्ज) कारण कोणताही नमुना दर त्या अर्ध्या दरापर्यंत वारंवारता बाळगू शकतो. . या सेटिंगचा चांगला वापर भाषण ओळखणे किंवा पिच एक्सगीतपट्टा्शनमध्ये आहे, जिथे आपण नेत्रहीन महत्त्वाचे नसलेले उच्च वारंवारता लपवू शकता आणि कमी वारंवारतेवर लक्ष केंद्रित करू शकता.
३० ३० Colors रंग
३१ ३१ Gain (dB): This enables you to increase / decrease the brightness of the display. गेन (डीबी): हे आपल्याला प्रदर्शनाची चमक वाढविण्यास / कमी करण्यास सक्षम करते.
३२ ३२ For small signals where the display is mostly "blue" (dark) you can increase this value to see brighter colors and give more detail. लहान सिग्नलसाठी जिथे प्रदर्शन बहुतेक "निळे" (गडद) असते तेथे उजळ रंग पाहण्यासाठी आणि अधिक तपशील देण्यासाठी आपण हे मूल्य वाढवू शकता.
३३ ३३ If the display has too much "white", decrease this value. जर डिस्प्लेमध्ये बरेच "पांढरे" असतील तर हे मूल्य कमी करा.
३४ ३४ The default is २०dB and corresponds to a -२० dB signal at a particular frequency being displayed as "white". This option has no effect and is grayed out when the Pitch (EAC) algorithm is selected. पूर्वनिर्धारित २० डीबी आहे आणि -२० डीबी सिग्नलशी संबंधित आहे विशिष्ट आवृत्ति "व्हाइट" म्हणून प्रदर्शित केल्या जात आहे. जेव्हा या पिच (ईएसी) अल्गोरिदमची निवड केली जाते तेव्हा या पर्यायाचा कोणताही परिणाम होत नाही आणि राखाडी आहे.
३५ ३५ Range (dB): Affects the range of signal sizes that will be displayed as colors. श्रेणी (डीबी): रंग म्हणून दर्शविल्या जाणार्‍या सिग्नल आकारांच्या श्रेणीवर परिणाम करते.
३६ ३६ The default is ८० dB and means that you will not see anything for signals ८० dB below the value set for "Gain". This option has no effect and is grayed out when the Pitch (EAC) algorithm is selected. पूर्वनिर्धारित d० डीबी आहे आणि याचा अर्थ असा आहे की आपण "गेन" साठी सेट केलेल्या मूल्याच्या खाली d० डीबी सिग्नलसाठी काहीही दिसणार नाही. जेव्हा या पिच (ईएसी) अल्गोरिदमची निवड केली जाते तेव्हा या पर्यायाचा कोणताही परिणाम होत नाही आणि राखाडी आहे.
३७ ३७ Frequency Gain (dB/dec): A positive value here gives some extra gain to higher frequencies (above १००० Hz), as they tend to be smaller and so cannot be seen as well. वारंवारता गेन (डीबी / डिक): इथले एक सकारात्मक मूल्य उच्च वारंवारताना (१००० हर्ट्झपेक्षा जास्त) काही अतिरिक्त वाढ देते कारण ते कमी असतात आणि म्हणूनही पाहिले जाऊ शकत नाही.
३८ ३८ You get less gain at lower frequencies as well. आपल्याला कमी वारंवारतावर देखील कमी नफा मिळतो.
३९ ३९ The default is ० dB. This option has no effect and is grayed out when the Pitch (EAC) algorithm is selected. पूर्वनिर्धारित ० डीबी आहे. जेव्हा या पिच (ईएसी) अल्गोरिदमची निवड केली जाते तेव्हा या पर्यायाचा कोणताही परिणाम होत नाही आणि राखाडी आहे.
४० ४० Grayscale: Shows gray shades in all spectrogram views instead of full color. ग्रेस्केल: संपूर्ण रंगाऐवजी सर्व स्पेक्ट्रोग्राम दृश्यांमध्ये करड्या छटा दाखवतात.
४१ ४१ Algorithm अल्गोरिदम
४२ ४२ Algorithm: अल्गोरिदम:
४३ ४३ Frequencies (default): Audio frequency determines the pitch of a sound. वारंवारता (पूर्वनिर्धारित): ध्वनी वारंवारता ध्वनीची खेळणी निश्चित करते.
४४ ४४ Measured in Hz, higher frequencies have higher pitch. हर्ट्झमध्ये मोजले जाणारे, उच्च वारंवारतामध्ये जास्त खेळपट्टी असते.
४५ ४५ See this Wikipedia article. हा विकिपीडिया लेख पहा.
४६ ४६ Reassignment: The method of reassignment sharpens blurry time-frequency data by relocating the data according to local estimates of instantaneous frequency and group delay. This mapping to reassigned time-frequency coordinates is very precise for signals that are separable in time and frequency with respect to the analysis window. पुन्हा नियुक्त करणे: पुन्हा नियुक्त करण्याची पद्धत त्वरित वारंवारता आणि गटाच्या विलंबाच्या स्थानिक अंदाजानुसार डेटा पुनर्स्थित करून अस्पष्ट वेळ-वारंवारता डेटा तीव्र करते. विश्लेषण केलेल्या विंडोच्या संदर्भात वेळ आणि वारंवारतेनुसार विभक्त होणार्‍या सिग्नलसाठी पुन्हा नियुक्त केलेल्या वेळ-वारंवारता समन्वयांचे हे मॅपिंग अगदी अचूक आहे.
४७ ४७ Pitch (EAC): Highlights the contour of the fundamental frequency (musical pitch) of the audio, using the Enhanced Autocorrelation (EAC) algorithm. The EAC Algorithm was developed to produce a mathematical representation of the changes of pitch in a piece of audio. The aim was to allow automated comparison of sound files so that two versions of the same tune could be recognized as being similar, even if played in different keys, or on different instruments. पीच (ईएसी): वर्धित ऑटोकॉरेलेशन (ईएसी) अल्गोरिदम वापरुन ध्वनीच्या मूलभूत वारंवारतेचे (संगीतमय पिच) समोच्च हायलाइट करते. ईएसी अल्गोरिदम ध्वनीच्या तुकड्यात खेळपट्टीच्या बदलांचे गणितीय प्रतिनिधित्व करण्यासाठी विकसित केले गेले. ध्वनी धारिकांच्या स्वयंचलित तुलनाची परवानगी देणे हाच हेतू होता जेणेकरून भिन्न टोनमध्ये किंवा भिन्न इंस्ट्रूमेंटवर प्ले केल्या गेल्या तरीही समान ट्यूनच्या दोन आवृत्त्या समान असल्याचे ओळखता येईल.
४८ ४८ Window Size: The dropdown menu lets you choose the size of the Fast Fourier Transform (FFT) window which affects how much vertical (frequency) detail you see. विंडोचा आकारः ड्रॉपडाउन मेनूमुळे आपल्याला फास्ट फूरियर ट्रान्सफॉर्म (एफएफटी) विंडोचा आकार निवडता येतो ज्यामुळे आपण किती उभ्या (वारंवारता) तपशीलांवर परिणाम करता.
४९ ४९ Larger FFT window sizes give more low frequency resolution and less temporal resolution, and are slower. मोठे एफएफटी विंडो आकार अधिक कमी वारंवारता रिझोल्यूशन आणि कमी ऐहिक रिझोल्यूशन देतात आणि हळू असतात.
५० ५० Window type: Determines precisely how the spectrogram is computed. विंडोचा प्रकारः स्पेक्ट्रोग्रामची गणना कशी केली जाते हे तंतोतंतपणे निर्धारित करते.
५१ ५१ 'Rectangular' is slightly faster than other methods, but introduces some artifacts. 'आयताकृती' इतर पद्धतींपेक्षा किंचित वेगवान आहे, परंतु काही कलाकृतींचा परिचय देते.
५२ ५२ All methods give broadly similar results. सर्व पद्धती मोठ्या प्रमाणात समान परिणाम देतात.

५६ ५६ Enable Spectral Selection वर्णक्रमीय निवड सक्षम करा
५७ ५७ Check this box "on" if you want to enable spectral selections. आपण वर्णक्रमीय निवडी सक्षम करू इच्छित असल्यास हा बॉक्स "चालू" पहा.
५८ ५८ These are used to make selections that include a frequency range as well as a time range on tracks in one of the Spectrogram views. Spectral selection is used with special spectral edit effects to make changes to the frequency content of the selected audio. Among other purposes, spectral selection and editing can be used for cleaning up unwanted sound, enhancing certain resonances, changing the quality of a voice or removing mouth sounds from voice work. हे निवडी करण्यासाठी वापरले जातात ज्यात स्पेक्ट्रोग्राम दृश्यांपैकी एकामधील वारंवारता श्रेणी तसेच गीतपट्टावर वेळ श्रेणी समाविष्ट असते. निवडलेल्या ध्वनीच्या वारंवारता सामग्रीमध्ये बदल करण्यासाठी स्पेक्ट्रल निवड विशेष वर्णक्रमीय संपादन प्रभावासह वापरली जाते. इतर कारणांपैकी अवांछित ध्वनी साफ करण्यासाठी, विशिष्ट अनुनादांना वर्धित करण्यासाठी, आवाजाची गुणवत्ता बदलण्यासाठी किंवा व्हॉइसच्या कामातून तोंडातील आवाज काढून टाकण्यासाठी वर्णक्रमीय निवड आणि संपादन वापरले जाऊ शकते.
५९ ५९ Setting this to "on" or "off" will only affect spectrogram displayed tracks which have not already been independently set for this purpose in their Track Control Panel's Spectrogram Preferences. The Track Control Panel setting will override this Preferences setting if Use Preferences is unchecked in the Track Control Panel's Spectrogram Preferences. हे "चालू" किंवा "बंद" वर सेट केल्याने केवळ स्पेक्ट्रोग्राम प्रदर्शित गीतपट्टावर परिणाम होईल जे त्यांच्या गीतपट्टा कंट्रोल पॅनेलच्या स्पेक्ट्रोग्राम प्राधान्यांमध्ये या हेतूसाठी स्वतंत्रपणे सेट केलेले नाहीत. गीतपट्टा कंट्रोल पॅनेलच्या स्पेक्ट्रोग्राम प्राधान्यांमध्ये वापर प्राधान्ये न तपासल्यास गीतपट्टा नियंत्रण पॅनेल सेटिंग ही प्राधान्ये सेटिंग अधिशून्य करेल.
