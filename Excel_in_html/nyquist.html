४ ४ Nyquist is a programming language for sound synthesis and analysis based on the Lisp programming language. It is an extension of the XLISP dialect of Lisp, and is named after Harry Nyquist. It can be used to write plug-in effects for Audacity. Nyquist लिस्प प्रोग्रामिंग भाषेच्या आधारे ध्वनी संश्लेषण आणि विश्लेषणासाठी प्रोग्रामिंग भाषा आहे. हे लिस्पच्या एक्सएलआयएसपी बोलीचा विस्तार आहे आणि हॅरी न्यक्विस्टच्या नावावर आहे. याचा उपयोग ओड्यासिटीसाठी प्लग-इन प्रभाव लिहिण्यासाठी केला जाऊ शकतो.
५ ५ The Nyquist programming language and interpreter were written by Roger Dannenberg at Carnegie Mellon University, with support from Yamaha Corporation and IBM. यमाहा कॉर्पोरेशन आणि आयबीएमच्या पाठिंब्याने कार्नेगी मेलॉन युनिव्हर्सिटीमध्ये रॉयर्स डॅन्नेनबर्ग यांनी नेक्विस्ट प्रोग्रामिंग भाषा आणि दुभाषिया लिहिले आहेत.

११ ११ Nyquist Programming Nyquist प्रोग्रामिंग
१2 १2 Nyquist was written by Roger B. Dannenberg and was intended to be used as a complete programming language for audio synthesis and analysis, with support for MIDI, audio recording and playback, file I/O, object-oriented programming, profiling, debugging and more. नायक्विस्ट रॉजर बी. डॅन्नेनबर्ग यांनी लिहिलेले होते आणि ध्वनी संश्लेषण आणि विश्लेषणासाठी एमआयडीआय, ध्वनी ध्वनीमुद्रणिंग आणि प्लेबॅक, धारिका आय / ओ, ऑब्जेक्ट-ओरिएंटेड प्रोग्रामिंग, प्रोधारिकािंग, डीबगिंग आणि बरेच काही यासाठी एक संपूर्ण प्रोग्रामिंग भाषा म्हणून वापरली जाण्याची इच्छा होती .
१३ १३ Audacity uses only a subset of Nyquist’s functionality, allowing you to take simple Nyquist functions and use them to process audio data. Audacity doesn’t include any support for debugging Nyquist code, so if you are trying to write a complicated plug-in, you may find it easier to get the full version of Nyquist and develop there, then turn it into an Audacity plug-in. Nyquist is available from the Carnegie Mellon University Computer Music Project. ओड्यासिटी केवळ Nyquist च्या कार्यक्षमतेचा उपसंच वापरते, आपल्याला साधी Nyquist कार्ये करण्याची आणि ध्वनी डेटावर प्रक्रिया करण्यासाठी त्यांचा वापर करण्याची परवानगी देते. ऑडीसिटीमध्ये एनक्यूइस्ट कोड डीबग करण्यासाठी कोणतेही समर्थन समाविष्ट नाही, म्हणून जर आपण एक गुंतागुंतीचे प्लग-इन लिहिण्याचा प्रयत्न करीत असाल तर आपल्याला Nyquist ची संपूर्ण आवृत्ती मिळवणे आणि तेथे विकसित करणे सुलभ वाटेल, तर त्यास ओड्यासिटी प्लग-इनमध्ये रूपांतरित करा. . कार्केगी मेलॉन युनिव्हर्सिटी संगणक संगीत प्रकल्पातून Nyquist उपलब्ध आहे.
१४ १४ You can download the full Nyquist from here: CMU Computer Music Project Software. आपण येथून संपूर्ण Nyquist डाउनलोड करू शकता: सीएमयू संगणक संगीत प्रकल्प सॉफ्टवेअर.
