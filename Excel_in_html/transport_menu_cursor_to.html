४ ४ These commands let you move the cursor to the start or end of the selection, track or any adjacent Clip that you may have. या आज्ञा्स तुम्हाला निवड, गीतपट्टा किंवा तुमच्या जवळच्या कोणत्याही क्लिपच्या सुरूवातीस किंवा अंतात कर्सर हलवू देतात.
५ ५ Click on the image to learn more अधिक जाणून घेण्यासाठी प्रतिमेवर क्लिक करा
६ ६ Selection Start निवड प्रारंभ
७ ७ When there is a selection, moves the cursor to the start of the selection and removes the selection without changing the zoom level. If the left edge of the selection is not visible, scrolls the display so that the cursor is visible. जेव्हा निवड असते तेव्हा कर्सर निवडीच्या प्रारंभाकडे हलवते आणि झूम पातळी बदलल्याशिवाय निवड काढून टाकते. जर निवडीची डावी बाजू दृश्यमान नसेल, तर प्रदर्शन स्क्रोल करा जेणेकरून कर्सर दृश्यमान होईल.
८ ८ Cursor moved to the start of the previously selected region with the selection removed. कर्सर पूर्वी निवडलेल्या प्रदेशाच्या सुरूवातीस निवड काढून टाकला.
९ ९ Selection End निवड समाप्त
१० १० When there is a selection, moves the cursor to the end of the selection and removes the selection without changing the zoom level. If the right edge of the selection is not visible, scrolls the display so that the cursor is visible. जेव्हा निवड असते, कर्सर निवडीच्या शेवटी हलवते आणि झूम पातळी बदलल्याशिवाय निवड काढून टाकते. जर निवडीची उजवी धार दिसत नसेल तर प्रदर्शनला स्क्रोल करा जेणेकरून कर्सर दृश्यमान होईल.
११ ११ Cursor moved to the end of the previously selected region with the selection removed. कर्सर पूर्वी निवडलेल्या प्रदेशाच्या टोकाकडे गेला आणि निवड हटविली.
१२ १२ Track Start  J गीतपट्टा प्रारंभ जे
१३ १३ Moves the cursor to the start of the selected track, removing any selection. If the left edge of the track is not visible, scrolls the display so that the cursor is visible. कोणतीही निवड काढून, निवडलेल्या गीतपट्टाच्या सुरूवातीस कर्सर हलवते. गीतपट्टाची डावी धार दिसत नसल्यास, प्रदर्शन स्क्रोल करते जेणेकरून कर्सर दृश्यमान होईल.
१४ १४ The shorter track was the one selected, so the cursor is moved to the ३८ second mark at the beginning of that track with the selection removed. छोटा गीतपट्टा निवडलेला होता, म्हणून कर्सर त्या गीतपट्टाच्या सुरूवातीस ३८ सेकंदांच्या मार्कवर नेऊन निवडला गेला.
१५ १५ Track End  K गीतपट्टा एंड के
१६ १६ Moves the cursor to the end of the selected track, removing any selection. If the right edge of the track is not visible, scrolls the display so that the cursor is visible. कोणतीही निवड काढून, निवडलेल्या गीतपट्टाच्या शेवटी कर्सर हलवते. गीतपट्टाची उजवी धार दिसत नसल्यास, प्रदर्शन स्क्रोल करते जेणेकरून कर्सर दृश्यमान होईल.
१७ १७ Operating on the same track selection as above, the cursor is moved to the १:२८ mark at the end of the shorter selected track with the selection removed. वरील प्रमाणेच गीतपट्टा निवडीवर कार्य करीत, कर्सर निवड निवडलेल्या गीतपट्टाच्या छोट्या निवडानंतर १:२८ मार्क वर हलविला जाईल.
१८ १८ Previous Clip Boundary मागील क्लिप सीमा
१९ १९ Moves the cursor position backwards to the previous clip boundary. मागील क्लिप सीमेवर कर्सर स्थिती मागे सरकते.
२० २० If the cursor is inside a clip, but not exactly on the left boundary, this command will move the cursor to the left boundary of the clip. जर कर्सर क्लिपच्या आत असेल, परंतु अगदी डाव्या सीमेवर नसेल तर, ही आज्ञा कर्सरला क्लिपच्या डाव्या सीमेवर हलवेल.
२१ २१ If the cursor is exactly on the left boundary of a clip, this command will move the cursor to the right boundary of the previous clip. जर कर्सर अगदी क्लिपच्या डाव्या सीमेवर असेल तर ही आज्ञा मागील क्लिपच्या उजव्या सीमेवर कर्सर हलवेल.
२२ २२ If the cursor is between clips, this command will move the cursor to the right boundary of the previous clip. जर कर्सर क्लिपच्या दरम्यान असेल तर ही आज्ञा मागील क्लिपच्या उजव्या सीमेवर कर्सर हलवेल.
२३ २३ If the cursor is at the left edge of the first clip in the track, this command does nothing. जर कर्सर गीतपट्टामधील पहिल्या क्लिपच्या डाव्या काठावर असेल तर, ही आज्ञा काहीही करत नाही.
२४ २४ Both Previous Clip Boundary and Next Clip Boundary (below) use the clips in the selected track or tracks. If no tracks are selected, the clips in all tracks are used. मागील क्लिप सीमा आणि पुढील क्लिप सीमा (खाली) दोन्ही निवडलेल्या गीतपट्टा किंवा गीतपट्टामध्ये क्लिप वापरतात. कोणतेही गीतपट्टा निवडलेले नसल्यास, सर्व गीतपट्टामधील क्लिप वापरल्या जातील.
२५ २५ Next Clip Boundary पुढील क्लिप सीमा
२६ २६ Moves the cursor position forwards to the next clip boundary. कर्सर स्थान पुढील क्लिप सीमेवर हलवते.
२७ २७ If the cursor is inside a clip, but not exactly on the right boundary, this command will move the cursor to the right boundary of the clip. जर कर्सर क्लिपच्या आत असेल, परंतु अगदी उजव्या सीमेवर नसेल तर, ही आज्ञा कर्सरला क्लिपच्या उजव्या सीमेवर हलवेल.
२८ २८ If the cursor is exactly on the right boundary of a clip, this command will move the cursor to the left boundary of the next clip. जर कर्सर अगदी क्लिपच्या उजव्या सीमेवर असेल तर, ही आज्ञा कर्सरला पुढील क्लिपच्या डाव्या सीमेवर हलवेल.
२९ २९ If the cursor is between clips, this command will move the cursor to the left boundary of the next clip. जर कर्सर क्लिप दरम्यान असेल तर ही आज्ञा पुढील क्लिपच्या डाव्या सीमेवर कर्सर हलवेल.
३० ३० If the cursor is at the right edge of the last clip in the track, this command does nothing. जर कर्सर गीतपट्टामधील शेवटच्या क्लिपच्या उजव्या काठावर असेल तर, ही आज्ञा काहीही करत नाही.
३१ ३१ Both Previous Clip Boundary and Next Clip Boundary will scroll the display to show the cursor if the result of the command moves the cursor out of view. आज्ञाचा परिणाम कर्सरला दृश्यास्पद बाहेर हलवित असल्यास मागील क्लिप सीमा आणि पुढील क्लिप सीमा दोन्ही कर्सर दर्शविण्यासाठी प्रदर्शन स्क्रोल करेल.
३२ ३२ Project Start  Home प्रकल्प स्टार्ट होम
३३ ३३ Moves the cursor to the beginning of the project. This is useful if you want to play everything, or record a new track starting from the beginning. Same as clicking the Skip to Start button प्रकल्पच्या सुरुवातीस कर्सर हलवते. आपण सर्वकाही प्ले करू इच्छित असल्यास, किंवा प्रारंभपासून नवीन गीतपट्टा ध्वनीमुद्रण करू इच्छित असल्यास हे उपयुक्त आहे. प्रारंभ करण्यासाठी वगळा बटणावर क्लिक करण्यासारखेच
३४ ३४ in Transport Toolbar. ट्रान्सपोर्ट साधनपट्टी मध्ये.
३५ ३५ Project End  End प्रकल्प एंड एंड
३६ ३६ Moves the cursor to the end of the project. Same as clicking the Skip to End button प्रकल्पच्या शेवटी कर्सर हलवते. वगळा बटणावर क्लिक करण्यासारखेच
३७ ३७ in Transport Toolbar. ट्रान्सपोर्ट साधनपट्टी मध्ये.
