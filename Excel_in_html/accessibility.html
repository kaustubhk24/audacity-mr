६ ६ Keyboard shortcuts कीबोर्ड आडमार्ग
७ ७ A large number of keyboard shortcuts are available for those who cannot use a mouse, and for those who find it quicker to use the keyboard. These keyboard shortcuts are described throughout the manual, and in the Keyboard Shortcut Reference. They can be customized on the Keyboard page of the Preferences dialog. ज्यांना माउस वापरू शकत नाही त्यांच्यासाठी आणि जे कीबोर्ड वापरणे जलद गतीने शोधतात त्यांच्यासाठी मोठ्या संख्येने कीबोर्ड आडमार्ग उपलब्ध आहेत. हे कीबोर्ड आडमार्ग संपूर्ण माहितीपुस्तिका आणि कीबोर्ड आडमार्ग संदर्भात वर्णन केले आहेत. ते पसंती संवाद च्या कीबोर्ड पृष्ठावर सानुकूलित केले जाऊ शकतात.

११ ११ Mac keyboard navigation मॅक कीबोर्ड निर्देशक
१२ १२ The following methods of keyboard navigation which are standard on Windows and Linux work differently on Mac. विंडोज आणि लिनक्स वर मानक असलेल्या कीबोर्ड निर्देशकच्या खालील पद्धती मॅकवर वेगळ्या पद्धतीने कार्य करतात.
१3 १3 Alt key navigation of menus and buttons is not supported by Apple. To access menus, hold Ctrl and press F२ then type the first few letters of the root menu item, then type the first few letters of the item required in that menu. अलद्वारे साधने आणि बटणाचे Alt कि निर्देशक समर्थित नाही. मेनूमध्ये प्रवेश करण्यासाठी, Ctrl दाबून ठेवा आणि F२ दाबा नंतर मूळ साधने प्रथम काही अक्षरे टाइप करा, त्यानंतर त्या मेनूमध्ये आवश्यक असलेल्या गोष्टींची प्रथम काही अक्षरे टाइप करा.
१4 १4 Tabbing in dialogs other than between text boxes and lists is off by default. Tabbing can be enabled by opening the System Keyboard Preferences, choose the "Keyboard Shortcuts" tab, then under "Full Keyboard Access", choose the radio button "All controls". मजकूर चौकटी आणि याद्या व्यतिरिक्त संवादांमधील टॅबिंग पूर्वनियोजनानुसार बंद आहे. प्रणाली कीबोर्ड प्राधान्ये उघडुन टॅबिंग सक्षम केले जाऊ शकते, "कीबोर्ड आडमार्ग" टॅब निवडा आणि नंतर "पूर्ण कीबोर्ड प्रवेश" अंतर्गत रेडिओ बटण "सर्व नियंत्रणे" निवडा.

२3 २3 On Mac मॅक वर
२4 २4 Unfortunately, Audacity २.3.० is not accessible for users of VoiceOver, and this was also the case for a number of recent releases. However, an accessible version of Audacity २.१.१ is available to download on the Mac page of the Audacity website. The name of this accessible version is "२.१.१-screen-reader" (DMG and ZIP downloads are available). It is hoped that future versions of Audacity will be accessible for users of VoiceOver. दुर्दैवाने, ओड्यासिटी २.3.० व्हॉइसओव्हरच्या वापरकर्त्यांसाठी प्रवेशयोग्य नाही आणि बर्‍याच अलीकडील रिलीझसाठीही हीच परिस्थिती होती. तथापि, ओड्यासिटी २.१.१ ची प्रवेशयोग्य आवृत्ती ओड्यासिटी वेबसाइटच्या मॅक पृष्ठावर डाउनलोड करण्यासाठी उपलब्ध आहे. या प्रवेशयोग्य आवृत्तीचे नाव "२.१.१-स्क्रीन-रीडर" आहे (डीएमजी आणि पिन डाउनलोड उपलब्ध आहेत). अशी आशा आहे की व्हॉइसओव्हरच्या वापरकर्त्यांसाठी ओड्यासिटीची भविष्यातील आवृत्ती प्रवेशयोग्य असेल.
