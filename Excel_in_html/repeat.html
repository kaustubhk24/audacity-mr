१ १ Repeat पुन्हा करा

४ ४ Repeat repeats the selection the specified number of times. पुनरावृत्ती पुनरावृत्ती निवडीची निर्दिष्ट वेळाची पुनरावृत्ती करते.
५ ५ Accessed by: Effect > Repeat द्वारा प्रवेश केलेले: प्रभाव> पुनरावृत्ती
६ ६ Number of repeats to add जोडण्यासाठी पुनरावृत्तीची संख्या
७ ७ Specifies the number of times the selected audio will be repeated. The default setting is १ repeat (that is you will than have two copies of the selected audio). निवडलेल्या ध्वनीची पुनरावृत्ती किती वेळा होईल हे निर्दिष्ट करते. पूर्वनिर्धारित समायोजन १ पुनरावृत्ती आहे (म्हणजे आपल्याकडे निवडलेल्या ध्वनीच्या दोन प्रती असतील).
८ ८ Current selection length सध्याची निवड लांबी
९ ९ Shows what the resultant selection length of the currently selected aufio is prior to the repeat. पुनरावृत्ती होण्यापूर्वी सध्या निवडलेल्या ऑफिओची परिणामी निवड लांबी किती आहे हे दर्शविते.
१० १० New selection length नवीन निवड लांबी
११ ११ Shows what the resultant selection length will be, depending on the specified number of repeats. The resultant selection includes both the originally selected audio and the inserted audio that has been repeated. पुनरावृत्तीच्या निर्दिष्ट संख्येनुसार, परिणामी निवडीची लांबी किती असेल हे दर्शविते. परिणामी निवडीमध्ये मूळ निवडलेला ध्वनी आणि घातलेला ध्वनी समाविष्ट आहे जो पुनरावृत्ती झाला आहे.
१२ १२ If the selection to be repeated borders the right edge of an audio clip, each repeat becomes a separate clip bordered by black vertical split lines. जर निवड पुन्हा पुन्हा करायची असेल तर ध्वनी क्लिपच्या उजव्या काठावर किनारी असल्यास, प्रत्येक पुनरावृत्ती काळ्या उभ्या विभाजित रेषांनी सीमित एक स्वतंत्र क्लिप बनते.
१३ १३ Any audio to right of the inserted repeats is moved to right, and so not overwritten. However the repeat will not be allowed if it would move other clips and "Editing a clip can move other clips" in Tracks Preferences is unchecked. घातलेल्या पुनरावृत्तीच्या उजवीकडील कोणताही ध्वनी उजवीकडे हलविला जातो आणि त्यामुळे अधिलेखित केला जात नाही. तथापि पुनरावृत्ती करण्यास अनुमती दिली जाणार नाही जर ते इतर क्लिप हलवित असेल आणि "क्लिप संपादित करणे इतर क्लिप हलवू शकते" गीतपट्टा पसंतींमध्ये अनचेक केले गेले आहे.
१४ १४ Repeat does not copy the original or resultant selection to the Audacity clipboard. पुनरावृत्ती मूळ किंवा परिणामी निवडीची ओड्यासिटी क्लिपबोर्डवर नक्कल करत नाही.

१६ १६ Clicking on the command buttons give the following results: आज्ञा बटणावर क्लिक केल्याने खालील परिणाम मिळतात:
१७ १७ Manage gives a dropdown menu enabling you to manage presets for the tool and to see some detail about the tool. For details see Manage presets व्यवस्थापित करा ड्रॉपडाउन मेनू आपल्याला साधनाचे पुर्वनिर्धारित सेट करण्यास आणि साधनबद्दल काही तपशील पाहण्यास सक्षम करते. तपशीलांसाठी पुर्वनिर्धारित्स व्यवस्थापित करा पहा
१८ १८ Preview plays a short preview of what the audio would sound like if the effect is applied with the current settings, without making actual changes to the audio. पूर्वावलोकन ध्वनीमध्ये वास्तविक बदल न करता, प्रभाव वर्तमान समायोजनसह लागू केल्यास ध्वनीला काय वाटेल हे एक लहान पूर्वावलोकन प्ले करते.
१९ १९ The length of preview is determined by your setting in Edit > Preferences > Playback, the default setting is ६ seconds. पूर्वावलोकनाची लांबी संपादन> प्राधान्ये> प्लेबॅक मधील आपल्या समायोजनद्वारे निश्चित केली जाते, पूर्वनिर्धारित समायोजन ६ सेकंद असते.
२० २० OK applies the effect to the selected audio with the current effect settings and closes the dialog ठीक आहे सध्याच्या प्रभाव समायोजनसह निवडलेल्या ध्वनीवर प्रभाव लागू करते आणि संवाद बंद करते
२१ २१ Cancel aborts the effect and leaves the audio unchanged, closing the dialog रद्द करा प्रभाव सोडतो आणि संवाद बंद केल्यामुळे ध्वनीमध्ये कोणताही बदल होत नाही
