3९ 3९ Export MIDI... एमआयडीआय निर्यात करा ...
४0 ४0 Export the currently selected Note Track सध्या निवडलेला टीप गीतपट्टा निर्यात करा
४1 ४1 as a MIDI or Allegro file (which is like a MIDI file except that the notes information is stored as plain text). You can choose in the Import / Export Preferences whether to export Allegro files with times and durations represented in seconds (default) or in beats. Some applications that can play MIDI files may not be able to play Allegro files. एमआयडीआय किंवा अ‍ॅलेग्रो धारिका म्हणून (जी नोट्स माहिती साध्या मजकूर म्हणून संग्रहित केलेली आहे त्याशिवाय एक एमआयडीआय धारिका प्रमाणे आहे). सेकंदात (पूर्वनिर्धारित) किंवा बीट्समध्ये प्रतिनिधित्व केलेल्या वेळा आणि कालावधीसह अल्लेरो धारिका निर्यात कराव्यात की नाही ते आपण आयात / निर्यात प्राधान्यांमध्ये निवडू शकता. एमआयडीआय धारिका खेळू शकणारे काही अनुप्रयोग अल्लेरो धारिका प्ले करण्यास सक्षम नसतील.
४2 ४2 Only one Note Track can be exported at a time. The entire Note Track will be exported regardless of any selection made in the track. If the Note Track has been time shifted to start at a point other than zero, empty MIDI bars will be added at the start. एका वेळी केवळ एक टीप गीतपट्टा निर्यात केला जाऊ शकतो. गीतपट्टामध्ये कोणत्याही निवडीची पर्वा न करता संपूर्ण टीप गीतपट्टा निर्यात केला जाईल. टीप गीतपट्टा शून्य व्यतिरिक्त इतर बिंदूवर प्रारंभ करण्यासाठी वेळ बदलल्यास, रिक्त एमआयडीआय बार सुरूवातीस जोडले जातील.
४3 ४3 To export an Allegro file the .gro file extension must be manually added to the end of the exported file name, except on Windows. एखादी अ‍ॅलेग्रो धारिका निर्यात करण्यासाठी विंडोज वगळता .gro धारिका एक्सटेंशन निर्यात केलेल्या धारिकाच्या शेवटी स्वहस्ते जोडले जाणे आवश्यक आहे.
