४ ४ The Ogg Vorbis export option uses the open-source compressed, lossy Vorbis codec inside an Ogg format container. Ogg format can be played in Windows Media Player and iTunes, but requires additional codecs to be downloaded. Ogg is a popular format for streaming applications. Audacity does not yet support use of the Speex low bit rate voice codec within Ogg. ओग व्हॉर्बिस एक्सपोर्ट पर्याय ओईजी फॉरमॅट कंटेनरमध्ये ओपन-सोर्स कॉम्प्रेस, लॉस व्हॉर्बिस कोडेक वापरतो. विंडोज मीडिया प्लेयर आणि आयट्यून्समध्ये ओग स्वरूपन प्ले केले जाऊ शकते, परंतु डाउनलोड करण्यासाठी अतिरिक्त कोडेक्स आवश्यक आहेत. प्रवाह अनुप्रयोगांसाठी ओग एक लोकप्रिय स्वरूप आहे. ओड्यासिटी अद्याप ओगमध्ये स्पीक्स कमी बिट दर व्हॉईस कोडेकच्या वापरास समर्थन देत नाही.
