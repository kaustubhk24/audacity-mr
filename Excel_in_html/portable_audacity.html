४ ४ This page describes how you can take Audacity with you (for example on a USB stick) and use it on other computers whilst keeping all your personal Audacity settings. हे पृष्ठ वर्णन करते की आपण आपल्यासह ओड्यासिटी कशी घेऊ शकता (उदाहरणार्थ यूएसबी स्टिकवर) आणि आपली सर्व वैयक्तिक ओड्यासिटी समायोजन ठेवत असताना अन्य संगणकांवर याचा वापर करा.
५ ५ Portable Settings in Audacity धृष्टता मध्ये पोर्टेबल समायोजन
६ ६ Audacity provides a way for you to keep your Preference settings with the application, rather than in a global place for user settings. It works as follows: ओड्यासिटी आपल्याला आपल्या पसंतीच्या समायोजन अनुप्रयोगासह ठेवण्याचा मार्ग प्रदान करते, त्याऐवजी वापरकर्त्याच्या समायोजनकरिता वैश्विक ठिकाणी न ठेवता. हे खालीलप्रमाणे कार्य करते:
७ ७ On launch, Audacity checks if there is a "Portable Settings" folder in the same directory as the executable (audacity.exe, Audacity.app or audacity). If "Portable Settings" exists, Audacity stores its settings there rather than in the user's home folder. Thus, a user can create a portable Audacity as follows: लाँच करताना, ओड्यासिटी कार्यवाही करण्यायोग्य (ओड्यासिटी.एक्सएसी, ओड्यासिटी.अॅप किंवा ओड्यासिटी) सारख्या त्याच निर्देशिकेत "पोर्टेबल समायोजन" फोल्डर असल्यास ते तपासते. जर "पोर्टेबल समायोजन" अस्तित्वात असतील तर ओड्यासिटी वापरकर्त्याच्या मुख्यपृष्ठाऐवजी तिची समायोजन तेथे संचयित करते. अशा प्रकारे, एक पोर्टेबल ऑडीसिटी खालीलप्रमाणे वापरकर्ता तयार करू शकतो:
८ ८ Extract the Audacity .zip or .dmg download to a USB stick or similar ओड्यासिटी .झिप किंवा .dmg डाउनलोड यूएसबी स्टिक किंवा तत्सम डाउनलोड करा
९ ९ Create a folder called "Portable Settings" on the USB stick. यूएसबी स्टिकवर "पोर्टेबल समायोजन" नावाचे फोल्डर तयार करा.

१४ १४ The settings are stored in an editable plain text file called audacity.cfg which contains your Preferences and other settings such as settings for some effects and generators. समायोजन ऑडीसिटी.सीएफजी नावाच्या संपादनयोग्य साध्या मजकूर धारिकामध्ये संग्रहित केल्या आहेत ज्यात आपले प्राधान्ये आणि काही प्रभाव आणि जनरेटरसाठी समायोजन यासारख्या इतर समायोजन आहेत.
१५ १५ Audacity Portable: ओड्यासिटी पोर्टेबल:
१६ १६ An independent third-party has created a portable version of Audacity called Audacity Portable. स्वतंत्र तृतीय-पक्षाने ओड्यासिटीची पोर्टेबल आवृत्ती तयार केली आहे ज्याला ओड्यासिटी पोर्टेबल म्हणतात.
१७ १७ Audacity Portable is not maintained by Audacity Team and we can therefore not give technical support for it. For assistance, please contact Audacity Portable Support. ओड्यासिटी पोर्टेबल ओड्यासिटी टीमद्वारे देखभाल केली जात नाही आणि म्हणून आम्ही त्यासाठी तांत्रिक आधार देऊ शकत नाही. सहाय्यासाठी, कृपया ओड्यासिटी पोर्टेबल समर्थनाशी संपर्क साधा.
१८ १८ Once the application is installed to the portable medium such as a Flash Drive, simply launch the AudacityPortable.exe file. When you're done, exit Audacity and then wait for the activity light on your removable drive to stop flashing. Then select the "Safely remove device" option from the icon in the System Tray. If you remove the drive while it is writing, you may lose data. With Audacity Portable, Audacity's temporary directory is placed on the local computer within the %TEMP% directory. If there are no problems, this directory is removed when you exit Audacity Portable, leaving no unwanted traces of data on the computer that was used. एकदा अनुप्रयोग फ्लॅश ड्राइव्हसारख्या पोर्टेबल माध्यमावर स्थापित झाल्यानंतर, केवळ ओड्यासिटीपोर्टेबल.एक्सई धारिका लाँच करा. आपण पूर्ण केल्यावर ओड्यासिटीच्या बाहेर पडा आणि नंतर फ्लॅशिंग थांबविण्यासाठी आपल्या काढण्यायोग्य ड्राइव्हवरील क्रियाकलाप प्रकाशाची प्रतीक्षा करा. त्यानंतर सिस्टम ट्रेमधील चिन्हावरून "सुरक्षितपणे उपकरण काढा" पर्याय निवडा. आपण हे लिहित असताना ड्राइव्ह काढून टाकल्यास आपण कदाचित डेटा गमावू शकता. ऑडीसिटी पोर्टेबलसह, ऑडिटीची तात्पुरती निर्देशिका स्थानिक संगणकावर% TEMP% निर्देशिकेत ठेवली जाते. काही अडचण नसल्यास, आपण वापरात असलेल्या संगणकावर डेटाचे अवांछित ट्रेस सोडत ओड्यासिटी पोर्टेबल बाहेर पडता तेव्हा ही निर्देशिका काढून टाकली जाते.
१९ १९ Audacity Portable is not a U३ compliant application. It can be installed on a U३ SmartDrive and launched from its location on the drive, but cannot be launched or shut down with the U३ Launchpad software. ओड्यासिटी पोर्टेबल एक यू ३ अनुरूप अनुप्रयोग नाही. हे यू ३ स्मार्ट ड्राईव्हवर स्थापित केले जाऊ शकते आणि ड्राइव्हवरील त्याच्या स्थानापासून लाँच केले जाऊ शकते, परंतु यू ३ लाँचपॅड सॉफ्टवेअरसह लॉन्च किंवा बंद करणे शक्य नाही.
