1८ 1८ Effect Options प्रभाव पर्याय
1९ 1९ These options let you sort or group effects that appear in the Generate, Effect or Analyze menus by effect name, type of effect or publisher. The "Grouped" options place effects into submenus within each group, so substantially shorten the menu. All the options place built-in effects above plug-in effects. हे पर्याय आपणास प्रभाव, नाव आणि प्रभावाच्या प्रकारानुसार मेनू व्युत्पन्न, प्रभाव किंवा विश्लेषण मध्ये दिसू शकणार्‍या प्रभावांचे क्रमवारी लावू शकतात किंवा गटबद्ध करू देतील. "गटबद्ध" पर्याय प्रत्येक गटात सबमेनसमध्ये प्रभाव ठेवतात, म्हणून मेनू लहान करा. सर्व पर्याय प्लग-इन प्रभावांच्या वरचे अंगभूत प्रभाव ठेवतात.
20 20 Effects in menus are: मेनूमधील प्रभावः
21 21 Sorted by Effect Name: Effects are listed in one long menu sorted by effect name, built-in effects above the divider then plug-ins below the divider. प्रभाव नावानुसार क्रमवारी लावली: प्रभावाच्या नावाने क्रमवारी लावलेल्या एका लांब मेनूमध्ये प्रभाव सूचीबद्ध केला जातो, डिव्हिडरच्या वरचे अंगभूत प्रभाव नंतर विभाजकच्या खाली प्लग-इन.
22 22 Sorted by Publisher and Effect Name: Effects are listed in one long menu, built-in effects sorted by name above the divider then plug-ins sorted by publisher then name below the divider. प्रकाशक आणि प्रभाव नावानुसार क्रमवारी लावली: प्रभाग एका लांब मेनूमध्ये सूचीबद्ध केले जातात, अंतर्निहित प्रभावाच्या नावाखाली क्रमवारी लावलेले असतात नंतर प्लग-इन प्रकाशकाद्वारे क्रमवारी लावलेले असतात आणि नंतर भागाच्या खाली नाव दिले जाते.
23 23 Sorted by Type and Effect Name: Effects are listed in one long menu, built-in effects sorted by name above the divider then plug-ins sorted by type then name below the divider. प्रकार आणि प्रभाव नावानुसार क्रमवारी लावली: प्रभाव एका लांब मेनूमध्ये सूचीबद्ध केलेला असतो, ते अंतर्निहित प्रभावाच्या नावाखाली क्रमवारी लावतात नंतर प्लग-इन क्रमवारी लावून क्रमवार नंतर भागाच्या खाली नाव.
24 24 Grouped by Publisher: Effects are grouped by effect publisher, with effects in each group in a submenu. The Audacity built-in effects group is above the divider and groups for plug-ins are below the divider. प्रकाशकाद्वारे गटबद्धः प्रभाव सबमेनूमधील प्रत्येक गटाच्या प्रभावांसह प्रभाव प्रकाशकाद्वारे गटबद्ध केला जातो. ओड्यासिटी बिल्ट-इन परिणामग्रुप डिव्हिडरच्या वर आहे आणि प्लग-इनसाठी गट डिव्हिडरच्या खाली आहेत.
2५ 2५ Grouped by Type: प्रकारानुसार गटबद्धः
2६ 2६ Effects are grouped by effect type, with effects in each group in a submenu. The Audacity built-in effects group, and the Nyquist Prompt, are above the divider and groups for plug-ins are below the divider. सबमेनूमधील प्रत्येक गटातील प्रभावांसह प्रभावांना प्रभाव प्रकारानुसार वर्गीकृत केले जाते. ओड्यासिटी बिल्ट-इन परिणामग्रुप आणि न्यक्विस्ट प्रॉम्प्ट डिव्हिडरच्या वर आहेत आणि प्लग-इनसाठी गट डिव्हिडरच्या खाली आहेत.
2७ 2७ Maximum effects per group (0 to disable): If enabled by setting a non-zero value, this controls for the "Sorted" options the maximum number of plug-ins allowed in the main menu before they are grouped into submenus. For the "Grouped" options this controls the maximum number of plug-ins allowed in a submenu before being grouped into a secondary submenu. Each Group is named according to the number of plug-ins it contains in numerical order. For example, if there were 10 plug-ins and the value was set to "५", there would be two groups "Plug-ins 1 to ५" and "Plug-ins ६ to 10". प्रति गटाचे जास्तीत जास्त प्रभाव (० अक्षम करण्यासाठी): शून्य नसलेले मूल्य सेट करून सक्षम केल्यास, मुख्य मेनूमध्ये सबमेनसमध्ये गटबद्ध होण्यापूर्वी जास्तीत जास्त प्लग-इनची अनुमती असलेल्या "क्रमवारी लावलेले" पर्यायासाठी हे नियंत्रित करते. "गटबद्ध" पर्यायांसाठी दुय्यम उपमेनूमध्ये गटबद्ध होण्यापूर्वी सबमेनूमध्ये अनुमत प्लगइनची जास्तीत जास्त संख्या नियंत्रित करते. प्रत्येक गटाचे संख्यात्मक क्रमाने असलेल्या प्लगइनच्या संख्येनुसार नाव दिले जाते. उदाहरणार्थ, जर तेथे 10 प्लगइन असतील आणि मूल्य "५" वर सेट केले असेल तर "प्लग-इन 1 ते ५" आणि "प्लग-इन ६ ते 10" असे दोन गट असतील.
