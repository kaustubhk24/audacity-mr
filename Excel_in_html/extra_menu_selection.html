२0 २0 Set (or Extend) Left Selection डावे निवड सेट करा (किंवा वाढवा)
२1 २1 When playing or recording, sets the left boundary of a potential selection by moving the cursor to the current position of the green playback cursor (or red recording cursor). The selection can then be drawn by using "Set (or Extend) Right Selection"(below). प्ले करताना किंवा ध्वनीमुद्रण करताना, कर्सरला ग्रीन प्लेबॅक कर्सरच्या (किंवा लाल ध्वनीमुद्रणकर्सर) सद्य स्थितीत हलवून संभाव्य निवडीची डावे सीमा सेट करते. त्यानंतर निवड "सेट (किंवा वाढवा) उजवी निवड" (खाली) वापरून रेखाटता येईल.
२२ २२ When stopped, increases the size of the selection by extending it to the left. The amount of increase is dependent on the zoom level. If there is no selection one is created starting at the cursor position. थांबविल्यावर निवडीचा आकार डावीकडे वाढवून वाढवितो. वाढ होण्याचे प्रमाण झूम स्तरावर अवलंबून असते. जर निवड नसेल तर कर्सर स्थानापासून प्रारंभ करुन एक तयार केली जाईल.
२3 २3 Set (or Extend) Right Selection योग्य निवड सेट करा (किंवा वाढवा)
२४ २४ When playing or recording, sets the right boundary of the selection, thus drawing the selection from the cursor position to the current position of the green playback cursor (or red recording cursor). This is a quick way to mark a selection for future editing while playing or recording. Selections can be labeled while playing or recording using Edit > Labels > Add Label at Selection or shortcut Ctrl + B. प्ले करताना किंवा ध्वनीमुद्रण करताना, निवडीची योग्य सीमा निश्चित करते, अशा प्रकारे निवड कर्सर स्थानावरून ग्रीन प्लेबॅक कर्सरच्या (किंवा रेड ध्वनीमुद्रणकर्सर) सद्य स्थितीवर आणते. खेळताना किंवा ध्वनीमुद्रणिंग करताना भविष्यातील संपादनासाठी निवड चिन्हांकित करण्याचा हा एक द्रुत मार्ग आहे. संपादन> नावपट्टीे> निवड येथे नावपट्टी जोडा किंवा शॉर्टकट Ctrl + B वापरुन निवडताना प्ले किंवा ध्वनीमुद्रणकरताना नावपट्टी दिले जाऊ शकतात.
२५ २५ When stopped, increases the size of the selection by extending it to the right. The amount of increase is dependent on the zoom level. If there is no selection one is created starting at the cursor position. थांबविल्यावर निवडीचा आकार उजवीकडे वाढवून वाढवते. वाढ होण्याचे प्रमाण झूम स्तरावर अवलंबून असते. जर निवड नसेल तर कर्सर स्थानापासून प्रारंभ करुन एक तयार केली जाईल.
२६ २६ Selection Contract Left  Ctrl + Shift + Right निवड करार डावा Ctrl + Shift + उजवीकडे
२७ २७ Decreases the size of the selection by contracting it from the right. The amount of decrease is dependent on the zoom level. If there is no selection no action is taken. उजवीकडून करार करून निवडीचा आकार कमी करते. घट होण्याचे प्रमाण झूम स्तरावर अवलंबून असते. निवड नसल्यास कोणतीही कारवाई केली जात नाही.

If there is no selection no action is taken. डावीकडून करार करून निवडीचा आकार कमी करते. घट होण्याचे प्रमाण झूम स्तरावर अवलंबून असते. निवड नसल्यास कोणतीही कारवाई केली जात नाही.
