२३ २३ Installing Nyquist plug-ins Nyquist प्लगइन स्थापित करीत आहे
२४ २४ Nyquist plug-ins provide most of the optional effects underneath the divider in the Effect menu. They are also used to provide some of Audacity's built-in audio generators and analysis tools. A wide range of additional Nyquist effect, generation and analysis plug-ins can be obtained from Download Nyquist Plug-ins on our Wiki. Nyquist प्लगइन प्रभाव मेनूमधील विभाजक खाली बरेच पर्यायी प्रभाव प्रदान करतात. ते ओड्यासिटीचे अंगभूत काही ध्वनी जनरेटर आणि विश्लेषण साधने प्रदान करण्यासाठी देखील वापरले जातात. आमच्या विकीवरील डाउनलोड न्युक्विस्ट प्लग-इन्समधून विस्तृत अतिरिक्त न्यक्युइस्ट इफेक्ट, जनरेशन आणि अ‍ॅनालिसिस प्लग-इन्स ची विस्तृत श्रृंखला मिळू शकते.
२५ २५ To install new Nyquist plug-ins, place them in Audacity's "Plug-Ins" folder at ~/Library/Application Support/audacity/Plug-Ins. नवीन Nyquist प्लगइन स्थापित करण्यासाठी, त्यांना ओड्यासिटीच्या "प्लग-इन" फोल्डरमध्ये ~ / लायब्ररी / अनुप्रयोग समर्थन / ओड्यासिटी / प्लग-इनमध्ये ठेवा.
२६ २६ To load the new effects into Audacity so they are available in the menu, use the Plug-in Manager: Effects, Generators and Analyzers dialog. ओड्यासिटीमध्ये नवीन प्रभाव लोड करण्यासाठी जेणेकरून ते मेनूमध्ये उपलब्ध असतील, प्लग-इन व्यवस्थापक: प्रभाव, जनरेटर आणि विश्लेषक संवाद वापरा.
२७ २७ Some Nyquist plug-ins could crash while processing very long audio selections (typically an hour or more). This is due to the plug-in using a large amount of memory and is a known issue in Audacity's current Nyquist implementation. Try using the plug-in on shorter selections instead. बर्‍याच लांब ध्वनी निवडी (विशेषत: एक तास किंवा त्याहून अधिक) प्रक्रिया करताना काही Nyquist प्लगइन क्रॅश होऊ शकतात. हे मोठ्या प्रमाणात मेमरी वापरुन प्लग-इन केल्यामुळे आहे आणि ओड्यासिटीच्या सध्याच्या Nyquist अंमलबजावणीची ज्ञात समस्या आहे. त्याऐवजी लहान निवडींवरील प्लग-इन वापरुन पहा.
२८ २८ Installing VST plug-ins व्हीएसटी प्लग-इन्स स्थापित करीत आहे
२९ २९ Audacity supports almost all VST effect plug-ins on Mac including "shell" VST's that host multiple VST effects. ओड्यासिटी मॅकवरील "शेल" व्हीएसटी च्या बहुविध व्हीएसटी प्रभावांना होस्ट केलेल्या जवळजवळ सर्व व्हीएसटी प्रभाव प्लगइनना समर्थन देते.
३० ३० To install new VST plug-ins, place them in Audacity's "Plug-Ins" folder at ~/Library/Application Support/audacity/Plug-Ins. नवीन व्हीएसटी प्लग-इन स्थापित करण्यासाठी, त्यांना ओड्यासिटीच्या "प्लग-इन" फोल्डरमध्ये ~ / लायब्ररी / अनुप्रयोग समर्थन / ओड्यासिटी / प्लग-इनमध्ये ठेवा.

३५ ३५ All the directories where you can place VST plug-ins are searched recursively, which means that plug-ins that are inside their own folder in that directory should still be found by Audacity. आपण व्हीएसटी प्लग-इन ठेवू शकता अशा सर्व डिरेक्टरीज पुनरावृत्ती शोधल्या जातात, ज्याचा अर्थ असा की त्या निर्देशिकेत त्यांच्या स्वत: च्या फोल्डरमध्ये असलेले प्लग-इन अद्याप ओड्यासिटी द्वारे शोधले जावेत.
३६ ३६ You can install new VST effects into Audacity by using the Effect > Add / Remove Plug-ins... menu item. This opens the Plug-in Manager: Effects, Generators and Analyzers dialog where you can select and enable the new effects then click OK to load them. Next time you launch Audacity the enabled effect(s) will be cached and you will not need to re-enable them. आपण प्रभाव> जोडा / काढा प्लगइन ... मेनू आयटमचा वापर करून ओड्यासिटीमध्ये नवीन व्हीएसटी प्रभाव स्थापित करू शकता. हे प्लग-इन व्यवस्थापक उघडते: प्रभाव, जनरेटर आणि विश्लेषक संवाद जेथे आपण नवीन प्रभाव निवडू आणि सक्षम करू शकता नंतर त्यांना लोड करण्यासाठी ओके क्लिक करा. पुढच्या वेळी आपण ओड्यासिटी लॉन्च करा सक्षम प्रभाव कॅश केला जाईल आणि आपल्याला त्यांना पुन्हा सक्षम करण्याची आवश्यकता नाही.
३७ ३७ The list of effects in the dialog will include any detected VST instrument (VSTi) plug-ins (such as synths) and any real-time VST effects that are capable of changing audio data while it's being written to disk. Neither of these are yet supported and they will not load even if you enable or re-enable them in the dialog. संवादातील प्रभावांच्या सूचीमध्ये कोणतीही आढळलेली व्हीएसटी इन्स्ट्रुमेंट (व्हीएसटीआय) प्लग-इन्स (जसे की सिंथ्स) आणि डिस्कवर लिहिताना ध्वनी डेटा बदलण्यात सक्षम असलेले कोणतेही रिअल-वेळ व्हीएसटी प्रभाव समाविष्ट असतील. यापैकी अद्याप समर्थित नाहीत आणि आपण संवादात त्यांना सक्षम किंवा पुन्हा-सक्षम केले तरीही ते लोड होणार नाहीत.

५५ ५५ Installing LV२ plug-ins एलव्ही २ प्लग-इन स्थापित करीत आहे

६७ ६७ Installing Vamp plug-ins व्हँप प्लग-इन स्थापित करीत आहे
६८ ६८ Vamp plug-ins are usually for analyzing audio so will appear under Audacity's Analyze Menu. You can do interesting things like attempt to track beats, note pitches, chords or frequencies. Any Vamp plug-ins whose output is suitable for a label track should work in Audacity on Mac. To add a new Vamp analysis tool, add the plug-in's DLL, DYLIB or SO file and any supplied category or RDF files to any of the Vamp search locations then enable the tool in the Plug-in Manager: Effects, Generators and Analyzers dialog. व्हँप प्लग-इन सहसा ध्वनी विश्लेषणासाठी असतात जेणेकरून ओड्यासिटीच्या विश्लेषण मेनू अंतर्गत दिसतील. आपण बीट्सचा मागोवा घेण्याचा प्रयत्न करणे, पिच नोट्स, जीवा किंवा वारंवारितांचा मागोवा घेणे यासारख्या मनोरंजक गोष्टी करू शकता. कोणतेही व्हँप प्लगइन ज्यांचे आउटपुट लेबल गीतपट्टासाठी योग्य असेल त्यांनी मॅकवरील ओड्यासिटीमध्ये कार्य केले पाहिजे. नवीन व्हँप एनालिसिस साधन जोडण्यासाठी, प्लग-इनची डीएलएल, डीवायआयएलबी किंवा एसओ धारिका आणि कोणत्याही पुरविलेल्या श्रेणी किंवा आरडीएफ धारिका व्हॅम्प शोध स्थानांवर जोडा त्यानंतर प्लग-इन व्यवस्थापकामधील साधन सक्षम करा: प्रभाव, जनरेटर आणि विश्लेषक संवाद .
