4 4 Use Delay to create a multiple echo effect with options for variable delay times between successive echoes and the number, amplitude and pitch of successive repeats. क्रमिक प्रतिध्वनी आणि क्रमिक पुनरावृत्तीची संख्या, मोठेपणा आणि खेळपट्टी दरम्यान चल विलंब वेळाकरिता पर्यायांसह एकाधिक प्रतिध्वनी प्रभाव तयार करण्यासाठी विलंब वापरा.
५ ५ Accessed by: Effect > Delay... याद्वारे प्रवेश केलेले: प्रभाव> विलंब ...
६ ६ Delay type विलंब प्रकार
७ ७ Regular: Each echo is delayed by the same amount. नियमितः प्रत्येक प्रतिध्वनी समान प्रमाणात विलंबित होतो.
८ ८ Bouncing ball: Makes the echoes occur increasingly close together (faster). उसळणारा चेंडू: प्रतिध्वनी एकत्र (वेगवान) जवळ येऊ लागतात.
९ ९ Reverse bouncing ball: Makes the echoes occur increasingly far apart (slower). उलट उसळणारा चेंडू : प्रतिध्वनी वाढत्या वेगळ्या (हळूवारपणे) होण्यास मदत करते.
१0 १0 Delay level per echo (dB) प्रतिध्वनी विलंब पातळी (डीबी)
११ ११ This controls the amount in dB by which each successive echo will change loudness. The more negative this value, the more the echoes will die away. Positive values cause successive echoes to be louder. When set to 0.0, all echoes will be the same volume. हे डीबीमधील प्रमाणात नियंत्रित करते ज्याद्वारे प्रत्येक अनुक्रमित प्रतिध्वनी मोठ्याने बदलते. हे मूल्य जितके नकारात्मक असेल तितके प्रतिध्वनी नष्ट होतील. सकारात्मक मूल्यांमुळे सलग प्रतिध्वनी जोरात होते. 0.0 वर सेट केल्यावर, सर्व प्रतिध्वनी एकसारखेच असतील.
१2 १2 Because echoes are often added to the original audio, the processed audio will often have a higher peak level than the original. If the output exceeds 0 dB (full track height) and so triggers the red clipping warning lights in Playback Meter Toolbar, you should open Effect > Amplify..., ensure "Amplification (dB)" shows a negative dB value (not "0.0 dB") then click OK. प्रतिध्वनी मूळ ध्वनीमध्ये अनेकदा जोडल्या गेल्यामुळे, प्रक्रिया केलेल्या ध्वनीमध्ये बर्‍याचदा मूळपेक्षा उच्च पातळी असते. जर आउटपुट 0 डीबी (पूर्ण गीतपट्टा उंची) पेक्षा जास्त असेल आणि म्हणून प्लेबॅक मीटर साधनपट्टीमध्ये रेड क्लिपिंग चेतावणी दिवे चालू केले तर आपण प्रभाव> एम्प्लिफाय ... उघडला पाहिजे, "एम्प्लिफिकेशन (डीबी)" नकारात्मक डीबी मूल्य दर्शविते (0.0 "नाही" डीबी ") नंतर ओके क्लिक करा.
१3 १3 If "Amplification (dB)" in Amplify shows "0.0 dB", the track does not contain 32-bit audio and the clipping cannot be repaired. In that case, Edit > Undo Delay, use Amplify to reduce the track to a lower amplitude then reapply Delay. जर एम्प्लीफाई मधील "एम्प्लिफिकेशन (डीबी)" "0.0 डीबी" दर्शवित असेल तर गीतपट्टामध्ये 32-बिट ध्वनी नसतो आणि क्लिपिंग दुरुस्त करता येणार नाही. त्या प्रकरणात, संपादन> विलंब पूर्ववत करा, गीतपट्टाला कमी आकारात कमी करण्यासाठी एम्प्लिफाय वापरा नंतर विलंब पुन्हा लागू करा.
१4 १4 Delay time (seconds) विलंब वेळ (सेकंद)
१५ १५ This controls the time between echoes. For the Bouncing Ball types of delay, this sets the delay time for the longest (slowest) echo. हे प्रतिध्वनी दरम्यानचा वेळ नियंत्रित करते. उसळणारा चेंडू प्रकारच्या विलंबांसाठी, हे सर्वात लांब (सर्वात धीमे) प्रतिध्वनीसाठी विलंब वेळ निश्चित करते.
१६ १६ Pitch change effect खेळपट्टीचा प्रभाव बदल
१७ १७ Pitch/Tempo: Changes the pitch in the same way that the Change Speed effect causes a change in pitch. The sound quality of each echo is very good, but an increase in pitch speeds up the delayed audio and a decrease in pitch slows it down. The echoes will be noticeably desynchronized with any comparable echoes in other tracks where "Pitch/Tempo" was not applied. खेळपट्टी / टेम्पो: बदलाच्या वेग परिणामामुळे खेळपट्टीवर बदल होऊ शकतो त्याच प्रकारे खेळपट्टी बदलते. प्रत्येक प्रतिध्वनीची ध्वनी गुणवत्ता खूप चांगली आहे, परंतु खेळपट्टीमध्ये वाढ झाल्याने उशीरा ध्वनी वेगवान होईल आणि खेळपट्टीमध्ये घट कमी होते. "खेळपट्टी / टेम्पो" लागू न झालेल्या इतर गीतपट्टामध्ये कोणत्याही प्रतिध्वनींसह प्रतिध्वनी स्पष्टपणे विभक्त केल्या जातील.
१८ १८ LQ Pitch Shift: Changes the pitch while maintaining the same tempo in a similar way to the Change Pitch effect. The sound quality of each echo is of lower quality than the "Pitch/Tempo" effect (including a short additional echo on percussive sounds) and the delay time will often be a little longer than specified. However, interesting "spiraling" effects can be produced. एलक्यू खेळपट्टीवर शिफ्ट : चेंज पिच इफेक्ट प्रमाणेच टेम्पो कायम ठेवत खेळपट्टी बदलते. प्रत्येक प्रतिध्वनीची ध्वनी गुणवत्ता "खेळपट्टी / टेंपो" प्रभावापेक्षा (दर्जेदार ध्वनीवरील अतिरिक्त अतिरिक्त प्रतिध्वनीसह) कमी गुणवत्तेची असते आणि विलंब वेळ बर्‍याचदा निर्दिष्ट केल्यापेक्षा थोडा जास्त असेल. तथापि, मनोरंजक "आवर्त" प्रभाव तयार केला जाऊ शकतो.
१९ १९ Pitch change per echo (semitones) प्रति प्रतिध्वनी खेळपट्टीवर बदल (सेमीटोन)
20 20 This controls the amount in semitones by which each successive echo will change pitch. The change can be set to fractions of a semitone which can produce a "glissando" type of delay. When set to zero, the pitch change effect is disabled. हे सेमिनेन्समधील रक्कम नियंत्रित करते ज्याद्वारे प्रत्येक अनुक्रमित प्रतिध्वनी बदलते. बदल सेमिटोनच्या अपूर्णांकांवर सेट केला जाऊ शकतो ज्यामुळे "ग्लिसॅन्डो" प्रकारचा विलंब होऊ शकतो. शून्यावर सेट केल्यास, पिच बदल प्रभाव अक्षम होईल.
2१ 2१ Number of echoes प्रतिध्वनींची संख्या
22 22 This controls how many echoes are produced. किती प्रतिध्वनी निर्माण होते हे हे नियंत्रित करते.
23 23 Allow duration to change कालावधी बदलू द्या
24 24 Yes: (Default). Allows the processed audio to lengthen so as to accommodate all the requested echoes. होय: (पूर्वनिर्धारित) सर्व विनंती केलेल्या प्रतिध्वनींना सामावून घेण्यासाठी प्रक्रियेच्या ध्वनीला वाढण्यास अनुमती देते.
2५ 2५ No: The processed audio is trimmed to the same length as the original selection. Any echoes that would have occurred after the end of the original selection will be deleted. Choose this option if you need to keep the delayed track synchronized with other tracks in the project. नाहीः प्रक्रिया केलेला ध्वनी मूळ निवडीच्या समान लांबीवर सुव्यवस्थित आहे. मूळ निवड संपल्यानंतर उद्भवलेली कोणतीही प्रतिध्वनी हटविली जाईल. आपल्याला प्रकल्पातील इतर गीतपट्टासह विलंबित गीतपट्टा संकालित करणे आवश्यक असल्यास हा पर्याय निवडा.
