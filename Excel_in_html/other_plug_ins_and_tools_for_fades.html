९ ९ Fade In and Out फेड इन अँड बाहेर
१० १० This is a Nyquist plug-in that can apply a linear fade in and fade out to the selected audio. हे एक Nyquist प्लग-इन आहे जे एक रेषीय फॅड इन लागू करू शकते आणि निवडलेल्या ध्वनीमध्ये फीडबाहेर होऊ शकते.
११ ११ Nyquist plug-ins extend the capabilities of Audacity and are supported on Windows, Mac and GNU/लिनक्स. Instructions for installing Nyquist plug-ins can be found on our Wiki page Nyquist प्लग-इन ओड्यासिटीची क्षमता वाढविते आणि Windows, Mac आणि GNU / लिनक्स वर समर्थित आहेत. आमच्या विकी पृष्ठावर Nyquist प्लगइन स्थापित करण्यासाठी सूचना आढळू शकतात
१२ १२ Download Nyquist Plug-ins. Nyquist प्लग-इन्स डाउनलोड करा.
१३ १३ This tool was written by David R Sky to provide a simple, accessible way for visually impaired and other non-mouse users to apply fade in and out effects. The current Audacity २.x has a Selection Toolbar providing a screen-reader friendly display of selection start time and duration (* except on लिनक्स) which can be used for similar purpose, but this effect may be found to be quicker and easier to use. हे साधन डेव्हिड आर स्काय यांनी दृष्टिबाधित आणि माउस नसलेल्या वापरकर्त्यांसाठी फॅड इन आणिबाहेर प्रभाव लागू करण्यासाठी सोपा, प्रवेशयोग्य मार्ग प्रदान करण्यासाठी लिहिले होते. वर्तमान ओड्यासिटी २.० मध्ये निवड साधनपट्टी आहे जो निवड प्रारंभ वेळ आणि कालावधीचे स्क्रीन-रीडर अनुकूल प्रदर्शन प्रदान करतो (* लिनक्स वगळता) जो समान उद्देशासाठी वापरला जाऊ शकतो, परंतु हा प्रभाव जलद आणि वापरण्यास सुलभ असल्याचे आढळू शकते .
१४ १४ Text Envelope मजकूर लिफाफा
१५ १५ This is a Nyquist plug-in that can apply multiple changes to the amplitude of the selected audio, fading from one level to another as required. हे एक Nyquist प्लग-इन आहे जे निवडलेल्या ध्वनीच्या विशालतेमध्ये एकाधिक बदलांसाठी आवश्यकतेनुसार एका पातळीवरून दुसर्‍या स्तरावर विलीन होऊ शकते.
१६ १६ This tool was written by Steve Daulton to provide an accessible alternative to the "Envelope Tool" for visually impaired and other users that do not use pointing devices. हे साधन स्टीव्ह डाॅल्टन यांनी दृष्टिहीन व इतर वापरकर्त्यांसाठी पॉईंटिंग उपकरण वापरत नाही अशा प्रवेशासाठी "लिफाफा साधन" चा प्रवेशयोग्य पर्याय उपलब्ध करुन देण्यासाठी लिहिले होते.

१८ १८ Nyquist is a programming language that is included in Audacity. It can be used to write plug-ins for Audacity that may provide new effects, generators and analysis plug-ins. Nyquist commands may also be run directly in Audacity by entering code into the Nyquist Prompt Effect. Below are a few short code samples that may be run in the Nyquist Prompt to produce fades to selected audio.
