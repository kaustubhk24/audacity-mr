१ १ Window Menu विंडो मेनू
२ २ From Audacity Development Manual ओड्यासिटी विकास माहितीपुस्तिकेकडून
३ ३ Jump to: navigation, search येथे जा: निर्देशक, शोध
४ ४ The Window menu appears only on macOS and Mac OS X. It contains commands for displaying and zooming project windows. विंडो मेनू केवळ मॅकोस आणि मॅक ओएस एक्स वर दिसून येतो. यात प्रकल्प विंडो प्रदर्शित आणि झूम करण्यासाठी आदेश आहेत.
५ ५ Minimize कमी करा
६ ६ Minimizes the front project window into the Dock. Same as clicking the yellow minimize button at the upper left of the window. समोर प्रकल्प विंडो डॉकमध्ये लहान करते. विंडोच्या वरच्या डाव्या बाजूला पिवळ्या मिनिमाइझ बटणावर क्लिक करण्यासारखेच.
७ ७ Zoom झूम करा
८ ८ If the front window does not fill the entire screen between the Dock and the Audacity Menu Bar, expands the window to fill that space. समोरची विंडो डॉक आणि ओड्यासिटी मेनू बार दरम्यान संपूर्ण स्क्रीन भरत नसल्यास, ती जागा भरण्यासाठी विंडो विस्तृत करते.
९ ९ If the window was previously expanded with this command, clicking the Zoom item again will return the window to its previous size and position. जर या आदेशासह विंडो पूर्वी विस्तारित केला गेला असेल तर पुन्हा झूम आयटमवर क्लिक केल्यास विंडो मागील आकार आणि स्थितीवर परत येईल.
१० १० This menu item is the same as Option-clicking the green zoom button at upper left of the project window (or unmodified-clicking the zoom button on OS X Snow Leopard). On OS X Lion and later, unmodified-clicking the green zoom button does the same as View > Enter / Exit Full Screen. हा मेनू आयटम प्रकल्प विंडोच्या वरच्या डाव्या बाजूस ग्रीन झूम बटणावर (किंवा ओएस एक्स स्नो लेपर्डवरील झूम बटणावर न बदललेल्या क्लिकवर) क्लिक करण्यासारखेच आहे. ओएस एक्स लायन वर आणि नंतर, ग्रीन झूम बटणावर विनाशब्द क्लिक केल्याने पहा> एंटर / एग्जिट पूर्ण स्क्रीन असेच होते.
११ ११ Bring All to Front सर्वांना समोर आणा
१२ १२ If any Audacity windows are minimized or covered by other windows, this command will make those windows visible. जर कुठल्याही ओड्यासिटी विंडोला कमीतकमी किंवा इतर विंडोने संरक्षित केले असेल तर ही आज्ञा त्या विंडोजा दृश्यमान करेल.
