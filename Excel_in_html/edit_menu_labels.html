७ ७ Edit Labels लेबले संपादित करा
८ ८ This command invokes the Labels Editor which lets you add or remove Label Tracks and edit their labels entirely using the keyboard, so is particularly useful for visually impaired users. ही आज्ञा लेबल एडिटरची विनंती करते जी आपल्याला कीबोर्ड वापरुन लेबल गीतपट्टा जोडू किंवा काढू देते आणि त्यांची लेबले पूर्णपणे संपादित करू देते, जेणेकरून दृष्टिहीन वापरकर्त्यांसाठी उपयुक्त आहे.
९ ९ It brings up a dialog box showing all of your labels in a keyboard-accessible tabular view. Handy buttons in the dialog let you insert or delete a label, or import and export labels to a file. 10 10 Add Label at Selection Ctrl + B निवड Ctrl + B वर लेबल जोडा
11 11 This command will place a new label at your current cursor position or selection. See this page for more details. ही आज्ञा आपल्या वर्तमान कर्सर स्थानावर किंवा निवडीस नवीन लेबल ठेवेल. अधिक माहितीसाठी हे पृष्ठ पहा.
12 12 Creates a new label at the position of the editing cursor or selection region. You can then title the label by typing with the keyboard and hitting "Enter" when you're done. When you click in the label later it will recall the position of the editing cursor or selection at which the label was created. See Label Tracks for more information on how to use label tracks. संपादन कर्सर किंवा निवड प्रदेशाच्या स्थानावर एक नवीन लेबल तयार करते. त्यानंतर आपण कीबोर्डसह टाइप करून आणि आपले कार्य पूर्ण झाल्यावर "एंटर" दाबून लेबलचे शीर्षक देऊ शकता. जेव्हा आपण नंतर लेबल क्लिक कराल तेव्हा ते संपादन कर्सर किंवा लेबल तयार केलेल्या निवडीची स्थिती आठवेल. लेबल गीतपट्टा कसे वापरावे याबद्दल अधिक माहितीसाठी लेबल गीतपट्टा पहा.
13 13 Add Label at Playback Position Ctrl + M प्लेबॅक पोझिशन Ctrl + M वर लेबल जोडा
14 14 This command will place a new label at your current playback or recording position. See this page for more details. ही आज्ञा आपल्या सद्य प्लेबॅक किंवा रेकॉर्डिंग स्थितीत एक नवीन लेबल ठेवेल. अधिक माहितीसाठी हे पृष्ठ पहा.
1५ 1५ Like Add Label at Selection but the label is added at the current position during playback or recording. निवडीवर लेबल जोडा परंतु लेबल प्लेबॅक किंवा रेकॉर्डिंग दरम्यान सद्य स्थितीत जोडले गेले आहे.
1६ 1६ On a Mac the shortcut for this command is (⌘ + .). मॅकवर या आदेशाचा शॉर्टकट (⌘ +.) आहे.
1७ 1७ Paste Text to New Label Ctrl + Alt + V नवीन लेबल Ctrl + Alt + V वर मजकूर पेस्ट करा
1८ 1८ Extra अतिरिक्त
1९ 1९ Pastes the text on the system clipboard (or text from a label stored in the Audacity clipboard) to a new label at the cursor or region position in the currently selected label track. If there is no selection in the label track a point label is created. If a range is selected in the label track a range label is created. A new label track is created if none exists. सिस्टम क्लिपबोर्डवरील मजकूर (किंवा ओड्यासिटी क्लिपबोर्डमध्ये संग्रहित लेबलवरील मजकूर) सध्या निवडलेल्या लेबल गीतपट्टामधील कर्सर किंवा प्रदेश स्थितीवरील नवीन लेबलवर पेस्ट करतो. जर लेबल गीतपट्टामध्ये निवड नसेल तर पॉईंट लेबल तयार केले जाईल. लेबल गीतपट्टामध्ये श्रेणी निवडल्यास श्रेणी लेबल तयार केले जाते. काहीही नसल्यास एक नवीन लेबल गीतपट्टा तयार केला जातो.
