४ ४ These commands help you to fit your project to the screen horizontally and vertically. हे आदेश आपल्याला आपला प्रकल्प क्षैतिज आणि अनुलंब स्क्रीनवर बसविण्यास मदत करतात.
५ ५ Click on the image to learn more अधिक जाणून घेण्यासाठी प्रतिमेवर क्लिक करा
६ ६ Fit to Width  Ctrl + F फिट ते रूंदी Ctrl + F
७ ७ Zooms out until the entire project just fits in the width of the project window. संपूर्ण प्रकल्प फक्त प्रकल्प विंडोच्या रुंदीमध्ये बसत नाही तोपर्यंत झूम कमी करते.
८ ८ Fit to Height  Ctrl + Shift + F फिट ते उंची Ctrl + Shift + F
९ ९ Adjusts the height of tracks so they fit in the vertical space of the project window if possible. All tracks that have not been collapsed using the Track Collapse Button or Collapse All Tracks menu item are resized so that each separate waveform has the same height. Accordingly mono tracks will have the same height as each other and stereo tracks will have the same height as each other (twice the height of the mono tracks). गीतपट्टाची उंची समायोजित करते जेणेकरून शक्य असेल तर ते प्रकल्प विंडोच्या उभ्या जागेत बसतील. गीतपट्टा कोलॅप्स बटण किंवा संकुचित सर्व गीतपट्टा मेनू आयटम वापरुन संकुचित न झालेल्या सर्व गीतपट्टाचे आकार बदलले जातात जेणेकरून प्रत्येक वेगळ्या वेव्हफॉर्मची उंची समान असेल. त्यानुसार मोनो गीतपट्टाची उंची एकमेकांइतकीच असेल आणि स्टीरिओ गीतपट्टाची उंची देखील एकमेकांप्रमाणे असेल (मोनो गीतपट्टाच्या दुप्पट उंची).
१० १० For example, if you have two mono tracks with considerable space underneath them you can use this command to expand each track to fill half the available vertical space. उदाहरणार्थ, आपल्या खाली दोन मोनो गीतपट्टा त्यांच्या खाली असलेल्या जागेसह, आपण अर्ध्या उपलब्ध उभ्या जागा भरण्यासाठी प्रत्येक गीतपट्टाचा विस्तार करण्यासाठी या आदेशाचा वापर करू शकता.
११ ११ If you want tracks to automatically fit the vertical space without the need to use the Fit Vertically command, enable Automatically fit tracks vertically zoomed in Tracks Preferences. आपण फिट अनुलंब आदेश न वापरता अनुलंब जागेत गीतपट्टा स्वयंचलितपणे फिट करू इच्छित असल्यास, गीतपट्टा्स प्राधान्यांमधील अनुलंब झूम केलेले स्वयंचलितपणे फिट गीतपट्टा सक्षम करा.
१२ १२ Collapse All Tracks  Ctrl + Shift सर्व गीतपट्टा संकुचित करा Ctrl + Shift
१३ १३ + C + सी
१४ १४ Collapses all tracks to take up the minimum amount of space. जागेची किमान रक्कम घेण्यासाठी सर्व गीतपट्टा संकुचित करते.
१५ १५ To collapse or expand an individual track, click on the Track Collapse Button with triangle icon at the bottom of the track. स्वतंत्र गीतपट्टा कोसळण्यासाठी किंवा विस्तृत करण्यासाठी गीतपट्टाच्या तळाशी असलेल्या त्रिकोण चिन्हासह गीतपट्टा कोलप्स बटणावर क्लिक करा.
१६ १६ Expand Collapsed Tracks  Ctrl + Shift + X संकुचित गीतपट्टा Ctrl + Shift + X विस्तृत करा
१७ १७ Expands all tracks that were collapsed using the Collapse button or Collapse All Tracks menu item to the height the tracks had before their last collapse. संकुचित बटण वापरून संकुचित केलेले सर्व गीतपट्टा विस्तृत करा किंवा गीतपट्टाच्या शेवटच्या कोसळण्यापूर्वी गीतपट्टाच्या सर्व गीतपट्टा मेनू आयटमची उंची वाढवा.
