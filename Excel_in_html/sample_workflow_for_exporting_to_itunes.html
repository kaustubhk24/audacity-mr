4 4 This tutorial outlines a set of example steps that can be used to get Audacity Projects (LP and tape transcriptions for example) into iTunes. या प्रशिक्षणमध्ये उदाहरणांच्या चरणांचा एक आराखडा आहे ज्याचा उपयोग ओड्यासिटी प्रोजेक्ट्स (उदाहरणार्थ एलपी आणि टेप ट्रान्सक्रिप्शन) आयट्यून्समध्ये मिळविण्यासाठी केला जाऊ शकतो.
5 5 The workflow concentrates on AAC music files because that is Apple's native, default, format. वर्कफ्लो एएसी संगीत धारिका्सवर केंद्रित आहे कारण ते Appleपलचे मूळ, पूर्वनिर्धारित, स्वरूप आहे.
6 6 Some users may prefer to use uncompressed WAVs or AIFFs or compressed Apple Lossless formats for their music files in iTunes. काही वापरकर्ते आयट्यून्समधील संगीत धारिका्ससाठी कंप्रप्रेस केलेले डब्ल्यूएव्ही किंवा एआयएफएफ किंवा संकुचित Appleपल लॉसलेस स्वरूप वापरण्यास प्राधान्य देऊ शकतात.
7 7 It is not the only way of working - there are many alternatives. काम करण्याचा हा एकमेव मार्ग नाही - असे बरेच पर्याय आहेत.
8 8 Like any recipe it can be adapted to suit your personal needs. You can work in AIFF rather than WAV if required. Many Apple Mac users would probably prefer to work in AIFF. कोणत्याही रेसिपीप्रमाणे आपल्या वैयक्तिक गरजा भागविण्यासाठी ते अनुकूल केले जाऊ शकते. आवश्यक असल्यास आपण डब्ल्यूएव्हीऐवजी एआयएफएफमध्ये काम करू शकता. बरेच Appleपल मॅक वापरकर्ते कदाचित एआयएफएफमध्ये काम करण्यास प्राधान्य देतील.
9 9 See also Tutorial - Exporting to iTunes which has further useful information on this topic. प्रशिक्षण देखील पहा - या विषयावर पुढील उपयुक्त माहिती असलेल्या आयट्यून्सवर निर्यात करणे.

22 22 Export the WAV files डब्ल्यूएव्ही धारिका निर्यात करा
23 23 Use Export Audio or Export Multiple to produce a WAV file or set of WAVs to any convenient location. With Audacity set to a project rate of 44100 Hz and 32-bit sample format (these are the default quality settings), exporting will by default produce 44.1kHz 16-bit PCM WAV files (or AIFF files on Mac). Audacity will down-sample from its 32-bit internal format to 16-bit, with shaped dither noise applied to cover any clicky noise that may result from the conversion to 16-bit. Advanced users can change the type of dither, or turn it off, in the Quality Preferences. कोणत्याही सोयीस्कर ठिकाणी डब्ल्यूएव्ही धारिका किंवा डब्ल्यूएव्हीचा संच तयार करण्यासाठी ध्वनी निर्यात किंवा एकाधिक निर्यात करा वापरा. ओड्यासिटीने 44100 हर्ट्ज आणि 32-बिट नमुना स्वरूपात (या पूर्वनिर्धारित गुणवत्ता समायोजन आहेत) च्या प्रोजेक्ट रेटवर सेट केल्याने, निर्यात केल्यास पूर्वनिर्धारित 44.1kHz 16-बिट पीसीएम डब्ल्यूएव्ही धारिका्स (किंवा मॅकवर एआयएफएफ धारिका्स) तयार होतात. ओड्यासिटी त्याच्या 32-बिट अंतर्गत स्वरुपापासून 16-बिट पर्यंत खाली नमुना घेईल, ज्यामध्ये रूपांतरणानंतरच्या 16-बिटवर परिणाम होऊ शकणार्‍या कोणत्याही क्लिकच्या आवाजासाठी आकार काढलेला आवाज लावला जाईल. प्रगत वापरकर्ते क्वालिटी प्राधान्यांमधे तिचा प्रकार बदलू किंवा बंद करू शकतात.
24 24 16-bit export will ensure the files are compatible with all versions of iTunes, They will also be compatible for conversion to AAC or other formats (or for CD burning) in iTunes. It is possible to export 32-bit PCM files too, by choosing "Other uncompressed files" in either export dialog, then clicking Options.... These files will theoretically be of marginally higher quality but twice the size of 16-bit files. Latest versions of iTunes will play them, but other players may not. 16-बिट एक्सपोर्ट केल्याने हे सुनिश्चित होईल की धारिका्स आयट्यून्सच्या सर्व आवृत्त्यांशी सुसंगत आहेत, ते आयएट्यून्समधील एएसी किंवा इतर स्वरूपांमध्ये (किंवा सीडी बर्न करण्यासाठी) सुसंगत असतील. एकतर निर्यात संवादामध्ये "इतर कंप्रप्रेस धारिका्स" निवडून नंतर पर्यायांवर क्लिक करून 32-बिट पीसीएम धारिका निर्यात करणे देखील शक्य आहे .... या धारिका सैद्धांतिकदृष्ट्या अत्यल्प गुणवत्तेच्या असतील परंतु 16-बीट धारिकाच्या आकाराच्या दुप्पट असतील. ITunes च्या नवीनतम आवृत्त्या त्या प्ले केल्या जातील, परंतु अन्य खेळाडू कदाचित प्ले करू शकणार नाहीत.
25 25 Add the WAV files into iTunes आयट्यून्समध्ये डब्ल्यूएव्ही धारिका जोडा
26 26 Add the WAV files to the iTunes library (using the iTunes command File > Add File to Library or File > Add Folder to Library. Note that even if iTunes Import Preferences are set to Import Using AAC Settings ( Edit > Preferences > General > Import Settings ), no conversion takes place as this preference only applies to importing from CD. आयट्यून्स लायब्ररीमध्ये डब्ल्यूएव्ही धारिका जोडा (आयट्यून्स आज्ञा धारिका वापरुन> धारिका लायब्ररीमध्ये जोडा किंवा धारिका लायब्ररीमध्ये फोल्डर जोडा. आयटीयन्स आयात प्राधान्ये एएसी समायोजन (संपादन> प्राधान्ये> सामान्य> आयात समायोजन आयात करुन आयात केली असल्यास सेट करा. ), कोणतेही रूपांतरण होत नाही कारण हे प्राधान्य फक्त सीडीवरून आयात करण्यासाठीच लागू होते.
27 27 Be aware that by default iTunes does not copy audio files into its own filespace, rather it makes a link to where you currently have the audio file stored. To change this in iTunes go to Edit > Preferences > Advanced and mark the checkbox that says "Copy files to iTunes Media folder when adding to library". It is strongly recommended that you make this setting. हे लक्षात घ्या की पूर्वनिर्धारितनुसार आयट्यून्स ध्वनी धारिका त्याच्या स्वत: च्या धारिकास्पेसमध्ये कॉपी करत नाही, त्याऐवजी आपल्याकडे सध्या ध्वनी धारिका कोठे संग्रहित आहे याचा दुवा बनविते. आयट्यून्समध्ये हे बदलण्यासाठी संपादन> प्राधान्ये> प्रगत वर जा आणि "लायब्ररीत जोडताना आयट्यून्स मीडिया फोल्डरमध्ये धारिका कॉपी करा" असे म्हणणारे चेकबॉक्स चिन्हांकित करा. आपण ही सेटिंग बनविली पाहिजे अशी जोरदार शिफारस केली जाते.
28 28 Use iTunes to edit the metadata for the album name and artist अल्बमचे नाव आणि कलाकारांसाठी मेटाडेटा संपादित करण्यासाठी आयट्यून्स वापरा
29 29 Make sure that one of the columns you have showing in your iTunes library is Kind. The imported files will show as WAV and can be listed together by clicking on the "Kind" column header. आपण आपल्या आयट्यून्स लायब्ररीमध्ये दाखवत असलेल्या स्तंभांपैकी एक प्रकारची आहे याची खात्री करा. आयात केलेल्या धारिका डब्ल्यूएव्हीच्या रुपात दर्शविल्या जातील आणि "प्रकारची" स्तंभ शीर्षलेख वर क्लिक करून एकत्र सूचीबद्ध केली जाऊ शकतात.
30 30 Select all the WAV files and use iTunes' metadata editor to enter the album name and the artist name so that you can easily find it later when editing the other metadata tags. You could alternatively set this metadata prior to Audacity Export using Audacity's metadata editor – some users report that it is easier to use iTunes rather than Audacity to manage the metadata. सर्व डब्ल्यूएव्ही धारिका निवडा आणि अल्बमचे नाव आणि कलाकाराचे नाव प्रविष्ट करण्यासाठी आयट्यून्सचे मेटाडेटा संपादक वापरा जेणेकरून इतर मेटाडेटा टॅग संपादित करताना आपल्याला नंतर सहज सापडेल. ऑडीसिटीच्या मेटाडेटा एडिटरचा वापर करून ओड्यासिटी एक्सपोर्ट करण्यापूर्वी आपण हा मेटाडेटा वैकल्पिकरित्या सेट करू शकता - काही वापरकर्ते रिपोर्ट करतात की मेटाडेटा व्यवस्थापित करण्यासाठी ओड्यासिटीऐवजी आयट्यून्स वापरणे सोपे आहे.
31 31 Use iTunes to make AAC copies in the library from the WAV files डब्ल्यूएव्ही धारिकांमधून ग्रंथालयात एएसी प्रती करण्यासाठी आयट्यून्स वापरा
32 32 Provided that you did not already have other WAV files in your iTunes library, the WAV files you need should still be the only selected tunes at this stage, providing that you have done no further clicking. आपल्याकडे आधीपासूनच आपल्या आयट्यून्स लायब्ररीमध्ये इतर डब्ल्यूएव्ही धारिका नसतील तर आपल्याला आवश्यक असलेल्या डब्ल्यूएव्ही धारिका अद्याप या टप्प्यात केवळ निवडलेल्या सूर असू शकतात, परंतु आपण यापुढे क्लिक न करता केली आहे.
33 33 Make sure that you have your iTunes Import Settings set to AAC at your required bitrate. आपण आपल्या आवश्यक बिटरेटवर आपल्या आयट्यून्स आयात समायोजन एएसी वर सेट केल्या असल्याचे सुनिश्चित करा.
34 34 Convert them to AAC by using the iTunes command File > Convert . Note carefully that this makes a “copy” not a straight conversion – so at this stage your library will contain both the AAC and the WAV copies of the songs. आयट्यून्स आज्ञा धारिका> कन्व्हर्ट वापरुन त्यांना एएसी मध्ये रूपांतरित करा. काळजीपूर्वक लक्षात घ्या की यामुळे एक “कॉपी” सरळ रूपांतरण बनत नाही - म्हणून या टप्प्यावर आपल्या लायब्ररीत एएसी आणि गाण्यांच्या डब्ल्यूएव्ही दोन्ही प्रती असतील.
35 35 If you have plenty of room you may wish to keep the WAV files in your iTunes library to for their lossless quality and not convert them to AAC. आपल्याकडे भरपूर जागा असल्यास आपण आपल्या आयट्यून्स लायब्ररीत डब्ल्यूएव्ही धारिका त्यांच्या लॉसलेस गुणवत्तेसाठी ठेवू शकता आणि त्यांना एएसीमध्ये रूपांतरित करू इच्छित नाही.
36 36 In that case you should omit this conversion step and the next step which deletes the WAV files from your iTunes library. Yet another option is to set iTunes to convert to Apple Lossless which has about half the file size of WAV. अशा परिस्थितीत आपण हे रूपांतरण चरण आणि पुढील चरण आपल्या आयट्यून्स लायब्ररीतून डब्ल्यूएव्ही धारिका हटवू शकता. अजून एक पर्याय म्हणजे Appleपल लॉसलेसमध्ये रूपांतरित करण्यासाठी आयट्यून्स सेट करणे ज्यामध्ये डब्ल्यूएव्हीचे अर्धे धारिका आकार आहे.
37 37 Or you may wish to use MP3 files in your iTunes library instead of AAC. किंवा आपण एसी ऐवजी आपल्या आयट्यून्स लायब्ररीमध्ये एमपी 3 धारिका्स वापरू इच्छित असाल.
38 38 In that case either export as MP3 from Audacity, to any convenient location, for import into iTunes or import WAVs and set iTunes to convert to MP3 rather than its default AAC. त्या प्रकरणात एकतर आयट्यून्समध्ये आयात करण्यासाठी ओड्यासिटी वरून कोणत्याही सोयीस्कर ठिकाणी एमपी 3 म्हणून निर्यात करा किंवा डब्ल्यूएव्ही आयात करा आणि आयट्यून्सला पूर्वनिर्धारित एएसी ऐवजी एमपी 3 मध्ये रूपांतरित करण्यासाठी सेट करा.
39 39 Note that for equivalent file size, AAC is generally reckoned to give better quality audio than MP3, but MP3 is a much safer choice if you want to send the file to someone else. लक्षात घ्या की समतुल्य धारिका आकारासाठी, एएसी सहसा एमपी 3 पेक्षा अधिक दर्जेदार ध्वनी देणे मानले जाते, परंतु आपण धारिका दुसर्‍यास पाठवायची असल्यास एमपी 3 एक अधिक सुरक्षित निवड आहे.
40 40 There are several advantages to using compressed format on iPods. The two main ones are that you can fit a lot more songs into the device (for 256 kbps files you can fit more than 8-9 times as many songs) and compressed files improve battery life (because disk reads are relatively heavy on battery power). आयपॉडवर कॉम्प्रेस्ड फॉरमॅट वापरण्याचे बरेच फायदे आहेत. दोन मुख्य म्हणजे आपण डिव्हाइसमध्ये बरीच गाणी बसवू शकता (256 केबीपीएस धारिकांसाठी तुम्ही अनेक गाण्यांपेक्षा 8-9 पट फिट करू शकता) आणि कॉम्प्रेस केलेल्या धारिका्समुळे बॅटरीचे आयुष्य सुधारते (कारण डिस्क रीड्स बॅटरी उर्जेवर तुलनेने भारी असतात ).
41 41 Delete the WAV files from the library लायब्ररीतून डब्ल्यूएव्ही धारिका हटवा
42 42 The WAV files should still remain as the selected tunes - just take extreme care at this stage (the AACs are created but selection is not forced by iTunes) So then just Delete the selected WAV files using the Delete key - and send the files to the wastebasket. डब्ल्यूएव्ही धारिका अजूनही निवडलेल्या ट्यून म्हणूनच राहिल्या पाहिजेत - या टप्प्यावर अत्यधिक काळजी घ्या (एएसी तयार केल्या जातात परंतु आयट्यून्सद्वारे निवड करणे भाग पाडले जात नाही) तर मग डिलीट की वापरुन निवडलेल्या डब्ल्यूएव्ही धारिका हटवा - आणि त्या धारिका पाठवा कचरा
43 43 Note carefully that this workflow assumes that you have iTunes is set to copy files in, NOT to reference external files - so the WAVs that are deleted are iTunes’ copies of the original source WAVs. This is set in Edit > Preferences > Advanced with the Copy tunes into iTunes folder when adding to library checkbox. काळजीपूर्वक लक्षात घ्या की या वर्कफ्लोने असे गृहीत धरले आहे की आपल्याकडे आयट्यून्स आहेत धारिका कॉपी करण्यासाठी सेट केल्या आहेत बाह्य धारिकांचा संदर्भ घेण्यासाठी नाही - म्हणून हटविल्या जाणार्‍या डब्ल्यूएव्ही आयट्यून्सच्या मूळ स्त्रोत डब्ल्यूएव्हीच्या प्रती आहेत. हे लायब्ररी चेकबॉक्समध्ये जोडताना आयट्यून्स फोल्डरमध्ये कॉपी ट्यूनसह एडिट> पसंती> प्रगत मध्ये सेट केले आहे.
44 44 Edit the metadata tags of the tracks/album गीतपट्टा / अल्बमचे मेटाडेटा टॅग संपादित करा
45 45 Edit the metadata tags for Song Name, track number and similar गाण्याचे नाव, गीतपट्टा नंबर आणि तत्समतेसाठी मेटाडेटा टॅग संपादित करा
46 46 The album should be easy to locate in iTunes as you have already edited the Album tag in a previous workflow step above. आपण आधीच्या मागील वर्कफ्लो चरणात अल्बम टॅग संपादित केला आहे म्हणून आयट्यून्समध्ये अल्बम शोधणे सोपे आहे.
47 47 You may wish to edit the song titles to remove the leading 01, 02 numbering series that was applied earlier to facilitate the correct ordering of the songs. गाण्यांची क्रमवारी सुलभ करण्यासाठी यापूर्वी लागू झालेल्या 01, 02 क्रमांकाची मालिका काढून टाकण्यासाठी आपण गाण्याचे शीर्षक संपादित करू इच्छित असाल.
48 48 Fetch the album artwork अल्बमची कलाकृती आणा
49 49 Find a copy of the album artwork - either directly via iTunes if the album is recognized by the Gracenote database CDDB (but note that Gracenote does not always carry the artwork), or by making a digital photo of the album cover, and add the artwork to the album. अल्बमला ग्रॅझोनेट डेटाबेस सीडीडीबीने मान्यता दिली असेल तर थेट आयट्यून्सद्वारे (परंतु लक्षात घ्या की ग्रॅझोनेट नेहमीच कलाकृती ठेवत नाही), किंवा अल्बम कव्हरचा डिजिटल फोटो बनवून, आणि कलाकृती जोडा अल्बममध्ये
50 50 You may be able to find a copy of the album artwork on Amazon, Wikipedia or the artist's own website. आपल्याला Amazonमेझॉन, विकिपीडिया किंवा कलाकारांच्या स्वत: च्या वेबसाइटवर अल्बमच्या आर्टवर्कची एक प्रत सापडेल.
51 51 If you are using WAV files in your iTunes library then you cannot associate album artwork with these files. आपण आपल्या आयट्यून्स लायब्ररीमध्ये डब्ल्यूएव्ही धारिका वापरत असल्यास आपण या धारिकांसह अल्बम कलाकृती संबद्ध करू शकत नाही.
52 52 Optionally make a CD – and create cover & playlist वैकल्पिकरित्या एक सीडी तयार करा - आणि कव्हर आणि प्लेलिस्ट तयार करा
53 53 You may wish to burn a CD from the WAV files. आपण डब्ल्यूएव्ही धारिकांमधून सीडी बर्न करू शकता.
54 54 You can use iTunes to print CD covers with track-list and album cover; to do this you will need to create a temporary iTunes playlist for the album. गीतपट्टा-यादी आणि अल्बम कव्हरसह सीडी कव्हर मुद्रित करण्यासाठी आपण आयट्यून्स वापरू शकता; हे करण्यासाठी आपल्याला अल्बमसाठी तात्पुरती आयट्यून्स प्लेलिस्ट तयार करण्याची आवश्यकता असेल.
55 55 Backup your updated iTunes library आपल्या अद्यतनित आयट्यून्स लायब्ररीचा बॅकअप घ्या
56 56 This is a critical step – you do not want to lose the valuable fruits of your labors, do you? ही एक गंभीर पायरी आहे - आपण आपल्या श्रमांचे मौल्यवान फळ गमावू इच्छित नाही, आपण?
57 57 We would recommend maintaining two separate backup copies of your library on external disks. आम्ही बाह्य डिस्कवर आपल्या लायब्ररीच्या दोन स्वतंत्र बॅकअप प्रती राखण्याची शिफारस करतो.
58 58 Keep the WAV files that you exported from Audacity and back those up as well (two copies) on external disks. आपण ओड्यासिटी वरून निर्यात केलेल्या डब्ल्यूएव्ही धारिका ठेवा आणि त्या (दोन प्रती) बाह्य डिस्कवर परत करा.
59 59 Delete the Audacity project and exported WAV files to release hard disk space ओड्यासिटी प्रोजेक्ट हटवा आणि हार्ड डिस्क स्पेस सोडण्यासाठी डब्ल्यूएव्ही धारिका निर्यात केल्या
60 60 After the backups are made you can then safely release space on your hard drive by deleting any Audacity project you created (delete both the .aup file and the _data folder with the same name) and deleting the originally exported WAV files. बॅकअप तयार झाल्यानंतर आपण तयार केलेला कोणताही ओड्यासिटी प्रोजेक्ट हटवून (त्याच नावाने .aup धारिका आणि _डेटा फोल्डर दोन्ही हटवा) आणि मूळ निर्यात केलेली डब्ल्यूएव्ही धारिका हटवून आपण आपल्या हार्ड ड्राइव्हवर सुरक्षितपणे रिक्त सोडू शकता.
61 61 Direct AAC export from Audacity ओड्यासिटी वरून डायरेक्ट एएसी निर्यात
62 62 You may if you wish export to AAC compressed format (M4A files) from Audacity, to any convenient location, for direct import into iTunes. आपण आयट्यून्समध्ये थेट आयात करण्यासाठी ऑएडीसी कडून कोणत्याही सोयीस्कर ठिकाणी एएसी कॉम्प्रेस्ड फॉरमॅट (एम 4 ए धारिका्स) निर्यात करू इच्छित असाल तर.
63 63 AAC export is one of the options available in the Export menus. To export to AAC, download and install the optional FFmpeg library. The export quality for M4A can be modified using the Options... button. The highest quality Audacity setting is roughly comparable to encoding to AAC in iTunes using VBR encoding at the highest bit rate preset. निर्यात मेनूमध्ये एएसी निर्यात हा एक पर्याय उपलब्ध आहे. एएसीवर निर्यात करण्यासाठी, पर्यायी एफएफएमपीईजी लायब्ररी डाउनलोड आणि स्थापित करा. एम 4 ए साठी निर्यात गुणवत्ता पर्याय ... बटण वापरून सुधारित केली जाऊ शकते. उच्चतम गुणवत्ता ओड्यासिटी सेटिंग उच्च बिट रेट प्रीसेटवर व्हीबीआर एन्कोडिंग वापरुन आयट्यून्समध्ये एएसीशी एन्कोडिंग करणे अंदाजे तुलना आहे.
