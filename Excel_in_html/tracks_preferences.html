८ ८ Display प्रदर्शन
९ ९ Auto-fit track height: Tracks will be automatically resized to fit in the vertical space of the project window if possible. All tracks that have not been collapsed using the Track Collapse Button or Collapse All Tracks menu item are resized so that each separate waveform has the same height. Accordingly mono tracks will have the same height as each other and stereo tracks will have the same height as each other (twice the height of the mono tracks). ऑटो फिट गीतपट्टा उंची: शक्य असल्यास प्रकल्प विंडोच्या उभ्या जागेत फिट होण्यासाठी गीतपट्टाचे स्वयंचलितपणे आकार बदलले जाईल. गीतपट्टा कोसळणे बटण किंवा संकुचित सर्व गीतपट्टा मेनू आयटम वापरुन संकुचित न झालेल्या सर्व गीतपट्टाचे आकार बदलले जातात जेणेकरून प्रत्येक वेगळ्या तरंगरूपची उंची समान असेल. त्यानुसार मोनो गीतपट्टाची उंची एकमेकांइतकीच असेल आणि स्टीरिओ गीतपट्टाची उंची देखील एकमेकांप्रमाणे असेल (मोनो गीतपट्टाच्या दुप्पट उंची).
१० १० Show audio track name as overlay: When this is enabled the audio track name is displayed superimposed in yellow at top left of all audio tracks. Note that the track name is always shown in the Track Control Panel but will be truncated if the name is too long to fit. आच्छादन म्हणून ध्वनी गीतपट्टाचे नाव दर्शवा: जेव्हा हे सक्षम केले जाते तेव्हा ध्वनी गीतपट्टाचे नाव सर्व ध्वनी गीतपट्टाच्या डावीकडे शीर्षस्थानी पिवळा मध्ये सुपरम्पोज केलेले प्रदर्शित केले जाते. लक्षात ठेवा की गीतपट्टा नाव नेहमीच गीतपट्टा नियंत्रण पॅनेलमध्ये दर्शविले जाते परंतु हे नाव फारच लांब असल्यास तो कापला जाईल.
११ ११ Use half-wave display when collapsed: With this preference on, collapsing the tracks switches to half-wave view and uncollapsing the tracks switches to full wave view.} संकुचित झाल्यावर अर्ध्या-वेव्ह प्रदर्शन वापरा: या पसंतीसह, अर्ध्या-वेव्ह दृश्यावर गीतपट्टा स्विच करणे आणि गीतपट्टा स्विच पूर्ण लाट दृश्यावर अनलॉक करणे.}
१२ १२ Auto-scroll if head unpinned: If Pinned Recording/Playback head (see above) is unchecked as per default, this option scrolls the Timeline leftwards during standard play or recording when the playback or recording cursor reaches the right-hand edge of the project. Most users would keep this setting enabled. Turning it off: हेड अनपिन केले असल्यास ऑटो-स्क्रोल करा: पूर्वनिर्धारितनुसार पिन केलेले ध्वनीमुद्रणिंग / प्लेबॅक हेड (वर पहा) अनियमित नसल्यास, प्लेबॅक किंवा ध्वनीमुद्रणिंग कर्सर जेव्हा प्रकल्पाच्या उजव्या-कडेला पोहोचतो तेव्हा मानक प्ले किंवा ध्वनीमुद्रणिंग दरम्यान हा पर्याय वेळलाइन डावीकडे स्क्रोल करतो. बरेच वापरकर्ते ही समयोजन सक्षम ठेवत. हे बंद करीत आहे:
१३ १३ may prevent interrupted playback or recording on slower computers हळू संगणकांवर व्यत्यय प्लेबॅक किंवा ध्वनीमुद्रणिंग प्रतिबंधित करू शकते
१४ १४ lets you drag the horizontal scrollbar at the bottom of the window to a new position then restart playback from there आपल्याला विंडोच्या खालच्या आडव्या स्क्रोलबारला नवीन स्थानावर ड्रॅग करू देतो आणि तेथून प्लेबॅक रीस्टार्ट करू देतो
१५ १५ halts Scrubbing and Seeking playback when the playhead reaches either edge of the visible project प्लेहेड दृश्यमान प्रकल्पच्या दोन्ही काठावर पोहोचते तेव्हा घासणेआणि प्लेबॅक शोधणे थांबवते
१६ १६ prevents losing sight of a selection edge when making loops लूप बनवताना निवड काठावरील दृष्टी गमावण्यापासून प्रतिबंधित करते
१७ १७ makes it easier to listen repeatedly from a particular start point without its visible surroundings changing.This preference may also be turned on and off during playback using the Timeline right-click menu. एखाद्या विशिष्ट प्रारंभ बिंदूतून त्याच्या आसपासचे दृश्यमान बदल न बदलता वारंवार ऐकणे सुलभ करते. वेळलाइन योग्य-क्लिक मेनूचा वापर करुन प्लेबॅक दरम्यान ही पसंती देखील चालू आणि बंद केली जाऊ शकते.
१८ १८ Default view mode: Choose the default view for all newly-created audio tracks. The choices are the same as those available in the Audio Track Dropdown Menu. पूर्वनिर्धारित दृश्य पदधत: सर्व नव्याने तयार केलेल्या ध्वनी गीतपट्टासाठी पूर्वनिर्धारित दृश्य निवडा. ध्वनी गीतपट्टा ड्रॉपडाउन मेनूमध्ये असलेल्या पर्यायांसारखेच आहेत.
१९ १९ Display samples: This setting changes how Waveform and Waveform dB views are displayed. नमुने प्रदर्शित कराः ही समयोजन तरंगरूप आणि तरंगरूप डीबी दृश्ये कशी प्रदर्शित केली जातात हे बदलते.
२० २० It only affects the appearance of the waveform when you are so far zoomed in that you can see the individual sample dots. जेव्हा आपण आतापर्यंत झूम केलेले असतो तेव्हा आपण वैयक्तिक नमुना ठिपके पाहू शकता तेव्हाच हे तरंगरूपच्या देखावावर परिणाम करते.
२१ २१ At lower zoom levels it makes no difference. खालच्या झूम स्तरावर काही फरक पडत नाही.
२२ २२ Stem plot: This is the default setting which draws a vertical line from the track center line to the sample dot, giving a clearer impression of the relative amplitude of the samples. As seen in the images below, when zoomed out close to the minimum for a stem plot, the horizontal distance between sample dots may be more uneven than seen with the connect dots default. Uneven spacing is due to "aliasing", but zooming in further will equalize the spacing whether choosing Connect dots or Stem plot. स्टेम प्लॉटः ही पूर्वनिर्धारित समयोजन आहे जी गीतपट्टा केंद्र लाइनपासून नमुना बिंदूपर्यंत अनुलंब रेषा ओढवते, जे नमुन्यांच्या सापेक्ष विशालतेची स्पष्ट माहिती देते. खाली दिलेल्या प्रतिमांमध्ये पाहिल्याप्रमाणे, जेव्हा स्टेम प्लॉटसाठी कमीतकमी जवळ झूम वाढविला जातो तेव्हा नमुना बिंदूंमधील क्षैतिज अंतर कनेक्ट बिंदू पूर्वनिर्धारितसह पाहिल्यापेक्षा जास्त असमान असू शकते. असमान अंतर "अलियासिंग" च्या कारणामुळे आहे, परंतु पुढील झूम केल्याने कनेक्ट बिंदू किंवा स्टेम प्लॉट निवडल्यास ते अंतर समान केले जाईल.
२३ २३ Connect dots: This is alternative setting yields a waveform where each sample dot is connected to the next sample by a line drawn between them. बिंदू कनेक्ट करा: ही पर्यायी समयोजन एक तरंगरूप देते जेथे प्रत्येक नमुना बिंदू त्यांच्या दरम्यान काढलेल्या रेषाने पुढील नमुनाशी जोडलेला असतो.
२४ २४ Default audio track name: The default name that Audacity will use for new audio tracks other than imported files. The Audacity default name is "Audio Track". पूर्वनिर्धारित ध्वनी गीतपट्टा नाव: ओड्यासिटी आयात केलेल्या धारिका व्यतिरिक्त नवीन ध्वनी गीतपट्टासाठी वापरेल असे पूर्वनिर्धारित नाव. ओड्यासिटीचे पूर्वनिर्धारित नाव "ध्वनी गीतपट्टा" आहे.
