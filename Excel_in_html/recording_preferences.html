८ ८ Options पर्याय
९ ९ Play other tracks while recording (overdub): Makes existing tracks in the project play when you click the Record button in Transport Toolbar. You can decide which tracks will play according to their mute/solo buttons. When Overdub is enabled, recording will stop when the recording cursor reaches the end of a selection region. Turn overdub off if you want to record past the end of the selection. ध्वनीमुद्रणिंग करताना इतर गीतपट्टा प्ले करा (ओव्हरडब): आपण परिवहन साधनपट्टीमधील ध्वनीमुद्रण बटणावर क्लिक करता तेव्हा प्रकल्प प्लेमध्ये विद्यमान गीतपट्टा तयार होतात. त्यांच्या नि: शब्द / एकल बटणानुसार कोणते गीतपट्टा प्ले होतील हे आपण ठरवू शकता. जेव्हा ओव्हरडब सक्षम केले जाते, जेव्हा ध्वनीमुद्रणिंग कर्सर निवड प्रदेशाच्या शेवटी पोहोचतो तेव्हा ध्वनीमुद्रणिंग थांबेल. आपण निवडीच्या शेवटी गेल्या ध्वनीमुद्रण करू इच्छित असल्यास ओव्हरडब बंद करा.
१० १० Use software to play other tracks: Allows you to hear the input by sending it to any chosen playback device. This permits the recording and playback device to be different. Expect some latency in the sound that you hear, due to the time taken for the computer to process the audio. This option may cause recording dropouts on older, slower computers. This must not be used for recording computer playback because it will create feedback echoes. इतर गीतपट्टा खेळण्यासाठी सॉफ्टवेअर वापरा: कोणत्याही निवडलेल्या प्लेबॅक उपकरणवर पाठवून आपणास इनपुट ऐकण्याची परवानगी देते. हे ध्वनीमुद्रणिंग आणि प्लेबॅक उपकरण भिन्न असू देते. संगणकाला ध्वनीवर प्रक्रिया करण्यासाठी लागणा time्या वेळेमुळे आपण ऐकत असलेल्या आवाजात काही विलंब करण्याची अपेक्षा करा. हा पर्याय जुन्या, हळू संगणकांवर ध्वनीमुद्रणिंग ड्रॉपआउटस कारणीभूत ठरू शकतो. हे संगणकाच्या प्लेबॅक ध्वनीमुद्रणिंगसाठी वापरले जाऊ नये कारण यामुळे अभिप्राय प्रतिध्वनी निर्माण होईल.
११ ११ Useful Tips: उपयुक्त टिप्स:
१२ १२ If you enable monitoring with software to play other tracks, you can listen to the input without recording it. आपण इतर गीतपट्टा खेळण्यासाठी सॉफ्टवेअरसह देखरेख सक्षम केल्यास आपण इनपुट ध्वनीमुद्रण केल्याशिवाय ऐकू शकता.
१३ १३ For some examples of setting up equipment to record without playthrough latency, see this Tutorial about Recording overdubs. प्लेथ्रू लेटेंसीशिवाय ध्वनीमुद्रण करण्यासाठी उपकरणे सेट करण्याच्या काही उदाहरणांसाठी, ध्वनीमुद्रणिंग ओव्हरडब्सबद्दलचे हे प्रशिक्षण पहा.
१४ १४ When recording from a microphone, listen in headphones so that the microphone hears only what you are actually recording. मायक्रोफोनवरून ध्वनीमुद्रणिंग करताना, हेडफोन्समध्ये ऐका जेणेकरून आपण प्रत्यक्षात ध्वनीमुद्रण करत असलेलेच मायक्रोफोन ऐकतो.
१५ १५ Record on a new track: Setting this to be "on" will cause Audacity to always record on a new track when you press the Record button नवीन गीतपट्टावर ध्वनीमुद्रण करा: जेव्हा आपण ध्वनीमुद्रण बटण दाबाल तेव्हा हे "चालू" असण्यामुळे ओड्यासिटी नेहमीच नवीन गीतपट्टावर ध्वनीमुद्रण होते.
१६ १६ on Transport Toolbar (or use the shortcut R).If you hold the Shift button down, the Record button in Transport Toolbar will temporarily change to . Then clicking on this modified Record button (or using the shortcut Shift + R) will cause ट्रान्सपोर्ट साधनपट्टीवर (किंवा शॉर्टकट आर वापरा). जर आपण शिफ्ट बटण दाबून ठेवले तर ट्रान्सपोर्ट साधनपट्टीमधील ध्वनीमुद्रण बटण तात्पुरते बदलले जाईल. मग या सुधारित ध्वनीमुद्रण बटणावर क्लिक केल्याने (किंवा शॉर्टकट शिफ्ट + आर वापरुन) होईल
१७ १७ Audacity to record at the end of the currently selected, or only, track. सध्या निवडलेल्या किंवा केवळ गीतपट्टाच्या शेवटी ध्वनीमुद्रण करण्याची धडपड.
१८ १८ Detect dropouts: When this setting is "on" (default setting) Audacity will detect dropouts (brief gaps in the recording) and will insert zeroes into the recording to keep the other good parts synchronized. ड्रॉपआउट्स शोधाः जेव्हा ही सेटिंग "चालू असते" (पूर्वनिर्धारित सेटिंग) ओड्यासिटी ड्रॉपआउट्स (ध्वनीमुद्रणिंगमधील संक्षिप्त अंतर) शोधते आणि इतर चांगले भाग सिंक्रोनाइझ ठेवण्यासाठी ध्वनीमुद्रणिंगमध्ये शूनो समाविष्ट करते.
१९ १९ When recording stops, a message box alerts the user and a label track, called "Dropouts", is added showing the lost parts, labeled with consecutive numbers. See the Recording page for more details. ध्वनीमुद्रणिंग थांबतेवेळी, संदेश बॉक्स वापरकर्त्यास सतर्क करते आणि "ड्रॉपआउट्स" नावाच्या लेबल गीतपट्टाला गमावलेला भाग दर्शवित जोडला जातो, सतत क्रमांकांसह लेबल. अधिक माहितीसाठी ध्वनीमुद्रणिंग पृष्ठ पहा.
२० २० Sound Activated Recording ध्वनी सक्रिय ध्वनीमुद्रणिंग
२१ २१ Sound Activated Recording: Enables or disables Sound Activated Recording. When enabled, click the Record button in Transport Toolbar to start the Sound Activated Recording session. Recording will now occur whenever the received input is above the Sound Activation Level you specify (see the item below). Recording will pause whenever the received input falls below that level. Press the Stop button to stop the recording session. ध्वनी सक्रिय ध्वनीमुद्रणिंग: ध्वनी सक्रिय ध्वनीमुद्रणिंग सक्षम किंवा अक्षम करते. सक्षम केलेले असल्यास, ध्वनी सक्रिय ध्वनीमुद्रणिंग सत्र प्रारंभ करण्यासाठी परिवहन साधनपट्टीमधील ध्वनीमुद्रण बटणावर क्लिक करा. जेव्हा आपण प्राप्त केलेले इनपुट आपण निर्दिष्ट केलेल्या आवाज सक्रियतेच्या पातळीवर असेल तेव्हा ध्वनीमुद्रणिंग होईल (खाली आयटम पहा). जेव्हा प्राप्त केलेले इनपुट त्या पातळीच्या खाली येते तेव्हा ध्वनीमुद्रणिंगला विराम दिला जाईल. ध्वनीमुद्रणिंग सत्र थांबविण्यासाठी थांबा बटण दाबा.
२२ २२ You cannot pause Sound Activated recordings manually using the Pause button or corresponding menu item or shortcut. आपण विराम द्या बटण किंवा संबंधित मेनू आयटम किंवा शॉर्टकट वापरुन स्वहस्ते ध्वनी सक्रिय ध्वनीमुद्रणिंगस विराम देऊ शकत नाही.
२३ २३ Sound Activation Level (dB): Sets the threshold level in dB which triggers Sound Activated Recording. The range of the scale is determined by the Meter / Waveform dB range currently set in Interface Preferences. Although dB are logarithmic units, the actual scale values are displayed linearly (equally spaced for each value). The currently set dB value is displayed dynamically above the center of the scale. ध्वनी ationक्टिव्हिटी लेव्हल (डीबी): डीबी मध्ये थ्रेशोल्ड पातळी सेट करते ज्यामुळे साउंड एक्टिवेटेड ध्वनीमुद्रणिंग चालू होते. स्केलची श्रेणी सध्या संवादपटल प्राधान्यांमध्ये सेट केलेल्या मीटर / तरंगरूप डीबी श्रेणीद्वारे निश्चित केली जाते. जरी डीबी लॉगॅरिथमिक युनिट्स आहेत, वास्तविक स्केल मूल्ये रेषात्मकपणे दर्शविली जातात (प्रत्येक मूल्यासाठी समान अंतर) सध्या सेट केलेले डीबी मूल्य स्केलच्या मध्यभागी डायनॅमिकली प्रदर्शित केले जाईल.
२४ २४ For easier access, Overdub, Software Playthrough and Sound Activated Recording can be toggled on and off by using their respective entries in Transport Menu, or use Keyboard Preferences to allocate a keyboard shortcut to them. सुलभ प्रवेशासाठी, ओव्हरडब, सॉफ्टवेअर प्लेथ्र्यू आणि साउंड atedक्टिवेटेड ध्वनीमुद्रणिंग ट्रान्सपोर्ट मेनूमध्ये त्यांच्या संबंधित नोंदी वापरुन किंवा बंद केले जाऊ शकतात किंवा कीबोर्ड शॉर्टकट वाटप करण्यासाठी कीबोर्ड प्राधान्ये वापरा.
२५ २५ Name newly recorded tracks नव्याने नोंदवलेल्या गीतपट्टाची नावे
२६ २६ Custom Track Name: Use this to replace Audacity's default name of Audio Track with one of your own choosing. सानुकूल गीतपट्टा नाव: ऑडिटीचे पूर्वनिर्धारित नाव ध्वनी गीतपट्टाचे आपल्या स्वत: च्या निवडीसह पुनर्स्थित करण्यासाठी याचा वापर करा.
२७ २७ Track Number: Appends a track number to each new track name. गीतपट्टा क्रमांक: प्रत्येक नवीन गीतपट्टा नावावर एक गीतपट्टा नंबर जोडतो.
२८ २८ System Date: Appends the date to each new track name, in year-month-day format. सिस्टम तारीख: वर्षाच्या-महिन्या-दिवसाच्या स्वरूपात प्रत्येक नवीन गीतपट्टा नावावर तारीख समाविष्ट करते.
२९ २९ System Time: Appends the start time of the recording to each new track name, in hours-minutes-seconds format, २४-hour clock. सिस्टम वेळः प्रत्येक नवीन गीतपट्टा नावावर ध्वनीमुद्रणिंगचा प्रारंभ वेळ, तास-मिनिट-सेकंद स्वरूपात, २४-तास घड्याळामध्ये जोडते.
३० ३० Punch and Roll Recording पंच आणि रोल ध्वनीमुद्रणिंग
३१ ३१ Pre-roll: The amount of existing audio, in seconds, that will be played before the repair recording starts. The default is ५ seconds. प्री-रोलः विद्यमान ध्वनीचे प्रमाण, सेकंदांमध्ये, दुरुस्ती ध्वनीमुद्रणिंग सुरू होण्यापूर्वी प्ले केले जाईल. पूर्वनिर्धारित ५ सेकंद आहे.

The default is १० milliseconds. क्रॉसफेड: क्रॉसफेडची लांबी, मिलिसेकंदांमध्ये, ती ओड्यासिटी स्प्लिस पॉईंटवर संक्रमण सुरळीत करण्यासाठी लागू होईल. पूर्वनिर्धारित १० मिलिसेकंद आहे.
