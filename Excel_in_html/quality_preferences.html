८ ८ Sampling नमुना
९ ९ This panel sets the Default Sample Rate and Default Sample Format. हे पॅनेल पूर्वनियोजित नमुना दर आणि पूर्वनियोजित नमुना स्वरूप जतन करते.
१० १० It is strongly recommended that you stick with the default settings. आपण पूर्वनियोजित सामायोजनासह रहा, अशी जोरदार शिफारस केली जाते.
११ ११ Default Sample Rate: This sets the Sample Rate which will be used each time Audacity is launched and also when a new project window is opened. पूर्वनियोजित नमुना दर: यामुळे नमुना दर निश्चित केला जातो जो प्रत्येक वेळी ओड्यासिटी सुरू होताना आणि नवीन प्रकल्प उघडल्यावर वापरला जाईल.
१२ १२ It offers a choice of १३ Sample Rates. हे १३ नमुना दरांची निवड देते.
१३ १३ Generally you will want to use the default value of ४४१०० Hz, which can can reproduce all frequencies that humans can hear (up to २२०५० Hz) and is the standard for audio CDs. To choose some other bespoke rate, click "Other..." in the dropdown and type your rate into the box to right of the dropdown. Sample rates between ४४१०० Hz and ९६००० Hz can produce higher quality but the quality benefits are increasingly small compared to rapidly increasing consumption of disk space and increasing risk of recording dropouts on slower computers. १९२००० Hz music recordings could have ultrasonic playback distortion that make them inferior to recordings at lower rates. See सामान्यत: आपणास ४४१०० हर्ट्जचे पूर्वनियोजित मूल्य वापरायचे आहे, जे मनुष्यांनी ऐकू शकणार्‍या सर्व वारंवारता पुनरुत्पादित करू शकते (२२०५० हर्ट्झ पर्यंत) आणि ध्वनी सीडीसाठी हे मानक आहे. काही अन्य बीस्पोक रेट निवडण्यासाठी, ड्रॉपडाऊनमधील "अन्य ..." वर क्लिक करा आणि ड्रॉपडाऊनच्या उजवीकडील बॉक्समध्ये आपला दर लिहा ४४१००० हर्ट्ज ९६०००० हर्ट्झ दरम्यानचे नमुने दर उच्च गुणवत्तेचे उत्पादन करू शकतात परंतु डिस्क जागेचा वेगाने वाढणारा वापर आणि हळू संगणकांवर सोडण्याच्या मुद्रणाच्या जोखीमच्या तुलनेत गुणवत्ता फायदे वाढत्या प्रमाणात कमी आहेत.१९२००० हर्ट्झ ध्वनी मुद्रणामध्ये प्रचंड कंपनसंख्या असलेल्या (ध्वनिलहरी) प्लेबॅक विकृती असू शकतात ज्यामुळे त्यांना कमी दरात मुद्रणापेक्षा निकृष्ट केले जाईल. पहा
१४ १४ "२४/१९२ Music Downloads are Very Silly Indeed" for a technical discussion of the issues involved. Very high rates do have some specialized uses such as ultrasonic wildlife recordings and high-speed tape capture.Changing this Preference setting will not change your project's Project Rate, so if you wish your current project to use the new sample rate you will need to reset that in the Selection Toolbar. "२४/१९२ संगीत डाउनलोड्स खूप मूर्ख आहेत खरोखर" गुंतलेल्या मुद्द्यांच्या तांत्रिक चर्चेसाठी. अतिशय उच्च दरामध्ये अल्ट्रासोनिक वाइल्डलाइफ ध्वनीमुद्रणिंग आणि हाय-स्पीड टेप कॅप्चरिंगसारखे काही खास उपयोग आहेत. या पसंती सेटिंगमध्ये बदल केल्याने आपल्या प्रकल्पाचा प्रकल्प दर बदलला जाणार नाही, तर आपला नवीन प्रकल्प नमुना दर वापरण्याची आपली इच्छा असल्यास आपण रीसेट करणे आवश्यक आहे ते सिलेक्शन साधनपट्टी मध्ये आहे.
१५ १५ If you just want to make a change to your current project's sample rate without changing the default rate, open the Project Rate dropdown at left of आपण पूर्वनिर्धारित दर न बदलता आपल्या वर्तमान प्रकल्पाच्या नमुना दरामध्ये बदल करू इच्छित असल्यास डावीकडील प्रकल्प दर ड्रॉपडाउन उघडा
१६ १६ Selection Toolbar and select or type in the rate you require. निवड साधनपट्टी आणि निवडा किंवा आपल्याला आवश्यक दर टाइप करा.
१७ १७ Default Sample Format: This sets the Sample Format (bit depth) which will be used each time Audacity is launched, or each time a new project window or track is opened. पूर्वनिर्धारित नमुना स्वरूप: हे नमुना स्वरूप (थोडी खोली) सेट करते जे प्रत्येक वेळी ओड्यासिटी सुरू होताना वापरले जाईल किंवा प्रत्येक वेळी नवीन प्रकल्प विंडो किंवा गीतपट्टा उघडले जाईल.
१८ १८ It offers a choice of three sample formats or bit-depths. This affects both imported and newly recorded material, either in a new or existing project (even if the other audio in an existing project is at some other quality). The default ३२-bit float resolution gives the highest quality of the three choices, but takes twice the storage space on disk compared to १६-bit resolution. हे तीन नमुना स्वरूप किंवा बिट-खोलीची निवड प्रदान करते. नवीन किंवा विद्यमान प्रकल्पमध्ये (विद्यमान प्रकल्पातील अन्य ध्वनी काही अन्य गुणवत्तेवर असला तरीही) आयात आणि नवीन ध्वनीमुद्रण केलेल्या सामग्रीवर याचा परिणाम होतो. पूर्वनिर्धारित ३२-बिट फ्लोट रेझोल्यूशन तीन निवडींमध्ये सर्वोच्च गुणवत्ता देते, परंतु १६-बिट रेझोल्यूशनच्या तुलनेत डिस्कवर दोनदा स्टोरेज स्पेस घेते.
१९ १९ It is strongly recommended that you work with this at the default ३२-bit float resolution if you have sufficient available disk space. आपल्याकडे अशी डिस्क रिक्त जागा उपलब्ध असल्यास आपण पूर्वनिर्धारित ३२-बिट फ्लोट रेझोल्यूशनसह कार्य करावे अशी जोरदार शिफारस केली जाते.
२० २० OGG will always import at १६-bit resolution using the standard OGG Vorbis importer, irrespective of Default Sample Format. This is due to the design of the OGG codec. However OGG can be imported at ३२-bit resolution using FFmpeg, as described at Using the file type dropdown menu on the Importing Audio page. पूर्वनिर्धारित नमुना स्वरुपाची पर्वा न करता OGG नेहमी मानक OGG Vorbis आयातक वापरुन १६-बिट रेझोल्यूशनवर आयात करेल. हे ओजीजी कोडेकच्या डिझाइनमुळे आहे. तथापि, आयात ध्वनी पृष्ठावरील धारिका प्रकार ड्रॉपडाउन मेनू वापरण्यामध्ये वर्णन केल्यानुसार, ओएफजी ३२-बिट रेझोल्यूशनवर एफएफएमपीजी वापरुन आयात केले जाऊ शकते.
२१ २१ Conversion रूपांतरण
२२ २२ These options concern the trade-offs that arise when converting from one sample rate to another and one sample format to another. हे पर्याय एका नमुना दरापासून दुसर्‍या आणि एका नमुना स्वरूपनात दुसर्‍यामध्ये रूपांतरित करताना उद्भवलेल्या व्यापार-विषयी संबंधित असतात.
२३ २३ For example, sample rate conversion is necessary whenever the sample rate of the track does not match the current project rate. Sample format conversion would be required upon export if you used the default ३२-bit float sample format but exported a १६-bit audio file. उदाहरणार्थ, गीतपट्टाचा नमुना दर सध्याच्या प्रकल्प दराशी जुळत नसल्यास नमुना दर रूपांतरण आवश्यक आहे. आपण पूर्वनिर्धारित ३२-बिट फ्लोट नमुना स्वरूप वापरला परंतु १६-बिट ध्वनी धारिका निर्यात केल्यास निर्यातीवर नमुना स्वरूप रूपांतरण आवश्यक असेल.
२४ २४ Sample rate conversion is also performed when using a Time Track so that the speed-changed audio can retain its current sample rate. However for Time Track resampling Audacity automatically selects a quality level tailored for this purpose and so changing the converter qualities below will not affect Time Tracks. वेळ गीतपट्टा वापरताना नमुना दर रूपांतरण देखील केले जाते जेणेकरुन वेग-बदललेला ध्वनी सध्याचा नमुना दर कायम ठेवू शकेल. तथापि वेळ गीतपट्टा रीमॅम्पलिंग ओड्यासिटीसाठी स्वयंचलितपणे या हेतूसाठी तयार केलेल्या गुणवत्तेची पातळी निवडली जाते आणि त्यामुळे खाली कनव्हर्टरचे गुण बदलल्याने वेळ गीतपट्टावर परिणाम होणार नाही.
२५ २५ Real-time and High-quality: Both the Sample Rate Converter and the Dither (for conversion between sample formats) have options for use in different circumstances. रिअल-वेळ आणि उच्च-गुणवत्ताः नमुना दर कनव्हर्टर आणि डाइटर (नमुना स्वरूपात रूपांतरणासाठी) दोन्हीकडे भिन्न परिस्थितींमध्ये वापरण्यासाठी पर्याय आहेत.
२६ २६ The "Real-time" options are only used when converting sample rates or formats for playback in Audacity. "रीअल-वेळ" पर्याय केवळ ओड्यासिटीमध्ये प्लेबॅकसाठी नमुना दर किंवा स्वरूपांमध्ये रूपांतरित करताना वापरले जातात.
२७ २७ The "High-quality" options are used for sound that is being converted for storage on disk, for example when rendering or exporting. "उच्च-गुणवत्तेचे" पर्याय ध्वनीसाठी वापरले जातात जे डिस्कवरील संचयनासाठी रूपांतरित केले जात आहेत, उदाहरणार्थ प्रस्तुत करताना किंवा निर्यात करताना.
२८ २८ Option dropdown menus: पर्याय ड्रॉपडाउन मेनू:
२९ २९ The Real-time Sample Rate Converter defaults to and should normally be left on "Medium Quality". If jerky or uneven playback is experienced when using a slow machine or working with many tracks, the setting can be changed to "Low Quality (Fastest)". On a fast machine, higher quality settings may be used but may cause jerky or uneven playback, particularly with large multi-track projects. रीअल-वेळ नमुना दर रूपांतरक पूर्वनिर्धारित आहे आणि सामान्यपणे "मध्यम गुणवत्ता" वर सोडले पाहिजे. हळू मशीन वापरताना किंवा बर्‍याच गीतपट्टासह काम करताना विचित्र किंवा असमान प्लेबॅकचा अनुभव घेतल्यास सेटिंग "लो क्वालिटी (सर्वात वेगवान)" मध्ये बदलली जाऊ शकते. वेगवान मशीनवर, उच्च गुणवत्तेची समायोजन वापरली जाऊ शकतात परंतु त्रासदायक किंवा असमान प्लेबॅक होऊ शकतात, विशेषत: मोठ्या मल्टि-गीतपट्टा प्रकल्पांसह.
३० ३० The High-quality Sample Rate Converter defaults to "Best Quality (Slowest)". Leave it at this setting for the best quality conversion when exporting. "उत्कृष्ट गुणवत्ता (सर्वात धीमे)" वर उच्च-गुणवत्तेचा नमुना दर कनव्हर्टर पूर्वनिर्धारित आहे. निर्यात करताना उत्कृष्ट गुणवत्तेच्या रूपांतरणासाठी या सेटिंगवर ते सोडा.
३१ ३१ The Dither options are different methods of adding random low-level noise to mask clicky rounding distortions when converting to a lower bit depth. Dither should normally be set to the default "None" for Real Time playback and enabled for High-quality Conversion. डाईथ पर्याय म्हणजे कमी थोड्या खोलीमध्ये रूपांतरित करताना क्लिकिंग राउंडिंग विकृतींना मुखवटा लावण्यासाठी यादृच्छिक निम्न-स्तरीय आवाज जोडण्याची भिन्न पद्धती आहेत. एकतर रीअल वेळ प्लेबॅकसाठी पूर्वनिर्धारित "काहीही नाही" वर सेट केले जावे आणि उच्च-गुणवत्तेच्या रूपांतरणासाठी सक्षम केले जावे.
३२ ३२ "Shaped" dither (default for High-quality conversion) concentrates the noise in the less easily heard higher frequencies. "आकाराचे" दिवे (उच्च-गुणवत्तेच्या रूपांतरणासाठी पूर्वनिर्धारित) आवाज सहजतेने ऐकल्या जाणार्‍या उच्च वारंवारतेमध्ये लक्ष केंद्रित करतो.
३३ ३३ "Triangle" and "Rectangle" dither concentrate the noise less strongly in higher frequencies. "Rectangle" does not add noise to absolute silence ( zero amplitude samples ) so can be considered if the audio includes extensive pure silence. "त्रिकोण" आणि "आयत" जास्त आवृत्त्यांमध्ये आवाज कमी जोरात केंद्रित करतात. "आयत" निरपेक्ष शांततेत आवाज जोडत नाही (शून्य मोठेपणाचे नमुने) म्हणून ध्वनीमध्ये विस्तृत शुद्ध मौन समाविष्ट केले असल्यास त्यास विचारात घेतले जाऊ शकते.
३४ ३४ For more detailed information on dither see the the Dither page. दिनांकडील अधिक तपशीलवार माहितीसाठी डाईट पृष्ठ पहा.
