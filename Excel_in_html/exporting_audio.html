५ ५ Export Commands आज्ञा निर्यात करा

७ ७ File > Export > Export Audio... exports all the audio tracks in the project. By default, multiple mono tracks are mixed down to a mono file, but if you pan any mono tracks to left or right or if there are any stereo tracks, the tracks are mixed to a stereo file.
