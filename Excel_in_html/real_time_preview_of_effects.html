४ ४ Real-time preview enables you to change effect settings while playing and listening to the result in real-time, rather than having to preview, tweak settings then preview again. A real-time preview effect therefore includes Play/Stop, Skip Backward and Skip Forward buttons instead of a Preview button. Currently not all effects shipped with Audacity support real-time preview, but you can download additional LADSPA, LV२, VST and Audio Unit (for Mac) effects, almost all of which will support real-time preview. रीअल-वेळ पूर्वावलोकन आपल्याला प्ले करताना आणि रीअल-वेळमधील निकाल ऐकताना परिणाम समायोजन बदलण्यास सक्षम करते, पूर्वावलोकन करण्याऐवजी, चिमटा समायोजन नंतर पुन्हा पूर्वावलोकन करा. रीअल-वेळ पूर्वावलोकन परिणामी पूर्वावलोकनाऐवजी प्ले / स्टॉप, बॅकवर्ड वगळा आणि अग्रेषित बटणे समाविष्ट करा. सध्या ओड्यासिटी रिअल-वेळ पूर्वावलोकनासह पाठविलेले सर्व प्रभाव नाहीत, परंतु आपण अतिरिक्त एलएडीएसपीए, एलव्ही २, व्हीएसटी आणि ध्वनी युनिट (मॅकसाठी) प्रभाव डाउनलोड करू शकता, जे जवळजवळ सर्वच रिअल-वेळ पूर्वावलोकनास समर्थन देतील.
५ ५ Real-time preview differences रीअल-वेळ पूर्वावलोकन फरक
६ ६ An effect that supports real-time preview is different from regular effects in the following ways. रीअल-वेळ पूर्वावलोकनास समर्थन देणारा प्रभाव खालील मार्गांनी नियमित प्रभावापेक्षा भिन्न आहे.
७ ७ It can be opened when audio is playing or paused. जेव्हा ध्वनी चालू असतो किंवा विराम दिला जातो तेव्हा ते उघडता येऊ शकते.
८ ८ It can be opened even when no audio is selected, as long as the track is selected. कोणताही ध्वनी निवडलेला नसतानाही तो उघडला जाऊ शकतो, जोपर्यंत गीतपट्टा निवडला जात नाही.
९ ९ One regular effect can be opened and applied while a real-time preview effect is open but not playing. रीअल-वेळ पूर्वावलोकन प्रभाव उघडलेला परंतु प्ले होत नसताना एक नियमित प्रभाव उघडला आणि लागू केला जाऊ शकतो.
१० १० Real-time preview settings can be changed while audio is playing. ध्वनी चालू असताना रीअल-वेळ पूर्वावलोकने समायोजन बदलली जाऊ शकतात.
११ ११ When in Graphical Mode, live meters or graphs should respond when audio is playing. ग्राफिकल मोडमध्ये असताना, ध्वनी प्ले होत असताना थेट मीटर किंवा आलेखांनी प्रतिसाद द्यावा.
१२ १२ You can start and stop audio playback, change the selection, apply gain or pan to tracks or Mute or Solo them while the effect dialog is open. You can use the shortcuts Alt + F६ आपण ध्वनी प्लेबॅक प्रारंभ आणि थांबवू शकता, निवड बदलू शकता, गीतपट्टावर वाढवू किंवा पॅन लावू शकता किंवा प्रभाव संवाद उघडलेला असताना त्यांना निःशब्द किंवा सोलो करू शकता. आपण Alt + F६ शॉर्टकट वापरू शकता

१४ १४ Extra (use the Option key instead of Alt on Mac) to move forward or backward respectively between the effect and the main project window. प्रभाव आणि मुख्य प्रकल्प विंडो दरम्यान अनुक्रमे पुढे किंवा मागे जाण्यासाठी अतिरिक्त (मॅक ऑल्ट ऐवजी ऑप्शन की वापरा).
१५ १५ A selection must be present to apply the effect. प्रभाव लागू करण्यासाठी निवड असणे आवश्यक आहे.
१६ १६ You will hear a preview of applying the effect to the selected track(s). Unselected tracks will not have the effect applied to them but unlike in regular effects, unselected tracks will still be heard unless you use the Mute/Solo state to silence them. आपण निवडलेल्या गीतपट्टावर प्रभाव लागू करण्याचे पूर्वावलोकन ऐकू येईल. निवडलेल्या गीतपट्टावर त्यांच्यावर प्रभाव लागू होणार नाही परंतु नियमित प्रभावाच्या विपरीत, निवड न केलेले गीतपट्टा अद्याप आपण शांत किंवा मौन / निःशब्द स्थितीचा वापर न करता ऐकल्या जातील.
१७ १७ The effect may be applied without closing the effect. प्रभाव बंद केल्याशिवाय लागू होऊ शकतो.
१८ १८ User presets may be saved and loaded. As long as you apply the effect, changes you make to an effect's settings will be restored for use next time you launch Audacity. वापरकर्ता पुर्वनिर्धारित जतन आणि लोड केले जाऊ शकतात. जोपर्यंत आपण प्रभाव लागू करेपर्यंत आपण प्रभावाच्या समायोजनमध्ये केलेले बदल आपण पुढच्या वेळी ओड्यासिटी लॉन्च करण्यासाठी वापरासाठी पुनर्संचयित केले जातील.
१९ १९ Real-Time preview with graphical interface ग्राफिकल संवादपटलसह रिअल-वेळ पूर्वावलोकन
२० २० When you open an effect that supports real-time preview with graphical interface it will have these control icons at the bottom of the effect dialog. जेव्हा आपण ग्राफिकल संवादपटलसह रिअल-वेळ पूर्वावलोकनास समर्थन देणारा प्रभाव उघडता तेव्हा प्रभाव संवादच्या तळाशी ही नियंत्रण चिन्हे असतील.

Select one of the user presets to load it into the effect. वापरकर्त्याचे पुर्वनिर्धारित: या परिणामासाठी कोणतेही वापरकर्ता पुर्वनिर्धारित अस्तित्वात असल्यास हे मेनू त्यांना सूचीबद्ध करेल. तो प्रभाव मध्ये लोड करण्यासाठी वापरकर्त्याच्या पुर्वनिर्धारितपैकी एक निवडा.
२३ २३ Factory Presets: If any factory presets exist for this effect this menu will list them. Select one of the factory presets to load it into the effect. फॅक्टरी पुर्वनिर्धारित: या कारणासाठी कोणतेही फॅक्टरी पुर्वनिर्धारित अस्तित्वात असल्यास हे मेनू त्यांना सूचीबद्ध करेल. कारखाना पुर्वनिर्धारित मध्ये एक तो प्रभाव मध्ये लोड करण्यासाठी निवडा.
२४ २४ Delete Preset: Select this option to delete a previously saved user preset. पुर्वनिर्धारित हटवा: पूर्वी जतन केलेला वापरकर्ता पुर्वनिर्धारित हटविण्यासाठी हा पर्याय निवडा.

२६ २६ Import... Select this option to import a presets file (or multiple files for Audio Unit effects on Mac). आयात करा ... पुर्वनिर्धारित धारिका (किंवा मॅकवरील ध्वनी युनिट प्रभावांसाठी एकाधिक धारिका्स) आयात करण्यासाठी हा पर्याय निवडा.
२७ २७ Export... Select this option to export a presets file (or multiple files for Audio Unit effects on Mac). Export lets you share presets for this effect with other users on your computer or to move them to another computer. निर्यात करा ... पुर्वनिर्धारित धारिका (किंवा मॅकवरील ध्वनी युनिट प्रभावांसाठी एकाधिक धारिका्स) निर्यात करण्यासाठी हा पर्याय निवडा. निर्यात आपल्याला या परिणामाच्या पुर्वनिर्धारित आपल्या संगणकावरील अन्य वापरकर्त्यांसह सामायिक करू देते किंवा त्या दुसर्‍या संगणकावर हलवू देते.

Please click the links in the Options... section below to see a full explanation of presets features for each type of real-time preview effect. बरेच व्हीएसटी प्रभाव एकाच एफएक्सबी धारिकामध्ये नामित फॅक्टरी पुर्वनिर्धारितच्या आयात आणि निर्यातीला समर्थन देतात, परंतु एलएडीएसपीए आणि एलव्ही २ प्रभाव पुर्वनिर्धारितच्या आयात आणि निर्यातीला समर्थन देत नाहीत. प्रत्येक प्रकारच्या रीअल-वेळ पूर्वावलोकन प्रभावासाठी पुर्वनिर्धारितच्या वैशिष्ट्यांचे संपूर्ण स्पष्टीकरण पाहण्यासाठी कृपया खालील पर्याय ... मधील दुव्यांवर क्लिक करा.

३७ ३७ If you stop playback inside a selection with this button then do want to restart playback from the start of the selection, either click the "Skip Backward" button (see below) followed by the real-time preview Play button, or use the Play button in Transport Toolbar in the main project window. आपण या बटणासह निवडीमध्ये प्लेबॅक थांबविल्यास निवड सुरू झाल्यापासून प्लेबॅक रीस्टार्ट करू इच्छित असल्यास, रीअल-वेळ पूर्वावलोकन प्ले बटणा नंतर "मागे जा" वगळा (खाली पहा) वर क्लिक करा किंवा प्ले बटण वापरा मुख्य प्रकल्प विंडोमध्ये परिवहन साधनपट्टीमध्ये.
३८ ३८ Skip Backward: When playback is stopped, clicking this button will cause the next playback in the effect to start from the start of the selection. During playback, each click of this button will perform a short seek back. You cannot seek back behind the editing cursor or behind the start of the selection. पाठीमागे जा: प्लेबॅक थांबविल्यावर, या बटणावर क्लिक केल्याने प्रभावातील पुढील प्लेबॅक निवड सुरू होण्यापासून सुरू होईल. प्लेबॅक दरम्यान, या बटणाचे प्रत्येक क्लिक एक लहान शोध मागे घेईल. आपण संपादन कर्सरच्या मागे किंवा निवडीच्या प्रारंभाच्या मागे शोधू शकत नाही.
३९ ३९ Skip Forward: When playback is stopped, clicking this button will cause the next playback in the effect to start from the end of the selection. You can use this to compare subsequent audio which you may not want to apply the effect to. During playback, each click of this button will perform a short seek forward. पुढे जा: प्लेबॅक थांबविल्यावर, या बटणावर क्लिक केल्याने परिणामामधील पुढील प्लेबॅक निवडीच्या शेवटीपासून सुरू होईल. आपण याचा वापर त्यानंतरच्या ध्वनीशी तुलना करण्यासाठी करू शकता ज्यावर आपण प्रभाव लागू करू इच्छित नाही. प्लेबॅक दरम्यान, या बटणाचे प्रत्येक क्लिक एक लहान शोध पुढे करेल.
४० ४० Enable: This checkbox contains a checkmark by default  Enable, which enables the effect. सक्षम करा: या चेकबॉक्समध्ये पूर्वनिर्धारित सक्षम करून चेकमार्क आहे सक्षम करा, जो प्रभाव सक्षम करतो.
४१ ४१ Remove the checkmark  Enable to disable the effect (also known as bypassing the effect). Disabling makes the audio play as if the effect was closed and the effect's controls will no longer change the audio. Disabling and enabling is a convenient way to compare the "dry" (non-affected) sound with the "wet" (affected) sound while it plays in real-time. प्रभाव अक्षम करण्यासाठी चेकमार्क सक्षम करा (प्रभाव स्थलांतरित म्हणून देखील ओळखले जाते). अक्षम केल्यामुळे ध्वनी प्ले जणू प्रभाव बंद झाला आहे आणि परिणामाची नियंत्रणे यापुढे ध्वनी बदलणार नाहीत. रिअल-वेळमध्ये प्ले होत असताना "कोरडे" (प्रभावित नसलेले) ध्वनी "ओले" (प्रभावित) ध्वनीशी तुलना करण्याचा अक्षम करणे आणि सक्षम करणे हा एक सोपा मार्ग आहे.
४२ ४२ Close Click this button to close the effect dialog. Closing the effect without pressing "Apply" (below) does not save the current settings of the effect for use next time you launch Audacity. बंद करा प्रभाव संवाद बंद करण्यासाठी या बटणावर क्लिक करा. "लागू करा" (खाली) दाबल्याशिवाय प्रभाव बंद केल्याने आपण पुढच्या वेळी ओड्यासिटी लॉन्च कराल तेव्हा परिणामाची वर्तमान समायोजन जतन होणार नाही.

The effect dialog will remain open after the effect is applied. लागू करा निवडलेल्या ध्वनीवर प्रभाव लागू करण्यासाठी या बटणावर क्लिक करा आणि पुढील वेळी आपण ओड्यासिटी लॉन्च कराल तेव्हा परिणामाच्या वर्तमान समायोजन जतन करा. प्रभाव लागू झाल्यानंतर प्रभाव संवाद खुले राहील.

५१ ५१ Notes and Limitations नोट्स आणि मर्यादा
५२ ५२ Currently, real-time preview does not compensate for the latency many plug-ins introduce in order to provide smooth processing of audio. Real-time preview therefore may have small gaps in the audio, or timing artifacts may be audible when previewing multiple tracks. However by default Audacity compensates for latency when applying the effect (unless you disable compensation in the options for the effect). Therefore if you hear latency problems in real-time preview, apply the effect and close it, then listen to the result. If you do not like the result you can simply use the Edit > Undo command, or its shortcut Ctrl + Z (on Mac ⌘ + Z), open the effect and try a different setting. सध्या, रिअल-वेळ पूर्वावलोकन ध्वनीची सहज प्रक्रिया करण्यासाठी अनेक प्लगइन सुरू केलेल्या विलंबपणाची भरपाई करीत नाहीत. रीअल-वेळ पूर्वावलोकन म्हणून ध्वनीमध्ये लहान अंतर असू शकतात किंवा एकाधिक गीतपट्टाचे पूर्वावलोकन करताना वेळेच्या कलाकृती ऐकण्यायोग्य असू शकतात. तथापि प्रभाव लागू करताना पूर्वनिर्धारित ओड्यासिटी विलंबपणाची भरपाई करते (जोपर्यंत आपण प्रभावाच्या पर्यायांमध्ये नुकसान भरपाई अक्षम करत नाही). म्हणूनच आपण रिअल-वेळ पूर्वावलोकनात उशीरा अडचणी ऐकल्यास प्रभाव लागू करा आणि बंद करा, नंतर निकाल ऐका. आपल्याला निकाल आवडत नसल्यास आपण केवळ संपादन> पूर्ववत करा आज्ञा किंवा त्याचा शॉर्टकट Ctrl + Z (मॅक Z + झेड वर) वापरू शकता, प्रभाव उघडा आणि भिन्न सेटिंग वापरून पहा.
५३ ५३ So that real-time preview plays how the audio will sound immediately after applying the effect, any existing volume envelopes or Time Tracks and the positions of gain or pan sliders are applied before preview is heard. म्हणूनच रिअल-वेळ पूर्वावलोकन प्रभाव लागू झाल्यानंतर ध्वनी कसा आवाज येईल हे प्ले करते, कोणतेही विद्यमान आवाज लिफाफे किंवा वेळ गीतपट्टा आणि गेन किंवा पॅन स्लाइडरची स्थिती पूर्वावलोकन ऐकण्यापूर्वी लागू केली जाते.
