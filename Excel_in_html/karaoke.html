४ ४ The karaoke feature shows a bouncing ball that follows along text as audio plays. कराओके वैशिष्ट्य एक बाउन्सिंग बॉल दर्शविते जो मजकूरासह ध्वनी प्ले म्हणून अनुसरण करतो.
५ ५ Accessed by: View > Karaoke... यावर प्रवेश: पहा> कराओके ...
६ ६ The Karaoke... command is enabled whenever you have at least one label track. If you have multiple label tracks, it uses only the first one. जेव्हा आपल्याकडे कमीतकमी एक लेबल गीतपट्टा असेल तेव्हा कराओके ... आज्ञा सक्षम केली जाते. आपल्याकडे एकाधिक लेबल गीतपट्टा असल्यास ते फक्त प्रथमच वापरते.
७ ७ The command brings up the Karaoke window, which displays the labels in a "bouncing ball" scrolling display. You can select a playback start point in the main tracks window and the Karaoke window will start from there. आज्ञा कराओके विंडो आणते, जी लेबले "बाउंसिंग बॉल" स्क्रोलिंग डिस्प्लेमध्ये दाखवते. आपण मुख्य गीतपट्टा विंडोमध्ये प्लेबॅक प्रारंभ बिंदू निवडू शकता आणि तेथून कराओके विंडो सुरू होईल.
८ ८ Audacity automatically puts a space between words, so you do not need to put them in your labels. धडधड स्वयंचलितपणे शब्दांदरम्यान एक स्थान ठेवते, म्हणून आपल्याला ती आपल्या लेबलांमध्ये ठेवण्याची आवश्यकता नाही.
९ ९ You may want to split up lyrics such that the ball bounces per syllable rather than just for the whole word. To do this, create a label for each syllable, and end each with a hyphen ("-"), except the last syllable. For example, "Aud-" "a-" "ci-" "ty". The hyphen tells Audacity to not put a space between the syllables, as it does between words. Note there may still be space between the syllables, depending on how far apart they are in the timeline and the size of the Karaoke window. आपल्याला केवळ संपूर्ण शब्दाऐवजी बॉल एका अक्षरेनुसार बाउन्स करतात इतके गीत विभाजित करू शकतात. हे करण्यासाठी, प्रत्येक अक्षरासाठी एक लेबल तयार करा आणि शेवटच्या अक्षरेखेरीज प्रत्येकास हायफन ("-") सह समाप्त करा. उदाहरणार्थ, "ऑड-" "ए-" "सीआय-" "टाय". हायफन ओड्यासिटीला शब्दांदरम्यान अक्षरे दरम्यान जागा न ठेवण्यास सांगते. लक्षात घ्या की अक्षरे आणि कराओके विंडोच्या आकारात किती अंतर आहेत यावर अवलंबून अक्षरे दरम्यान अद्याप जागा असू शकते.
