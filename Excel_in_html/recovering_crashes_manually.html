४ ४ If Audacity crashes or the computer loses power, Audacity should normally recover the projects automatically when you restart. The steps on this page describe how to recover audio from the project's _data folder or Audacity's temporary folder if: जर ओड्यासिटी क्रॅश झाली किंवा संगणकाची शक्ती गमावली तर आपण पुन्हा सुरू केल्यावर ओड्यासिटी सहसा प्रकल्प आपोआप पुनर्प्राप्त करा. या पृष्ठावरील चरणांमध्ये वर्णन केले आहे की प्रकल्पच्या _डेटा फोल्डर किंवा ओड्यासिटीच्या तात्पुरत्या फोल्डरमधून ध्वनी पुनर्प्राप्त कसे करावे:
५ ५ Automatic Crash Recovery does not work correctly स्वयंचलित क्रॅश पुनर्प्राप्ती योग्यरित्या कार्य करत नाही
६ ६ You saved a project without apparent crash, but Audacity was shut down before the AUP project file was written or you do not have a usable AUP file. आपण एखादा प्रकल्प क्रॅश न करता जतन केला, परंतु एयूपी प्रकल्प धारिका लिहिण्यापूर्वी ओड्यासिटी बंद केली गेली किंवा आपल्याकडे वापरण्यायोग्य एयूपी धारिका नाही.
७ ७ The steps on this page will only recover unedited recordings (and may recover stereo recordings with some sections of left and right channel in the wrong channel). या पृष्ठावरील चरणांमुळे केवळ सुधारित ध्वनीमुद्रणिंग पुनर्प्राप्त होईल (आणि चुकीच्या वाहिनीमधील डाव्या आणि उजव्या वाहिनीच्या काही विभागांसह स्टीरिओ ध्वनीमुद्रणिंग पुनर्प्राप्त करू शकतात).
८ ८ If the Automatic Crash Recovery dialog has appeared but recovery is incorrect, do not close the project or save any changes until manual recovery using these instructions is complete. स्वयंचलित क्रॅश पुनर्प्राप्ती संवाद आढळल्यास परंतु पुनर्प्राप्ती चुकीची असल्यास, या सूचनांचा वापर करून व्यक्तिचलित पुनर्प्राप्ती पूर्ण होईपर्यंत प्रकल्प बंद करू नका किंवा कोणतेही बदल जतन करू नका.
९ ९ If you need to shut down the computer before manually fixing an incorrect Automatic Crash Recovery, force quit Audacity in the system task manager. This will preserve the temporary data for next launch of Audacity. अयोग्य स्वयंचलित क्रॅश पुनर्प्राप्ती व्यक्तिचलितपणे निराकरण करण्यापूर्वी आपल्याला संगणक बंद करण्याची आवश्यकता असल्यास, सिस्टम टास्क मॅनेजरमध्ये ओड्यासिटी सोडा. हे ओड्यासिटीच्या पुढील लाँचसाठी तात्पुरते डेटाचे जतन करेल.
१० १० If you had successfully saved the project just before the crash, it may be simpler to discard the unsaved project in the Automatic Crash Recovery dialog then open the saved AUP file. आपण क्रॅश होण्यापूर्वी हा प्रकल्प यशस्वीरित्या जतन केला असेल तर स्वयंचलित क्रॅश पुनर्प्राप्ती संवादात जतन न केलेला प्रकल्प टाकणे सोपे असेल तर जतन केलेली AUP धारिका उघडा.

२२ २२ Location of the project _data folder प्रकल्पाचे स्थान _डेटा फोल्डर
२३ २३ If you saved an AUP project file, the _data folder you need to recover from will be in the directory to which you originally saved the AUP file. आपण एक एयूपी प्रकल्प धारिका जतन केली असल्यास, आपणास पुनर्प्राप्त करणे आवश्यक आहे _डेटा फोल्डर आपण ज्या एयूपी धारिकाची मूळतः जतन केली त्या निर्देशिकेत असेल.
२४ २४ Location of Temporary Folder तात्पुरते फोल्डरचे स्थान
२५ २५ If you never saved an AUP project file, you need to recover from Audacity's temporary folder. You can see the exact location in Directories Preferences, otherwise you can see the location in the "TempDir" line in the audacity.cfg settings file. By default the Audacity temporary folder should be as follows. आपण कधीही AUP प्रकल्प धारिका जतन केली नसेल तर आपल्याला ओड्यासिटीच्या तात्पुरत्या फोल्डरमधून पुनर्प्राप्त करण्याची आवश्यकता आहे. आपण निर्देशिका प्राधान्यांमधील अचूक स्थान पाहू शकता, अन्यथा आपण ओड्यासिटी.सीएफजी समायोजन धारिकामध्ये "टेम्पडिअर" लाइनमध्ये स्थान पाहू शकता. पूर्वनिर्धारितनुसार ओड्यासिटी तात्पुरते फोल्डर खालीलप्रमाणे असावे.

३० ३० In order to see the default location of the Audacity temp folder on Windows or Mac you need to show hidden files and folders or type the folder location into your file manager's address bar. विंडोज किंवा मॅकवरील ओड्यासिटी टेम्प फोल्डरचे पूर्वनिर्धारित स्थान पाहण्यासाठी आपल्याला लपविलेले धारिका आणि फोल्डर्स दर्शविणे आवश्यक आहे किंवा आपल्या धारिका व्यवस्थापकाच्या अ‍ॅड्रेस बारमध्ये फोल्डरचे स्थान टाइप करावे लागेल.

Otherwise, see how to show hidden files and folders. विंडोजः एक्सप्लोररच्या डावीकडील झाडावर, "वापरकर्ते" वर डबल क्लिक करा त्यानंतर उजवीकडे, Dपडाटा किंवा अनुप्रयोग डेटा फोल्डर उघडा, नंतर "स्थानिक" किंवा "लोकॅल समायोजन" फोल्डर आणि शेवटी "टेम्प" फोल्डर. आपण ते फोल्‍डर पाहू शकत नसल्यास, आपण एक्सप्लोरर अ‍ॅड्रेस बारमध्ये% लोकलॅपडाटा% टाइप करू शकता आणि नंतर "टेम्प" फोल्डर शोधा. अन्यथा, लपवलेल्या धारिका आणि फोल्डर्स कसे दर्शवायचे ते पहा.

३७ ३७ Manual recovery व्यक्तिचलित पुनर्प्राप्ती
३८ ३८ If you only have a few AU files to recover, use the "Import Audio" command to open all the .au files from the Audacity temporary folder or project _data folder. आपल्याकडे पुनर्प्राप्त करण्यासाठी फक्त काही एयू धारिका असल्यास, ओड्यासिटी तात्पुरते फोल्डर किंवा प्रकल्प _डेटा फोल्डरमधून सर्व .au धारिका उघडण्यासाठी "ध्वनी आयात करा" आदेश वापरा.
३९ ३९ Use shift-click or control-click to select multiple files. एकाधिक धारिका निवडण्यासाठी शिफ्ट-क्लिक किंवा नियंत्रण-क्लिक वापरा.
४० ४० The files will be in approximately ६-second chunks and will be on separate tracks in the Audacity screen. धारिका अंदाजे ६-सेकंद भागांमध्ये असतील आणि ओड्यासिटी स्क्रीनमध्ये वेगळ्या गीतपट्टावर असतील.
४१ ४१ Files from stereo tracks will alternate between left and right channels. स्टीरिओ गीतपट्टावरील धारिका डाव्या आणि उजव्या वाहिनी दरम्यान वैकल्पिक बनतील.
४२ ४२ Click in the Track Control Panel (by the Mute/Solo buttons) of the topmost track. सर्वात वरच्या गीतपट्टाच्या गीतपट्टा नियंत्रण पॅनेलमध्ये (निःशब्द / एकल बटणाद्वारे) क्लिक करा.
४३ ४३ Shortcut Z or Select > Region > Ends to Zero Crossings. आडमार्ग झेड किंवा निवडा> क्षेत्र> शून्य क्रॉसिंग्जवर समाप्त.
४४ ४४ Edit > Remove Special > Trim. संपादित करा> विशेष> ट्रिम काढा.
४५ ४५ Click in the Track Control Panel of the first track you want to join to the topmost track. आपण सर्वात वरच्या गीतपट्टावर सामील होऊ इच्छित असलेल्या पहिल्या गीतपट्टाच्या गीतपट्टा नियंत्रण पॅनेलवर क्लिक करा.
४६ ४६ Shortcut Z or Edit > Find Zero Crossings. आडमार्ग झेड किंवा संपादन> शून्य क्रॉसिंग शोधा.
४७ ४७ Shortcut Z or Select > Region > Ends to Zero Crossings. आडमार्ग झेड किंवा निवडा> क्षेत्र> शून्य क्रॉसिंग्जवर समाप्त.
४८ ४८ Click after the end of the topmost track. सर्वात वरच्या गीतपट्टाच्या शेवटी क्लिक करा.
४९ ४९ Press End on the keyboard. कळफलक वरील एंड दाबा.
५० ५० Click Edit > Paste and the cut track will attach to the end of the topmost track. संपादन> पेस्ट वर क्लिक करा आणि कट गीतपट्टा सर्वात वरच्या गीतपट्टाच्या शेवटी जोडेल.
५१ ५१ Repeat as necessary to join all the tracks onto the topmost track. सर्वात वरच्या गीतपट्टावर सर्व गीतपट्टामध्ये सामील होण्यासाठी आवश्यकतेनुसार पुन्हा करा.
५२ ५२ Play the result. निकाल प्ले.
५३ ५३ Cut and paste as needed if any files are found to be in the wrong order or the wrong channel. कोणत्याही धारिका चुकीच्या क्रमाने किंवा चुकीच्या वाहिनीमध्ये असल्याचे आढळल्यास आवश्यकतेनुसार कट आणि पेस्ट करा.
५४ ५४ Automatic recovery tools स्वयंचलित पुनर्प्राप्ती साधने
५५ ५५ Tools written for legacy Audacity १.२ can automate data recovery. लेगसी ओड्यासिटी १.२ साठी लिहिलेली साधने डेटा पुनर्प्राप्ती स्वयंचलित करू शकतात.
५६ ५६ All these utilities require the .au files to be input in consecutive alphanumerical order. Audacity names files randomly so files will need to renamed. या सर्व उपयुक्ततांसाठी .au धारिका सलग अक्षरांक क्रमाने इनपुट असणे आवश्यक आहे. ओड्यासिटी धारिका्सना यादृच्छिकरित्या धारिका्स म्हणतात त्यामुळे धारिका्सचे नाव बदलणे आवश्यक आहे.

Sort the files by timestamp order (earliest first), then rename them using a consecutive alphanumerical sequence, lowest number first.
