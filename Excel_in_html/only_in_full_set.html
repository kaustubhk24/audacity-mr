४ ४ When you see a shortcut labeled like Ctrl + J जेव्हा आपण Ctrl + J सारख्या लेबल असलेले आडमार्ग पहाल
५ ५ Extra i.e. with the word 'Extra' as a superscript, it means that the shortcut is only available if you choose the full set of shortcuts. अतिरिक्त म्हणजे सुपरस्ट्राफ्ट म्हणून 'एक्स्ट्रा' या शब्दाचा अर्थ असा आहे की आपण आडमार्गचा पूर्ण सेट निवडल्यासच आडमार्ग उपलब्ध होईल.
६ ६ The standard default set of shortcuts does not include it आडमार्गच्या मानक पूर्वनिर्धारित सेटमध्ये हे समाविष्ट नाही
७ ७ Defaults: There are two sets of default shortcuts available in Audacity. पूर्वनिर्धारितः ओड्यासिटीमध्ये पूर्वनिर्धारित आडमार्गचे दोन संच उपलब्ध आहेत.
८ ८ The default set is "Standard". पूर्वनिर्धारित संच "मानक" आहे.
९ ९ This set is a reduced set to simplify the set of shortcuts somewhat and to provide greater flexibilty for users who wish to create their own shortcuts. आडमार्गचा संच काहीसा सुलभ करण्यासाठी आणि स्वत: चा आडमार्ग तयार करू इच्छिणा users्या वापरकर्त्यांना जास्त फ्लेक्सबिलिटी प्रदान करण्यासाठी हा सेट कमी केलेला संच आहे.
१० १० The other available set of shortcuts is "Full". आडमार्गचा इतर उपलब्ध संच "पूर्ण" आहे.
११ ११ This set is, as the name suggests the full and extensive set of shortcuts that were available in Audacity २.१.३ and earlier. हा सेट आहे, जसे नावाने ओड्यासिटी २.१..३ आणि पूर्वीच्या काळात उपलब्ध आडमार्गचा पूर्ण आणि विस्तृत सेट सूचित केला आहे.
१२ १२ You can choose to revert to the full set of shortcuts by selecting "Full" from the dropdown menu accessed from the Defaults button in the Keyboard Preferences dialog. आपण कीबोर्ड प्राधान्ये संवादातील पूर्वनिर्धारित बटणामधून प्रवेश केलेल्या ड्रॉपडाउन मेनूमधून "पूर्ण" निवडून आडमार्गच्या पूर्ण संचावर परत जाणे निवडू शकता.
१३ १३ You can use the Defaults button to switch between the two provided default sets of shortcuts at any time. आडमार्गच्या दोन प्रदान केलेल्या पूर्वनिर्धारित सेट दरम्यान कधीही स्विच करण्यासाठी आपण पूर्वनिर्धारित बटण वापरू शकता.
१४ १४ This button is also handy if you are making custom shortcuts and decide you have not got your changes right and want to start over again. आपण सानुकूल आडमार्ग बनवत असल्यास आणि आपले बदल योग्य न झाल्याचे आणि पुन्हा प्रारंभ करू इच्छित असल्यास हे बटण सुलभ आहे.
