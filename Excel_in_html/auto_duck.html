१ १ Auto Duck ऑटो डक
२ २ From Audacity Development Manual ओड्यासिटी विकास माहितीपुस्तिकेकडून
३ ३ Jump to: navigation, search येथे जा: निर्देशक, शोध
४ ४ Auto Duck reduces (ducks) the volume of one or more selected tracks whenever the volume of a single unselected "control track" placed underneath reaches a particular threshold level. The effect can be used to create voice-overs for podcasts or DJ sets, for automatic "ramping" of background music in radio productions and for turning down a voice in original language as soon as its translation kicks in. खाली ठेवलेल्या एकाच निवडलेल्या "कंट्रोल गीतपट्टा" ची मात्रा जेव्हा विशिष्ट उंबरठा स्तरावर पोहोचते तेव्हा ऑटो डक एक किंवा अधिक निवडलेल्या गीतपट्टाचा आवाज कमी करते (बदके). याचा उपयोग पॉडकास्ट किंवा डीजे सेटसाठी व्हॉईस-ओव्हर्स तयार करण्यासाठी, रेडिओ प्रॉडक्शनमध्ये पार्श्वभूमी संगीताच्या स्वयंचलित "रॅम्पिंग" आणि भाषांतर सुरू होताच मूळ भाषेत व्हॉइस डाउन करण्यासाठी केला जाऊ शकतो.
५ ५ Accessed by: Effect > Auto Duck... यावर प्रवेशः प्रभाव> ऑटो डक ...
६ ६ Preparation तयारी
७ ७ Select the track(s) or region(s) whose volume you want to modify (for example, a background music track). ज्याचा आवाज आपण सुधारित करू इच्छिता त्याचा गीतपट्टा किंवा प्रदेश निवडा (उदाहरणार्थ, पार्श्वभूमी संगीत गीतपट्टा).
८ ८ Deselect the track that is to act as the control track (for example, a spoken commentary). Note that the first unselected track underneath the selected track(s) acts as the control track. नियंत्रण गीतपट्टा म्हणून कार्य करण्यासाठी गीतपट्टाची निवड रद्द करा (उदाहरणार्थ, स्पोकन टीका). लक्षात घ्या की निवडलेल्या गीतपट्टाखालील प्रथम निवडलेला गीतपट्टा नियंत्रण गीतपट्टा म्हणून कार्य करतो.
९ ९ Place the tracks so that at least one of the selected tracks to be modified is above the unselected control track. Any other tracks below the unselected control track will also be modified if they are selected. You can use the Audio Track Dropdown Menu to move tracks up or down if needed. गीतपट्टा ठेवा जेणेकरून सुधारित करण्यासाठी निवडलेल्या किमान एक गीतपट्टा न निवडलेल्या नियंत्रण गीतपट्टाच्या वर असेल. न निवडलेले नियंत्रण गीतपट्टा खालील इतर कोणतेही गीतपट्टा निवडल्यास ते सुधारित केले जातील. आवश्यक असल्यास गीतपट्टा वर किंवा खाली हलविण्यासाठी आपण ध्वनी गीतपट्टा ड्रॉपडाउन मेनू वापरू शकता.
१0 १0 The control track should be properly synchronized with the tracks to be "ducked" - you can use Time Shift Tool to align the tracks with each other as required. कंट्रोल गीतपट्टा गीतपट्टा बरोबर "डक्ड" होण्यासाठी योग्यरित्या समक्रमित केला पाहिजे - आपण आवश्यकतेनुसार गीतपट्टा संरेखित करण्यासाठी वेळ शिफ्ट साधन वापरू शकता.
११ ११ Choose "Auto Duck" from the Effect Menu. प्रभाव मेनूमधून "ऑटो डक" निवडा.
१२ १२ Auto Duck analyzes the control track first of all, then applies the effect to the selected track(s). This second stage takes longer to complete than the first, so the estimated time remaining in the progress bar may be an underestimate until the later stages of processing. ऑटो डक सर्वप्रथम कंट्रोल गीतपट्टाचे विश्लेषण करते, त्यानंतर निवडलेल्या गीतपट्टावर प्रभाव लागू करते. हा दुसरा टप्पा पहिल्यापेक्षा पूर्ण होण्यास अधिक वेळ घेईल, म्हणून प्रक्रियेच्या नंतरच्या टप्प्यांपर्यंत प्रगती बारमध्ये उर्वरित अंदाजे वेळ कमी लेखू शकतो.
१३ १३ Graph and Parameters ग्राफ आणि पॅरामीटर्स
१४ १४ All the effect parameters can be set by inputting values in the boxes. The graph above the input boxes is a convenient visual representation of the changes made to the volume of the ducked track when Auto Duck takes effect. If preferred, you can modify the Duck amount and Fade Down and Fade Up lengths by dragging the "control points" (the small white circles) on the graph. Simply hover over a control point with the mouse and drag in the direction indicated by the cursor arrows. सर्व प्रभाव मापदंड बॉक्समधील मूल्ये इनपुट करुन सेट केले जाऊ शकतात. इनपुट बॉक्स वरील आलेख हे ऑटो डक प्रभावी झाल्यावर डिक गीतपट्टाच्या व्हॉल्यूममध्ये झालेल्या बदलांचे एक सोयीस्कर दृश्य प्रतिनिधित्व आहे. प्राधान्य दिल्यास, आपण आलेखवरील "कंट्रोल पॉइंट्स" (लहान पांढरे मंडळे) ड्रॅग करून डक रक्कम आणि फीड डाउन आणि फॅड अप लांबी सुधारित करू शकता. फक्त माउससह नियंत्रण बिंदूवर फिरवा आणि कर्सर बाणांनी दर्शविलेल्या दिशेने ड्रॅग करा.
१५ १५ Duck Amount: The amount (default: -१२ dB) of volume reduction in the selected track when a signal above the threshold level is detected बदकाची रक्कमः उंबरठा स्तरावरील सिग्नल आढळल्यास निवडलेल्या गीतपट्टामध्ये आवाज कपातची रक्कम (पूर्वनिर्धारित: -१२ डीबी)
१६ १६ Maximum Pause: Avoids over-rapid volume changes by setting a length of time following the ramp down that must be exceeded before the volume is ramped up again. The default value is १ second, but irrespective of the setting, volume will not ramp up unless the pause is at least as long as the fade down length plus the fade up length. कमाल विराम द्या: उताराचा मागोवा खाली उतरत जास्तीत जास्त वेगवान व्हॉल्यूममध्ये होणारे बदल टाळतात ज्याचा आवाज पुन्हा वाढण्यापूर्वी ओलांडणे आवश्यक आहे. पूर्वनिर्धारित मूल्य १ सेकंद आहे, परंतु सेटिंगची पर्वा न करता, विराम कमीतकमी जोपर्यंत फेड डाउन लांबी आणि फॅड अप लांबीपर्यंत विराम देत नाही तोपर्यंत खंड वाढणार नाही.
१७ १७ Fade Down Lengths: The length of time over which the selected tracks fade down to the ducked volume (default: 0.५ seconds). फिकट डाउन लांबी: निवडक गीतपट्टा डिक केलेल्या आवाज (पूर्वनिर्धारित: 0.५ सेकंद) पर्यंत कमी होण्याच्या कालावधीची लांबी.
१८ १८ Fade Up Lengths: The length of time over which the selected tracks fade up from the ducked volume (default: 0.५ seconds) फिकट लांबी: निवडलेल्या गीतपट्टा डाक केलेल्या व्हॉल्यूममधून किती काळ संपतात (पूर्वनिर्धारित: 0.५ सेकंद)
१९ १९ The "outer" and "inner" fade lengths both adjust the perceived speed of the fade. In effect, the inner length controls the length of time the ducked track is attenuated by the full Duck amount. If you play with the fade lengths you will be able to see this in the graph. "बाह्य" आणि "आतील" फीड लांबी दोन्ही फीडच्या अपेक्षित वेग समायोजित करतात. प्रत्यक्षात, डक गीतपट्टा पूर्ण डक रकमेद्वारे क्षमतेच्या वेळेची लांबी आतील लांबी नियंत्रित करते. जर आपण फिकट लांबीने खेळत असाल तर आपल्याला हे आलेखामध्ये दिसण्यात सक्षम असेल.
२0 २0 Threshold: The level (default: -३0 dB) used for detecting a signal in the control track. उंबरठा: नियंत्रण गीतपट्टामधील सिग्नल शोधण्यासाठी वापरलेले स्तर (पूर्वनिर्धारित: -३0 डीबी).
