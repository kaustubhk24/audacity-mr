१ १ Regular Interval Labels नियमित अंतराल लेबले

४ ४ Regular Interval Labels places point labels in a label track so as to divide the associated audio into smaller, equally-sized segments. This can be useful for distributing a large file on the internet or ensuring that all exported files are the same size. You can either choose the number of labels to be created or the interval between them. Each label produced can contain specified label text and the labels can be given sequential numbers before or after the text. नियमित अंतराल लेबले लेबल गीतपट्टामध्ये पॉइंट लेबले ठेवतात जेणेकरून संबंधित ध्वनी लहान, समान-आकारातील विभागांमध्ये विभाजित करता येईल. इंटरनेटवर मोठी धारिका वितरित करण्यासाठी किंवा सर्व निर्यात केलेल्या धारिका समान आकाराच्या आहेत हे सुनिश्चित करण्यासाठी हे उपयुक्त ठरेल. आपण एकतर तयार केलेली लेबलची संख्या किंवा त्या दरम्यान मध्यांतर निवडू शकता. प्रत्येक उत्पादित लेबलमध्ये निर्दिष्ट लेबल मजकूर असू शकतो आणि लेबल मजकूराच्या आधी किंवा नंतर अनुक्रमांक दिले जाऊ शकतात.
५ ५ Accessed by: Tools > Regular Interval Labels... याद्वारे प्रवेश केलेलेः साधने> नियमित अंतराल लेबले ...
६ ६ After running "Regular Interval Labels", you can choose File > Save Other > Export Multiple... to export all the files in one process based on the label points. "नियमित अंतराल लेबले" चालवल्यानंतर, आपण लेबल पॉईंट्सच्या आधारावर सर्व धारिका एकाच प्रक्रियेमध्ये निर्यात करण्यासाठी धारिका> इतर जतन करा> एकाधिक निर्यात करा ... निवडू शकता.
७ ७ If you already have a label track and that is included in the selection that you make for the analysis then these labels will be added to your selected label track - possibly causing confusion with your existing labels. आपल्याकडे आधीपासूनच लेबल गीतपट्टा असल्यास आणि त्या विश्लेषणासाठी आपण केलेल्या निवडीमध्ये ते समाविष्ट केले असल्यास ही लेबले आपल्या निवडलेल्या लेबल गीतपट्टामध्ये जोडली जातील - शक्यतो आपल्या विद्यमान लेबलांसह गोंधळ होऊ शकेल.
८ ८ Workaround: Ensure you just select audio track(s) and include no label track in the selection for the analysis. कार्यवाही: आपण फक्त ध्वनी गीतपट्टा निवडल्याचे सुनिश्चित करा आणि विश्लेषणासाठी निवडीमध्ये कोणत्याही लेबल गीतपट्टाचा समावेश नाही.
९ ९ Use "Number of labels" OR "Label interval" "लेबलची संख्या" किंवा "लेबल मध्यांतर" वापरा
१० १० This enables you to choose the label placement method: हे आपल्याला लेबल प्लेसमेंट पद्धत निवडण्यास सक्षम करते:
११ ११ To create a chosen number of labels within the selection, select Number of Labels   from the dropdown (this is the default setting). निवडीमध्ये निवडलेल्या लेबलांची संख्या तयार करण्यासाठी ड्रॉपडाउन वरून लेबलची संख्या निवडा (ही पूर्वनिर्धारित समायोजन आहे).
१२ १२ To place labels within the selection a chosen distance apart, select Label Interval   from the dropdown then enter that distance in Label interval (seconds). निवडीमध्ये निवडलेल्या अंतराच्या अंतरावर लेबले ठेवण्यासाठी, ड्रॉपडाउनमधून लेबल अंतराल निवडा नंतर ते अंतर लेबल मध्यांतर (सेकंद) मध्ये प्रविष्ट करा.
१३ १३ Number of labels लेबलची संख्या
१४ १४ If you selected Number of Labels   (the default) then your specified number of labels (default is १०) will be created. The number can be changed by typing in the input box, the allowed range is between २ and १००० labels. आपण लेबलची संख्या (पूर्वनिर्धारित) निवडल्यास आपल्या निर्दिष्ट लेबलांची संख्या (पूर्वनिर्धारित १०) तयार केली जाईल. संख्या इनपुट बॉक्समध्ये टाइप करुन बदलली जाऊ शकते, अनुमत श्रेणी २ आणि १००० लेबल दरम्यान आहे.
१५ १५ Label interval लेबल मध्यांतर
१६ १६ If you selected Label Interval   then the labels will be placed at your specified interval within the selected audio. आपण लेबल मध्यांतर निवडल्यास निवडलेल्या ध्वनीमध्ये लेबले आपल्या निर्दिष्ट अंतराने ठेवल्या जातील.
१७ १७ The default interval is ६०.० seconds. The allowed range is between ०.००१ and ३६०० seconds. पूर्वनिर्धारित मध्यांतर ६०.० सेकंद आहे. परवानगी दिलेली श्रेणी ०.००१ आणि ३६०० सेकंद दरम्यान आहे.
१८ १८ Adjust label interval to fit length लांबी फिट करण्यासाठी लेबल मध्यांतर समायोजित करा
१९ १९ This only operates if you have chosen the Label interval placement method. Number of Labels आपण केवळ लेबल अंतराल नियुक्ती पद्धत निवडल्यास हे चालते. लेबलची संख्या
२० २० If you select Yes   from the dropdown for this control, it adjusts your chosen label interval if necessary to make all the audio segments of equal length, including the final segment between the last label and the end of the selection. आपण या नियंत्रणासाठी ड्रॉपडाऊनमधून येस निवडल्यास, शेवटच्या लेबल आणि निवडीच्या शेवटी असलेल्या शेवटच्या विभागासह समान लांबीचे सर्व ध्वनी विभाग तयार करणे आवश्यक असल्यास ते आपले निवडलेले लेबल मध्यांतर समायोजित करते.
२१ २१ When set to No  , the interval between labels will be exactly the length specified in the Label interval control, but the final segment from the last label to the end of the selection may be different (depending on whether the selection length is exactly divisible by the specified interval duration. नाही वर सेट केल्यावर लेबलांमधील मध्यांतर लेबल अंतराच्या नियंत्रणामध्ये निश्चित केलेली लांबी असेल, परंतु शेवटच्या लेबलपासून निवडीच्या शेवटापर्यंत अंतिम विभाग भिन्न असू शकतो (निवडीची लांबी नेमकी विभाजनीय आहे की नाही यावर अवलंबून) निर्दिष्ट अंतराल कालावधी.
२२ २२ When Number of Labels   is selected, the label interval will always be the length of the selection divided by the number of labels, irrespective of this setting. जेव्हा लेबलची संख्या निवडली जाते, तेव्हा लेबल मध्यांतर या समायोजनची पर्वा न करता लेबलच्या संख्येने विभाजित केलेल्या निवडीची लांबी नेहमीच असेल.
२३ २३ The default setting for this control is No  . या नियंत्रणासाठी पूर्वनिर्धारित समायोजन नाही.
२४ २४ Label text लेबल मजकूर
२५ २५ The text that will be included in each label. The default setting is Label but this can be replaced by any text (or no text) by using the input box. प्रत्येक लेबलमध्ये समाविष्ट केलेला मजकूर. पूर्वनिर्धारित समायोजन लेबल आहे परंतु इनपुट बॉक्स वापरुन हे कोणत्याही मजकूराद्वारे (किंवा कोणताही मजकूर नाही) पुनर्स्थित केले जाऊ शकते.
२६ २६ The label text may include spaces and/or punctuation characters, but note that if you intend to use the labels with Export Multiple, some characters may not be valid for file names. लेबल मजकूरामध्ये रिक्त स्थान आणि / किंवा विरामचिन्हे समाविष्ट असू शकतात परंतु लक्षात घ्या की आपण एकाधिक निर्यातसह लेबले वापरण्याचा विचार करीत असाल तर काही नावे धारिका नावांसाठी वैध नसू शकतात.
२७ २७ Minimum number of digits in label लेबलमध्ये अंकांची किमान संख्या
२८ २८ Choose the minimum number of number of digits in the label numbers and whether the number is placed before or after the label text. The default setting is २(before label)  . With that setting, if you had १० labels with "Label" as the label text, the first label would be "०१Label", the next "०२Label" and the last would be "१०Label". लेबल क्रमांकामधील अंकांची किमान संख्या आणि लेबल मजकूराच्या आधी किंवा नंतर संख्या घातली आहे की नाही ते निवडा. पूर्वनिर्धारित समायोजन २ असते (लेबलच्या आधी). त्या समायोजनसह, आपल्याकडे लेबल मजकूर म्हणून "लेबल" असलेली १० लेबल असल्यास, प्रथम लेबल "०१ लेबल", पुढील "०२ लेबल" आणि अंतिम "१० लेबल" असेल.
२९ २९ If you select None - text only   then no numbering will be applied to the labels. आपण काहीही नाही - मजकूर केवळ निवडल्यास लेबलवर क्रमांकिंग लागू केली जाणार नाही.
३० ३० If you choose this and set no label text then you will produce a set of empty labels. आपण हे निवडल्यास आणि कोणतेही लेबल मजकूर सेट न केल्यास आपण रिक्त लेबलचा संच तयार कराल.
३१ ३१ Begin numbering from पासून क्रमांकन प्रारंभ करा
३२ ३२ If a minimum number of digits has been chosen in the control above, enter the number from which the sequential numbering will start. The default value of १ can be changed by typing in the input box. You can enter any whole number equal or greater than ०. उपरोक्त नियंत्रणात अंकांची किमान संख्या निवडली असल्यास, अनुक्रमांक क्रमांकन सुरू होण्यास प्रारंभ होणारी संख्या प्रविष्ट करा. १ ची पूर्वनिर्धारित मूल्य इनपुट बॉक्समध्ये टाइप करुन बदलली जाऊ शकते. आपण कोणतीही पूर्ण संख्या ० किंवा त्यापेक्षा मोठी प्रविष्ट करू शकता.
३३ ३३ The number of digits actually produced in the label is determined by your choice in Minimum number of digits in label above. For example, if you type "१०" in Begin numbering from and your minimum number of digits is ३, then the first label will be numbered from "०१०". लेबलमध्ये प्रत्यक्षात उत्पादित केलेल्या अंकांची संख्या वरील लेबलच्या किमान संख्येमध्ये आपल्या निवडीद्वारे निश्चित केली जाते. उदाहरणार्थ, आपण प्रारंभ क्रमांकापासून "१०" टाइप केल्यास आणि आपल्या अंकांची किमान संख्या ३ असेल तर प्रथम लेबल "०१०" वरून क्रमांकित केले जाईल.
