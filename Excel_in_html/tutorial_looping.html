१० १० Creating an audio loop ध्वनी लूप तयार करत आहे
११ ११ To create an audio loop from an audio track, the first step is to find a suitable section. Ideally the sound will be reasonably constant for as long as possible. Looping is much easier with mono tracks, so if you have a stereo recording but a mono track would be sufficient, the track could first be converted to mono using Tracks > Mix > Mix Stereo down to Mono. ध्वनी गीतपट्टा वरून ध्वनी लूप तयार करण्यासाठी, योग्य विभाग शोधणे ही पहिली पायरी आहे. शक्यतोपर्यंत आवाज योग्यरित्या स्थिर असेल. मोनो गीतपट्टासह पळवाट करणे अधिक सुलभ आहे, म्हणून आपल्याकडे स्टिरिओ ध्वनीमुद्रणिंग असल्यास परंतु मोनो गीतपट्टा पुरेसा असेल तर गीतपट्टा> मिक्स> मिक्स स्टिरीओ डाऊन मोनो पर्यंत गीतपट्टा प्रथम मोनोमध्ये रूपांतरित केला जाऊ शकतो.
१२ १२ The start of the selection should match (look like) the end of the selection as closely as possible and both selection edges should be positioned so that the blue line of the waveform seems to cross the selection edge at zero amplitude (where the horizontal line appears). निवडीची सुरूवात शक्य तितक्या जवळच्या निवडीच्या शेवटी (जसे दिसते) जुळली पाहिजे आणि दोन्ही निवड कडा स्थित असाव्यात जेणेकरुन वेव्हफॉर्मची निळी रेषा शून्य मोठेपणावर (जेथे क्षैतिज रेखा दिसते तेथे) निवड धार पार करेल. ).
१३ १३ There are items in the View menu that allow jumping from one end of the selection to the other while zoomed in: व्ह्यू मेनूमध्ये असे काही आयटम आहेत जे झूम वाढवताना निवडीच्या एका टोकापासून दुस to्या बाजूला जाण्यासाठी परवानगी देतात:
१४ १४ View > Skip to > Selection Start ( shortcut is Ctrl + [  ) पहा> येथे जा> निवड प्रारंभ (शॉर्टकट Ctrl + [आहे)
१५ १५ View > Skip to > Go to Selection End ( shortcut is Ctrl + ]  ) पहा> येथे जा> निवडीच्या शेवटी जा (शॉर्टकट Ctrl +] आहे)
१६ १६ To look for appropriate loop points, zoom in horizontally on the start of the selection and then on the end of the selection. To zoom, place the mouse pointer at the selection edge and scroll with the wheel or ball. योग्य लूप पॉईंट्स शोधण्यासाठी, निवड सुरू झाल्यावर आणि नंतर निवडीच्या शेवटी क्षैतिज झूम करा. झूम करण्यासाठी, निवड काठावर माउस पॉईंटर ठेवा आणि चाक किंवा बॉलसह स्क्रोल करा.
१७ १७ Drag the selection edges leftwards or rightwards as necessary so as to find loop points that have a similar shape, amplitude and slope so that the end of the loop may flow smoothly into the start of the loop. It may also be useful to zoom in on the vertical scale to left of the waveform, as shown in the following images: आवश्यकतेनुसार निवड कडा डावीकडे किंवा उजवीकडे ड्रॅग करा जेणेकरून समान आकार, मोठेपणा आणि उतार असलेले लूप पॉईंट्स शोधा जेणेकरून लूपचा शेवट लूपच्या सुरूवातीस सहजतेने वाहू शकेल. खाली दिलेल्या प्रतिमांप्रमाणेच वेव्हफॉर्मच्या डावीकडून उभ्या प्रमाणात झूम वाढविणे देखील उपयुक्त ठरू शकेल:

२४ २४ Using loops in Audacity ओड्यासिटी मध्ये पळवाट वापरणे
२५ २५ If the loop is to be used in Audacity, the selection may be trimmed: Ctrl + T. जर लूप ओड्यासिटीमध्ये वापरायचा असेल तर निवड ट्रिम केली जाऊ शकतेः Ctrl + T.
२६ २६ The loop may be repeated using Copy and Paste, though it is usually more convenient to use the Repeat function: Effect > Repeat.... कॉपी आणि पेस्ट वापरुन लूपची पुनरावृत्ती होऊ शकते, तथापि हे पुन्हा वापराचे कार्य करणे अधिक सोयीस्कर असेल: प्रभाव> पुनरावृत्ती ....
२७ २७ Exporting Loops निर्यात पळवाट
२८ २८ The selection may be exported using File > Export > Export Selected Audio.... धारिका धारिका> निर्यात> निवडलेला ध्वनी वापरुन निवड निर्यात केली जाऊ शकते ....
२९ २९ The lossless PCM WAV format is the best format for loops. Choose "WAV (Microsoft) signed १६-bit PCM" when exporting. Many lossy, size-compressed formats like MP३, WMA and ADPCM WAV suffer from added silence at the the start or end of the file or other issues that do not respect the exact length. लूपलेस पीसीएम डब्ल्यूएव्ही स्वरूप लूपसाठी सर्वोत्तम स्वरूप आहे. निर्यात करताना "डब्ल्यूएव्ही (मायक्रोसॉफ्ट) ने १६-बिट पीसीएम स्वाक्षरित" निवडा. एमपी ३, डब्ल्यूएमए आणि एडीपीसीएम डब्ल्यूएव्ही सारख्या बर्‍याच हानीकारक, आकार-संकुचित स्वरूपामध्ये धारिकाच्या सुरूवातीस किंवा शेवटी समाप्ती किंवा इतर लांबीचा त्रास होतो ज्या अचूक लांबीचा आदर करत नाहीत.
३० ३० If a lossy compressed audio format is required then OGG Vorbis may be a better choice as it does not suffer from length issues. जर हानीकारक कॉम्प्रेस केलेले ध्वनी स्वरूपन आवश्यक असेल तर लांबीच्या समस्येमुळे त्रास होत नाही म्हणून ओजीजी व्हॉर्बिस एक चांगली निवड असू शकते.
३१ ३१ Using Timeline Quick-Play to create loops लूप तयार करण्यासाठी वेळलाइन क्विक-प्ले वापरणे
३२ ३२ The above process can be aided by using the Timeline to help create a seamless loop: सीमलेस लूप तयार करण्यात मदत करण्यासाठी वरील प्रक्रियेस वेळलाइन वापरुन सहाय्य केले जाऊ शकते:
३३ ३३ Ensure that Quick-Play is enabled in the Timeline right-click context menu. वेळलाइन राइट-क्लिक संदर्भ मेनूमध्ये क्विक-प्ले सक्षम केले असल्याचे सुनिश्चित करा.
३४ ३४ Select "Enable dragging selection" in the Timeline right-click context menu. वेळलाइन उजवे-क्लिक संदर्भ मेनूमध्ये "ड्रॅगिंग निवड सक्षम करा" निवडा.
३५ ३५ Make an approximate selection of the audio to be looped. पळवून नेण्यासाठी ध्वनीची अंदाजे निवड करा.
३६ ३६ Shift + Left-Click within the selection to loop play the selection. निवडीला पळवाट देण्यासाठी निवड मध्ये शिफ्ट + डावे क्लिक करा.
३७ ३७ Shift + Left-Click and drag the start/end of the selection as necessary to adjust the selection closer to "seamless" looping positions. शिफ्ट + डावे क्लिक करा आणि निवड "अखंड" वळण स्थितीच्या जवळ समायोजित करण्यासाठी आवश्यकतेनुसार निवडीचा प्रारंभ / अंत ड्रॅग करा.
३८ ३८ If necessary, zoom in on the start / end positions to make adjustments to the play region and selection. आवश्यक असल्यास, खेळाच्या प्रदेशात आणि निवडीमध्ये mentsडजस्ट करण्यासाठी स्टार्ट / एंड पोझिशन्सवर झूम वाढवा.
३९ ३९ Note that if zooming in causes the selected region to extend beyond the visible track area, Audacity will, by default, cause the waveform to scroll. The scrolling behavior can be disabled from the Timeline right-click context menu. लक्षात ठेवा जर झूम केल्यामुळे निवडलेले प्रदेश दृश्यमान गीतपट्टा क्षेत्राच्या पलीकडे वाढवित असेल तर ओड्यासिटी पूर्वनिर्धारितनुसार वेव्हफॉर्मला स्क्रोल करण्यास कारणीभूत ठरेल. स्क्रोलिंग वर्तन वेळलाइन राइट-क्लिक संदर्भ मेनूमधून अक्षम केले जाऊ शकते.
४० ४० While zoomed in, use Ctrl + [ and Ctrl + ] to view the start / end of the selection. झूम वाढवताना, निवडीचा प्रारंभ / शेवट पाहण्यासाठी Ctrl + [आणि Ctrl +] वापरा.
४१ ४१ If a long selection is being looped, playback may be stopped at any time by pressing the Space key, then looped Quick-Play may be restarted from a short way before the end of the selection with Shift + Click on the Timeline एखादी लांब पळवाट सोडली जात असल्यास, स्पेस की दाबून कोणत्याही वेळी प्लेबॅक थांबविला जाऊ शकतो, नंतर पळवाट क्विक-प्लेची निवड शिफ्ट + सह निवड संपण्यापूर्वी थोड्या मार्गाने पुन्हा सुरू केली जाऊ शकते + वेळलाइनवर क्लिक करा.
४२ ४२ Then Optionally label the loop region (use Ctrl + B) नंतर वैकल्पिकरित्या लूप प्रदेशाला लेबल करा (Ctrl + B वापरा)
४३ ४३ When the selection loops seamlessly, playback may be stopped (using Stop button or its Space shortcut) and the selection exported or trimmed as required (use Ctrl + T to trim to the selection). जेव्हा निवड अखंडपणे लूप करते, तेव्हा प्लेबॅक थांबविला जाऊ शकतो (स्टॉप बटण किंवा त्याचे स्पेस शॉर्टकट वापरुन) आणि निवड निर्यात केली किंवा आवश्यकतेनुसार सुव्यवस्थित करा (निवडीमध्ये ट्रिम करण्यासाठी Ctrl + T वापरा).
४४ ४४ Seamless loops should normally be exported in a lossless format such as WAV. Note that seamless loops should NOT be exported in MP३ format because encoder/decoder overall delay is not defined, which means there is no official provision for gapless playback. सीमलेस पळवाट सामान्यत: डब्ल्यूएव्हीसारख्या लॉसलेस स्वरूपात निर्यात केले जावे. लक्षात घ्या की सीमलेस लूप एमपी ३ फॉरमॅटमध्ये एक्सपोर्ट करू नये कारण एन्कोडर / डीकोडर एकंदर विलंब परिभाषित केलेला नाही, ज्याचा अर्थ असा की गॅपलेस प्लेबॅकसाठी कोणतीही अधिकृत तरतूद नाही.
४५ ४५ If a compressed file format is required, try OGG Vorbis (though understand that because OGG Vorbis is a lossy format, the encoded audio is not exactly identical to the original audio). OGG Vorbis loops will generally perform acceptably provided that the start / end of the loop are at zero crossing points. संकुचित धारिका स्वरूपन आवश्यक असल्यास, OGG Vorbis वापरून पहा (जरी हे समजून घ्या की OGG Vorbis एक हानीकारक स्वरूप आहे, परंतु एन्कोड केलेला ध्वनी मूळ ध्वनीसारखेच नाही). ओ.जी.जी. व्होर्बिस लूप सामान्यत: स्वीकारल्या जातील परंतु लूपचा प्रारंभ / शेवट शून्य क्रॉसिंग पॉईंटवर असेल तर.
