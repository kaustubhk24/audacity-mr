४ ४ AC३ is a compressed audio format developed by Dolby Laboratories, often used in DVD production. एसी ३ हे डॉल्बी प्रयोगशाळांद्वारे विकसित केलेले संकुचित ध्वनी स्वरूप आहे, जे बहुधा डीव्हीडी उत्पादनामध्ये वापरले जाते.
५ ५ Accessed by: File > Export > Export Audio... then choosing AC३ Files (FFmpeg) from the Save as type dropdown menu. याद्वारे प्रवेश केलेले: धारिका> निर्यात> ध्वनी निर्यात करा ... नंतर जतन म्हणून टाइप ड्रॉपडाउन मेनूमधून एसी ३ धारिका (एफएफएमपीएग) निवडणे.
६ ६ Also accessed by: File > Export > Export Multiple... then choosing AC३ Files (FFmpeg) याद्वारे देखील प्रवेश केलेले: धारिका> निर्यात> एकाधिक निर्यात करा ... त्यानंतर AC३ धारिका (एफएफएमपीजी) निवडणे
७ ७ from the Save as type dropdown menu. जतन असे टाइप ड्रॉपडाउन मेनूमधून.
८ ८ In this case the options dialog will appear in the center of the Export Multiple dialog. अशावेळी निर्यात संवाद एकाधिक संवादाच्या मध्यभागी दिसेल.
९ ९ FFmpeg is not shipped with Audacity but you can download it and then export as AC३ by following these instructions. एफएफम्पेग ओड्यासिटीसह पाठविला जात नाही परंतु आपण त्या डाउनलोड करुन नंतर या सूचनांचे अनुसरण करून एसी ३ म्हणून निर्यात करू शकता.

११ ११ Bit Rate: Choose a bit rate from ३२ kbps to ६४० kbps (defaults to १६० kbps). Larger bit rates give higher quality but produce larger files. बिट रेट: ३२ केपीएस ते ६४० केबीपीएस पर्यंत पूर्वनिर्धारित (१६० केबीपीएस पर्यंत पूर्वनिर्धारित) निवडा. मोठे बिट दर उच्च गुणवत्ता देतात परंतु मोठ्या धारिका तयार करतात.
