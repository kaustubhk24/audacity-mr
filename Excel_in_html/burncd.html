४ ४ To burn your work to an audio CD for standalone CD players, first export it as WAV or AIFF. स्टँडअलोन सीडी प्लेयरसाठी ध्वनी सीडीवर आपले कार्य बर्न करण्यासाठी प्रथम त्यास डब्ल्यूएव्ही किंवा एआयएफएफ म्हणून निर्यात करा.
५ ५ The exported WAV or AIFF file should be: निर्यात केलेली डब्ल्यूएव्ही किंवा एआयएफएफ धारिका अशी असावी:
६ ६ stereo स्टिरीओ
७ ७ ४४100 Hz (set at Project Rate at bottom left of the project window) ४४100 हर्ट्ज (प्रकल्प खिडकीच्या डावीकडे तळाशी असलेल्या प्रकल्प दरावर सेट)
८ ८ 1६-bit (chosen using the "Save as type" menu in the File Export Dialog ). 1६-बिट (धारिका निर्यात संवादातील "प्रकार म्हणून जतन करा" मेनू वापरुन निवडलेले)
९ ९ These are Audacity's default settings, so normally you will not need to change these before exporting. या ओड्यासिटीच्या पूर्वनिर्धारित समायोजन आहेत, म्हणून सामान्यत: निर्यात करण्यापूर्वी आपल्याला या बदलण्याची आवश्यकता नसते.
10 10 Burn the exported file to an "audio CD" - not a "data CD" or "MP३ CD" - with a CD burning application like iTunes or Windows Media Player. आयट्यून्स किंवा विंडोज मीडिया प्लेयर सारख्या सीडी बर्निंग अनुप्रयोगासह निर्यात केलेली धारिका "ध्वनी सीडी" वर बर्न करा - "डेटा सीडी" किंवा "एमपी ३ सीडी" नाही.
