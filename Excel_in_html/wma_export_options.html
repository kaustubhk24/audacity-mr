४ ४ WMA is a compressed, lossy audio format developed by Microsoft. It is the default audio format for Windows Media Player. डब्ल्यूएमए मायक्रोसॉफ्टने विकसित केलेले एक संकुचित, हानीकारक ध्वनी स्वरूप आहे. हे विंडोज मीडिया प्लेयरसाठी पूर्वनिर्धारित ध्वनी स्वरूप आहे.
५ ५ Accessed by: File > Export > Export Audio... then choosing WMA (version२) Files (FFmpeg) from the Save as type dropdown menu. याद्वारे प्रवेश केलेले: धारिका> निर्यात> ध्वनी निर्यात करा ... त्यानंतर जतन म्हणून टाइप ड्रॉपडाऊन मेनूमधून डब्ल्यूएमए (आवृत्ती २) धारिका (एफएफएमपीजी) निवडणे.
६ ६ Also accessed by: File > Save Other > Export Multiple... then choosing WMA (version२) Files (FFmpeg) from the Save as type dropdown menu. याद्वारे देखील प्रवेश केलेले: धारिका> अन्य जतन करा> एकाधिक निर्यात करा ... त्यानंतर जतन म्हणून टाइप ड्रॉपडाऊन मेनूमधून डब्ल्यूएमए (आवृत्ती २) धारिका (एफएफम्पेग) निवडणे.
७ ७ In this case the options dialog will appear in the center of the Export Multiple dialog. अशावेळी निर्यात संवाद एकाधिक संवादाच्या मध्यभागी दिसेल.
८ ८ FFmpeg is not shipped with Audacity but you can download it and then export as WMA (version २) by following these instructions. एफएफएमपेग ओड्यासिटीसह पाठविला जात नाही परंतु आपण त्या डाउनलोड करुन नंतर या निर्देशांचे पालन करून डब्ल्यूएमए (आवृत्ती २) म्हणून निर्यात करू शकता.

१० १० Bit Rate: Choose a bit rate from २४ kbps to ५७६ kbps. The default is १८० kbps. This produces a larger file of higher quality than default १२८ kbps MP३ encoding, the general consensus being that WMA v२ and MP३ produce comparable quality for the same file size.
