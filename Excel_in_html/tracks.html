४ ४ Anything that is added to the Project Window will be placed inside a track container, underneath the Timeline of the project. There are four different types of tracks: प्रकल्प विंडोमध्ये जोडलेली कोणतीही गोष्ट प्रकल्पाच्या वेळलाइनच्या खाली गीतपट्टा कंटेनरमध्ये ठेवली जाईल. चार वेगवेगळ्या प्रकारचे गीतपट्टा आहेत:
५ ५ Audio Tracks, ध्वनी गीतपट्टा,
६ ६ Label Tracks, लेबल गीतपट्टा,
७ ७ Time Tracks and वेळ मागोवा आणि

९ ९ Audio tracks ध्वनी गीतपट्टा
१० १० Audio Tracks contain digitally sampled sounds. ध्वनी गीतपट्टामध्ये डिजिटल नमुना घेतलेले आवाज असतात.

१५ १५ Label tracks लेबल गीतपट्टा
१६ १६ A Label Track is an additional track that can be created in your project; it is used in conjunction with audio tracks but contains no audio itself. It can be used to mark specific points or ranges in the project with point labels or range labels. एक लेबल गीतपट्टा एक अतिरिक्त गीतपट्टा आहे जो आपल्या प्रकल्पात तयार केला जाऊ शकतो; ते ध्वनी गीतपट्टाच्या संयोगाने वापरले जाते परंतु त्यात कोणताही ध्वनी नसतो. याचा उपयोग पॉईंट लेबले किंवा श्रेणी लेबलांसह प्रकल्पातील विशिष्ट बिंदू किंवा श्रेणी चिन्हांकित करण्यासाठी केला जाऊ शकतो.
१७ १७ Time Tracks वेळ मागोवा
१८ १८ A Time Track is used in conjunction with one or more audio tracks to progressively increase or decrease playback speed (and pitch) over the length of the audio. Speed changes are controlled by manipulating the blue "time warp" line with the Envelope Tool used for making gradual volume changes. If the blue line is dragged above the horizontal, the audio plays faster; if dragged below, it plays slower. ध्वनीच्या लांबीवर प्लेबॅक गती (आणि खेळपट्टी) क्रमाने वाढविण्यासाठी किंवा कमी करण्यासाठी एक किंवा अधिक ध्वनी गीतपट्टाच्या संयोगाने एक वेळ गीतपट्टा वापरला जातो. हळूहळू व्हॉल्यूम बदल करण्यासाठी वापरल्या गेलेल्या लिफाफा उपकरणासह निळ्या "वेळ वार्प" लाइनमध्ये फेरफार करून वेग बदलला जातो. जर निळ्या रेषा आडव्या वर ड्रॅग केली असेल तर ध्वनी वेगवान प्ले करतो; जर खाली ड्रॅग केले तर ते हळू वाजवते.
१९ १९ Note Tracks नोट्स गीतपट्टा
२० २० Note Tracks display data from MIDI files - these can be imported with the File > Import > Import MIDI... command. टीप एमआयडीआय धारिकांमधून डेटा प्रदर्शित करते - हे धारिका> आयात> एमआयडीआय आयात करा ... आदेशासह आयात केले जाऊ शकते.
२१ २१ Audacity can play these MIDI files, but note that although this should just work on Windows on Mac and Linux additional software may be required (for details please see this page). धृष्टता या एमआयडीआय धारिका प्ले करू शकते, परंतु लक्षात घ्या की हे फक्त मॅक आणि लिनक्सवर विंडोजवर कार्य करत असले तरी अतिरिक्त सॉफ्टवेअर आवश्यक असू शकते (तपशीलांसाठी कृपया हे पृष्ठ पहा).
२२ २२ Editing these Note tracks is possible, but very limited. या टीप गीतपट्टाचे संपादन करणे शक्य आहे, परंतु अत्यंत मर्यादित आहे.
२३ २३ For more details please see: Note Tracks. अधिक माहितीसाठी कृपया पहा: नोट्स गीतपट्टा.
२४ २४ Track focus गीतपट्टा फोकस
२५ २५ You may have noticed in the images on this page that Note track has a yellow border around it. This border indicates that this track has focus. या पृष्ठावरील प्रतिमांमध्ये आपणास लक्षात आले असेल की टीप गीतपट्टाला आजुबाजुला पिवळी किनार आहे. ही सीमा सूचित करते की या गीतपट्टावर लक्ष केंद्रित आहे.
२६ २६ The track that has focus is the track that accepts any command whose name includes "focused track". These commands include commands such as "Close focused track" (Shift + C), "Mute/Unmute focused track" (Shift + U) and "Toggle focused track" (Return or Enter, which toggles whether the focused track is selected or not). लक्ष असणारा गीतपट्टा असा गीतपट्टा आहे ज्याच्या नावावर "केंद्रित गीतपट्टा" समाविष्ट असलेली कोणतीही आज्ञा स्वीकारली जाते. या आज्ञा्समध्ये "क्लोस्क्स्ड गीतपट्टा गीतपट्टा" (शिफ्ट + सी), "निःशब्द / सशब्द केंद्रित गीतपट्टा" (शिफ्ट + यू) आणि "टॉगल केंद्रित गीतपट्टा" (रिटर्न किंवा एंटर, जे केंद्रित गीतपट्टा निवडलेले आहे की नाही हे टॉगल करते) सारख्या आदेशांचा समावेश आहे. ).
२७ २७ The complete list of "focused track" commands can be seen by searching for "focused track" (without quotes) in Keyboard Preferences or by viewing the description of these commands in Keyboard Shortcut Reference. "फोकस गीतपट्टा" आदेशांची संपूर्ण यादी कीबोर्ड प्राधान्यांमध्ये "केंद्रित गीतपट्टा" (कोटेशिवाय) शोधून किंवा कीबोर्ड शॉर्टकट संदर्भात या आदेशांचे वर्णन पाहून दिसून येते.
२८ २८ Track focus is used to select audio and to extend or move the selection region into other tracks using the keyboard. गीतपट्टा फोकसचा वापर ध्वनी निवडण्यासाठी आणि निवड प्रदेश विस्तृत करण्यासाठी किंवा कीबोर्डचा वापर करून इतर गीतपट्टामध्ये हलविण्यासाठी केला जातो.
