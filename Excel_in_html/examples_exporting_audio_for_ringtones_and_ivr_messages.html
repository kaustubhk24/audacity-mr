11 11 Look at the Project Rate control on the Selection Toolbar (by default at bottom left of the Audacity Project window); if it is already showing "44100" skip to Step 2 below. Otherwise, select the "44100" option from the dropdown menu; if there is not a 44100 (Hz) option select Other ... and type 44100 in the box that pops up (in Audacity 2.x select and type over the existing value). सिलेक्शन टूलबारवरील प्रोजेक्ट रेट कंट्रोल पहा (ऑडसिटी प्रोजेक्ट विंडोच्या डावीकडे खाली डीफॉल्टनुसार); जर ते आधीपासूनच खाली चरण 2 वर "44100" वगळत असेल तर. अन्यथा, ड्रॉपडाउन मेनूमधून "44100" पर्याय निवडा; 44100 (हर्ट्ज) पर्याय नसल्यास अन्य ... निवडा आणि पॉप अप झालेल्या बॉक्समध्ये 44100 टाइप करा (ऑडसिटी २.x मध्ये निवडा आणि विद्यमान मूल्यावर टाइप करा).
12 12 Select menu item File > Export > Export Audio...; in the "Export File" dialog choose WAV (Microsoft) signed 16-bit PCM in the "Save as type" dropdown, then enter a file. Click OK then click Save. If the Metadata Editor pops up at any stage, click OK. मेनू आयटम फाइल निवडा> निर्यात करा> ऑडिओ निर्यात करा ...; "फाइल निर्यात करा" संवादात डब्ल्यूएव्ही (मायक्रोसॉफ्ट) ने "टाइप म्हणून जतन करा" ड्रॉपडाउनमध्ये 16-बिट पीसीएम स्वाक्षरित केले आहे, त्यानंतर एक फाईल प्रविष्ट करा. ओके क्लिक करा आणि नंतर सेव्ह क्लिक करा. मेटाडेटा संपादक कोणत्याही टप्प्यावर पॉप अप करत असल्यास, ओके क्लिक करा.
13 13 Metadata tags are not needed for WAV files in mobile phones. मोबाइल फोनमध्ये डब्ल्यूएव्ही फायलींसाठी मेटाडेटा टॅगची आवश्यकता नाही.
14 14 8000 Hz 8-bit WAV 8000 हर्ट्ज 8-बिट डब्ल्यूएव्ही
15 15 WAV (Microsoft) Bit depth: 8-bit; Sample Rate: 8000 Hz; Channels: 1 (mono) डब्ल्यूएव्ही (मायक्रोसॉफ्ट) बिट खोली: 8-बिट; नमुना दर: 8000 हर्ट्ज; चॅनेल: 1 (मोनो)
16 16 This is an example of exporting an uncompressed but small size WAV file specific to a particular type of phone. The low sample rate of 8000 Hz means that the file will not contain frequencies higher than 4000 Hz, so this file may not give the best results on other phones that have more capable speakers. एका विशिष्ट प्रकारच्या फोनशी संबंधित एक कंप्रेस केलेले परंतु लहान आकाराच्या डब्ल्यूएव्ही फाईल निर्यात करण्याचे हे उदाहरण आहे. 8000 हर्ट्झचा कमी नमुना दर असा आहे की फाईलमध्ये 4000 हर्ट्झपेक्षा जास्त फ्रिक्वेन्सी नसतील, म्हणून कदाचित या फाईलमध्ये अधिक सक्षम स्पीकर्स असलेल्या इतर फोनवर सर्वोत्कृष्ट परिणाम मिळणार नाही.

If your phone has the same requirements as this, the instructions below should work for your phone. रिंगटोनसाठी 8-बिट 8000 हर्ट्ज मोनो डब्ल्यूएव्ही फायली असणे आवश्यक आहे. आपल्या फोनला यासारख्याच आवश्यकता असल्यास, खाली दिलेल्या सूचनांनी आपल्या फोनसाठी कार्य केले पाहिजे.
20 20 If the only information you have is that the WAV needs to have a bit rate of 64 kbps, these instructions will also probably work for your phone, because in a WAV file the bit rate is always the (bit depth) multiplied by the (sample rate), multiplied by the (number of channels) so the WAV in our example is (8 * 8 * 1) = 64 kbps. आपल्याकडे फक्त माहिती असल्यास डब्ल्यूएव्हीला 64 केबीपीएसचा बिट रेट असणे आवश्यक आहे, या सूचना कदाचित आपल्या फोनसाठी देखील कार्य करतील कारण डब्ल्यूएव्ही फाइलमध्ये बिट रेट नेहमीच (नमुनाने) ने गुणाकार करते रेट), (चॅनेलची संख्या) सह गुणाकार जे आमच्या उदाहरणातील डब्ल्यूएव्ही (8 * 8 * 1) = 64 केबीपीएस आहे.

It's assumed you've already made the track mono as per "Convert stereo to mono". यापेक्षा थोड्या वेगळ्या वैशिष्ट्यांसह आपल्या फोनला डब्ल्यूएव्ही फायली आवश्यक असल्यास आपण खाली दिलेल्या सूचना योग्यरित्या समायोजित करू शकता. असे मानले आहे की आपण "कन्व्हर्ट स्टिरिओ टू मोनो" नुसार ट्रॅक मोनो बनविला आहे.
22 22 Look at the Project Rate control on the Selection Toolbar (by default at bottom left of the Audacity Project window); if it is already showing "8000", skip to Step 2 below; otherwise, select the "8000" option from the dropdown menu. If there is not an 8000 (Hz) option, select "Other ..." and type 8000 in the box that pops up (in Audacity 2.x select and type over the existing value). सिलेक्शन टूलबारवरील प्रोजेक्ट रेट कंट्रोल पहा (ऑडसिटी प्रोजेक्ट विंडोच्या डावीकडे खाली डीफॉल्टनुसार); हे आधीपासूनच "8000" दर्शवित असल्यास, खाली चरण 2 वर जा; अन्यथा, ड्रॉपडाउन मेनूमधून "8000" पर्याय निवडा. 8000 (हर्ट्ज) पर्याय नसल्यास, "अन्य ..." निवडा आणि पॉप अप झालेल्या बॉक्समध्ये 8000 टाइप करा (ऑडसिटी २.x मध्ये निवडा आणि विद्यमान मूल्यावर टाइप करा).
23 23 Select menu item File > Export > Export Audio... , in the "Export File" dialog choose other uncompressed files from the "Save as type" dropdown, then enter a file name. Click Options, a dialog will open. In the "Header" dropdown, select "WAV (Microsoft)" and in the "Encoding" dropdown choose "Unsigned 8-bit PCM"; click OK then Save. If the Metadata Editor pops up at any stage just click OK; metadata tags are not needed for WAV files in mobile phones. मेनू आयटम फाइल निवडा> निर्यात करा> ऑडिओ निर्यात करा ... "फाइल निर्यात करा" संवादामध्ये "प्रकार म्हणून जतन करा" ड्रॉपडाऊनमधून इतर कंप्रेस फाइल फाइल्स निवडा, त्यानंतर एक फाइल नाव प्रविष्ट करा. पर्याय क्लिक करा, एक संवाद उघडेल. "शीर्षलेख" ड्रॉपडाऊनमध्ये, "डब्ल्यूएव्ही (मायक्रोसॉफ्ट)" निवडा आणि "एन्कोडिंग" ड्रॉपडाऊनमध्ये "अस्वाक्षरित 8-बिट पीसीएम" निवडा; ओके क्लिक करा नंतर सेव्ह करा. जर मेटाडेटा संपादक कोणत्याही टप्प्यावर पॉप अप करत असेल तर ओके क्लिक करा; मोबाइल फोनमध्ये डब्ल्यूएव्ही फायलींसाठी मेटाडेटा टॅगची आवश्यकता नाही.

"ऑडिओ फाईल एक्सपोर्ट करा" संवादात, "टाइप म्हणून जतन करा" ड्रॉपडाउन वरून एमपी 3 फायली निवडा, नंतर फाइल नाव प्रविष्ट करा. पर्याय क्लिक करा, एक संवाद उघडेल. बिट रेट मोडला कॉन्स्टंट वर सेट करा; "गुणवत्ता" ड्रॉपडाऊनमध्ये "32 केबीपीएस" निवडा; "चॅनेल मोड" ला "स्टीरिओ" वर सेट करा - तरीही आपला ऑडेसिटी ट्रॅक स्टिरीओ असल्यास एकल चॅनेल (मोनो) फाइल तयार केली जाईल; ओके क्लिक करा आणि नंतर सेव्ह क्लिक करा.
