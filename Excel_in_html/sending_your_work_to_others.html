१३ १३ Sending Projects to others इतरांना प्रकल्प पाठवित आहे
१४ १४ It's quite possible to send an Audacity Project to another person or open it on another computer of your own, so that the Project can be opened in another copy of Audacity with all the tracks, label and envelope information in situ. However when opening the AUP Project file you need to have its associated _data folder with the same name as the AUP file present in the same folder. For example, to send a Project to someone else you could send the AUP file and _data folder in a compressed .zip archive. Free Windows tools to create .zip archives include ७-Zip or IZArc. Free Mac tools include Keka or Apple's built-in compression utility. Zip compression is lossless, so does not reduce the space on disk very much. Ask your recipient to extract both the AUP file and _data folder to the same directory then open the AUP file from inside that folder. दुसर्या व्यक्तीला ओड्यासिटी प्रकल्प पाठविणे किंवा आपल्या स्वत: च्या दुसर्‍या संगणकावर उघडणे शक्य आहे, जेणेकरून प्रकल्प ओड्यासिटीच्या दुसर्‍या प्रतात सर्व गीतपट्टा, लेबल आणि लिफाफ्याच्या माहितीसह परिस्थितीत उघडता येईल. तथापि एओपी प्रकल्प धारिका उघडताना आपणास त्याचे संबंद्ध _डेटा निर्देशिका त्याच निर्देशिकामध्ये एयूपी धारिकाच्या समान नावाने असणे आवश्यक आहे. उदाहरणार्थ, एखाद्या दुसर्‍यास प्रकल्प पाठविण्यासाठी आपण संकुचित .zip संग्रहणामध्ये AUP धारिका आणि _डेटा निर्देशिका पाठवू शकता. .Zip संग्रहण तयार करण्यासाठी विनामूल्य Windows साधनांमध्ये ७-झिप किंवा IZArc चा समावेश आहे. विनामूल्य मॅक साधन्समध्ये केका किंवा Appleपलची अंगभूत कॉम्प्रेशन युटिलिटी समाविष्ट आहे. झिप कॉम्प्रेशन हानिरहित आहे, म्हणून डिस्कवरील जागा फारच कमी करत नाही. आपल्या प्राप्तकर्त्यास समान निर्देशिका मध्ये AUP धारिका आणि _डेटा निर्देशिका दोन्ही काढण्यास सांगा आणि त्या निर्देशिकाच्या आत AUP धारिका उघडा.
१५ १५ In practice, the problem with sending such a folder to someone else is the size of the _data folder. At Audacity's default ३२-bit float sample format / ४४१०० Hz sample rate using lossless uncompressed audio, stereo Projects take २० MB of space per minute, which rules out sending the project by email. सराव मध्ये, असे निर्देशिका दुसर्‍या कोणास पाठविण्यातील अडचण म्हणजे _डेटा निर्देशिकाचा आकार. ओड्यासिटीच्या पूर्वनिर्धारित ३२-बिट फ्लोट नमुना स्वरूपात / ४४१०० हर्ट्ज नमुना दर लॉलेसलेस कॉम्प्रप्रेसित ध्वनी वापरुन, स्टीरिओ प्रकल्प्स प्रति मिनिट २० एमबी जागा घेतात, जे ईमेलद्वारे प्रकल्प पाठविण्यास नकार देतात.
१६ १६ To send your project, first close it (saving changes) so that Audacity can discard the surplus data files which allow undo and redo of edits while projects are open. This way, only the audio for the current state of the project is included. If you forget to close the project before zipping, Audacity will complain about these surplus files, calling them "orphan block files". If this happens, the safest course is to tell Audacity to "continue without deleting" the orphans, at least until you are sure the audio of the project is correct. आपला प्रकल्प पाठविण्यासाठी प्रथम ते बंद करा (बदल जतन करा) जेणेकरून ओड्यासिटी अतिरेकी डेटा धारिका टाकू शकेल ज्या प्रकल्पांना उघडे असताना संपादन पूर्ववत आणि पुन्हा करण्याची परवानगी देतील. अशाप्रकारे, प्रकल्पाच्या सद्य स्थितीसाठी केवळ ध्वनी समाविष्ट केला आहे. आपण झिप करण्यापूर्वी प्रकल्प बंद करणे विसरल्यास ओड्यासिटी या अतिरीक्त धारिकांबद्दल तक्रार करेल, त्यांना "अनाथ ब्लॉक धारिका" म्हणत असेल. असे झाल्यास, सर्वात सुरक्षित मार्ग म्हणजे ओड्यासिटीला अनाथांना "हटविल्याशिवाय पुढे जा" असे सांगावे लागेल, जोपर्यंत आपल्याला खात्री नाही की प्रकल्पातील ध्वनी बरोबर आहे.
१७ १७ Then use a zip application like those above to place the AUP file and _data folder inside a zip file. Then find a suitable free internet file transfer service to send your zip file. Recommendable services include the following. Check the limitations for free accounts on size and number of uploads, maximum permitted number of downloads, and length of time files are stored for. त्यानंतर झिप धारिकामध्ये एयूपी धारिका आणि _डेटा निर्देशिका ठेवण्यासाठी वरीलप्रमाणे झिप अनुप्रयोग वापरा. नंतर आपली झिप धारिका पाठविण्यासाठी एक योग्य विनामूल्य इंटरनेट धारिका ट्रान्सफर सेवा शोधा. शिफारस केलेल्या सेवांमध्ये पुढील गोष्टींचा समावेश आहे. आकार आणि अपलोडवरील संख्येवर विनामूल्य खात्यांसाठी मर्यादा तपासा, डाउनलोडची जास्तीत जास्त परवानगी संख्या आणि धारिका ज्या कालावधीसाठी संग्रहित आहेत त्या तपासा.

२४ २४ Don't forget old-fashioned solutions too if you are still on slow dial-up internet. You can always combine your AUP Project file and _data folder in a .zip file, burn this to a data CD with Windows Media Player or similar then send it by postal mail. आपण अद्याप स्लो डायल-अप इंटरनेट वर असल्यास जुन्या पद्धतीची निराकरणे देखील विसरू नका. आपण आपल्या एओपी प्रकल्प धारिका आणि _डाटा फोल्‍डरला नेहमीच एक .zip धारिकामध्ये एकत्र करू शकता, विंडोज मीडिया प्लेयर किंवा तत्सम असलेल्या डेटा सीडीमध्ये बर्न करा किंवा पोस्टल मेलद्वारे पाठवा.
२५ २५ You can produce a small, compressed project for distribution to others using File > Save Project > Save Compressed Copy of Project.... आपण धारिका> जतन प्रकल्प> प्रकल्पची संकुचित प्रत जतन करा .... वापरून इतरांना वितरणासाठी एक छोटा, संकुचित प्रकल्प तयार करू शकता.
२६ २६ The audio in the _data folder is in the form of one OGG file per track of the project. _डेटा निर्देशिकामधील ध्वनी प्रकल्पच्या प्रत्येक गीतपट्टावर एक ओजीजी धारिकाच्या रूपात आहे.
२७ २७ The OGG files are very slightly lossy, whereas when saving a project by File > Save Project > Save Project or File > Save Project > Save Project As... the .au files in the _data folder are lossless. ओजीजी धारिका खूपच हानीकारक असतात, परंतु धारिका> प्रकल्प जतन करा> प्रकल्प जतन करा किंवा धारिका जतन करा> प्रकल्प जतन करा> प्रकल्प म्हणून जतन करा ... _डाटा निर्देशिकामधील .au धारिका्स हानीकारक आहेत.
२८ २८ Dependencies on other audio files इतर ध्वनी धारिकांवर अवलंबन
२९ २९ If you save a standard uncompressed project which contains audio imported into Audacity from a WAV or AIFF file, you must have previously set Audacity to make a copy of that original audio when you imported it, so that it's included in the _data folder. आपण एक प्रमाणित संकुचित प्रकल्प जतन केले ज्यामध्ये डब्ल्यूएव्ही किंवा एआयएफएफ धारिकामधून ओड्यासिटीमध्ये ध्वनी आयात केलेला असेल तर आपण आयात केल्यावर त्या मूळ ध्वनीची प्रत तयार करण्यासाठी आपण यापूर्वी ओड्यासिटी सेट केली पाहिजे, जेणेकरून ते _डेटा निर्देशिकामध्ये समाविष्ट असेल.
३० ३० Note that when importing compressed audio like MP३ or OGG, Audacity always copies the audio into the _data folder. लक्षात ठेवा एमपी ३ किंवा ओजीजी सारख्या संकुचित ध्वनीची आयात करताना ओड्यासिटी नेहमी ध्वनी _data निर्देशिकामध्ये नक्कल करते.
३१ ३१ When you import a WAV or AIFF there will by default be a warning where you can choose to copy in the file or not. जेव्हा आपण डब्ल्यूएव्ही किंवा एआयएफएफ आयात करता तेव्हा पूर्वनिर्धारितनुसार एक चेतावणी होते जिथे आपण धारिकामध्ये नक्कल करणे निवडू शकता किंवा नाही.
३२ ३२ You can always check if you need to copy in files to the current project, and copy them in if necessary at Help > Diagnostics > Check Dependencies.... आपणास सद्य प्रकल्पमध्ये धारिकांमध्ये नक्कल करण्याची आवश्यकता असल्यास आपण नेहमी तपासू शकता आणि मदत> निदान> चेक अवलंबित्व वर आवश्यक असल्यास त्यामध्ये नक्कल करा. ....
३३ ३३ You can change your preference for future copying in or not in Import/Export Preferences, and you can turn the warning on or off in Warnings Preferences. आपण आयात / निर्यात प्राधान्ये मध्ये भविष्यकाळात नक्कल करण्यासाठी आपले प्राधान्य बदलू शकता आणि आपण चेतावणी प्राधान्यांमध्ये चेतावणी चालू किंवा बंद करू शकता.
३४ ३४ Sending an exported audio file निर्यात केलेली ध्वनी धारिका पाठवित आहे
३५ ३५ You can also send an exported audio file from your Project to the other user. Only Project–specific information such as label and envelope data will be lost by exporting instead of saving a project. आपण आपल्या प्रकल्पमधून निर्यात केलेली ध्वनी धारिका अन्य वापरकर्त्याला देखील पाठवू शकता. केवळ प्रकल्प label विशिष्ट माहिती जसे की लेबल आणि लिफाफा डेटा प्रकल्प जतन करण्याऐवजी निर्यात करुन गमावला जाईल.
३६ ३६ To export your Project as an audio file, choose the File > Export command. Choices: आपला प्रकल्प ध्वनी धारिका म्हणून निर्यात करण्यासाठी धारिका> निर्यात आदेश निवडा. निवडी:
३७ ३७ Exporting as a WAV or AIFF file creates a file with no loss of audio quality, but this takes १० MB of disk space per minute for CD quality (४४१०० Hz, १६-bit stereo). डब्ल्यूएव्ही किंवा एआयएफएफ धारिका म्हणून निर्यात केल्याने ध्वनी गुणवत्तेची हानी नसलेली एक धारिका तयार होते, परंतु सीडी गुणवत्तेसाठी प्रति मिनिट १० एमबी डिस्क स्पेस (४४१०० हर्ट्ज, १६-बिट स्टिरिओ) घेते.
३८ ३८ Exporting from Audacity as FLAC provides lossless compression and reliably reduces file size by about ४०% compared to WAV or AIFF. ओड्यासिटीकडून एफएलएसी म्हणून निर्यात करणे हानीकारक कॉम्प्रेशन प्रदान करते आणि डब्ल्यूएव्ही किंवा एआयएफएफच्या तुलनेत धारिकाचे आकार सुमारे ४०% ने कमीतकमी कमी करते.
३९ ३९ Exporting as an OGG or MP३ greatly reduces the size of the exported file, at the expense of some quality loss. ओजीजी किंवा एमपी ३ म्हणून निर्यात केल्याने काही गुणवत्तेच्या तोटाच्या किंमतीवर निर्यात केलेल्या धारिकाचा आकार मोठ्या प्रमाणात कमी होतो.
४० ४० OGG tends to have slightly higher quality than MP३ for the same file size, but not all software players can accept OGG, and most websites hosting audio files expect MP३. ओजीजीकडे समान धारिका आकारासाठी एमपी ३ पेक्षा किंचित उच्च दर्जाची आहे परंतु सर्व सॉफ्टवेअर प्लेयर ओजीजी स्वीकारू शकत नाहीत आणि ध्वनी धारिका होस्ट करणार्‍या बर्‍याच वेबसाइट एमपी ३ ची अपेक्षा करतात.
४१ ४१ Audacity's default MP३ export bit rate of १२८ kbps gives very reasonable sound quality for about १ MB of space per minute. To compress the MP३ to a still smaller file (at the cost of further quality loss), choose "MP३ Files" in the Export File window, press the Options button then reduce the bit rate in the "Quality" dropdown.
