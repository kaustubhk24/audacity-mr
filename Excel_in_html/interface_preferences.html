२२ २२ Options पर्याय
२३ २३ Show 'How to get Help' message at launch: Uncheck this to disable the default start up dialog that lists our help resources. लाँच करताना 'मदत कशी मिळवायची' संदेश दर्शवा: आमची मदत संसाधने सूचीबद्ध करणार्‍या पूर्वनिर्धारित स्टार्ट अप संवाद अक्षम करण्यासाठी हे अनचेक करा.
२४ २४ Show extra menus: Check this to see the extended menu bar. अतिरिक्त मेनू दर्शवा: विस्तारित मेनू बार पाहण्यासाठी हे तपासा.
२५ २५ Beep on completion of longer activities: Makes Audacity play a beep when activities lasting longer than a minute are completed. Visually impaired users may find this useful. जास्त क्रियाकलाप पूर्ण केल्यावर बीप: एका मिनिटापेक्षा जास्त काळ क्रियाकलाप पूर्ण झाल्यावर ओड्यासिटीला बीप प्ले करते. दृष्टिहीन वापरकर्त्यांना हे उपयुक्त वाटू शकते.
२६ २६ Retain labels if selection snaps to a label: निवड लेबलवर स्नॅप केल्यास लेबल पुन्हा ठेवा:
२७ २७ By default this option is unchecked, in which case a point label is deleted when you delete a selection that snaps exactly up to the label edge (as shown by the yellow Boundary Snap Guides) and includes the label in the selection. Similarly, a region label included in the selection is deleted when deleting a selection that extends from or outside one edge and snaps to the other edge. पूर्वनिर्धारितनुसार या पर्यायाची तपासणी केली जात नाही, अशा वेळी जेव्हा आपण लेबलच्या काठावर अचूकपणे निवडलेली एखादी निवड (जेव्हा पिवळ्या सीमेच्या स्नॅप मार्गदर्शकाने दर्शविली आहे) हटविली जाते आणि निवडीमध्ये लेबलचा समावेश असेल तेव्हा पॉईंट लेबल हटविला जाईल. त्याचप्रमाणे, एका काठाच्या बाहेरील किंवा बाहेरील बाजूने आणि दुसर्‍या काठावर स्नॅपिंग केलेली निवड हटविताना निवडीमध्ये समाविष्ट केलेला प्रदेश लेबल हटविला जातो.
२८ २८ When this option is checked, a label included in the selection is always retained if deleting a selection that snaps to one or both label edges. Even if this option is checked, the label will still be deleted if you select over a point label or over both edges of a region label. Checking this option also means that if you Effect > Repeat... an audio selection defined by a region label (for example, by clicking in the label to select it), or extend that selection past one edge only, the selection will be extended to include a single label. Without this option checked the selection would be extended to include extra labels. जेव्हा हा पर्याय तपासला जातो, तेव्हा निवडात समाविष्ट असलेले लेबल नेहमीच एकतर किंवा दोन्ही लेबलच्या काठावर स्नॅपिंग केलेली निवड हटविते. जरी हा पर्याय तपासला गेला, तरीही आपण पॉइंट लेबलवर किंवा प्रदेश लेबलच्या दोन्ही कडांवर निवडल्यास हे लेबल हटविले जाईल. हा पर्याय तपासण्याचाही अर्थ असा आहे की आपण प्रभाव> पुनरावृत्ती ... प्रदेश लेबलद्वारे परिभाषित केलेली ध्वनी निवड (उदाहरणार्थ, निवडण्यासाठी लेबलवर क्लिक करून), किंवा निवड एका काठाच्या मागील बाजूने वाढविल्यास, निवड वाढविली जाईल एकच लेबल समाविष्ट करण्यासाठी. हा पर्याय न तपासता अतिरिक्त लेबले समाविष्ट करण्यासाठी निवड वाढविली जाईल.
