४ ४ Label Tracks can be exported to, and imported from plain text files (.txt extension). The structure of the file is a tab-delimited plain text format that can be opened by any text editor or spreadsheet application and edited there. लेबल गीतपट्टा निर्यात आणि साध्या मजकूर धारिका (.txt विस्तार) मधून आयात केले जाऊ शकतात. धारिकाची रचना एक टॅब-मर्यादित साधा मजकूर स्वरूप आहे जी कोणत्याही मजकूर संपादकाद्वारे किंवा स्प्रेडशीट अनुप्रयोगाद्वारे उघडली जाऊ शकते आणि तेथे संपादित केली जाऊ शकते.

2० 2० Sometimes it may be useful to use these labels in another application (for example if somebody wanted to know the time where each piece of an interview starts or stops), or to import a labels file that someone has shared with you. कधीकधी ही लेबले दुसर्‍या अनुप्रयोगात वापरणे उपयुक्त ठरू शकते (उदाहरणार्थ एखाद्या मुलाखतीचा प्रत्येक तुकडा कोठे सुरू होतो किंवा थांबत असावे अशी वेळ एखाद्यास जाणून घ्यायची असेल तर) किंवा एखाद्याने आपल्यासह सामायिक केलेली लेबल्स धारिका आयात करण्यासाठी.
2१ 2१ You can export labels using the Edit Labels window mentioned above, or by choosing File > Export > Export Labels.... By default the exported file has the same name as the current name of the label track. If there are multiple label tracks, the labels of all those tracks are combined into a single label track in the exported file, which by default has the same name as the current name of the last label track. आपण वर नमूद केलेल्या संपादन लेबले विंडोचा वापर करून किंवा धारिका> निर्यात> निर्यात लेबले निवडून लेबलांची निर्यात करू शकता .... पूर्वनिर्धारितनुसार निर्यात केलेल्या धारिकाला लेबल गीतपट्टाच्या विद्यमान नावासारखेच नाव आहे. एकाधिक लेबल गीतपट्टा असल्यास, त्या सर्व गीतपट्टाची लेबले निर्यात केलेल्या धारिकामध्ये एकाच लेबल गीतपट्टामध्ये एकत्रित केली जातात, ज्याचे डिफॉल्टनुसार शेवटच्या लेबल गीतपट्टाच्या विद्यमान नावासारखेच नाव असते.
22 22 Labels can be imported using the Edit Labels window, or by choosing File > Import > Labels.... Imported label tracks are added to the bottom of all existing track types in the project, just as when you import an audio track. लेबल संपादित करा विंडो वापरून किंवा धारिका> आयात> लेबले निवडून .... आयातित लेबल गीतपट्टा प्रकल्पात सर्व विद्यमान गीतपट्टा प्रकारांच्या तळाशी जोडले जातात, जसे आपण ध्वनी गीतपट्टा आयात करता तेव्हाच.
2३ 2३ If you import labels created by some application other than Audacity, make sure the application supports UTF-८ or use only ASCII (Latin unaccented) characters and numbers. Otherwise, accented or any other non-Latin characters may cause the file to not import. आपण ओड्यासिटी व्यतिरिक्त काही अनुप्रयोगाद्वारे तयार केलेली लेबले आयात करत असल्यास, अनुप्रयोग UTF-८ चे समर्थन करत असल्याचे सुनिश्चित करा किंवा फक्त एएससीआयआय (लॅटिन अनचेन्सेन्ट) वर्ण आणि क्रमांक वापरा. अन्यथा, उच्चारण किंवा इतर कोणतीही लॅटिन-वर्णांमुळे धारिका आयात होऊ शकत नाही.
2४ 2४ If you use a Windows application to create the file, you may have to use "Save As" and look for a specific option that saves with UTF-८ encoding. जर आपण धारिका तयार करण्यासाठी Windows अनुप्रयोग वापरत असाल तर आपणास "या रूपात जतन करा" वापरावे लागेल आणि UTF-८ एन्कोडिंगसह जतन करणारा विशिष्ट पर्याय शोधावा लागेल.
