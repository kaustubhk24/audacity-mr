१ १ Rhythm Track ताल गीतपट्टा
२ २ From Audacity Development Manual ओड्यासिटी विकास माहितीपुस्तिकेकडून
३ ३ Jump to: navigation, search येथे जा: निर्देशक, शोध
४ ४ Rhythm Track generates a sequence of regular pulses at the selected tempo, beats per bar and either number of bars or track duration, using a selectable sound. It can thus be used like a metronome for setting a steady beat against which an overdub recording can be made. रिदम गीतपट्टा निवडलेल्या टेम्पोमध्ये नियमित डाळींचा क्रम तयार करतो, निवडक आवाज वापरुन प्रति बार मारतो आणि एकतर बारची संख्या किंवा गीतपट्टा कालावधी. हे स्थिर बीट सेट करण्यासाठी मेट्रोनोमसारखे वापरले जाऊ शकते ज्याच्या विरूद्ध ओव्हरडब ध्वनीमुद्रणिंग केले जाऊ शकते.

६ ६ Tempo (bpm) टेम्पो (बीपीएम)
७ ७ The total number of beats, (clicks) per minute, default is १२०. बीट्सची एकूण संख्या (क्लिक) प्रति मिनिट, पूर्वनिर्धारित १२० आहे.
८ ८ Beats per bar प्रति बार बीट्स
९ ९ The first beat of each bar is always louder than the remaining beats in the bar. So for example, three beats per bar will sound like a waltz in ३/४ time signature, and four beats per bar (the default) will sound like the ४/४ of a march or like most pop songs. प्रत्येक बारचा पहिला ठोका बारमधील उर्वरित बीट्सपेक्षा नेहमीच जोरात असतो. उदाहरणार्थ, प्रति बार तीन बीट्स ३/४ वेळ सहीमध्ये वॉल्ट्जसारखे वाटतील आणि प्रति बार चार बीट्स (पूर्वनिर्धारित) मार्चच्या ४/४ सारखे किंवा बर्‍याच पॉप गाण्यांसारखे दिसतील.
१० १० Swing amount स्विंग रक्कम
११ ११ When set to zero, each beat has the exact length specified by the Tempo (beats per minute). When set to a non-zero amount, alternate beats are delayed or advanced to give a swing feel. At maximum / minimum settings the rhythm plays with triplet timing. शून्यावर सेट केल्यास, प्रत्येक बीटमध्ये टेम्पोने निर्दिष्ट केलेली अचूक लांबी असते (प्रति मिनिट बीट्स). शून्य-नसलेल्या रकमेवर सेट केल्यावर स्विंग अनुभूती देण्यासाठी वैकल्पिक बीट्स उशीर होतो किंवा उन्नत होतो. जास्तीत जास्त / किमान समायोजनमध्ये ताल तिप्पट वेळेसह प्ले होते.
१२ १२ Number of bars बारची संख्या
१३ १३ Beats per bar is repeated for the chosen number of bars. The default is १६ bars. The combination of tempo, beats per bar and number of bars determines the length of the generated track unless the rhythm track duration is specified (see the next control). निवडलेल्या बारसाठी प्रति बार बीट्सची पुनरावृत्ती होते. पूर्वनिर्धारित १६ बार आहे. लिप गीतपट्टाचा कालावधी निर्दिष्ट केल्याशिवाय टेम्पो, बीट्स प्रति बार आणि बारची संख्या एकत्रित गीतपट्टाची लांबी निर्धारित करते (पुढील नियंत्रण पहा).
१४ १४ Rhythm track duration ताल गीतपट्टा कालावधी
१५ १५ If you enter a value into this time control, the generated rhythm track will be at or slightly longer than this duration. आपण या वेळेच्या नियंत्रणामध्ये मूल्य प्रविष्ट केल्यास व्युत्पन्न केलेला ताल गीतपट्टा या कालावधीपेक्षा थोडा किंवा जास्त लांब असेल.
१६ १६ The end of the track is extended into a whole bar if the entered duration does not produce a complete final bar. प्रविष्ट कालावधी पूर्ण अंतिम बार तयार करत नसल्यास गीतपट्टाचा शेवट संपूर्ण बारमध्ये वाढविला जातो.
१७ १७ Start time offset प्रारंभ वेळ ऑफसेट
१८ १८ Makes the rhythm track start at a later point on the Timeline than the very beginning (zero seconds). The maximum is ३० seconds and the default is zero. अगदी सुरुवातीच्या (शून्य सेकंद) वेळेपेक्षा वेळलाइनवर नंतरच्या वेळी लय गीतपट्टाची सुरूवात करते. कमाल ३० सेकंद आणि पूर्वनिर्धारित शून्य आहे.
१९ १९ Beat sound बीट आवाज
२० २० Choose which sound to use for the beats. The default is "Metronome tick". बीट्ससाठी कोणता आवाज वापरायचा ते निवडा. पूर्वनिर्धारित "मेट्रोनोम टिक" आहे.
२१ २१ MIDI pitch of strong beat जोरदार बीटची मिडी खेळपट्टी
२२ २२ The pitch of the first beat in each bar. The MIDI values indicate what pitch to use. Examples: प्रत्येक बारमधील प्रथम बीटची खेळपट्टी. एमआयडीआय मूल्ये कोणती पिच वापरायची हे दर्शवितात. उदाहरणे:
२३ २३ C-notes are २४, ३६, ४८, ६० (middle C), ७२, ८४, ९६, १०८ सी-नोट्स २४, ३६, ४८, ६० (मध्यम सी), ७२, ८४, ९६, १०८ आहेत
२४ २४ C# (C sharp) above middle C is ६१ सी # (सी शार्प) मधल्या सी च्या वर ६१ आहे
२५ २५ The default is ९२ (G#). पूर्वनिर्धारित ९२ (जी #) आहे.
२६ २६ MIDI pitch of weak beat कमकुवत ताची मिडी खेळपट्टी
२७ २७ The pitch of the remaining beats in each bar. The default is ८० (G# an octave below the strong click). प्रत्येक बारमधील उर्वरित बीट्सची खेळपट्टी. पूर्वनिर्धारित ८० (मजबूत क्लिक खाली जी # एक आठवडा) आहे.
२८ २८ Tips: टिपा:
२९ २९ Once created, the track can be edited (for example, its volume changed) like any audio track. एकदा तयार झाल्यावर, कोणत्याही ध्वनी गीतपट्टाप्रमाणे गीतपट्टा संपादित केला जाऊ शकतो (उदाहरणार्थ, त्याचे आवाज बदलले).
३० ३० When exporting your finished work, use the Track Control Panel to either close or mute the Rhythm Track, so that it's not audible in the exported file. आपले तयार काम निर्यात करताना, एकतर लय गीतपट्टा बंद किंवा नि: शब्द करण्यासाठी गीतपट्टा नियंत्रण पॅनेल वापरा, जेणेकरून ती निर्यात केलेल्या धारिकामध्ये ऐकू न येईल.
