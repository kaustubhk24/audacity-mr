४ ४ The Mute/Unmute submenu enables you to mute or unmute all of the tracks in your project in one go, without having to use the buttons Mute and Unmute in each track's Track Control Panel. निःशब्द / सशब्द सबमेनू आपल्याला प्रत्येक गीतपट्टाच्या गीतपट्टा कंट्रोल पॅनेलमधील निःशब्द आणि सशब्द बटणे न वापरता एकाच वेळी आपल्या प्रकल्पमधील सर्व गीतपट्टा निःशब्द किंवा सशब्द करण्यास सक्षम करते.
५ ५ Click on the image to learn more अधिक जाणून घेण्यासाठी प्रतिमेवर क्लिक करा
६ ६ If a track is muted, it cannot be heard in the project and is not exported from it. एखादा गीतपट्टा नि: शब्द केल्यास तो प्रकल्पात ऐकू येत नाही आणि त्यातून निर्यात केला जात नाही.
७ ७ The track mute and solo buttons are usually used to change which tracks are muted, rather than these menu items. गीतपट्टा निःशब्द आणि एकल बटणे सामान्यत: या मेनू आयटमऐवजी कोणते गीतपट्टा नि: शब्द केले जातात ते बदलण्यासाठी वापरली जातात.
८ ८ Mute All Tracks  Ctrl + U सर्व गीतपट्टा Ctrl + U नि: शब्द करा
९ ९ Mutes all the audio tracks in the project, as if you had used the Mute buttons from the Track Control Panel on each track. प्रकल्पमधील सर्व ध्वनी गीतपट्टा नि: शब्द करते, जसे की आपण प्रत्येक गीतपट्टावरील गीतपट्टा नियंत्रण पॅनेलमधील निःशब्द बटणे वापरली आहेत.
१० १० Unmute All Tracks  Ctrl + Shift + U सर्व गीतपट्टा सशब्द करा Ctrl + Shift + U
११ ११ Unmutes all the audio tracks in the project, as if you had used the Mute buttons from the Track Control Panel on each track. प्रकल्पमधील सर्व ध्वनी गीतपट्टा सशब्द करा, जणू की आपण प्रत्येक गीतपट्टावरील गीतपट्टा नियंत्रण पॅनेलमधील निःशब्द बटणे वापरली असतील.
