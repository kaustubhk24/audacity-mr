१ १ Tutorial - Punch-in repair of recordings प्रशिक्षण - ध्वनीमुद्रणिंगची पंच-इन दुरुस्ती

४ ४ This mini-tutorial describes a simple technique for making a "punch-in" to make a correction to a part of a recording., useful when you make a short fluff or cough. या मिनी-प्रशिक्षणमध्ये ध्वनीमुद्रणिंगच्या भागामध्ये सुधारणा करण्यासाठी "पंच-इन" बनविण्याच्या सोप्या तंत्राचे वर्णन केले आहे.
५ ५ Good Practice Tip: Always make a backup copy of a recording by exporting it as WAV or FLAC, immediately after you stop the recording, before you work on the punch-in. चांगली सराव टिप: पंच-इन वर काम करण्यापूर्वी ध्वनीमुद्रणिंगची प्रत काढल्यानंतर लगेच ध्वनीमुद्रणिंगची डब्ल्यूएव्ही किंवा एफएलएसी म्हणून निर्यात करुन त्याची नोंद घ्या.
६ ६ To "punch-in" a correction to a recorded track, we use a new track for each take. This allows us to hear a "lead in" and "lead out" (pre-roll / post roll) and, if necessary, to make multiple attempts at correcting, from which we can select and use the one that we like best. ध्वनीमुद्रण केलेल्या गीतपट्टामध्ये सुधारणा करण्यासाठी "पंच-इन" करण्यासाठी आम्ही प्रत्येक टेकसाठी नवीन गीतपट्टा वापरतो. हे आम्हाला "लीड इन" आणि "लीड आउट" (प्री-रोल / पोस्ट रोल) ऐकण्याची आणि आवश्यक असल्यास दुरुस्त करण्याचे अनेक प्रयत्न करण्यास अनुमती देते, ज्यामधून आम्हाला आमच्या आवडीच्या निवडीचा वापर करू शकतो.
७ ७ Backup the track गीतपट्टाचा बॅकअप घ्या
८ ८ The first thing to do is to make a backup copy of the track, so that if you mess up, you have something to go back to. प्रथम आपण गीतपट्टाची बॅकअप प्रत बनविणे म्हणजे आपण गोंधळल्यास आपल्याकडे परत जाण्यासाठी काहीतरी आहे.
९ ९ In this illustration the track has been duplicated, using the Ctrl + D shortcut, and muted या चित्रात गीतपट्टा Ctrl + D शॉर्टकट वापरून डुप्लिकेट केले गेले आहे आणि निःशब्द केले गेले आहे
१० १० using the Mute button. The backup copy is muted so that it won't play. निःशब्द बटण वापरुन. बॅकअप प्रत नि: शब्द केली आहे जेणेकरून ती प्ले होणार नाही.
११ ११ You may find it useful to collapse this track to save screen space by clicking the Track Collapse button . गीतपट्टा संकुचित करा बटणावर क्लिक करून स्क्रीनची बचत करण्यासाठी हा गीतपट्टा कोसळणे आपल्याला उपयुक्त ठरू शकेल.
१२ १२ In the example the part to be changed is marked with a label, which is not really necessary, but is for illustration purposes. उदाहरणात बदलला जाणारा भाग लेबलने चिन्हांकित केला आहे, जो खरोखर आवश्यक नाही, परंतु स्पष्टीकरण हेतूंसाठी आहे.
१३ १३ The "bad part" has been silenced. The easiest way to do this is to select the bad part, then use the Ctrl + L shortcut. "वाईट भाग" शांत केला आहे. सर्वात सोपा मार्ग म्हणजे खराब भाग निवडणे, त्यानंतर Ctrl + L शॉर्टकट वापरा.
१४ १४ But only do this in the working copy of the track - be careful not to edit the muted backup copy. परंतु केवळ गीतपट्टाच्या कार्यरत प्रतिमध्ये हे करा - नि: शब्द बॅकअप प्रत संपादित न करण्याची काळजी घ्या.
१५ १५ Overdub ओव्हरडब
१६ १६ Ensure that you have Audacity set for overdubbing in the Recording Preferences (this is the default setting). आपणास ध्वनीमुद्रणिंग प्राधान्यांमध्ये ओव्हरडबिंगसाठी ओड्यासिटी सेट असल्याची खात्री करा (ही पूर्वनिर्धारित सेटिंग आहे).
१७ १७ Also ensure there that "Software Playthrough" is turned off (default setting). तेथेही हे सुनिश्चित करा की "सॉफ्टवेअर प्लेथ्रू" बंद आहे (पूर्वनिर्धारित सेटिंग).
१८ १८ As you will be using overdubbing, ensure that you have made the necessary Latency correction setting आपण ओव्हरडबिंग वापरत असताना, आपण आवश्यक लेटन्सी दुरुस्ती सेटिंग केली असल्याची खात्री करा
१९ १९ Lead-in Lead-out लीड-इन लीड-आउट
२० २० Next you need to extend the selection a bit so that there is some lead-in/lead-out (pre-roll/post-roll): पुढे आपल्याला निवड थोडी वाढविणे आवश्यक आहे जेणेकरून तेथे काही लीड-इन / लीड-आउट (प्री-रोल / पोस्ट-रोल) असेलः
२१ २१ Make the punch-in पंच-इन करा
२२ २२ And now you are आणि आता आपण आहात
२३ २३ ready to record the "drop in" (punch-in). "ड्रॉप इन" (पंच-इन) ध्वनीमुद्रण करण्यास सज्ज.
२४ २४ Tip: For working with multi-track projects, it can be useful to set the Solo button behavior to "Multi-track" in Tracks Behaviors Preferences टीपः मल्टी-गीतपट्टा प्रकल्पसह कार्य करण्यासाठी, गीतपट्टा वर्तणूक प्राधान्यांमध्ये "मल्टी-गीतपट्टा" वर सोलो बटण वर्तन सेट करणे उपयुक्त ठरू शकते.
२५ २५ You will need to record on a new track, so just hold down the Shift button and the Record button will change to the alternate आपल्याला नवीन गीतपट्टावर ध्वनीमुद्रण करणे आवश्यक आहे, म्हणून फक्त शिफ्ट बटण दाबून ठेवा आणि ध्वनीमुद्रण बटण वैकल्पिकात बदलेल
२६ २६ Record New Track button. नवीन गीतपट्टा बटण ध्वनीमुद्रण करा.
२७ २७ Now click that Record New Track button (or use the Shift + R shortcut) and Audacity will start recording on a new track, and will stop automatically when it reaches the end of the selection. आता त्या नवीन गीतपट्टा ध्वनीमुद्रण बटणावर क्लिक करा (किंवा शिफ्ट + आर शॉर्टकट वापरा) आणि ओड्यासिटी नवीन गीतपट्टावर ध्वनीमुद्रणिंग सुरू करेल आणि जेव्हा ते निवडीच्या शेवटी पोहोचते तेव्हा आपोआप थांबेल.
२८ २८ Audition your punch-in repair आपल्या पंच-इन दुरुस्तीचे ऑडिशन करा
२९ २९ Now is the time to listen to the punch-in repair you have made. आपण केलेली पंच-इन दुरुस्ती ऐकण्याची वेळ आता आली आहे.
३० ३० Select around your punch-in repair (leaving the backup track muted) and press the आपल्या पंच-इन दुरुस्तीभोवती निवडा (बॅकअप गीतपट्टा नि: शब्द सोडून) आणि दाबा
३१ ३१ play button. प्ले बटण.
३२ ३२ If you are happy with the repair you can delete the backup track from the project. जर आपण दुरुस्तीबद्दल आनंदी असाल तर आपण प्रकल्पातील बॅकअप गीतपट्टा हटवू शकता.
३३ ३३ Optional extras पर्यायी अतिरिक्त
३४ ३४ You don't have to mix down the tracks as they will be mixed down automatically on export (muted tracks are not included in the mix), but if you want to mix the tracks, select the tracks that you want to mix down and use Tracks > Mix > Mix and Render. आपल्याला गीतपट्टा खाली मिसळण्याची आवश्यकता नाही कारण ते निर्यातीवर आपोआप मिसळले जातील (नि: शब्द गीतपट्टा मिक्समध्ये समाविष्ट नाहीत), परंतु जर आपल्याला गीतपट्टा मिसळायचे असतील तर आपण तयार करू इच्छित गीतपट्टा निवडा आणि वापरा गीतपट्टा> मिक्स> मिक्स आणि रेंडर.
३५ ३५ If you want to practice the drop-in before recording it, just use the आपण ध्वनीमुद्रण करण्यापूर्वी ड्रॉप-इनचा सराव करू इच्छित असल्यास, फक्त वापरा
३६ ३६ play button space rather than the record button. ध्वनीमुद्रण बटणाऐवजी बटण जागा प्ले करा.
३७ ३७ After the recording stops, press Space to listen back to it. If you're not happy, us shortcut Ctrl+Z to undo. ध्वनीमुद्रणिंग थांबल्यानंतर, हे ऐकण्यासाठी Space दाबा. आपण आनंदी नसल्यास, पूर्ववत करण्यासाठी आम्ही Ctrl + Z शॉर्टकट.
३८ ३८ If your timing is slightly off, you can use the Time Shift tool, in the Tools Toolbar, to adjust the position of the new clip. जर आपला वेळ थोडासा बंद असेल तर आपण नवीन क्लिपची स्थिती समायोजित करण्यासाठी साधन्स साधनपट्टीमध्ये वेळ शिफ्ट साधन वापरू शकता.
३९ ३९ The dropped in recording has some lead-in/lead out silence, which may contain noise. That can be trimmed, faded, or fully silenced if required. ध्वनीमुद्रणिंगमध्ये सोडण्यात काही लीड-इन / लीड आउट शांतता असते, ज्यात आवाज असू शकतो. आवश्यक असल्यास ते सुव्यवस्थित, फिकट किंवा पूर्णपणे शांत केले जाऊ शकते.
४० ४० For some types of recording, crossfading between the original recording and the punch track can help to create seamless edits. काही प्रकारच्या ध्वनीमुद्रणिंगसाठी, मूळ ध्वनीमुद्रणिंग आणि पंच गीतपट्टा दरम्यान क्रॉसफॅडिंग अखंड संपादने तयार करण्यात मदत करू शकते.
