४ ४ Keyboard Preferences lists, searches for and sets the keyboard shortcuts which can be used for numerous commands in Audacity as an alternative to using Menus, Toolbars or mouse gestures. A full list of the commands with their default shortcut key bindings is available at Keyboard Shortcut Reference. कीबोर्ड प्राधान्ये सूची, मेनू, साधनपट्टी किंवा माऊस जेश्चरचा पर्याय म्हणून ओड्यासिटीमध्ये असंख्य आदेशांसाठी वापरल्या जाणार्‍या कीबोर्ड शॉर्टकट शोधतात आणि सेट करतात. कीबोर्ड शॉर्टकट संदर्भात पूर्वनिर्धारित शॉर्टकट की बाइंडिंगसह आदेशांची पूर्ण यादी उपलब्ध आहे.
५ ५ The default set of shortcut key bindings can be added to, changed or removed here if desired. शॉर्टकट की बाइंडिंगचा पूर्वनिर्धारित सेट इच्छित असल्यास जोडू, बदलला किंवा काढला जाऊ शकतो.
६ ६ For menu commands that are enabled (not grayed-out) if you Shift and click on the submenu item, instead of executing the command the Keyboard Preferences opens at the chosen command. मेनू आज्ञासाठी जे सक्षम आहेत (राखाडी-आऊट नाहीत) जर आपण शिफ्ट आणि सबमेनू आयटमवर क्लिक केल्यास आज्ञा कार्यान्वित करण्याऐवजी निवडलेल्या आज्ञावर कीबोर्ड प्राधान्यता उघडेल.
७ ७ Accessed by: Edit > Preferences > Keyboard याद्वारे प्रवेशः संपादित करा> प्राधान्ये> कीबोर्ड

११ ११ View by द्वारा पहा
१२ १२ There are three ways of viewing the shortcut keys or "bindings" which you can select by clicking on the appropriate radio button. The view chosen when you click OK to close Preferences is remembered when you access Keyboard Preferences next time. शॉर्टकट की किंवा “बाइंडिंग” पाहण्याचे तीन मार्ग आहेत जे आपण योग्य रेडिओ बटणावर क्लिक करून निवडू शकता. आपण प्राधान्ये बंद करण्यासाठी ओके क्लिक करता तेव्हा निवडलेले दृश्य आपण पुढच्या वेळी कीबोर्ड प्राधान्यांमध्ये प्रवेश करता तेव्हा लक्षात ठेवले जाईल.
१३ १३ Tree झाड
१४ १४ This is the default view. On the left-hand side, it lists all the commands to which keyboard shortcuts can be assigned. On the right-hand side, the currently assigned shortcut for each command is shown (if any). The commands are grouped then listed within each group in the same order in which they appear in the Audacity Menu Bar. If a group of items are in a submenu within their Menu Bar, that group appears in the tree indented to right as a sub-group within their group. There is a final "Command" category listed in alphabetical order for commands which do not have a Menu Bar entry. हे पूर्वनिर्धारित दृश्य आहे. डाव्या बाजूस, कीबोर्ड शॉर्टकट नियुक्त केला जाऊ शकतो अशा सर्व आदेशांची सूची. उजवीकडील, प्रत्येक आदेशासाठी सध्या नियुक्त केलेले शॉर्टकट दर्शविला आहे (असल्यास). आज्ञा्स प्रत्येक गटात त्याच क्रमाने सूचीबद्ध केल्या जातात ज्यामध्ये ते ओड्यासिटी मेनू बारमध्ये दिसतात. आयटमचा गट त्यांच्या मेनू बारमध्ये सबमेनूमध्ये असल्यास, तो गट वृक्षात त्यांच्या गटात उप-गट म्हणून उजवीकडे दाबलेला दिसतो. मेनू बार प्रविष्टी नसलेल्या आदेशांच्या वर्णमाला क्रमानुसार अंतिम "आज्ञा" श्रेणी आहे.
१५ १५ The menu groups and sub-groups ("branches" in this "tree" view) always open fully expanded. मेनू गट आणि उप-गट (या "वृक्ष" दृश्यामधील "शाखा") नेहमीच विस्तृतपणे उघडलेले असतात.
१६ १६ You can click on any branch's आपण कोणत्याही शाखेत क्लिक करू शकता
१७ १७ collapse button to collapse the branch. Any branch which is collapsed can be expanded by clicking on its शाखा कोसळण्यासाठी संकुचित बटण. कोसळलेली कोणतीही शाखा त्यावर क्लिक करून वाढविली जाऊ शकते
१८ १८ expand button. बटण विस्तृत करा.
१९ १९ Keyboard use: कीबोर्ड वापर:
२० २० Use Left arrow when on a group or sub-group's title row to collapse the group or sub-group, or when on an item in an expanded group or sub-group to jump to the title row of गट किंवा उप-गटाच्या कोसळण्यासाठी गट किंवा उप-गटाच्या शीर्षक पंक्तीवर किंवा विस्तारित गटाच्या किंवा उपसमूहातील आयटमवर असताना शीर्षक पंक्तीवर जाण्यासाठी डावे बाण वापरा
२१ २१ that group or sub-group. तो गट किंवा उप-गट.
२२ २२ Use Right arrow to expand a collapsed group or sub-group. संकुचित गट किंवा उप-गट विस्तृत करण्यासाठी उजवा बाण वापरा.
२३ २३ Up arrow and Down arrow navigate up and down the items in "Tree" view and also do so in the "Name" and "Key" views. वर आणि बाण वर बाण "ट्री" दृश्यात आयटम वर आणि खाली नेव्हिगेट करतात आणि ते "नाव" आणि "की" दृश्यांमध्ये देखील करतात.
२४ २४ To navigate through all commands and title rows starting with a particular letter, type that letter. विशिष्ट अक्षरापासून प्रारंभ होणार्‍या सर्व आदेश आणि शीर्षक पंक्तींमध्ये नेव्हिगेट करण्यासाठी ते अक्षर टाइप करा.
२५ २५ Search box: शोध बॉक्स:
२६ २६ Typing in the "Tree" view search box looks for that text string in a command name, title row or shortcut. You can enter text in upper or lower case (search is not case-sensitive). "ट्री" व्यू शोध बॉक्समध्ये टाइप करणे आज्ञाचे नाव, शीर्षक पंक्ती किंवा शॉर्टकटमध्ये मजकूर अक्षर शोधते. आपण अप्पर किंवा लोअर केसमध्ये मजकूर प्रविष्ट करू शकता (शोध केस-सेन्सेटिव्ह नाही).
२७ २७ Tree View Search Examples वृक्ष दृश्य शोध उदाहरणे
२८ २८ Entering zoom finds the four Zoom items in View Menu and the Zoom Tool command. झूममध्ये प्रवेश केल्यावर व्यू मेनू आणि झूम साधन आदेशामध्ये चार झूम आयटम आढळतात.
२९ २९ Entering just z finds all menu items or commands that include the letter z, plus shortcuts that include Z. फक्त झेडमध्ये प्रवेश केल्यास सर्व मेनू आयटम किंवा आज्ञा ज्यात अक्षरे झेड समाविष्ट आहेत, तसेच शॉर्टकट ज्यात झेड समाविष्ट आहे.
३० ३० Entering +j finds only shortcuts that include a modified "J", so by default Ctrl + J, Ctrl + Alt + J, + J मध्ये प्रवेश करताना केवळ सुधारित "J" समाविष्ट असलेले शॉर्टकट सापडतात, म्हणून पूर्वनिर्धारित Ctrl + J, Ctrl + Alt + J,
३१ ३१ Alt + J, Alt + J,
३२ ३२ Alt + Shift + J and Shift + J. "Tree" View is the only way to find all shortcuts that use a given modifier letter. अल्ट + शिफ्ट + जे आणि शिफ्ट + जे. दिलेली सुधारक अक्षरे वापरणारे सर्व शॉर्टकट शोधण्याचा "ट्री" व्ह्यू हा एकमेव मार्ग आहे.
३३ ३३ If you have non-default shortcuts and press the Defaults button as below to restore the default shortcut set, the search results for your entered text will be updated. Empty the text in the Search box to restore the complete Tree list. आपल्याकडे पूर्वनिर्धारित नसलेले शॉर्टकट असल्यास आणि पूर्वनिर्धारित शॉर्टकट सेट पुनर्संचयित करण्यासाठी खालील प्रमाणे पूर्वनिर्धारित बटण दाबा तर आपल्या प्रविष्ट केलेल्या मजकूरासाठी शोध परिणाम अद्यतनित केले जातील. संपूर्ण वृक्ष सूची पुनर्संचयित करण्यासाठी शोध बॉक्समधील मजकूर रिक्त करा.
३४ ३४ Name नाव
३५ ३५ "Name" view lists on the left-hand side all the commands in alphabetical order, with no grouping. On the right-hand side, the currently assigned shortcut for each command is shown (if any). "नाव" दृश्य डावीकडील सर्व आदेशांची वर्णमाला क्रमानुसार क्रमवारी आहे, कोणतीही गट न करता. उजवीकडील, प्रत्येक आदेशासाठी सध्या नियुक्त केलेले शॉर्टकट दर्शविला आहे (असल्यास).
३६ ३६ To navigate through all commands starting with a particular letter, type that letter. एका विशिष्ट पत्रापासून सुरू होणार्‍या सर्व आदेशांमधून नेव्हिगेट करण्यासाठी ते अक्षर टाइप करा.
३७ ३७ Search box: शोध बॉक्स:
३८ ३८ Typing in the "Name" view search box works similarly to search in "Tree" view, except that the text string you type is searched for only in command names and title rows and not in shortcuts. This can give more focused search results when you really only want to search in the command name. "नाव" दृश्य शोध बॉक्समध्ये टाइप करणे "वृक्ष" दृश्यामध्ये शोधण्यासाठी असेच कार्य करते, याशिवाय आपण टाइप केलेला मजकूर शब्द केवळ आज्ञा नावे आणि शीर्षक पंक्तींमध्ये शोधला जातो आणि शॉर्टकटमध्ये नाही. जेव्हा आपल्याला खरोखरच आदेश नावामध्ये शोधायचे असेल तेव्हा हे अधिक केंद्रित शोध परिणाम देऊ शकेल.
३९ ३९ Name View Search Examples नाव पहा शोध उदाहरणे
४० ४० Entering Shift finds the Effect menu item "Sliding Time Scale / Pitch Shift" and the "Time Shift Tool" command, but excludes all the shortcuts that include the Shift modifier. शिफ्टमध्ये प्रवेश करताना प्रभाव मेनू आयटम "स्लाइडिंग वेळ स्केल / पिच शिफ्ट" आणि "वेळ शिफ्ट साधन" आज्ञा आढळतो, परंतु शिफ्ट सुधारक समाविष्ट करणारे सर्व शॉर्टकट वगळते.
४१ ४१ Entering just b finds all menu items or commands that include the letter "b", but does not find the "Play to Selection" command with its default shortcut of "B". "Tree" view would find it but the quickest way to find what the "B" command does would be to type it into the "Hotkey" box when in "Key" view (see next section). फक्त बी प्रविष्ट केल्याने सर्व मेनू आयटम किंवा आज्ञा आढळतात ज्यामध्ये "बी" अक्षराचा समावेश आहे, परंतु "बी" च्या पूर्वनिर्धारित शॉर्टकटसह "प्ले टू सिलेक्शन" आज्ञा सापडत नाही. "ट्री" व्ह्यू पहायला मिळेल परंतु "की" व्ह्यूमध्ये असताना "बी" आज्ञा काय करेल हे शोधण्याचा सर्वात वेगवान मार्ग म्हणजे "हॉटकी" बॉक्समध्ये टाइप करा (पुढील विभाग पहा).
४२ ४२ Key की
४३ ४३ "Key" view is the easiest to use to ascertain which keys or key combinations are currently assigned and which are not. By default it lists on the left-hand side all the currently assigned shortcuts in alphabetical order of the shortcut, followed by all the commands that currently have no shortcut assigned to them (in alphabetical order of the command). सध्या कोणत्या की किंवा की संयोजन एकत्रित केल्या आहेत आणि कोणत्या नाहीत याची खात्री करण्यासाठी "की" दृश्य वापरणे सर्वात सुलभ आहे. पूर्वनिर्धारितनुसार शॉर्टकटच्या वर्णक्रमानुसार सध्या नियुक्त केलेले सर्व शॉर्टकट डाव्या बाजूस सूचीबद्ध होते, त्यापुढे सर्व आज्ञा्स ज्याना त्यांना नेमलेला कोणताही शॉर्टकट नाही (आज्ञाच्या अक्षराच्या क्रमानुसार).
४४ ४४ To navigate through all shortcuts starting with a particular letter, type that letter. विशिष्ट अक्षरापासून प्रारंभ होणार्‍या सर्व शॉर्टकटमध्ये नेव्हिगेट करण्यासाठी, ते अक्षर टाइप करा.
४५ ४५ The "Key" view search box is called Hotkey. To use this, click in or Tab into the box then press the shortcut combination you are searching for, just as you would press the shortcut combination you want to set in the shortcut input box. You can also search for one letter only in the shortcut itself (this excludes searching for that letter in the modifier). "की" दृश्य शोध बॉक्सला हॉटकी म्हणतात. हे वापरण्यासाठी, बॉक्समध्ये टॅब क्लिक करा किंवा नंतर आपण शोधत असलेले शॉर्टकट संयोजन दाबा, जसे की आपण शॉर्टकट इनपुट बॉक्समध्ये सेट करू इच्छित शॉर्टकट संयोजन दाबा. आपण केवळ शॉर्टकटमध्येच एका पत्राचा शोध घेऊ शकता (हे त्या पत्रासाठी सुधारकातील शोध वगळता).

५३ ५३ you can then use the Delete or any other method to empty the box and display the complete list. आपण बॉक्स रिक्त करण्यासाठी आणि संपूर्ण यादी प्रदर्शित करण्यासाठी डिलीट किंवा इतर कोणत्याही पद्धतीचा वापर करू शकता.
५४ ५४ Input box, Set and Clear buttons इनपुट बॉक्स, सेट आणि साफ बटणे
५५ ५५ To change or add a shortcut binding: शॉर्टकट बाइंडिंग बदलण्यासाठी किंवा जोडण्यासाठी:
५६ ५६ Click in the list window to select the item you want to change. The entry will be highlighted and the current key combination for that command (if any) will now appear in the input box to left of the Set button. आपण बदलू इच्छित आयटम निवडण्यासाठी सूची विंडोमध्ये क्लिक करा. प्रविष्टी ठळकपणे दर्शविली जाईल आणि त्या आज्ञासाठी सध्याचे की संयोजन (असल्यास) सेट बटणाच्या डावीकडील इनपुट बॉक्समध्ये दिसेल.
५७ ५७ Either click in the input box, or press Tab once to switch focus to it. एकतर इनपुट बॉक्समध्ये क्लिक करा किंवा त्यावर लक्ष केंद्रित करण्यासाठी एकदा टॅब दाबा.
५८ ५८ Choose your binding by either holding any modifier key (or combination of modifiers) then pressing any non-modifier key, or press any non-modifier key on its own. Valid modifiers are: एकतर कोणतीही मॉडिफायर की (किंवा सुधारकांचे संयोजन) धरून आपले बंधन निवडा, त्यानंतर कोणतीही नॉन-मॉडिफायर की दाबून, किंवा कोणतीही स्वत: ची नस-सुधारक की दाबा. वैध सुधारक हे आहेत:
५९ ५९ Shift शिफ्ट
६० ६० Ctrl ( ⌘ on a Mac) Ctrl (Mac मॅक वर)
६१ ६१ Alt ( Option on a Mac) Alt (मॅकवरील पर्याय)
६२ ६२ Altgr or Right Alt (on Windows keyboards, this key sends a combined Ctrl + Alt modifier and can thus be used as an alternative to call any binding that includes Ctrl + Alt). </ul> Altgr किंवा राइट Alt (विंडोज कीबोर्डवर, ही की एकत्रित Ctrl + Alt सुधारक पाठवते आणि अशा प्रकारे Ctrl + Alt समाविष्ट असलेल्या कोणत्याही बंधन्यास कॉल करण्यासाठी पर्यायी म्हणून वापरले जाऊ शकते). </ul>

Your chosen key or key combination will now replace the previous content of the input box. उदाहरणार्थ, शिफ्ट +६ किंवा सीटीआरएल + शिफ्ट + पी हे दोन्ही वैध बंधन आहेत. आपली निवडलेली की किंवा की संयोजन आता इनपुट बॉक्समधील मागील सामग्री पुनर्स्थित करेल.
६४ ६४ Click Set to apply your desired change. आपला इच्छित बदल लागू करण्यासाठी सेट क्लिक करा.
६५ ६५ If your chosen shortcut is already assigned to a command, Audacity will show a dialog asking you to confirm the change. Click OK to assign your chosen shortcut to the chosen command. This will remove the shortcut for the command that originally used it, leaving that command with no shortcut. Click Cancel to make no change of shortcut. जर आपला निवडलेला शॉर्टकट आधीपासूनच आज्ञाला नियुक्त केला असेल तर ओड्यासिटी आपल्याला एक बदल दर्शविण्यास सांगत एक संवाद दर्शवेल. निवडलेल्या आज्ञाला तुमचा निवडलेला शॉर्टकट लागू करण्यासाठी ओके क्लिक करा. हे आज्ञाला शॉर्टकट न ठेवता मूळ रूपात वापरलेल्या आज्ञाचे शॉर्टकट काढून टाकेल. शॉर्टकट न बदलण्यासाठी रद्द करा क्लिक करा.

६७ ६७ If you omit this step then the changes will be aborted. आपण हे चरण वगळल्यास बदल निरस्त केले जातील.
६८ ६८ Clear: To remove a shortcut binding, select the item as in step १ above, then press Clear. साफ करा: शॉर्टकट बाइंडिंग काढण्यासाठी, वरील चरण १ प्रमाणे आयटम निवडा, नंतर साफ दाबा.

७० ७० Export..., Import... and Defaults buttons निर्यात करा ..., आयात करा ... आणि पूर्वनिर्धारित बटणे
७१ ७१ Export... and Import...: You can export and import shortcut bindings (for example to keep a safe copy of your custom settings, or to transfer them to Audacity on another machine). The Export... button exports your bindings as an XML file, and Import... imports the XML file back again. You can also edit the XML file in any text editor and re-import it, as an alternative to editing the shortcuts in Preferences. निर्यात करा ... आणि आयात करा: आपण शॉर्टकट बाइंडिंग्ज निर्यात आणि आयात करू शकता (उदाहरणार्थ आपल्या सानुकूल समायोजनची सुरक्षित प्रत ठेवण्यासाठी किंवा दुसर्‍या मशीनवर ओड्यासिटीमध्ये हस्तांतरित करण्यासाठी). निर्यात ... बटण आपल्या बाइंडिंगची एक्सएमएल धारिका म्हणून निर्यात करते आणि आयात ... एक्सएमएल धारिका परत आयात करते. प्राधान्यांमधील शॉर्टकट संपादित करण्यासाठी पर्यायी म्हणून आपण कोणत्याही मजकूर संपादकात एक्सएमएल धारिका देखील संपादित करू शकता आणि ती पुन्हा आयात करू शकता.
७२ ७२ Audacity does not check for shortcuts which are allocated to more than one command when importing the file, so edit carefully. The only entries you can usefully edit are the actual key bindings themselves which are in the "key" string at the end of each command. For example, to set the key binding for "New" to Shift + N, the command in the XML file should read:<command name="New" label="New" key="Shift+N"/>Be sure not to delete the closing "/>" characters at the end of the command. ओड्यासिटी शॉर्टकटची तपासणी करत नाही जे धारिका आयात करताना एकापेक्षा अधिक आदेशांना वाटप केले आहे, म्हणून काळजीपूर्वक संपादित करा. केवळ आपण प्रविष्ट करू शकता अशा नोंदी ही वास्तविक की बाइंडिंग्ज आहेत जी प्रत्येक आदेशाच्या शेवटी "की" स्ट्रिंगमध्ये आहेत. उदाहरणार्थ, "नवीन" शी शिफ्ट + एन वर की बाइंडिंग सेट करण्यासाठी, एक्सएमएल धारिका मधील आज्ञा वाचली पाहिजे: <आज्ञा नाव = "नवीन" लेबल = "नवीन" की = "शिफ्ट + एन" /> खात्री नाही आदेशाच्या शेवटी बंद होणारे "/>" वर्ण हटविण्यासाठी.
७३ ७३ Defaults: There are two sets of default shortcuts available in Audacity. पूर्वनिर्धारितः ओड्यासिटीमध्ये पूर्वनिर्धारित शॉर्टकटचे दोन संच उपलब्ध आहेत.
७४ ७४ The default set is "Standard". पूर्वनिर्धारित संच "मानक" आहे.
७५ ७५ This set is a reduced set to simplify the set of shortcuts somewhat and to provide greater flexibilty for users who wish to create their own shortcuts. शॉर्टकटचा संच काहीसा सुलभ करण्यासाठी आणि स्वत: चा शॉर्टकट तयार करू इच्छिणा users्या वापरकर्त्यांना जास्त फ्लेक्सबिलिटी प्रदान करण्यासाठी हा सेट कमी केलेला संच आहे.
७६ ७६ The other available set of shortcuts is "Full". शॉर्टकटचा इतर उपलब्ध संच "पूर्ण" आहे.
७७ ७७ This set is, as the name suggests the full and extensive set of shortcuts that were available in Audacity २.१.३ and earlier. हा सेट आहे, जसे नावाने ओड्यासिटी २.१..३ आणि पूर्वीच्या काळात उपलब्ध शॉर्टकटचा पूर्ण आणि विस्तृत सेट सूचित केला आहे.
७८ ७८ You can choose to revert to the full set of shortcuts by selecting "Full" from the dropdown menu accessed from the Defaults button in the Keyboard Preferences dialog. आपण कीबोर्ड प्राधान्ये संवादातील पूर्वनिर्धारित बटणामधून प्रवेश केलेल्या ड्रॉपडाउन मेनूमधून "पूर्ण" निवडून शॉर्टकटच्या पूर्ण संचावर परत जाणे निवडू शकता.
७९ ७९ You can use the Defaults button to switch between the two provided default sets of shortcuts at any time. शॉर्टकटच्या दोन प्रदान केलेल्या पूर्वनिर्धारित सेट दरम्यान कधीही स्विच करण्यासाठी आपण पूर्वनिर्धारित बटण वापरू शकता.
८० ८० This button is also handy if you are making custom shortcuts and decide you have not got your changes right and want to start over again. आपण सानुकूल शॉर्टकट बनवत असल्यास आणि आपले बदल योग्य न झाल्याचे आणि पुन्हा प्रारंभ करू इच्छित असल्यास हे बटण सुलभ आहे.
८१ ८१ OK and Cancel buttons ठीक आणि रद्द करा बटणे
८२ ८२ OK: When you have completed your edits press the OK button to save the new shortcut bindings and exit the Preferences dialog. ठीक: आपण आपली संपादने पूर्ण केल्यावर नवीन शॉर्टकट बाइंडिंग जतन करण्यासाठी ओके बटण दाबा आणि प्राधान्ये संवादातून बाहेर पडा.
८३ ८३ Cancel: If you press the Cancel button this aborts the editing. All shortcuts will remain as they were before you started to edit them and the Preferences dialog will be exited. रद्द करा: आपण रद्द करा बटण दाबल्यास हे संपादन रद्द करते. सर्व शॉर्टकट आपण त्यांचे संपादन करण्यास प्रारंभ करण्यापूर्वी त्याप्रमाणेच राहतील आणि प्राधान्ये संवाद बाहेर पडतील.
