८ ८ When saving a project that depends on other audio files इतर ध्वनी धारिकांवर अवलंबून असलेला एखादा प्रकल्प जतन करताना
९ ९ If you import uncompressed files into your project with Import / Export Preferences set to "Read uncompressed audio files directly from the original (faster)", you create a dependency on those files remaining accessible. This option decides what happens the first time you attempt to save that dependent project. आपण आपल्या प्रकल्पात आयात / निर्यात प्राधान्ये सह संकुचित धारिका आयात केल्यास "मूळ (वेगवान) वरून थेट संकुचित ध्वनी धारिका वाचा" "वर सेट केल्यास आपण प्रवेश करण्यायोग्य त्या धारिकांवर अवलंबन तयार करता. आपण हा अवलंबून असलेला प्रकल्प जतन करण्याचा प्रयत्न करता तेव्हा प्रथम काय होते हे हा पर्याय ठरवितो.
१० १० Copy all audio into project (safest): All imported files that your project depends on will be copied automatically into the saved project. You can then safely move, rename or delete those imported files if you wish. सर्व ध्वनी प्रकल्पमध्ये कॉपी करा (सर्वात सुरक्षित): आपला प्रकल्प ज्या सर्व आयात केलेल्या धारिका्सवर अवलंबून आहे त्या जतन केलेल्या प्रकल्पात स्वयंचलितपणे कॉपी केल्या जातील. त्यानंतर आपली इच्छा असल्यास आपण त्या आयात केलेल्या धारिका सुरक्षितपणे हलवू, पुनर्नामित करू किंवा हटवू शकता.
११ ११ Do not copy any audio: Every project will be saved automatically, but will always depend on the continued presence of the imported files. You must not move, rename or delete those files, or the folder they are in. कोणताही ध्वनी कॉपी करु नका: प्रत्येक प्रकल्प स्वयंचलितपणे जतन होईल, परंतु आयात केलेल्या धारिकांच्या सतत उपस्थितीवर अवलंबून असेल. आपण त्या धारिका किंवा त्यामध्ये असलेले फोल्डर हलविणे, पुनर्नामित करणे किंवा हटविणे आवश्यक नाही.
१२ १२ Ask: The Dependencies Dialog will always appear when you first save a project that depends on imported files. This dialog lets you choose which if any of the dependent files should be copied into the project before saving it. विचारा: आपण आयात केलेल्या धारिकांवर अवलंबून असलेला एखादा प्रकल्प प्रथम जतन करता तेव्हा अवलंबन संवाद नेहमीच दिसून येईल. हा संवाद आपणास यापैकी कोणत्याही धारिका जतन करण्यापूर्वी प्रकल्पमध्ये कॉपी करू इच्छित असल्यास ती निवडू देतो.
१३ १३ You can check at any time if your project depends on imported files, and copy them in if desired. To do this, click Help > Diagnostics > Check Dependencies... to open Dependencies Dialog. आपला प्रकल्प आयात केलेल्या धारिकांवर अवलंबून असल्यास आपण कधीही तपासू शकता आणि इच्छित असल्यास त्यामध्ये कॉपी करू शकता. हे करण्यासाठी, मदत> निदान> अवलंबित्व तपासा ... अवलंबित्व संवाद उघडण्यासाठी क्लिक करा.
