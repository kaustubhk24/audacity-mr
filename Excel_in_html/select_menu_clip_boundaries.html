४ ४ A set of commands to enable you to make selections based on any Clips that you may have in the project. आपण प्रकल्पात असलेल्या कोणत्याही क्लिपच्या आधारे निवड करण्यास सक्षम करण्यासाठी आज्ञा्सचा एक सेट.
५ ५ Click on the image to learn more अधिक जाणून घेण्यासाठी प्रतिमेवर क्लिक करा
६ ६ Clips and what you can do with them are explained on page Audacity Tracks and Clips. क्लिप आणि आपण त्यांच्यासह काय करू शकता हे पृष्ठ ओड्यासिटी गीतपट्टा आणि क्लिपवर स्पष्ट केले आहे.
७ ७ Previous Clip Boundary to Cursor मागील क्लिप सीमा ते कर्सर
८ ८ Previous Clip Boundary to Cursor and the three related commands below Cursor to Next Clip Boundary, Previous Clip and Next Clip use the clips in the selected track or tracks. If no tracks are selected, the clips in all tracks are used. मागील क्लिप सीमा ते कर्सर आणि खाली तीन संबंधित आदेश कर्सर ते नेक्स्ट क्लिप सीमा, मागील क्लिप आणि नेक्स्ट क्लिप निवडलेल्या गीतपट्टा किंवा गीतपट्टामध्ये क्लिप वापरतात. कोणतेही गीतपट्टा निवडलेले नसल्यास, सर्व गीतपट्टामधील क्लिप वापरल्या जातील.
९ ९ Selects from the current position of the editing cursor or selection backwards to the previous clip boundary, so extending the selection with each use to include further clips (or blank space) until the start of the track is reached. संपादन कर्सरच्या सद्य स्थितीपासून मागील निवडीच्या सीमेवर निवडीची निवड करते, म्हणून गीतपट्टाचा प्रारंभ होईपर्यंत पुढील क्लिप्स (किंवा रिक्त जागा) समाविष्ट करण्यासाठी प्रत्येक वापरासह निवड वाढविते.
१० १० Example: उदाहरणः
११ ११ Starting with this selection ... या निवडीपासून प्रारंभ करीत आहे ...
१२ १२ after the first command the selection is extended from the editing cursor position to the start of the clip: पहिल्या आदेशानंतर निवड संपादनाच्या कर्सर स्थानापासून क्लिपच्या सुरूवातीस वाढविण्यात आलीः
१३ १३ after the second command the selection is extended to the end of the previous clip: दुसर्‍या आदेशानंतर निवड मागील क्लिपच्या शेवटी वाढविली जाते:
१४ १४ and after the third command the selection is extended to the start of the previous clip: आणि तिसर्‍या आदेशानंतर निवड मागील क्लिपच्या सुरूवातीस वाढविण्यात आलीः
१५ १५ Cursor to Next Clip Boundary पुढील क्लिप सीमेवर कर्सर
१६ १६ Selects from the current position of the editing cursor or selection forwards to the next next clip boundary, so extending the selection with each use to include further clips (or blank space) until the final clip is reached. संपादन कर्सरच्या सद्य स्थितीपासून निवड किंवा पुढील पुढील क्लिप सीमेवरील निवड अग्रेषित करते, जेणेकरून अंतिम क्लिप पूर्ण होईपर्यंत पुढील क्लिप्स (किंवा रिक्त जागा) समाविष्ट करण्यासाठी प्रत्येक वापरासह निवड विस्तृत करा.
१७ १७ Example: उदाहरणः
१८ १८ Starting with this selection ... या निवडीपासून प्रारंभ करीत आहे ...
१९ १९ after the first command the selection is extended from the editing cursor position to the end of the clip: पहिल्या आदेशानंतर निवड संपादनाच्या कर्सर स्थानापासून क्लिपच्या शेवटी वाढविली जाते:
२० २० after the second command the selection is extended to the start of the next clip: दुसर्‍या आदेशानंतर निवड पुढील क्लिपच्या सुरूवातीस वाढविण्यात आलीः
२१ २१ and after the third command the selection is extended to the end of the next clip: आणि तिसर्‍या आदेशानंतर निवड पुढील क्लिपच्या शेवटी वाढविली जाते:
२२ २२ Previous Clip  Alt + , मागील क्लिप Alt +,
२३ २३ Moves or creates a selection so as to fully select in the current clip where the editing cursor or selection is. However if that cursor is already exactly at the start of the current clip or if the current clip is already fully selected, this command will fully select in the previous clip. Further use of Previous Clip then fully selects in the previous clip until the first clip is reached. वर्तमान क्लिपमध्ये जेथे संपादन कर्सर किंवा निवड आहे तेथे पूर्णपणे निवडण्यासाठी निवड हलवते किंवा तयार करते. तथापि, सध्याचा क्लिप सुरू होण्यापूर्वी तो कर्सर आधीपासून किंवा सध्याची क्लिप आधीच निवडलेली असेल तर ही आज्ञा मागील क्लिपमध्ये पूर्णपणे निवडेल. मागील क्लिपचा पुढील वापर नंतर प्रथम क्लिप पूर्ण होईपर्यंत मागील क्लिपमध्ये पूर्णपणे निवडतो.
२४ २४ Example: उदाहरणः
२५ २५ Starting with this selection ... या निवडीपासून प्रारंभ करीत आहे ...
२६ २६ after the first command the clip containing the cursor is selected: पहिल्या आदेशानंतर कर्सर असलेली क्लिप निवडली:
२७ २७ after the second command the previous clip is selected: दुसर्‍या आदेशानंतर मागील क्लिप निवडली:
२८ २८ Subsequent commands will do nothing since the first clip in the track is selected. गीतपट्टामधील प्रथम क्लिप निवडल्या नंतरच्या आदेशांमुळे काहीही होणार नाही.
२९ २९ Next Clip  Alt + . पुढील क्लिप Alt +.
३० ३० Moves or creates a selection so as to fully select in the next clip. If the editing cursor is exactly at the start of a clip, this command selects all of that current clip. पुढील क्लिपमध्ये पूर्णपणे निवडण्यासाठी निवड हलवते किंवा तयार करते. संपादन कर्सर क्लिपच्या सुरूवातीस असल्यास, ही आज्ञा त्या वर्तमान क्लिपची निवड करते.
३१ ३१ Example: उदाहरणः
३२ ३२ Starting with this selection ... या निवडीपासून प्रारंभ करीत आहे ...
३३ ३३ after the first command the next clip is selected: पहिल्या आदेशानंतर पुढील क्लिप निवडली:
३४ ३४ after the second command the next clip is selected: दुसर्‍या आदेशानंतर पुढील क्लिप निवडली:
३५ ३५ Subsequent commands will do nothing since the last clip in the track is selected. गीतपट्टामधील शेवटची क्लिप निवडल्यापासून त्यानंतरच्या आज्ञा्स काहीही करणार नाहीत.
३६ ३६ To move the editing cursor to the start of the next or previous clip without selecting its audio, see the Transport > Cursor to commands for Previous Clip Boundary and Next Clip Boundary. पुढील ध्वनी किंवा मागील क्लिपच्या ध्वनीची निवड न करता संपादन कर्सरच्या प्रारंभावर हलविण्यासाठी, मागील क्लिप सीमा आणि पुढील क्लिप सीमारेषेच्या आदेशासाठी परिवहन> कर्सर पहा.
