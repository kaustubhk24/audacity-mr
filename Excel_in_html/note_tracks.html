४ ४ Note Tracks contain data from MIDI (and Allegro) files - these can be imported with the File > Import > MIDI... command. टीप गीतपट्टामध्ये एमआयडीआय (आणि अल्लेग्रो) धारिकांमधील डेटा असतो - हे धारिका> आयात> एमआयडीआय ... आदेशासह आयात केले जाऊ शकतात.
५ ५ Note Tracks can be useful to visually compare a given MIDI (or Allegro) file with a sampled audio file (such as WAV) of the same music. टीप त्याच संगीताच्या नमुना असलेल्या ध्वनी धारिका (जसे की डब्ल्यूएव्ही) सह दिलेल्या एमआयडीआय (किंवा अ‍ॅलेग्रो) धारिकाची दृष्टि तुलना करण्यास गीतपट्टा उपयुक्त ठरू शकतात.
६ ६ They can be cut-and-paste edited. ते संपादित आणि कट-पेस्ट केले जाऊ शकतात.
७ ७ They can then be exported using the File > Export > Export MIDI... command. When exporting so as to overwrite an existing GRO, MID or MIDI file, the original file is renamed with ".bak" suffix, for example "keyboards.mid.bak". त्यानंतर ते धारिका> निर्यात> निर्यात एमआयडीआय ... आदेशाचा वापर करुन निर्यात करता येऊ शकते. अस्तित्वातील जीआरओ, एमआयडी किंवा एमआयडीआय धारिका अधिलिखित करण्यासाठी निर्यात करताना मूळ धारिकाचे नाव ".bak" प्रत्यय सह बदलले जाते, उदाहरणार्थ "कळफलक. मिड.बाक".

१८ १८ A Note Track एक नोट गीतपट्टा
१९ १९ Note tracks can be created by importing MIDI (or Allegro) files with File > Import > MIDI.... धारिका> आयात> एमआयडीआय सह एमआयडीआय (किंवा द्रुतगतीने) धारिका आयात करुन टीप गीतपट्टा तयार केले जाऊ शकतात.
2० 2० Note carefully that when a MIDI or Allegro is imported you are unlikely to see all of the notes that are in the track with the initial view restricted to two octaves centered around the average note value. काळजीपूर्वक लक्षात घ्या की जेव्हा एखादी एमआयडीआय किंवा अल्लेग्रो आयात केली जाते तेव्हा आपणास सुरुवातीच्या दृश्यासह गीतपट्टामध्ये असलेल्या सर्व नोट्स सरासरी नोट मूल्याच्या भोवती केंद्रित केलेल्या दोन अष्टकांपर्यंत प्रतिबंधित नसतात.
2१ 2१ Observe carefully the little black marks at the bottom right of the image. These indicate that there are additional notes in the Note Track below them. प्रतिमेच्या तळाशी उजवीकडे असलेल्या काळ्या रंगाच्या चिन्हे काळजीपूर्वक पहा. हे सूचित करते की त्यांच्याकडे असलेल्या टीप गीतपट्टामध्ये अतिरिक्त नोट्स आहेत.

2४ 2४ MIDI and Allegro playback मिडी आणि अ‍ॅलेग्रो प्लेबॅक
2५ 2५ Note tracks containing MIDI data (or Allegro data) can now be played in Audacity. एमआयडीआय डेटा (किंवा अल्लेरो डेटा) असलेले टीप गीतपट्टा आता ओड्यासिटीमध्ये प्ले केले जाऊ शकतात.
2६ 2६ Loop Play and Play Cut Preview also work with Note tracks. लूप प्ले आणि प्ले कट पूर्वावलोकन टीप गीतपट्टासह देखील कार्य करते.
2७ 2७ But note that there will no use of the Playback meter while Note tracks are played. परंतु लक्षात घ्या की नोट गीतपट्टा प्ले होत असताना प्लेबॅक मीटरचा वापर होणार नाही.
2८ 2८ In order to achieve MIDI playback on Mac and Linux you will need to install, and run, additional software, a software synthesizer मॅक आणि लिनक्सवर एमआयडीआय प्लेबॅक साध्य करण्यासाठी आपल्याला अतिरिक्त अनुप्रयोग, एक सॉफ्टवेयर सिंथेसाइजर स्थापित करणे आणि चालविणे आवश्यक आहे.

३2 ३2 Editing Note Tracks संपादन टीप गीतपट्टा
३३ ३३ Only limited editing can be performed on Note tracks such as cutting, pasting and deleting. केवळ कटिंग, पेस्ट करणे आणि हटविणे यासारख्या टीप गीतपट्टावर मर्यादित संपादन केले जाऊ शकते.
३४ ३४ Track Control Panel गीतपट्टा नियंत्रण पॅनेल
३५ ३५ The Track Control Panel at the left of the note track has controls for this track. Left-clicking other than on a control performs track selection and move as follows. टीप गीतपट्टाच्या डावीकडील गीतपट्टा नियंत्रण पॅनेलमध्ये या गीतपट्टासाठी नियंत्रणे आहेत. नियंत्रणाशिवाय अन्य डावे क्लिक गीतपट्टा निवड करते आणि खालीलप्रमाणे हलते.
३६ ३६ Clicking selects all this track's audio, deselecting all other tracks. इतर सर्व गीतपट्टाची निवड रद्द करून या क्लिकचा ध्वनी निवडतो.
३७ ३७ Holding Shift while clicking selects all this track's audio, keeping already selected tracks selected. आधीपासून निवडलेले गीतपट्टा निवडलेले ठेवून या सर्व गीतपट्टाचा ध्वनी निवडताना क्लिक करताना शिफ्ट होल्डिंग.
३८ ३८ (keyboard: use Up or Down to give the track focus, then Enter to select it; hold Shift then use Up or Down to extend selection into another track. Read keyboard selection for more). (कळफलक: गीतपट्टाला फोकस देण्यासाठी वर किंवा खाली वापरा, नंतर ते निवडण्यासाठी एन्टर करा; शिफ्ट दाबून धरा आणि दुसर्‍या गीतपट्टावर निवड विस्तृत करण्यासाठी वर किंवा खाली वापरा. ​​अधिकसाठी कळफलक निवड वाचा)
३९ ३९ Click-and-drag moves this track up or down when there are other tracks (keyboard: choose one of the "Move Track" commands in Note Track Dropdown Menu, opened by Shift + M on the currently focused track ). जेव्हा इतर गीतपट्टा असतात तेव्हा क्लिक आणि ड्रॅग हा गीतपट्टा वर किंवा खाली हलवितो (कळफलक: नोटिस गीतपट्टा ड्रॉपडाउन मेनूमधील “गीतपट्टा गीतपट्टा” आज्ञापैकी एक निवडा, सध्या फोकस केलेल्या गीतपट्टावर शिफ्ट + एम द्वारे उघडलेला)
४० ४० Controls नियंत्रणे
४१ ४१ Close Button: बटण बंद करा:
४2 ४2 Clicking this button will close the track, removing it from the project. या बटणावर क्लिक केल्याने हा प्रकल्पातून तो काढून टाकला जाईल.
४३ ४३ Note Track Dropdown Menu: नोट गीतपट्टा ड्रॉपडाउन मेनू:
४४ ४४ Clicking the track name or using Shift + M on the currently focused track opens the Note Track Dropdown Menu giving options that affect this track only. गीतपट्टाचे नाव क्लिक करणे किंवा सध्या फोकस केलेल्या गीतपट्टावर Shift + M वापरणे केवळ या गीतपट्टावर परिणाम करणारे टीप गीतपट्टा ड्रॉपडाउन मेनू देते.
४५ ४५ Newly created tracks are named "Note Track". नव्याने तयार केलेल्या गीतपट्टाचे नाव "टीप गीतपट्टा" आहे.
४६ ४६ Mute Button: बटण नि: शब्द करा
४७ ४७ Click to silence this track when playing. By default, the track will only be silenced if no Solo buttons are depressed. Click again to unsilence. खेळताना हा गीतपट्टा शांत करण्यासाठी क्लिक करा. पूर्वनिर्धारितनुसार, कोणतीही सोलो बटणे उदासीन नसल्यास केवळ गीतपट्टा शांत केला जाईल. पुन्हा कधीही क्लिक करा
४८ ४८ Tracks on which the Mute button is pressed are never exported. ज्या गीतपट्टावर निःशब्द बटण दाबले जाते ते कधीही निर्यात केले जात नाहीत.
४९ ४९ Solo Button: एकल बटण:
५० ५० Click to play just this track. फक्त हा गीतपट्टा खेळण्यासाठी क्लिक करा.
५१ ५१ Click again to release the button. बटण सोडण्यासाठी पुन्हा क्लिक करा.
५2 ५2 Whenever playing audio, Solo takes precedence over Mute by default - the Mute buttons have no effect whilst any Solo button is down. जेव्हा जेव्हा ध्वनी प्ले होतो तेव्हा पूर्वनिर्धारितनुसार एकल नि: शब्द करण्यापेक्षा सोलो अग्रक्रम घेते - कोणतेही एकल बटण खाली नसल्यास निःशब्द बटणांवर कोणताही प्रभाव पडत नाही.
५३ ५३ There are two other options for Solo button behavior in the Tracks Preferences. गीतपट्टा पसंतींमध्ये सोलो बटणाच्या वर्तनासाठी इतर दोन पर्याय आहेत.
५४ ५४ One of these is to remove the Solo button, leaving only a Mute button. यापैकी फक्त एक निःशब्द बटण सोडून सोलो बटण काढणे आहे.
५५ ५५ Channel toggles: वाहिनी टॉगल:
५६ ५६ Shows enabled channels, color coded to match the actual notes in the channels. Individual channels may be toggled by left-clicking, or all other channels can be muted/unmuted by right-clicking. वाहिनीमधील वास्तविक नोट्सशी जुळण्यासाठी सक्षम केलेले वाहिनी, रंग कोड कोड दर्शविते. डावी-क्लिक करून वैयक्तिक वाहिनी टॉगल केले जाऊ शकतात किंवा अन्य सर्व वाहिनी उजवे-क्लिक करून नि: शब्द / सशब्द केली जाऊ शकतात.
५७ ५७ Channel numbers start at १; channel १० reserved for percussion by the General MIDI specification. वाहिनी क्रमांक १ वाजता प्रारंभ होतात; जनरल एमआयडीआय स्पेसिफिकेशनद्वारे चक्र १० आरक्षित करण्यासाठी राखीव आहे.
५८ ५८ Velocity Slider: वेग स्लाइडर:
५९ ५९ Set the velocity offset for this track. Hold down the shift key while dragging to adjust the slider in finer increments. Double-click the slider to bring up a window where you can make precise adjustments or enter a velocity value. या गीतपट्टासाठी वेग ऑफसेट सेट करा. बारीक वाढीमध्ये स्लायडर समायोजित करण्यासाठी ड्रॅग करताना शिफ्ट की दाबून ठेवा. विंडो आणण्यासाठी स्लाइडरवर डबल-क्लिक करा जिथे आपण अचूक समायोजन करू शकता किंवा वेग मूल्य प्रविष्ट करू शकता.
६० ६० Velocity is in units of MIDI velocity, ranging from १ (lowest) to १2७ (highest). वेग एमआयडीआय वेगाच्या युनिटमध्ये आहे, ते १ (सर्वात कमी) ते १2७ (सर्वोच्च) पर्यंत आहेत.
६१ ६१ The offset from this slider (ranging from -५० to +५०) is combined with each note's individual velocity to produce the resultant velocity. या स्लाइडरमधील ऑफसेट (-५० ते +५० पर्यंतचे) परिणामी वेग तयार करण्यासाठी प्रत्येक नोटच्या वैयक्तिक वेगळ्यासह एकत्र केले जाते.
६2 ६2 Track Collapse Button: गीतपट्टा संकुचित बटण:
६३ ६३ Click on this to make the track 'fold up' into a smaller size. गीतपट्टाला लहान आकारात 'फोल्ड अप' करण्यासाठी या क्लिक करा.
६४ ६४ Click again, or drag the lower edge of the track to restore the size. पुन्हा क्लिक करा किंवा आकार पुनर्संचयित करण्यासाठी गीतपट्टाच्या खालच्या काठावर ड्रॅग करा.
६५ ६५ Status Indicators स्थिती निर्देशक
६६ ६६ Sync-Lock Indicator: संकालन-लॉक सूचक:
६७ ६७ When present (at bottom-right of the Track Control Panel, adjacent to the Track Collapse button), indicates that this track is part of a Sync-Locked Track Group. उपस्थित असल्यास (गीतपट्टा संक्षिप्त करा बटणालगत गीतपट्टा नियंत्रण पॅनेलच्या उजवीकडे) दर्शवितो की हा गीतपट्टा समक्रमित-लॉक केलेला गीतपट्टा गटाचा भाग आहे.
६८ ६८ Vertical Scale अनुलंब स्केल
६९ ६९ The vertical scale displays note pitches like a piano's keys, ranging from the lowest to the highest possible MIDI notes (C -१ to G ९). अनुलंब प्रमाणात पियानोच्या किज सारख्या नोटांचे खेळपट्ट्या दर्शवतात, ज्यामध्ये कमीतकमी सर्वात कमी एमआयडीआय नोट्स (सी -१ ते जी ९ पर्यंत) असतात.
७० ७० Each octave is labeled at C (and the notes above are in that octave). प्रत्येक अष्टक सी वर लेबल केलेले आहे (आणि वरील नोट्स त्या अष्टकातील आहेत)
७१ ७१ The Vertical Scale can be used to control Vertical Zooming. व्हर्टिकल स्केलचा वापर व्हर्टिकल झूमिंग नियंत्रित करण्यासाठी केला जाऊ शकतो.
७2 ७2 Observe carefully the little black marks above and below the colored notes. These indicate that there are additional notes in the Note Track above and below them. रंगीत नोटांच्या वर आणि खाली थोडे काळ्या खुणा काळजीपूर्वक पहा. हे सूचित करते की त्यांच्याकडे व त्याखालील टीप गीतपट्टामध्ये अतिरिक्त नोट्स आहेत.
७३ ७३ More notes can me made visible by using Vertical Zooming. व्हर्टिकल झूमिंगद्वारे अधिक नोट्स मला दृश्यमान केल्या जाऊ शकतात.
७४ ७४ Note Track Dropdown Menu नोंद ड्रॉपडाउन मेनू
७५ ७५ Clicking the note track's name by the downward-pointing triangle (or using shortcut Shift + M or the keyboard Menu key when the note track has focus) opens the Note Track Dropdown Menu: डाउन-पॉइंटिंग त्रिकोणाद्वारे नोट गीतपट्टाचे नाव क्लिक करणे (किंवा आडमार्ग शिफ्ट + एम किंवा नोट गीतपट्टावर लक्ष केंद्रित करतेवेळी कळफलक मेनू की वापरुन) नोट गीतपट्टा ड्रॉपडाउन मेनू उघडेल:
७६ ७६ Name नाव
७७ ७७ Displays the "Track Name" dialog where you can give the track a new name. Useful in multi-track projects to provide a visual indication of the content of each track. "गीतपट्टा नेम" संवाद प्रदर्शित करते जेथे आपण गीतपट्टाला एक नवीन नाव देऊ शकता. प्रत्येक गीतपट्टाच्या सामग्रीचे व्हिज्युअल संकेत देण्यासाठी मल्टी-गीतपट्टा प्रकल्पांमध्ये उपयुक्त.
७८ ७८ Up Octave and Down Octave अप ऑक्टोव्ह आणि डाऊन ऑक्टाव्ह
७९ ७९ Moves the vertical scale up or down one octave. अनुलंब स्केल एका अष्टकोनी वर किंवा खाली हलवते.
८० ८० Moving Tracks गीतपट्टा हलवित आहे
८१ ८१ Tracks can be moved up and down using the menu items. मेनू आयटम वापरून गीतपट्टा वर आणि खाली हलविले जाऊ शकतात.
८2 ८2 Move Track Up: Moves track up. गीतपट्टा अप हलवा: गीतपट्टा अप हलवते.
८३ ८३ Move Track Down: Moves track down. गीतपट्टा डाउन हलवा: गीतपट्टा डाउन हलवते.
८४ ८४ Move Track to Top: Moves track to become the topmost track in the project. गीतपट्टा वर जा: प्रकल्प मधील सर्वात वरचा गीतपट्टा होण्यासाठी गीतपट्टा हलवेल.
८५ ८५ Move Track to Bottom: Moves track to become the bottom track in the project. गीतपट्टा तळाशी हलवा: प्रकल्पातील तळाशी गीतपट्टा होण्यासाठी गीतपट्टा हलवते.
८६ ८६ Note tracks can also be moved up or down by clicking between the controls in the track's Track Control Panel then dragging upwards or downwards. गीतपट्टाच्या गीतपट्टा नियंत्रण पॅनेलमधील नियंत्रणे दरम्यान वर किंवा खाली ड्रॅग करून नोट गीतपट्टा वर किंवा खाली हलविले जाऊ शकतात.
८७ ८७ Note display टीप प्रदर्शन
८८ ८८ The notes from the MIDI sequence are displayed within the track. एमआयडीआय सीक्वेन्सवरील नोट्स गीतपट्टामध्ये प्रदर्शित केल्या आहेत.
८९ ८९ Colors indicate the MIDI channels that the given note is in, and match those seen in the channel toggles. रंग दिलेली टीप असलेली एमआयडीआय वाहिनी दर्शवितात आणि वाहिनी टॉगलमध्ये दिसणार्‍याशी जुळतात.
९० ९० Each channel has an instrument assigned to it (not currently shown), although channels can change instruments partway through. प्रत्येक वाहिनीकडे एक साधन नियुक्त केले आहे (सध्या दर्शविलेले नाही), जरी वाहिनी अंतरात वाद्ये बदलू शकतात.
९१ ९१ Channel १० (gray) is reserved by most MIDI devices for percussion. वाहिनी १० (राखाडी) बहुतेक एमआयडीआय उपकरणांनी पर्कशनसाठी आरक्षित केले आहे.
९2 ९2 Some allegro files and MIDI files have more than १६ channels, in that case they will be displayed as a different gray. काही बीफ्रो धारिका आणि एमआयडीआय धारिकांमध्ये १६ पेक्षा जास्त वाहिनी आहेत, त्या बाबतीत त्या भिन्न ग्रे म्हणून प्रदर्शित केल्या जातील.
९३ ९३ Notes that are above or below the visible area will be drawn in black in the top or bottom rows (no matter how far above or below they are), to indicate that there is something to be scrolled to. ज्या ठिकाणी स्क्रोल केले जावे असे दर्शविण्यासाठी, दृश्यमान क्षेत्राच्या वर किंवा खाली असलेल्या नोट्स वरच्या किंवा खालच्या ओळीत (ते कितीही वर किंवा खाली असले तरी) काळ्या रंगात रेखाटल्या जातील.
९४ ९४ Blue vertical lines indicate measures. निळ्या उभ्या रेषा उपाय दर्शवितात.
९५ ९५ Note that some MIDI files do not have accurate tempo information, in which case the lines may not match actual measures in the song. लक्षात घ्या की काही एमआयडीआय धारिकांमध्ये अचूक टेम्पो माहिती नाही, अशा प्रकरणात ओळी गाण्यातील वास्तविक उपायांशी जुळत नाहीत.
९६ ९६ Track focus गीतपट्टा फोकस
९७ ९७ You may have noticed in the images on this page that the tracks have a yellow border around them. This border indicates that this या पृष्ठावरील प्रतिमांमध्ये आपल्या लक्षात आले असेल की गीतपट्टाला आजुबाजुला पिवळी किनार आहे. ही सीमा दर्शवते
९८ ९८ track has focus. गीतपट्टावर फोकस आहे.
९९ ९९ The track that has focus is the track that accepts any command whose name includes "focused track". These commands include commands such as "Close focused track" (Shift + C), "Mute/Unmute focused track" (Shift + U) and "Toggle focused track" (RETURN or Enter, which toggles whether the focused track is selected or not). लक्ष असणारा गीतपट्टा असा गीतपट्टा आहे ज्याच्या नावावर "केंद्रित गीतपट्टा" समाविष्ट असलेली कोणतीही आज्ञा स्वीकारली जाते. या आज्ञा्समध्ये "क्लोस्क्स्ड गीतपट्टा गीतपट्टा" (शिफ्ट + सी), "निःशब्द / सशब्द केंद्रित गीतपट्टा" (शिफ्ट + यू) आणि "टॉगल फोकस गीतपट्टा" (रिटर्न किंवा एन्टर) या आज्ञा समाविष्ट आहेत जे लक्षित गीतपट्टा निवडलेले आहे की नाही हे टॉगल करते. ).
१०० १०० The complete list of "focused track" commands can be seen by searching for "focused track" (without quotes) in Keyboard Preferences or by viewing the description of these commands in Keyboard Shortcut Reference. "फोकस गीतपट्टा" आदेशांची संपूर्ण यादी कळफलक प्राधान्यांमध्ये "केंद्रित गीतपट्टा" (कोटेशिवाय) शोधून किंवा कळफलक आडमार्ग संदर्भात या आदेशांचे वर्णन पाहून दिसून येते.
