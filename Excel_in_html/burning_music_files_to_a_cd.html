६ ६ Audio CDs ध्वनी सीडी
७ ७ There are two main types of CDs that you can create with CD burning software: audio CDs and data CDs. सीडीचे दोन मुख्य प्रकार आहेत जे आपण सीडी बर्णिंग सॉफ्टवेअरसह तयार करू शकता: ध्वनी सीडी आणि डेटा सीडी.
८ ८ To ensure that you create a CD that will play anywhere it is important to choose the option to burn an "Audio CD" आपण कोठेही प्ले होईल अशी सीडी तयार केली आहे हे सुनिश्चित करण्यासाठी "ध्वनी सीडी" बर्न करण्याचा पर्याय निवडणे महत्वाचे आहे
९ ९ or "Music CD" and not a "Data CD". A data CD containing for example MP३ or WAV files will play happily on your computer but is unlikely to play in a standalone CD player or in-car CD player (note that some modern CD players will play data CDs). किंवा "संगीत सीडी" आणि "डेटा सीडी" नाही. उदाहरणार्थ एमपी ३ किंवा डब्ल्यूएव्ही धारिका्स असलेली डेटा सीडी आपल्या संगणकावर आनंदाने प्ले होईल परंतु स्टँडअलोन सीडी प्लेयर किंवा कार-सीडी प्लेयरमध्ये प्ले होण्याची शक्यता नाही (काही आधुनिक सीडी प्लेयर डेटा सीडी प्ले करतील हे लक्षात घ्या).
10 10 An audio CD will play on any standalone or in-car CD player and in your computer and in modern DVD players. ध्वनी सीडी कोणत्याही स्वतंत्र किंवा कार-मधील सीडी प्लेयरवर आणि आपल्या संगणकावर आणि आधुनिक डीव्हीडी प्लेयरवर प्ले होईल.
11 11 Audio CDs do not have files or a file system like data CDs and other computer storage media, but consist essentially of a stream of bits on the disc in a single spiral "track" with a TOC (Table of Contents) index. ध्वनी सीडीमध्ये डेटा सीडी आणि इतर संगणक स्टोअरेज मीडिया सारख्या धारिका किंवा धारिका सिस्टम नसते, परंतु त्यामध्ये टीओसी (अनुक्रमणिका सारणी) निर्देशांकासह सिंगल सर्पिल "गीतपट्टा" मध्ये डिस्कवरील बिटचा प्रवाह असतो.
12 12 Audio CDs are generally limited to ७४ minutes playing time on a ६५0 MB disc ("Red Book Standard") or ८0 minutes on a ७00 MB disc. ध्वनी सीडी सामान्यत: ६५0 एमबी डिस्क ("रेड बुक स्टँडर्ड") किंवा ७00 एमबी डिस्कवरील ८0 मिनिटांवर खेळण्याच्या वेळेसाठी मर्यादित असतात.

५४ ५४ Wrong sample rate: चुकीचा नमुना दर:
५५ ५५ Most CD burning software expects all files to be at ४४100 Hz. बर्‍याच सीडी बर्न सॉफ्टवेयर सर्व धारिका्स ४४100 हर्ट्जवर असण्याची अपेक्षा करतात.
५६ ५६ Unsupported formats: असमर्थित स्वरूप:
५७ ५७ Some CD burning software does not support compressed formats such as MP३, OGG, or M४A/AAC. काही सीडी बर्निंग सॉफ्टवेयर एमपी ३, ओजीजी, किंवा एम ४ ए / एएसी सारख्या संकुचित स्वरूपनास समर्थन देत नाहीत.
५८ ५८ Metadata and CD-Text मेटाडेटा आणि सीडी-मजकूर
५९ ५९ Even if you use Audacity's Metadata Tags editor to add metadata such as Title, Artist and Genre to your exported WAV files, that metadata does not get automatically burned to the CD. You can set iTunes and some other burning applications (but not Windows Media Player) to burn metadata to each track as separate CD-Text. Providing your CD burner supports CD-Text, a standalone CD player that supports CD-Text will then be able to display each track's metadata. जरी आपण आपल्या निर्यात केलेल्या डब्ल्यूएव्ही धारिकांमध्ये शीर्षक, कलाकार आणि शैली सारखे मेटाडेटा जोडण्यासाठी ऑडिटीचा मेटाडेटा टॅग संपादक वापरत असलात तरीही तो मेटाडेटा आपोआप सीडीवर बर्न होत नाही. आपण प्रत्येक गीतपट्टावर स्वतंत्र सीडी-मजकूर म्हणून मेटाडेटा बर्न करण्यासाठी आयट्यून्स आणि काही इतर बर्णिंग अनुप्रयोग (परंतु विंडोज मीडिया प्लेयर नाही) सेट करू शकता. आपला सीडी बर्नर प्रदान करणे सीडी-मजकूरला समर्थन देते, सीडी-मजकूरला समर्थन देणारी एक स्वतंत्र सीडी प्लेयर नंतर प्रत्येक गीतपट्टाचा मेटाडेटा प्रदर्शित करण्यास सक्षम असेल.
६0 ६0 You can also use a Cue Sheet to add CD-Text in burning applications that support Cue Sheets. क्यू शीटस समर्थन देणार्‍या बर्निंग अनुप्रयोगांमध्ये सीडी-मजकूर जोडण्यासाठी आपण क्यू शीट देखील वापरू शकता.
६1 ६1 Gapless burning गॅपलेस बर्निंग
६2 ६2 By default, many CD burning applications add a two-second gap between CD tracks as part of the standard for audio CDs. So be aware of CD track gaps when placing labels between album tracks for Export Multiple and consider deleting excess silences between tracks. पूर्वनिर्धारितनुसार, बर्‍याच सीडी बर्निंग audioप्लिकेशन्स ध्वनी सीडीच्या मानक भाग म्हणून सीडी गीतपट्टा दरम्यान दोन-सेकंद अंतर जोडतात. म्हणून एक्सपोर्ट मल्टिपलसाठी अल्बम गीतपट्टा दरम्यान लेबल ठेवत असताना सीडी गीतपट्टाच्या अंतरांबद्दल जागरूक रहा आणि गीतपट्टा दरम्यान जास्तीत जास्त शांतता हटवण्याचा विचार करा.
६३ ६३ However most CD burning applications have an option to burn the CD with no gaps between tracks. This is useful for recordings such as live concerts, allowing the CD to play continuously if the player supports gapless playback while still permitting skipping to individual CD tracks. If burning a gapless CD, you will need to place the Audacity labels exactly where you intend the burner to mark the track splits. Some CD burning applications (for example, older versions of Windows Media Player) have no option to burn without gaps. Gapless burning is also only available if the optical drive supports Disc-At-Once (DAO). तथापि बर्‍याच सीडी बर्निंग applicationsप्लिकेशन्सकडे सीडी बर्न करण्याचा पर्याय असतो गीतपट्टामध्ये अंतर नसते. हे थेट कॉन्सर्टसारख्या ध्वनीमुद्रणिंगसाठी उपयुक्त आहे, प्लेयर गॅपलेस प्लेबॅकला समर्थन देत असतानाही सीडी सतत प्ले करण्यास परवानगी देतो, तरीही वैयक्तिक सीडी गीतपट्टावर वगळण्याची परवानगी देत ​​आहे. जर गॅपलेस सीडी बर्न करत असेल तर गीतपट्टाचे विभाजन चिन्हांकित करण्यासाठी आपल्याला बर्नरचा हेतू असेल तेथे ओड्यासिटी लेबल ठेवावी लागतील. काही सीडी बर्निंग applicationsप्लिकेशन्स (उदाहरणार्थ, विंडोज मीडिया प्लेयरच्या जुन्या आवृत्त्या) मध्ये अंतर न करता बर्न करण्याचा पर्याय नाही. ऑप्टिकल ड्राइव्ह डिस्क-अट-वन (डीएओ) ला समर्थन देत असल्यासच गॅपलेस बर्णिंग देखील उपलब्ध आहे.
६४ ६४ Additionally, make sure you do not export MP३ files for burning to your gapless CD, even if you're burning a "data CD", because MP३s have inherent silence padding due to a restriction of the MP३ format. याव्यतिरिक्त, हे सुनिश्चित करा की आपण आपल्या "डेटा सीडी" जळत असला तरीही आपल्या गॅपलेस सीडीवर बर्न करण्यासाठी एमपी ३ धारिका निर्यात करत नाही, कारण एमपी ३ मध्ये एमपी ३ स्वरुपाच्या प्रतिबंधामुळे अंतर्भूत मौन पॅडिंग आहे.
६५ ६५ Note you may still hear the most momentary gaps between tracks even on DAO burned CDs, unless the CD player supports true gapless playback. Many CD players do not support gapless playback properly because the hardware simply does not buffer the audio data properly. However quite a few computer-based media players can use DSP effects when playing back audio to give proper gapless playback, even of CDs not burned with DAO. An example of such a player is Apple's iTunes. लक्षात ठेवा आपण डीएओ बर्न सीडी वर गीतपट्टा दरम्यान सर्वात क्षणिक अंतर अद्याप ऐकू शकता, जोपर्यंत सीडी प्लेयर खर्‍या अंतरविरहित प्लेबॅकला समर्थन देत नाही. बरेच सीडी प्लेयर गॅपलेस प्लेबॅक व्यवस्थित समर्थन देत नाहीत कारण हार्डवेअर ध्वनी डेटा योग्यरित्या बफर करत नाही. तथापि डीएओसह बर्न न झालेल्या सीडी देखील योग्य अंतराळ प्लेबॅक देण्यासाठी ध्वनी बॅक प्ले करताना संगणक-आधारित काही मीडिया प्लेयर डीएसपी प्रभाव वापरू शकतात. Aपलचे आयट्यून्स असे प्लेअरचे उदाहरण आहे.
६६ ६६ Cue Sheets क्यू पत्रके
६७ ६७ An alternative solution is to not split the track up at all in Audacity. The best way to get apparently gapless playback of a live concert recording or live album is to burn as one WAV file with with DAO and a "cue sheet" that marks the starting times of each CD track. Cue sheets can be created as simple plain text files. ओड्यासिटी मध्ये गीतपट्टा अप अजिबात विभाजित न करणे हा पर्यायी उपाय आहे. थेट मैफलीचे ध्वनीमुद्रणिंग किंवा थेट अल्बमचे स्पष्टपणे प्लेसबॅक मिळविण्याचा उत्तम मार्ग म्हणजे डीएओसह डब्ल्यूएव्ही धारिका म्हणून बर्न करणे आणि प्रत्येक सीडी गीतपट्टाच्या सुरूवातीच्या वेळेस चिन्हांकित करणारी "क्यू शीट". क्यू पत्रके साध्या साध्या मजकूर धारिका म्हणून तयार केल्या जाऊ शकतात.
६८ ६८ Each track in a Cue Sheet is of this format: क्यू शीटमधील प्रत्येक गीतपट्टा या स्वरूपाचा आहे:
६९ ६९ TRACK (number) AUDIO गीतपट्टा (संख्या) ध्वनी
७0 ७0 TITLE "(name)" शीर्षक "(नाव)"
७1 ७1 PERFORMER "(name)" परफॉर्मर "(नाव)"
७2 ७2 INDEX 01 (start time) in the format minutes:seconds:frames with the first track always being 00:00:00 इंडेक्स 01 (प्रारंभ वेळ) स्वरूपात मिनिटे: सेकंद: पहिल्या गीतपट्टासह फ्रेम नेहमी 00:00:00
७३ ७३ Audacity cannot currently export Cue Sheets but it is possible to use the File > Export > Export Labels... command to export a text file representing the label positions, then on Windows and Linux use label2cue to convert the minutes and seconds data in the text file to minutes, seconds and frames data laid out in cue sheet format. ओड्यासिटी सध्या क्यू शीट्सची निर्यात करू शकत नाही परंतु लेबल स्थानांची प्रतिनिधित्व करणारी मजकूर धारिका निर्यात करण्यासाठी धारिका> निर्यात> निर्यात निर्यात लेबल ... आज्ञा आणि मिनिटात सेकंद डेटा रूपांतरित करण्यासाठी Windows आणि Linux वर लेबल 2cue वापरा. मिनिट, सेकंद आणि फ्रेम डेटा धारिका क्यू शीट स्वरूपनात बाहेर घातली.
७४ ७४ You can read more about cue sheets on Wikipedia here. आपण विकिपीडियावर क्यू शीट्सबद्दल अधिक वाचू शकता.
७५ ७५ Burning longer recordings to disc डिस्कवर लांब ध्वनीमुद्रणिंग बर्न करत आहे

(डेटा सीडी सामान्यपणे जुन्या स्वतंत्र सीडी प्लेयरवर प्ले होणार नाही. हे संगणक, बहुतेक डीव्हीडी प्लेयर आणि एमपी ३ सीडी प्लेयरवर प्ले होईल).
७८ ७८ Data CDs and Data DVDs डेटा सीडी आणि डेटा डीव्हीडी

८३ ८३ Overburning जळजळ
