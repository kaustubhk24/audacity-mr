४ ४ This tutorial shows how to divide a long recording in one Audacity track into separate songs for export as one audio file per song. You can then burn these files to an audio CD (enabling you to skip from track to track on the CD) or play the files directly on a portable music player. We describe the basic concepts of removing unwanted audio, identifying and labeling boundaries between songs, maximizing volume then using Export Multiple.If you already have each song in Audacity tracks one above the other, you can still follow this tutorial. Instead of labeling as described here, name each song using that track's dropdown menu. At the Export Multiple step, split files based on Tracks instead of Labels. हे प्रशिक्षण दर्शविते की एका गाण्याचे प्रति ध्वनी धारिका म्हणून निर्यातीसाठी एका वेगळ्या गाण्यांमध्ये एका ध्वनीमुद्रणिंग गीतपट्टामध्ये लांब ध्वनीमुद्रणिंग कसे विभाजित करावे. त्यानंतर आपण या धारिका ध्वनी सीडीवर बर्न करू शकता (सीडी वर गीतपट्टा करण्यास गीतपट्टावर जाण्यास सक्षम करते) किंवा पोर्टेबल संगीत प्लेयरवर थेट धारिका प्ले करू शकता. आम्ही अवांछित ध्वनी काढून टाकणे, गाणी दरम्यान सीमा ओळखणे आणि लेबलिंग करणे, खंड वाढविणे आणि नंतर निर्यातमल्टिपल वापरणे या मूलभूत संकल्पनांचे वर्णन करतो. जर आपल्याकडे आधीपासूनच ओड्यासिटीमधील प्रत्येक गाणे एकापेक्षा एक वरील गीतपट्टा असतील तर आपण अद्याप या प्रशिक्षणचे अनुसरण करू शकता. येथे वर्णन केल्यानुसार लेबलिंग करण्याऐवजी त्या गाण्याचे नाव ड्रॉपडाउन मेनूच्या नावाने द्या. एकाधिक चरणात निर्यात करा, लेबलऐवजी गीतपट्टावर आधारित धारिका विभाजित करा.

६२ ६२ Include audio before first label should be unchecked, as there is no audio before the first label प्रथम लेबलच्या आधी कोणताही ध्वनी नसल्यामुळे प्रथम लेबल अनचेक होण्यापूर्वी ध्वनी समाविष्ट करा
६३ ६३ Under Name Files: नावाच्या धारिका अंतर्गत:
६४ ६४ Using Label/Track Name should be checked. लेबल / गीतपट्टा नाव वापरुन तपासले पाहिजे.
६५ ६५ Click the Export button. निर्यात बटणावर क्लिक करा.
६६ ६६ Metadata Editor will appear for the first song. The Track Title and Track Number will be pre-filled from the labels, but you can enter any additional information for that song that you wish (for example, Artist Name and Album Title). पहिल्या गाण्यासाठी मेटाडेटा संपादक दिसेल. गीतपट्टा शीर्षक आणि गीतपट्टा क्रमांक लेबलेमधून पूर्व-भरले जातील, परंतु आपण त्या गाण्यासाठी कोणतीही अतिरिक्त माहिती प्रविष्ट करू शकता (उदाहरणार्थ, कलाकाराचे नाव आणि अल्बम शीर्षक).
६७ ६७ Click the OK button in the Metatdata Editor (not the Save button). मेटाडेटा संपादकात ठीक बटणावर क्लिक करा (जतन बटणावर नाही).
६८ ६८ Metadata Editor will appear for the next and the subsequent songs; as before, enter any additional information and click "OK" for each window. When you click "OK" on the window for the last song, all the files will export. पुढील आणि त्यानंतरच्या गाण्यांसाठी मेटाडेटा संपादक दिसतील; पूर्वीप्रमाणे, कोणतीही अतिरिक्त माहिती प्रविष्ट करा आणि प्रत्येक विंडोसाठी "ओके" क्लिक करा. जेव्हा आपण शेवटच्या गाण्यासाठी विंडोवरील "ओके" क्लिक करता तेव्हा सर्व धारिका निर्यात होतील.
६९ ६९ Metadata in this context refers to information "tags" stored in the audio file such as Artist, Album and Song Title. या संदर्भातील मेटाडेटा कलाकार, अल्बम आणि गाणे शीर्षक यासारख्या ध्वनी धारिकामध्ये संग्रहित माहिती "टॅग" संदर्भित करते.
७० ७० Often you may have an album where most tags (such as Artist Name) are common to all tracks. If you choose Edit > Metatdata... before exporting and add the common tags there, Audacity will pre-fill these when the metadata window for each song appears. बर्‍याचदा आपल्याकडे असा अल्बम असू शकतो जिथे बहुतेक टॅग (जसे की आर्टिस्ट नेम) सर्व गीतपट्टासाठी सामान्य असतात. आपण संपादन> मेटाडेटा ... निवडण्यापूर्वी आणि तेथे सामान्य टॅग जोडण्यापूर्वी, प्रत्येक गाण्यासाठी मेटाडेटा विंडो दिसेल तेव्हा ओड्यासिटी हे पूर्व-भरेल.
७१ ७१ If all your tags are common to each song (except of course for each song's Track Title and Track Number), you can enter the common tags before export as above but export silently without the metadata window appearing. To do this, choose जर आपले सर्व टॅग प्रत्येक गाण्यासाठी सामान्य असतील (प्रत्येक गाण्याच्या गीतपट्टा शीर्षक आणि गीतपट्टा क्रमांकाशिवाय) आपण निर्यात करण्यापूर्वी सामान्य टॅग प्रविष्ट करू शकता परंतु मेटाडेटा विंडो दिसल्याशिवाय शांतपणे निर्यात करू शकता. हे करण्यासाठी, निवडा
७२ ७२ Preferences, then the Import / Export tab, and uncheck Show Metadata Editor prior to export step. प्राधान्ये, नंतर आयात / निर्यात टॅब, आणि निर्यात चरणापूर्वी मेटाडाटा संपादक दर्शवा अनचेक करा.
७३ ७३ Hiding Metadata Editor at the export step can also be useful if you mainly export WAV files for which many audio players don't recognize tag information. Use AIFF (in iTunes or VLC) or FLAC if you want full metadata in lossless files. निर्यातस्टेपवर मेटाडेटा सुधारणेर लपविणे देखील उपयोगी ठरू शकते जर आपण प्रामुख्याने डब्ल्यूएव्ही धारिका निर्यात केल्या ज्यासाठी बरेच ध्वनी प्लेअर टॅग माहिती ओळखत नाहीत. आपणास हानीकारक धारिकांमध्ये पूर्ण मेटाडेटा हवा असेल तर एआयएफएफ (आयट्यून्स किंवा व्हीएलसीमध्ये) किंवा एफएलएसी वापरा.
७४ ७४ A progress dialog might appear if the process takes more than a second or two. When the process is finished a confirmation dialog will appear listing the files that were created. प्रक्रियेस एक किंवा दोनपेक्षा जास्त वेळ लागल्यास एक प्रगती संवाद दिसून येईल. प्रक्रिया समाप्त झाल्यावर तयार केलेल्या धारिकांची यादी तयार करुन एक पुष्टीकरण संवाद दिसेल.
७५ ७५ The tracks are now ready to import into the CD burning software of your choice. गीतपट्टा आता आपल्या पसंतीच्या सीडी बर्निंग सॉफ्टवेअरमध्ये आयात करण्यास तयार आहेत.
७६ ७६ If you burn a CD with a gap of zero seconds between tracks (if your burning software supports this), you will end up with an exact replica of the LP or cassette. If, when burning the CD, you add the standard २-second gap between each track you will be adding an additional २ seconds to the existing silence between tracks, and the vinyl surface noise or cassette hiss will suddenly disappear which some listeners may find annoying. See Refinements, below, for ways of dealing with this. आपण गीतपट्टा दरम्यान शून्य सेकंदांच्या अंतरांसह सीडी बर्न केल्यास (जर आपल्या बर्न सॉफ्टवेअरने यास समर्थन दिले तर), आपण एलपी किंवा कॅसेटची अचूक प्रतिकृती तयार कराल. जर, सीडी जळत असताना, आपण प्रत्येक गीतपट्टा दरम्यान मानक २-सेकंद अंतर जोडल्यास आपण गीतपट्टा दरम्यान विद्यमान शांततेत अतिरिक्त २ सेकंद जोडत असाल आणि विनाइल पृष्ठभागावरील आवाज किंवा कॅसेट हिस अचानक अदृश्य होईल जे काही श्रोतांना त्रासदायक वाटेल. . यावर व्यवहार करण्याच्या पद्धतींसाठी खाली, परिष्करण पहा.
७७ ७७ Step ५: Backup चरण ५: बॅकअप
७८ ७८ Backup your exported WAV or MP३ files - you don't want to lose all that valuable work and have to do it all over again, do you? Computer hard drives can fail, destroying all data. आपल्या निर्यात केलेल्या डब्ल्यूएव्ही किंवा एमपी ३ धारिकांचा बॅकअप घ्या - आपण ते सर्व मौल्यवान काम गमावू इच्छित नाही आणि हे सर्व पुन्हा पुन्हा करावे लागेल, नाही का? संगणक हार्ड ड्राइव्ह अयशस्वी होऊ शकतात, सर्व डेटा नष्ट करते.
७९ ७९ Ideally use a dedicated drive (१+ TB external magnetic drives are convenient and economical), or upload to an online (cloud) storage service to store the WAVs or MP३s. Better still is to make two copies on different external devices and even better is to hold an online backup as well as the local copies. आदर्शपणे एक समर्पित ड्राइव्ह वापरा (१+ टीबी बाह्य चुंबकीय ड्राइव्ह सोयीस्कर आणि किफायतशीर आहेत) किंवा डब्ल्यूएव्ही किंवा एमपी ३ संचयित करण्यासाठी ऑनलाइन (क्लाऊड) स्टोरेज सेवेवर अपलोड करा. वेगवेगळ्या बाह्य उपकरणांवर दोन प्रती बनविणे अद्याप चांगले आहे आणि त्याशिवाय स्थानिक प्रती तसेच ऑनलाइन बॅकअप ठेवणे हे अधिक चांगले आहे.
८० ८० You may want to create a taxonomic file structure - for example each album can be stored in its own folder (named for the album) within a folder named for the artist (or, perhaps, composer for classical music) to make searching and retrieval easier. आपण एक वर्गीकरण धारिका रचना तयार करू शकता - उदाहरणार्थ प्रत्येक अल्बम त्याच्या स्वत: च्या निर्देशिकामध्ये संग्रहित केला जाऊ शकतो (अल्बमचे नाव दिले गेले आहे) कलाकारासाठी नामांकित निर्देशिकामध्ये (किंवा, कदाचित शास्त्रीय संगीतासाठी संगीतकार) शोध आणि पुनर्प्राप्ती सुलभ करण्यासाठी .
८१ ८१ Refinements परिष्कृत
८२ ८२ Remove the "silence" between songs गाण्यांमधील "शांतता" काढा
८३ ८३ Of course, the space between songs is not really "silent" - it contains vinyl surface noise or cassette hiss. Using standard editing techniques you can select and delete these portions of the recording, then insert whatever gap (silent portion) you want when burning the CD. नक्कीच, गाण्यांमधील जागा खरोखर "मूक" नाही - यात विनाइल पृष्ठभागावरील आवाज किंवा कॅसेट हिस आहे. मानक संपादन तंत्राचा वापर करून आपण ध्वनीमुद्रणिंगचे हे भाग निवडू किंवा हटवू शकता, नंतर सीडी बर्न करताना आपल्याला पाहिजे असलेले अंतर (मूक भाग) घाला.
८४ ८४ If you will be listening to the songs primarily on a computer or MP३ player you may prefer to have some silence at the end of each song. In this case select the silent portion then click on Edit > Remove Special > Silence Audio. Now you can edit the length of the silent portion to your taste. Alternatively you can delete the silent portion, then click on Generate > Silence and specify exactly how much silence you want. आपण प्रामुख्याने संगणक किंवा एमपी ३ प्लेयरवर गाणी ऐकत असल्यास आपण प्रत्येक गाण्याच्या शेवटी काही शांतता ठेवणे पसंत करू शकता. या प्रकरणात मूक भाग निवडा नंतर संपादन> विशेष काढा> मौन ध्वनीवर क्लिक करा. आता आपण मूक भागाची लांबी आपल्या आवडीनुसार संपादित करू शकता. वैकल्पिकरित्या आपण मूक भाग हटवू शकता, नंतर जनरेट> सायलेन्स वर क्लिक करा आणि आपल्याला किती शांतता हवी आहे हे निर्दिष्ट करा.
८५ ८५ Avoiding a sudden cessation of vinyl surface noise or cassette hiss विनाइल पृष्ठभागावरील ध्वनी किंवा कॅसेट ची हिसका अचानक बंद करणे टाळणे
८६ ८६ Both of the above techniques will result in the vinyl surface noise or cassette hiss suddenly stopping, which some listeners may find annoying. You can avoid this by using वरील दोन्ही तंत्रे विनाइल पृष्ठभागाच्या आवाजामुळे किंवा कॅसेट हिस अचानक थांबतील ज्यामुळे काही श्रोत्यांना त्रासदायक वाटेल. आपण याचा वापर करून हे टाळू शकता
८७ ८७ Effect > Fade Out at the end of each song. You can do the fade-out after the music ends (in effect fading out the noise). प्रभाव> प्रत्येक गाण्याच्या शेवटी फिकट. संगीत संपल्यानंतर आपण फेड-आउट करू शकता (परिणामी आवाज संपत असताना).
८८ ८८ Alternatively you can fade out the last second or so of the song if that works with the music. If fading the last seconds of the music you may prefer to use Effect > Studio Fade Out as that should give you a smoother and more "musical" sounding fade out. वैकल्पिकरित्या, जर ते संगीतासह कार्य करत असेल तर शेवटचे सेकंद किंवा शेवटचे गीत पुसून टाकू शकता. संगीताचे शेवटचे सेकंद लुप्त होत असल्यास आपण प्रभाव> स्टुडिओ फेडआउट वापरण्यास प्राधान्य देऊ शकता कारण यामुळे आपल्याला एक नितळ आणि अधिक "संगीत" सुशोभित करणे आवश्यक आहे.
८९ ८९ Experiment and listen to the results to decide what works for you and for the music you are working on. आपल्यासाठी आणि आपण ज्या संगीतावर कार्य करत आहात त्याकरिता काय कार्य करते हे ठरविण्यासाठी परिणाम वापरा आणि ऐका.
