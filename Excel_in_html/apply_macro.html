४ ४ Macros Palette automatically applies a series of preconfigured commands in a set order. The dialog allows you to select a Macro (which is a sequence of effect and/or export commands created using Tools > Macros then apply that Macro to either the entirety of the current project or to selected audio files in a single directory. मॅक्रो फळी स्वयंचलितपणे स्थापित केलेल्या क्रमामध्ये पूर्व-कॉन्फिगर केलेल्या आदेशांची मालिका लागू करते. संवाद आपल्याला एक मॅक्रो निवडण्याची परवानगी देतो (जो प्रभावीतेचा क्रम आहे आणि / किंवा साधने> मॅक्रोचा वापर करून निर्मित आज्ञा तयार केले गेले आहेत) नंतर त्या मॅक्रोला सध्याच्या प्रकल्पाच्या संपूर्णतेवर किंवा एकाच निर्देशिकेतील निवडलेल्या ध्वनी धारीकांवर लागू करा.

९ ९ Select Macro मॅक्रो निवडा
१० १० In the "Macro" list, left-click on a Macro (or use Up or Down keyboard arrow) to select the Macro you want to apply. "मॅक्रो" यादीमध्ये आपण लागू करू इच्छित मॅक्रो निवडण्यासाठी मॅक्रोवर (किंवा वर किंवा खाली कीबोर्ड बाण वापरा) वर डावे क्लिक करा.
११ ११ Apply Macro to यावर मॅक्रो लागू करा
१२ १२ Both the Macros dialog and the Macro Palette dialog have 'Apply Macro to' Buttons. दोन्ही मॅक्रो संवाद आणि मॅक्रो पॅलेट संवादात 'मॅक्रो ला लागू करा' बटणे आहेत.
१३ १३ Project: Use the Project button to apply the selected Macro to the current project window. प्रकल्प: सध्याच्या प्रकल्प विंडोमध्ये निवडलेला मॅक्रो लागू करण्यासाठी प्रकल्प बटण वापरा.
१४ १४ Files: Use the Files... button to apply the selected Macro to selected external audio files that are in a single directory. धारिका: एकेरी निर्देशिकेत असलेल्या निवडलेल्या बाह्य ध्वनी धारीकांवर निवडलेला मॅक्रो लागू करण्यासाठी धारिका ... बटण वापरा.
१५ १५ If you already have audio in the current project window, you must save and close that project using File > Close before applying a Macro to files. आपल्याकडे आधीपासूनच सध्याच्या प्रकल्प विंडोमध्ये ध्वनी असल्यास, धारीकांमध्ये मॅक्रो लागू करण्यापूर्वी आपण तो प्रकल्प धारिका> बंद वापरुन जतन आणि बंद करणे आवश्यक आहे.
१६ १६ You cannot process multi-channel audio files (for example, ५.१ surround sound files) using Macros, even if Import / Export Preferences has been set to "Use custom mix". Any multi-channel files you import will be mixed down on export. आयात / निर्यात प्राधान्ये "सानुकूल मिश्रणाचा वापर करा" वर स्थापित केली असल्यासदेखील आपण मॅक्रोझचा वापर करून बहु-वाहिनी ध्वनी धारिका (उदाहरणार्थ ५.१ आसपासच्या ध्वनी धारिका) वर प्रक्रिया करू शकत नाही. आपण आयात करता त्या कोणत्याही बहु-वाहिनी धारिका निर्यातीत मिसळल्या जातील.
१७ १७ Macros will work on pre-existing selections you make in your project prior to running the Macro. मॅक्रो चालविण्यापूर्वी आपण आपल्या प्रकल्पात घेतलेल्या पूर्व-विद्यमान निवडींवर कार्य करेल.
१८ १८ But the selection can be over-ridden by your Macro itself as there are macro commands available to effect selections in the audio. परंतु ध्वनीमधील निवडीवर परिणाम करण्यासाठी मॅक्रो आदेश उपलब्ध असल्याने आपल्या मॅक्रोद्वारे ही निवड ओव्हरड्राईड होऊ शकते.
१९ १९ In particular All(Select All) will select the entire project and Select which is parameterizable (see the provided Fade Ends Macro for an example where the first and last one seconds of the audio are selected for the fades). विशेषतः सर्व (सर्व निवडा) संपूर्ण प्रकल्प निवडेल आणि मापदंडनिय निवडेल (उदाहरणार्थ फिकटसाठी पहिल्या आणि शेवटच्या एक सेकंदाच्या ध्वनीची निवड केली गेलेल्या उदाहरणासाठी प्रदान केलेले फिकट शेवट मॅक्रो पहा).
२० २० Apply Macro to यावर मॅक्रो लागू करा
२१ २१ Project प्रकल्प
२२ २२ Use the Project to apply the selected Macro to the current project window. सध्याच्या प्रकल्प विंडोमध्ये निवडलेला मॅक्रो लागू करण्यासाठी प्रकल्प वापरा.
२३ २३ The typical purpose of this option is effect automation - apply a sequence of effects to the project, using effect parameters and an order of applying the effects that you have found to work well for the type of audio you are processing. This saves time and provides consistency in your workflow. या पर्यायचा विशिष्ट हेतू म्हणजे प्रभाव स्वयंचलन - आपण प्रभावित करत असलेल्या ध्वनीच्या प्रकारासाठी आपल्याला योग्यरित्या कार्य करण्यासाठी आढळलेल्या परिणामाचा वापर आणि परिणाम लागू करण्याची आज्ञा, प्रकल्पवर परिणामांचा क्रम लागू करा. हे वेळेची बचत करते आणि आपल्या कार्यप्रवाहात सुसंगतता प्रदान करते.
२४ २४ The effects in the Macro are applied to the selected region of waveform in the selected audio tracks. Usually the selected Macro would not include an export command so as to use the greater flexibility of the Export Audio Dialog. मॅक्रोमधील प्रभाव निवडलेल्या ध्वनी गीतपट्टामधील तरंगांच्या निवडलेल्या प्रदेशात लागू होतात. निर्यात ध्वनी संवादची अधिक लवचिकता वापरण्यासाठी सामान्यतः निवडलेल्या मॅक्रोमध्ये निर्यात आज्ञेचा समावेश नसतो.
२५ २५ If an export command is included in the Macro, the entire project audio is exported irrespective of track or region selection. Therefore if the project contains multiple audio tracks, they will be mixed together, unless any of the tracks are muted on the Track Control Panel. मॅक्रोमध्ये निर्यात आज्ञा समाविष्ट केल्यास, गीतपट्टा किंवा प्रदेश निवडीचा विचार न करता संपूर्ण प्रकल्प ध्वनी निर्यात केला जातो. म्हणून जर या प्रकल्पात अनेक ध्वनी गीतपट्टा असतील तर ते गीतपट्टा नियंत्रण पॅनेलवर कोणतेही गीतपट्टा नि: शब्द केल्याशिवाय एकत्रित केले जातील.
२६ २६ The exported files will be saved in a folder named "cleaned". निर्यात केलेल्या धारिका “साफ” नावाच्या फोल्डरमध्ये जतन केल्या जातील.
२७ २७ If the audio in the project came from an imported file or files, the "cleaned" folder will be inside the directory from which the first file was imported. The original files are not altered. प्रकल्पमधील ध्वनी आयात केलेल्या धारिका किंवा धारिकांकडून आला असेल तर, "साफ केलेले" फोल्डर त्या निर्देशिकेच्या आत असेल जिथून प्रथम धारिका आयात केली गेली. मूळ धारिका बदलल्या नाहीत.
२८ २८ Otherwise, a message prompt will indicate the location of the "cleaned" folder that will contain the exported file. The "cleaned" folder will be in the last location that the Macro process exported to. If the Macro process has never been used for export before, the "cleaned" folder will be in the location at which the File Export Dialog currently opens. अन्यथा, संदेश प्रॉम्प्टमध्ये "साफ" फोल्डरचे स्थान सूचित केले जाईल ज्यात निर्यात केलेली धारिका असेल. "साफ केलेले" फोल्डर मॅक्रो प्रक्रियेने निर्यात केलेल्या शेवटच्या ठिकाणी असेल. यापूर्वी कधीही निर्यात करण्यासाठी मॅक्रो प्रक्रिया वापरली नसल्यास, "निर्यात केलेले" फोल्डर सध्या ज्या ठिकाणी धारिका निर्यात संवाद उघडेल त्या ठिकाणी असेल.
२९ २९ Files... धारिका ...
३० ३० Use the Files... button to apply the selected Macro to selected external audio files that are in a single directory. If you already have audio in the current project window, you must save and close that project using File > Close before applying a Macro to files. एकेरी निर्देशिकेत असलेल्या निवडलेल्या बाह्य ध्वनी धारिकांवर निवडलेला मॅक्रो लागू करण्यासाठी धारिका ... बटण वापरा. आपल्याकडे आधीपासूनच सध्याच्या प्रकल्प विंडोमध्ये ध्वनी असल्यास, धारिकांमध्ये मॅक्रो लागू करण्यापूर्वी आपण तो प्रकल्प धारिका> बंद वापरुन जतन आणि बंद करणे आवश्यक आहे.
३१ ३१ The typical purpose of this option is batch processing - apply the Macro to multiple audio files so as to apply one or more effects to them, and/or convert them to another file format. You can convert from any file format supported by Audacity to WAV, MP३, OGG or FLAC. The selected Macro must include an "export" step, or the processed audio will not be saved. या पर्यायाचा विशिष्ट हेतू फळी प्रक्रिया करणे - एकाधिक ध्वनी धारिकांवर मॅक्रो लागू करा जेणेकरून त्यांना एक किंवा अधिक प्रभाव लागू करा आणि / किंवा त्यांना दुसर्‍या धारिका स्वरूपनात रूपांतरित करा. ओड्यासिटी द्वारा समर्थित कोणत्याही धारिका स्वरूपनातून आपण डब्ल्यूएव्ही, एमपी ३, ओजीजी किंवा एफएलएसीमध्ये रूपांतरित करू शकता. निवडलेल्या मॅक्रोमध्ये "निर्यात" चरण समाविष्ट करणे आवश्यक आहे किंवा प्रक्रिया केलेला ध्वनी जतन केला जाणार नाही.
३२ ३२ It is recommended not to process more than ५०० files at a time. एका वेळी ५०० पेक्षा जास्त धारिकांवर प्रक्रिया न करण्याची शिफारस केली जाते.
३३ ३३ A standard File Open dialog box will appear. Choose a directory then you can select one or any number of supported audio files in that directory, but not Audacity AUP project files. You cannot select files outside that directory, and files in folders inside that directory will not be processed. It is convenient therefore to first put all the audio files you want to process into one folder before applying the Macro. एक मानक धारिका उघड संवाद बॉक्स दिसेल. निर्देशिका निवडा त्यानंतर आपण त्या निर्देशिकेत एक किंवा अनेक समर्थित ध्वनी धारिका निवडू शकता, परंतु ओड्यासिटी एओपी प्रकल्प धारिका नाही. आपण त्या निर्देशिकेच्या बाहेरील धारिका निवडू शकत नाही आणि त्या निर्देशिकेत असलेल्या फोल्डरमधील धारिकांवर प्रक्रिया केली जाणार नाही. म्हणूनच आपण मॅक्रो लागू करण्यापूर्वी सर्व फोल्डर ध्वनी धारिका एका फोल्डरमध्ये ठेवणे सोयीचे आहे.
३४ ३४ After selecting the audio files you want to process, choose "Open". आपण प्रक्रिया करू इच्छित ध्वनी धारिका निवडल्यानंतर, "उघडा" निवडा.
३५ ३५ Each file will be imported into Audacity and processed, exported to the format you chose in the Macro, then the processed audio will be removed so as to clear the temporary disk space that had been used. प्रत्येक धारिका ओड्यासिटीमध्ये आयात केली जाईल आणि प्रक्रिया केली जाईल, आपण मॅक्रोमध्ये निवडलेल्या स्वरूपात निर्यात केली जाईल, त्यानंतर प्रक्रिया केलेले ध्वनी काढले जाईल जेणेकरून वापरण्यात आलेली तात्पुरती डिस्क जागा साफ होईल.
३६ ३६ The exported files will be saved in a folder named "cleaned" in the same folder that the original files came from. The original files are not altered. निर्यात केलेल्या धारिका ज्या मूळ धारिका्स आल्या त्याच फोल्डरमध्ये “साफ” नावाच्या फोल्डरमध्ये जतन केल्या जातील. मूळ धारिका बदलल्या नाहीत.
३७ ३७ When a Macro is applied to files, the only option is to import and process the entire file. Thus the entire file will be exported unless an action or effect in the Macro (such as Cut or Truncate Silence) removes some audio. जेव्हा धारिकांवर मॅक्रो लागू केला जातो, तेव्हा संपूर्ण धारिका आयात करणे आणि त्यावर प्रक्रिया करणे हा एकच पर्याय असतो. अशा प्रकारे मॅक्रोमधील एखादी क्रिया किंवा प्रभाव (जसे की कापणे किंवा ट्रंककेट शांतता) काही ध्वनी काढत नाही तोपर्यंत संपूर्ण धारिका निर्यात केली जाईल.
३८ ३८ If modifying the audio before exporting, the Select Time function may be useful to select audio to remove or modify. निर्यात करण्यापूर्वी ध्वनीमध्ये बदल करत असल्यास, वेळ निवडा कार्य ध्वनी निवडण्यासाठी किंवा सुधारित करण्यासाठी निवडण्यासाठी उपयुक्त ठरू शकेल.
३९ ३९ Some optional Nyquist plug-ins have parameters to काही पर्यायी Nyquist प्लगइन मध्ये मापदंड आहेत
४० ४० trim or extend audio by a specific length and Nyquist can do calculations too, so it may be helpful to use Nyquist plug-ins from within a macro. विशिष्ट लांबीद्वारे ध्वनी ट्रिम किंवा वाढवा आणि Nyquist गणना देखील करू शकते, म्हणून मॅक्रोमधून Nyquist प्लग-इन वापरणे उपयुक्त ठरेल.
४१ ४१ Expand विस्तृत करा
४२ ४२ Use the Expand to return to the full-size, full-function Manage Macros dialog. पूर्ण-आकारात, पूर्ण-चलित मॅक्रो मॅक्रो संवादवर परत जाण्यासाठी विस्तार वापरा.
४३ ४३ The apply Macro buttons are also available on the Manage Macros dialog - so all Macro operations can be made from that full dialog. मॅक्रो व्यवस्थापित करा संवादवर लागू मॅक्रो बटणे देखील उपलब्ध आहेत - जेणेकरून सर्व मॅक्रो कार्यांना त्या पूर्ण संवादामधून केले जाऊ शकतात.
४४ ४४ Presets for effects प्रभाव पुर्वनिर्धारित
४५ ४५ Since it is quite possible to have just a single command in a Macro you can use Macros to store your regularly used effects along with your favored settings. मॅक्रोमध्ये फक्त एकच आज्ञा असणे शक्य असल्याने आपण आपल्या पसंतीच्या समायोजनसह नियमितपणे वापरलेले प्रभाव संग्रहित करण्यासाठी मॅक्रो वापरू शकता.
४६ ४६ This Macros Palette is useful for presets for effects. It stays open after applying a macro, and you can pick another and apply that. हे मॅक्रोस पॅलेट प्रभाव असलेल्या पुर्वनिर्धारितसाठी उपयुक्त आहे. मॅक्रो लावल्यानंतर ते खुले राहते आणि आपण दुसरे निवडू शकता आणि ते लागू करू शकता.
४७ ४७ If you want, the presets can also adjust the selection. आपण इच्छित असल्यास, प्रीस्थापित देखील निवडी समायोजित करू शकतात.
४८ ४८ See Macros Examples for some suggested examples of this. याच्या काही सुचविलेल्या उदाहरणांसाठी मॅक्रो उदाहरणे पहा.
४९ ४९ Cancel to exit the dialog संवादातून बाहेर पडण्यासाठी रद्द करा
५० ५० The dialog will remain open on the screen, but allowing you to perform other Audacity functions. संवाद स्क्रीनवर खुले राहील, परंतु आपल्याला इतर ओड्यासिटी कार्ये करण्याची परवानगी दिली जाईल.
५१ ५१ To dismiss the dialog simply click on the Cancel button संवाद रद्द करण्यासाठी फक्त रद्द करा बटणावर क्लिक करा
५२ ५२ Macros Examples मॅक्रो उदाहरणे
५३ ५३ See the Macros Examples page for examples of using Macros. मॅक्रो वापरण्याच्या उदाहरणांसाठी मॅक्रोची उदाहरणे पृष्ठ पहा.
