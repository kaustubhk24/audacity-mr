१ १ Tracks Menu: Align Tracks गीतपट्टा मेनू: गीतपट्टा संरेखित करा
२ २ From Audacity Development Manual ओड्यासिटी विकास माहितीपुस्तीकेवरून
३ ३ Jump to: navigation, search येथे जा: निर्देशक, शोध
४ ४ The Align Tracks submenu provides a set of commands to help you align your selected tracks in various ways. संरेखित गीतपट्टा सबमेनू आपल्याला निवडक गीतपट्टा विविध प्रकारे संरेखित करण्यात मदत करण्यासाठी आज्ञाचा एक संच प्रदान करते.
५ ५ In particular there are two very useful commands that enable you to align tracks end-to-end, one after the other, or align them all together. विशेषतः दोन अतिशय उपयुक्त आज्ञा्स आहेत जे आपल्याला मागोमाग एक मागून एक गीतपट्टा संरेखित करण्यास सक्षम करतात किंवा त्या सर्व एकत्र संरेखित करतात.
६ ६ Click on the image to learn more अधिक जाणून घेण्यासाठी प्रतिमेवर क्लिक करा
७ ७ These commands provide an automatic way of aligning selected tracks with the cursor, the selection, or with the start of the project. हे आदेश कर्सर, निवड किंवा प्रकल्पच्या सुरूवातीस निवडलेल्या गीतपट्टा संरेखित करण्याचा स्वयंचलित मार्ग प्रदान करतात.
८ ८ With the exception of the first two commands ("Align End to End" and "Align Together"), each selected track is moved along equally so they remain synchronized with each other. If a label track is included in the selection, its labels will also move equally so they remain synchronized. पहिल्या दोन आज्ञास वगळता ("अलाइन टू एंड एंड" आणि "एकत्र संरेखित करा"), प्रत्येक निवडलेला गीतपट्टा समान रीतीने हलविला गेला आहे जेणेकरून ते एकमेकांशी समक्रमित राहतील. लेबल गीतपट्टा निवडीमध्ये समाविष्ट केल्यास त्याची लेबले देखील तितकीच हलतील जेणेकरून ते संकालित होतील.
९ ९ When aligning end to end or aligning together, the tracks are always moved individually to enforce the requested alignment (as described below). Label tracks are not affected by these two commands. शेवट संरेखित करताना किंवा एकत्र संरेखित करताना, विनंती केलेले संरेखन (खाली वर्णन केल्यानुसार) अंमलात आणण्यासाठी गीतपट्टा नेहमीच स्वतंत्रपणे हलवले जातात. या दोन आदेशांद्वारे लेबल गीतपट्टावर परिणाम होणार नाही.
१० १० When there is a selection, the commands that align tracks with the cursor/selection will align with the start of the selection, and when there is a cursor, the commands that align tracks with the cursor/selection will align with the cursor. जेव्हा सिलेक्शन असेल तेव्हा कर्सर / सेलेक्शनद्वारे गीतपट्टा संरेखित करणार्‍या आज्ञास सिलेक्शनच्या सुरूवातीस संरेखित होतील आणि जेव्हा कर्सर असेल तेव्हा कर्सर / सिलेक्शनद्वारे गीतपट्टा संरेखित करणार्‍या आज्ञा्स कर्सरसह संरेखित होतील.
११ ११ If Sync-Lock Tracks is enabled, unselected tracks in a Sync-Locked Track Group will also move so that all the tracks in that group remain synchronized, except for the Align "End to End" and "Together" commands where only the selected tracks will move. Aligning end to end or together overrides Sync-Lock Tracks if necessary, so the selected tracks in that group may become desynchronized. संकालन-लॉक गीतपट्टा सक्षम केलेले असल्यास, संकालित-लॉक केलेला गीतपट्टा गटातील निवड रद्द केलेले गीतपट्टा देखील हलतील जेणेकरून त्या गटातील सर्व गीतपट्टा संकालित राहतील, "एंड टू एंड" संरेखित करा आणि फक्त निवडलेले गीतपट्टा हलवेल समाप्तीची समाप्ती संरेखित करणे किंवा एकत्रितपणे आवश्यक असल्यास समक्रमण-लॉक गीतपट्टा अधिलिखित करते, जेणेकरून त्या गटामधील निवडलेले गीतपट्टा विभाजित होऊ शकतात.
१२ १२ If a track contains multiple clips and one of the Align commands is applied to that track then all the clips move and maintain their relative positions to each other within the track. The start of the first clip is the "start of the track" and the end of the last clip is the "end of the track". एखाद्या गीतपट्टामध्ये एकाधिक क्लिप्स असतील आणि त्या गीतपट्टावर संरेखित आज्ञा लागू केली गेली असेल तर सर्व क्लिप गीतपट्टामध्ये एकमेकांशी संबंधित स्थाने हलवतात आणि देखरेख करतात. पहिल्या क्लिपची सुरूवात "गीतपट्टाची सुरुवात" असते आणि शेवटच्या क्लिपचा शेवट "गीतपट्टाचा शेवट" असतो.
१३ १३ Align End to End समाप्तीस समाप्ती संरेखित करा
१४ १४ Aligns the selected tracks one after the other, based on their top-to-bottom order in the project window. The first selected track remains where it is, the start of the second selected track aligns with the end of the first selected track and so on. If you name a group of imported files from an album so that their alphabetical order is the same as the order you want to align them in, Align End to End will make the imported files play in their album order. निवडलेले गीतपट्टा एकामागून एक प्रकल्प विंडोमध्ये त्यांच्या टॉप-ते-डाउन ऑर्डरच्या आधारावर संरेखित करते. पहिला निवडलेला गीतपट्टा जिथे आहे तिथेच राहतो, दुस selected्या निवडलेल्या गीतपट्टाची सुरूवात पहिल्या निवडलेल्या गीतपट्टाच्या शेवटी संरेखित होते आणि याप्रमाणे. आपण अल्बममधून आयात केलेल्या धारिकांच्या एका गटास नाव दिले जेणेकरून त्यांची वर्णमाला क्रमानुसार आपण त्यांना संरेखित करू इच्छित असलेल्या ऑर्डर प्रमाणेच असल्यास, संरेखित करा एंड टू एंड धारिका आयात केलेल्या धारिका त्यांच्या अल्बमच्या क्रमानुसार प्ले करेल.
१५ १५ After the align, all tracks in the project (not just the selected tracks) will be fitted to the window so that the end of the last track will be visible in the project window. संरेखनानंतर, प्रकल्पातील सर्व गीतपट्टा (केवळ निवडलेले गीतपट्टाच नाहीत) विंडोमध्ये बसविण्यात येतील जेणेकरून शेवटच्या गीतपट्टाचा शेवट प्रकल्प विंडोमध्ये दिसून येईल.
१६ १६ Before doing Tracks > Align Tracks > Align End to End गीतपट्टा करण्यापूर्वी> गीतपट्टा संरेखित करा> संरेखित करा शेवट
१७ १७ After doing Tracks > Align Tracks > Align End to End गीतपट्टा केल्यावर> गीतपट्टा संरेखित करा> संरेखित करा शेवट
१८ १८ Note that zoom level has changed to display the entire project, and that the original selection has been retained. संपूर्ण प्रकल्प प्रदर्शित करण्यासाठी झूम पातळी बदलली आहे आणि मूळ निवड कायम ठेवली आहे हे लक्षात घ्या.
१९ १९ Align Together एकत्र संरेखित करा
२० २० If you select multiple tracks, this command moves the tracks so that they all start at the same time, so audio within them changes its synchronization. The resultant start time is an average of all of the tracks' original starting times. To adjust the starting times of tracks with complete flexibility, use Time Shift Tool. आपण एकाधिक गीतपट्टा निवडल्यास, ही आज्ञा गीतपट्टा हलवते जेणेकरून ते सर्व एकाच वेळी प्रारंभ होतील, म्हणून त्यांच्यामधील ध्वनी त्याचे संकालन बदलू शकेल. परिणामी प्रारंभ वेळ सर्व गीतपट्टाच्या मूळ सुरूवातीच्या वेळेची सरासरी असते. पूर्ण लवचिकतेसह गीतपट्टाच्या सुरूवातीच्या वेळेस समायोजित करण्यासाठी, वेळ शिफ्ट साधन वापरा.
२१ २१ Before doing Tracks > Align Tracks > Align Together गीतपट्टा करण्यापूर्वी> गीतपट्टा संरेखित करा> एकत्र संरेखित करा
२२ २२ After doing Tracks > Align Tracks > Align Together गीतपट्टा केल्यावर> गीतपट्टा संरेखित करा> एकत्र संरेखित करा
२३ २३ Note that the three tracks now start at the same time, and the start time is the average of the original start times of the three tracks. लक्षात घ्या की तीन गीतपट्टा आता एकाच वेळी प्रारंभ होतात आणि प्रारंभ वेळ तीन गीतपट्टाच्या मूळ प्रारंभ वेळेची सरासरी आहे.
२४ २४ Start to Zero झिरोला प्रारंभ करा
२५ २५ Aligns the start of selected tracks with the start of the project. प्रकल्पच्या सुरूवातीस निवडलेल्या गीतपट्टाची सुरूवात संरेखित करते.
२६ २६ Start to Cursor/Selection Start कर्सर प्रारंभ / निवड प्रारंभ
२७ २७ Aligns the start of selected tracks with the current cursor position or with the start of the current selection. सध्याच्या कर्सर स्थानासह किंवा सद्य निवडीच्या सुरूवातीसह निवडलेल्या गीतपट्टाची सुरूवात संरेखित करते.
२८ २८ Start to Selection End सिलेक्शन एंड टू सिलेक्शन एन्ड
२९ २९ Aligns the start of selected tracks with the end of the current selection. सद्य निवडीच्या समाप्तीसह निवडलेल्या गीतपट्टाचा प्रारंभ संरेखित करते.
३० ३० End to Cursor/Selection Start कर्सर / निवड प्रारंभ
३१ ३१ Aligns the end of selected tracks with the current cursor position or with the start of the current selection. सध्याच्या कर्सर स्थानासह किंवा सद्य निवडीच्या सुरूवातीसह निवडलेल्या गीतपट्टाचा शेवट संरेखित करते.
३२ ३२ End to Selection End निवडीचा शेवट
३३ ३३ Aligns the end of selected tracks with the end of the current selection. सद्य निवडीच्या समाप्तीसह निवडलेल्या गीतपट्टाचा शेवट संरेखित करते.
३४ ३४ Move Selection with Tracks (on/off) निवड गीतपट्टासह हलवा (चालू / बंद)
३५ ३५ When this option is off (not checked), the cursor or selection remains in its position relative to the Timeline. When it is on, the cursor or selection is moved to maintain its position relative to the tracks being moved. For example, Tracks >Align Tracks > Start to Cursor/Selection Start will align the selected tracks with the current cursor position or selection start, but then also move the cursor or selection along so that it remains at the same position relative to the new position of the aligned tracks. जेव्हा हा पर्याय बंद असतो (चेक केलेला नाही), तेव्हा वेळलाइनच्या तुलनेत कर्सर किंवा निवड त्याच्या स्थितीत राहील. ते चालू असताना, गीतपट्टा हलविण्याशी संबंधित नसल्यास कर्सर किंवा निवड त्याचे स्थान राखण्यासाठी हलविली जाते. उदाहरणार्थ, गीतपट्टा> गीतपट्टा संरेखित करा> प्रारंभ करणे कर्सर / निवड प्रारंभ निवडलेले गीतपट्टा वर्तमान कर्सर स्थानासह किंवा निवड प्रारंभसह संरेखित करेल, परंतु नंतर कर्सर किंवा निवड देखील हलवेल जेणेकरून ते नवीन स्थानाशी संबंधित त्याच स्थितीत राहील. संरेखित गीतपट्टाचे.
३६ ३६ This option has no effect on the Align End to End and the Align Together commands. संरेखित समाप्तीची समाप्ती आणि संरेखित एकत्र आदेशांवर या पर्यायाचा कोणताही परिणाम होणार नाही.
३७ ३७ Before doing Tracks > Align Tracks > Start to Zero गीतपट्टा करण्यापूर्वी> गीतपट्टा संरेखित करा> झिरोपासून प्रारंभ करा
३८ ३८ After doing Tracks > Align Tracks > Start to Zero, with Move Selection with Tracks off. गीतपट्टा केल्यावर> गीतपट्टा संरेखित करा> गीतपट्टासह हलवा निवडीसह झिरोपासून प्रारंभ करा.
३९ ३९ Note that the cursor remains at the same position relative to the Time Line. लक्षात घ्या की वेळर लाईनच्या तुलनेत कर्सर त्याच स्थितीत राहील.
४० ४० After doing Tracks > Align Tracks > Start to Zero, with Move Selection with Tracks on. गीतपट्टा केल्यावर> गीतपट्टा संरेखित करा> गीतपट्टासह मूव्ह सिलेक्शन चालू करून झिरोपासून प्रारंभ करा.
४१ ४१ Note that the cursor has moved to retain the same position relative to the audio. लक्षात घ्या की कर्सर ध्वनीच्या तुलनेत तीच स्थिती राखण्यासाठी हलविला आहे.
