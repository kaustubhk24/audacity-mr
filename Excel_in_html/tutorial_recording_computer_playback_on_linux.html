६ ६ PulseAudio पल्स ऑडियो
७ ७ PulseAudio is a sound server providing professional audio features such as mixing and input/output routing. It is the default audio device on many लिनक्स distributions including Ubuntu, लिनक्स Mint, Open Suse and many Debian/Ubuntu-based distributions. PulseAudio operates as a proxy between sound अनुप्रयोग and the audio hardware (usually via ALSA). PulseAudio Volume Control पल्स ध्वनी एक ध्वनी सर्व्हर आहे जो मिक्सिंग आणि इनपुट /बाहेरपुट रूटिंग यासारखी व्यावसायिक ध्वनी वैशिष्ट्ये प्रदान करतो. हे उबंटू, लिनक्स मिंट, ओपन सुसे आणि बर्‍याच डेबियन / उबंटू-आधारित वितरणासह बर्‍याच लिनक्स वितरणावर पूर्वनिर्धारित ध्वनी उपकरण आहे. पल्स ध्वनी ध्वनी अनुप्रयोग आणि ध्वनी हार्डवेअर (सहसा ALSA मार्गे) दरम्यान प्रॉक्सी म्हणून कार्य करते. पल्स ध्वनी खंड नियंत्रण

२२ २२ JACK जॅक
२३ २३ Jack Audio System is commonly used in लिनक्स distributions that specialise in media creation such as Ubuntu Studio and Studio ६४. जॅक ध्वनी उपकरण सामान्यत: लिनक्स वितरणामध्ये वापरली जातात ज्यात उबंटू स्टुडिओ आणि स्टुडिओ ६४ सारख्या मीडिया निर्मितीमध्ये तज्ञ असतात.
२४ २४ To use Audacity with Jack it is useful or in some cases essential to have a few additional packages installed. If a Mix device is shown as a capture device, set this as the active device to enable recording sounds that are playing on the computer. अलसॅमिकर सुरू करण्यासाठी टर्मिनल विंडोमध्ये अल्समिक्सर टाइप करा. एखादा मिक्स उपकरण कॅप्चर उपकरण म्हणून दर्शविला गेला असेल तर संगणकावर प्ले होत असलेले ध्वनीमुद्रणिंग आवाज सक्षम करण्यासाठी हे सक्रिय उपकरण म्हणून सेट करा.
३३ ३३ OSS ओएसएस

ALSA २.५ सह प्रारंभ केला गेला, त्या आवृत्तींमध्ये, लिनक्स कर्नल लेखकांनी ओएसएसला नापसंत म्हणून चिन्हांकित केले.
३५ ३५ Development of OSS was continued by ४Front Technology, initially as proprietary software but later released as partially open source. Possible solutions to recording audio playing on a computer using OSS are described here. ओएसएस चा विकास ४ समोर तंत्रज्ञानाद्वारे सुरू ठेवण्यात आला, सुरुवातीला मालकीचे सॉफ्टवेअर म्हणून परंतु नंतर अर्धवट मुक्त स्त्रोत म्हणून सोडले गेले. ओएसएस वापरुन संगणकावर ध्वनी प्ले करणे ध्वनीमुद्रण करण्याचे संभाव्य निराकरण येथे वर्णन केले आहे.
३६ ३६ Other Methods इतर पद्धती
३७ ३७ Alternative programs for recording computer playback संगणक प्लेबॅक ध्वनीमुद्रण करण्यासाठी पर्यायी प्रोग्राम
३८ ३८ An alternative to recording streaming audio from the Internet is to download the data directly to disk, either by using command-line tools such as streamripper or mplayer, or with a browser plug-in (many available for Firefox), or writing the stream to disk from a media player such as VLC or xmms, or using a graphical down-loader such as streamtuner. इंटरनेटवरून स्ट्रीमिंग ध्वनी ध्वनीमुद्रण करण्याचा पर्याय म्हणजे थेट डिस्कवर डेटा डाउनलोड करणे, एकतर स्ट्रिम्रीपर किंवा एमप्लेअर या आज्ञा-लाइन साधनांचा वापर करून किंवा ब्राउझर प्लग-इन (फायरफॉक्ससाठी बरेच उपलब्ध) किंवा प्रवाह यावर लिहिणे. व्हीएलसी किंवा एक्सएमएमएस सारख्या मीडिया प्लेयरवरील डिस्क किंवा स्ट्रीमट्यूनर सारख्या ग्राफिकल डाउन-लोडरचा वापर करून.
३९ ३९ Using the ALSA PCM file ALSA PCM धारिका वापरणे
४० ४० It is not necessary to use Audacity to record sounds playing on the computer. You can capture the sound from the application producing it before it reaches the sound device, which avoids capturing unwanted system sounds. This solution can be used even if there is no sound device. Note: this solution may not work on a system using PulseAudio but you can use PulseAudio Volume Control instead. संगणकावर प्ले होणारे ध्वनी ध्वनीमुद्रण करण्यासाठी ओड्यासिटी वापरणे आवश्यक नाही. आपण ध्वनी उपकरणापर्यंत पोहोचण्यापूर्वी तो तयार करीत असलेल्या ध्वनीद्वारे ध्वनी कॅप्चर करू शकता, जे अवांछित उपकरण ध्वनी मिळविणे टाळते. ध्वनी नसल्यासही हे समाधान वापरले जाऊ शकते. टीपः हे समाधान पल्स ऑडियो वापरुन उपकरणवर कार्य करू शकत नाही परंतु त्याऐवजी आपण पल्स ध्वनी आवाज नियंत्रण वापरू शकता.

६८ ६८ Use a loopback cable लूपबॅक केबल वापरा
६९ ६९ This is a simple and effective method of recording what is playing on the computer. It has the disadvantage that the audio signal is converted from digital to analog, then from analog to digital, instead of staying in the digital domain. This may result in some slight increase in noise and distortion. संगणकावर काय चालले आहे ते ध्वनीमुद्रण करण्याची ही एक सोपी आणि प्रभावी पद्धत आहे. ध्वनी सिग्नल डिजिटल डोमेनमध्ये न राहता डिजिटलमधून एनालॉगमध्ये, नंतर एनालॉगवरून डिजिटलमध्ये रूपांतरित केल्याचा तोटा आहे. यामुळे आवाज आणि विकृतीत थोडीशी वाढ होऊ शकते.
