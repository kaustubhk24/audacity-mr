४ ४ These commands extend the current selection up and/or down into all tracks, or Sync-Locked tracks in the project. या आदेशांद्वारे सध्याची निवड अप आणि / किंवा खाली सर्व गीतपट्टामध्ये किंवा प्रोजेक्टमधील सिंक-लॉक केलेले गीतपट्टा वाढवते.

६ ६ In All Tracks  Ctrl + Shift + K सर्व गीतपट्टामध्ये Ctrl + Shift + K
७ ७ Extends the current selection up and/or down into all tracks in the project. प्रोजेक्टमधील सर्व गीतपट्टामध्ये सध्याची निवड वर आणि / किंवा खाली वाढवते.
८ ८ In All Sync-Locked Tracks  Ctrl + Shift + Y सर्व समक्रमित-लॉक केलेल्या गीतपट्टामध्ये Ctrl + Shift + Y
९ ९ Extends the current selection up and/or down into all Sync-Locked tracks in the currently selected track group. सध्या निवडलेल्या गीतपट्टा गटामधील सिंक-लॉक केलेल्या गीतपट्टामध्ये सध्याची निवड वर आणि / किंवा खाली वाढवते.
