७ ७ Launches a file selection window where you can choose to import one or more audio files into the current Audacity project. The file(s) will always be added as a new track to the project. This lets you mix two or more files together. धारिका निवड विंडो लाँच करते जिथे आपण चालू ओड्यासिटी प्रकल्पमध्ये एक किंवा अधिक ध्वनी धारिका आयात करणे निवडू शकता. धारिका नेहमी या प्रकल्पात नवीन गीतपट्टा म्हणून जोडली जातील. हे आपल्याला दोन किंवा अधिक धारिका एकत्रित करू देते.
८ ८ See Importing Audio for more information. अधिक माहितीसाठी ध्वनी आयात करणे पहा.
९ ९ Labels... लेबले ...
१0 १0 Launches a file selection window where you can choose to import a single text file into the project containing point or region labels. For more information about the syntax for labels files, see Importing and Exporting Labels. धारिका निवड विंडो लाँच करते जेथे आपण पॉईंट किंवा प्रदेश लेबले असलेल्या प्रकल्पात एकच मजकूर धारिका आयात करणे निवडू शकता. लेबल धारिकांसाठी वाक्यरचना बद्दल अधिक माहितीसाठी, लेबले आयात करणे आणि निर्यात करणे पहा.

१५ १५ Raw Data... कच्ची माहिती...
१६ १६ Attempts to import an uncompressed audio file that might be "raw" data without any headers to define its format, might have incorrect headers or be otherwise partially corrupted, or might be in a format that Audacity is unable to recognize. Raw data in textual format cannot be imported. एक संकुचित ध्वनी धारिका आयात करण्याचा प्रयत्न ज्याचे स्वरूप निश्चित करण्यासाठी कोणत्याही शीर्षलेखांशिवाय "कच्चा" डेटा असू शकतो, चुकीचे शीर्षलेख असू शकतात किंवा अन्यथा अंशतः दूषित होऊ शकतात किंवा ओड्यासिटी ओळखण्यास अक्षम असलेल्या स्वरूपात असू शकतात. मजकूर स्वरूपात कच्चा डेटा आयात केला जाऊ शकत नाही.
१७ १७ First, select the file in question in the "Select any uncompressed audio file" dialog. Then select appropriate parameters to assist Audacity in the formatting of the data. The fields in the dialog require you select: प्रथम, "कोणतीही संकुचित ध्वनी धारिका निवडा" संवादातील प्रश्नातील धारिका निवडा. नंतर डेटाच्या स्वरूपणात ओड्यासिटीला मदत करण्यासाठी योग्य मापदंड निवडा. संवादातील फील्डसाठी आपण निवडणे आवश्यक आहे:
१८ १८ Encoding (PCM, ADPCM, float...) एन्कोडिंग (पीसीएम, एडीपीसीएम, फ्लोट ...)
१९ १९ Byte order (this is almost always Little-endian if the file was created on Windows) बाइट ऑर्डर (ही धारिका विंडोजवर तयार केली असल्यास हे नेहमीच लिटल-एन्डियन असते)
20 20 Number of channels (expected to be found in the file and created as a result) वाहिनीची संख्या (धारिकामध्ये सापडण्याची आणि परिणामी तयार केलेली)
