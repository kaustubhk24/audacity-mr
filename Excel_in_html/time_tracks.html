१ १ Time Tracks वेळ मागोवा
२ २ From Audacity Development Manual ओड्यासिटी विकास माहितीपुस्तिकेकडून
३ ३ Jump to: navigation, search येथे जा: निर्देशक, शोध
४ ४ A Time Track is used in conjunction with one or more audio tracks to progressively increase or decrease playback speed (and pitch) over the length of the audio. Speed changes are controlled by manipulating the blue "time warp" line with the Envelope Tool used for making gradual volume changes. If the blue line is dragged above the horizontal, the audio plays faster; if dragged below, it plays slower. ध्वनीच्या लांबीवर प्लेबॅक गती (आणि खेळपट्टी) क्रमाने वाढविण्यासाठी किंवा कमी करण्यासाठी एक किंवा अधिक ध्वनी गीतपट्टाच्या संयोगाने एक वेळ गीतपट्टा वापरला जातो. हळूहळू व्हॉल्यूम बदल करण्यासाठी वापरल्या गेलेल्या लिफाफा उपकरणासह निळ्या "वेळ वार्प" लाइनमध्ये फेरफार करून वेग बदलला जातो. जर निळ्या रेषा आडव्या वर ड्रॅग केली असेल तर ध्वनी वेगवान प्ले करतो; जर खाली ड्रॅग केले तर ते हळू वाजवते.
५ ५ General method of use: वापरण्याची सामान्य पद्धत:
६ ६ Hover your cursor over the warp line until the envelope cursor    appears, then click (and optionally, drag up or down) to create a control point. लिफाफा कर्सर दिसत नाही तोपर्यंत आपल्या कर्सरला तानाच्या ओळीवर फिरवा, आणि नियंत्रण बिंदू तयार करण्यासाठी (आणि वैकल्पिकरित्या ड्रॅग अप किंवा डाउन) क्लिक करा.
७ ७ To remove a control point, drag it outside the track, or use Edit > Undo to undo envelope changes in sequence. एखादा नियंत्रण बिंदू काढण्यासाठी त्यास गीतपट्टाच्या बाहेर ड्रॅग करा किंवा क्रमवारीत लिफाफा बदल पूर्ववत करण्यासाठी संपादन> पूर्ववत करा वापरा.
८ ८ By default you can warp playback speed by a maximum increase of १०% (११० on the vertical scale to left of the warp line) or a maximum decrease of १०% (९० on the vertical scale). पूर्वनिर्धारितनुसार आपण प्लेबॅकची गती जास्तीत जास्त १०% वाढवून (११० वर अनुलंब प्रमाणात डाव्या रेषाच्या डावीकडे) किंवा जास्तीत जास्त १०% (उभ्या प्रमाणात ९०) कमी करू शकता.
९ ९ To warp the speed by greater amounts, click in the name of the Time Track (in the panel that contains the downward pointing arrow) and choose "Set Range". Here you can specify the lower and upper speed limit of the vertical scale: वेग जास्त प्रमाणात वाढवण्यासाठी, वेळ गीतपट्टाच्या नावावर क्लिक करा (खाली असलेल्या दिशेने असलेल्या बाण असलेल्या पॅनेलमध्ये) आणि "श्रेणी सेट करा" निवडा. येथे आपण अनुलंब प्रमाणात कमी आणि अपर वेग मर्यादा निर्दिष्ट करू शकता:
१० १० Changing the range rescales the Time Track to the new limits, preserving the warp value of any envelope points that were there before. Having changed the range, you can now move the warp line up or down anywhere within the new upper or lower limit of the range. श्रेणी बदलणे यापूर्वी असलेल्या कोणत्याही लिफाफा बिंदूंचे तानाचे मूल्य जपून नवीन मर्यादेपर्यंत वेळ गीतपट्टाची पुनर्वापर करते. श्रेणी बदलल्यानंतर, आपण आता श्रेणीच्या नवीन वरच्या किंवा खालच्या मर्यादेत तणाव रेषा वर किंवा खाली कोठेही हलवू शकता.
११ ११ Only one Time Track can be used in a project, and it affects all audio tracks on screen. प्रकल्पमध्ये फक्त एक टाईम गीतपट्टा वापरला जाऊ शकतो आणि तो स्क्रीनवरील सर्व ध्वनी गीतपट्टावर परिणाम करतो.
१२ १२ Time Track has no effect when audio is speed-adjusted using the Play-at-Speed slider in Play-at-Speed Toolbar. प्ले--ट-स्पीड साधनपट्टीमध्ये प्ले--ट-स्पीड स्लायडर वापरुन ध्वनी वेग-समायोजित केला जातो तेव्हा वेळ गीतपट्टाचा कोणताही परिणाम होत नाही.
१३ १३ Time Track Dropdown Menu वेळ गीतपट्टा ड्रॉपडाउन मेनू
१४ १४ Clicking in the panel containing the downward-pointing triangle (to the left of the track) opens the Time Track Dropdown Menu containing display choices for the track: खाली-दर्शविणारा त्रिकोण (गीतपट्टाच्या डावीकडे) असलेल्या पॅनेलमध्ये क्लिक केल्याने गीतपट्टासाठी प्रदर्शन पर्याय असलेले वेळ गीतपट्टा ड्रॉपडाउन मेनू उघडेल:
१५ १५ Name...: Displays the "Track Name" dialog where you can give the track a new name. नाव ...: "गीतपट्टा नाव" संवाद प्रदर्शित करते जेथे आपण गीतपट्टाला एक नवीन नाव देऊ शकता.
१६ १६ Move Track Up: Moves the track up. गीतपट्टा अप हलवा: गीतपट्टा वर हलवते.
१७ १७ Move Track Down: Moves the track down. गीतपट्टा डाउन हलवा: गीतपट्टा खाली हलवते.
१८ १८ Move Track to Top: Moves track to become the topmost track in the project. गीतपट्टा वर जा: प्रकल्प मधील सर्वात वरचा गीतपट्टा होण्यासाठी गीतपट्टा हलवेल.
१९ १९ Move Track to Bottom: Moves track to become the bottom track in the project. गीतपट्टा तळाशी हलवा: प्रकल्पातील तळाशी गीतपट्टा होण्यासाठी गीतपट्टा हलवते.
२० २० Linear scale: Sets the vertical scale of Time Track to a linear display; this is the default. रेषात्मक स्केल: वेळ गीतपट्टाचे अनुलंब स्केल रेखीय प्रदर्शनावर सेट करते; हे पूर्वनिर्धारित आहे.
२१ २१ Logarithmic scale: Sets the vertical ruler to a logarithmic display. लोगारिथमिक स्केल: अनुलंब शासकास लॉगॅरिथमिक प्रदर्शनावर सेट करते.
२२ २२ Range...: Opens dialogs where you can first set the minimum and then the maximum speed change value to be displayed on the vertical scale. Values are expressed in percentages relative to the "no change" value of १००%. Only integer values (whole numbers) can be set. The lowest speed change value that can be set is १०% and the highest speed change value that can be set is १०००% . श्रेणी ...: आपण प्रथम अनुलंब स्केलवर प्रदर्शित करण्यासाठी किमान आणि नंतर जास्तीत जास्त वेग बदलण्याचे मूल्य सेट करू शकता अशा संवाद उघडतात. १००% च्या "न बदल" मूल्याच्या तुलनेत टक्केवारीत मूल्ये व्यक्त केली जातात. केवळ पूर्णांक मूल्ये (संपूर्ण संख्या) सेट केली जाऊ शकतात. सेट केले जाणारे सर्वात कमी वेग बदलण्याचे मूल्य १०% आहे आणि सेट केले जाणारे सर्वाधिक वेग बदलण्याचे मूल्य १०००% आहे.
२३ २३ Logarithmic Interpolation: The curve between control points is by default calculated using logarithmic interpolation. Uncheck this option to use linear interpolation between control points, so drawing an exactly straight line between points (assuming the vertical scale is set to "Linear"). लोगारिथमिक इंटरपोलेशन: कंट्रोल पॉईंट्स दरम्यानचे वक्र पूर्वनिर्धारितनुसार लोगारिथमिक इंटरपोलेशन वापरून मोजले जाते. नियंत्रण बिंदू दरम्यान रेषात्मक प्रक्षेप वापरण्यासाठी हा पर्याय अनचेक करा, म्हणून बिंदूंदरम्यान अगदी सरळ रेषा रेखाटणे (अनुलंब स्केल "रेखीय" असे सेट करणे गृहीत धरून).
२४ २४ Label tracks can also be moved up or down by clicking between the controls in the track's Track Control Panel then dragging upwards or downwards. गीतपट्टाच्या गीतपट्टा कंट्रोल पॅनेलमधील नियंत्रणे दरम्यान वर किंवा खाली ड्रॅग करून लेबल गीतपट्टा वर किंवा खाली हलविले जाऊ शकतात.
२५ २५ A Tutorial on Time Warping टाईम वॉरिंग वरील प्रशिक्षण
२६ २६ Key points about the image above: वरील प्रतिमेबद्दल मुख्य मुद्देः
२७ २७ The Time Track curves are drawn with default display types (linear vertical scaling with logarithmic interpolation). वेळ गीतपट्टाचे वक्र पूर्वनिर्धारित प्रदर्शन प्रकारांद्वारे काढले जातात (लॉगरिथमिक प्रक्षेप सह रेखीय अनुलंब स्केलिंग).
२८ २८ The range of the vertical scale has been increased to a lower limit of ५० and an upper limit of २००. अनुलंब स्केलची श्रेणी ५० च्या खालच्या मर्यादेपर्यंत वाढविली गेली आहे आणि २०० ची वरची मर्यादा.
२९ २९ The horizontal ruler underneath the Timeline warps as you change the warp line. It shows when the speed-changed playback will reach each time point in the audio. Thus the green playback cursor moves faster or slower than normal as it moves along Time Track's horizontal ruler. वेळलाइनच्या खाली क्षैतिज शासक वर्प लाइन बदलता तेव्हा. हे दर्शविते की वेगवान-बदललेला प्लेबॅक ध्वनीमधील प्रत्येक वेळी पोहचेल तेव्हा. टाईम गीतपट्टाच्या क्षैतिज शासकासह तो ग्रीन प्लेबॅक कर्सर सामान्यपेक्षा वेगवान किंवा हळू फिरतो.
३० ३० In our example: आमच्या उदाहरणात:
३१ ३१ we started playing twice as fast as normal आम्ही सामान्यपेक्षा दोनदा वेगवान खेळायला सुरुवात केली
३२ ३२ we slowed down fairly suddenly to a speed twice as slow as normal आम्ही सामान्यपेक्षा दुप्पट वेगात वेग वेगवान केला
३३ ३३ then increased very gradually to a steady speed a little faster than the normal speed of १००%.After all the changes, the resultant audio ends up with only a few seconds shorter playing time than it had before (about ३ minutes ४२ seconds, where the end of the audio lies against the Time Track horizontal ruler). नंतर १००% च्या सामान्य गतीपेक्षा थोडा वेगवान स्थिर वेगाने हळूहळू वाढला. सर्व बदलानंतर, परिणामी ध्वनी संपण्यापूर्वी फक्त काही सेकंद कमी वेळ घालवतात (सुमारे ३ मिनिट ४२ सेकंद, जिथे ध्वनीचा शेवट वेळ गीतपट्टा क्षैतिज शासकाच्या विरूद्ध आहे).
३४ ३४ Important Steps: महत्त्वपूर्ण चरणे:
३५ ३५ You must first have at least one audio track to manipulate. हाताळण्यासाठी आपल्याकडे प्रथम कमीत कमी एक ध्वनी गीतपट्टा असणे आवश्यक आहे.
३६ ३६ Choose Tracks > Add New > New Time Track. गीतपट्टा निवडा> नवीन जोडा> नवीन वेळ गीतपट्टा.
३७ ३७ If you want to warp the speed by more than plus or minus १०%, open the Time Track Dropdown Menu and choose Set Range.... The dialog will appear twice, the first time to choose the lower limit of the range and the second time to choose the upper limit of the range. जर आपल्याला वेगापेक्षा अधिक किंवा वजा १०% ने अधिक वेगाने वाढवायचे असेल तर वेळ गीतपट्टा ड्रॉपडाउन मेनू उघडा आणि श्रेणी सेट करा निवडा .... पहिल्यांदा श्रेणीची खालची मर्यादा निवडण्यासाठी आणि दुसरा संवाद निवडा. श्रेणीची वरची मर्यादा निवडण्याची वेळ.
३८ ३८ For example if you set a Lower Speed Limit of ५०%, you can make the audio play up to twice as slowly (an envelope point at the bottom of the Time Track would make the playback twice as slow, just like a Change Speed effect of -५०%). उदाहरणार्थ आपण कमी वेगवान मर्यादा ५०% निश्चित केल्यास आपण ध्वनी प्ले दुप्पट हळू करू शकता (वेळ गीतपट्टाच्या तळाशी असलेला एक लिफाफा बिंदू, बदलाच्या स्पीड प्रभावाप्रमाणेच दुप्पट प्लेबॅक करेल -५०%).
३९ ३९ If you set an Upper Speed Limit of २००%, you can make the audio play up to twice as fast (an envelope point at the top of the Time Track would double the playback speed, like a Change Speed effect of १००%). आपण २००% ची उच्च वेग मर्यादा सेट केल्यास आपण ध्वनी प्ले दुप्पट वेगवान बनवू शकता (वेळ गीतपट्टाच्या शीर्षस्थानी असलेला एक लिफाफा बिंदू प्लेबॅक गती दुप्पट करेल, जसे १००% चा वेगवान परिणाम).
४० ४० When you hover your cursor over the warp line, the cursor changes to the envelope cursor   , which lets you shape the time warp line. जेव्हा आपण आपला कर्सर जाळे ओळीवर फिरवाल, तेव्हा कर्सर लिफाफा कर्सरमध्ये बदलला जाईल, ज्यामुळे तुम्हाला टाईम वॅप लाइन आकारता येईल.
४१ ४१ Creating one envelope point only changes the speed of the whole track by the same amount, just like the Change Speed effect. To make a speed change that varies over time you must create at least one more point. Thereafter when you drag a point the line on each side of that point moves to reflect your change. एक लिफाफा बिंदू तयार करणे केवळ बदल गती प्रभावाप्रमाणेच संपूर्ण गीतपट्टाची गती समान प्रमाणात बदलवते. कालांतराने वेग बदलणारा वेग कमी करण्यासाठी आपण कमीतकमी आणखी एक बिंदू तयार केला पाहिजे. त्यानंतर जेव्हा आपण त्या बिंदूच्या प्रत्येक बाजूला बिंदू ड्रॅग करता तेव्हा आपला बदल प्रतिबिंबित करण्यासाठी हलविला जातो.
४२ ४२ You can press Space at any time to listen to how the warping sounds and change the points on the warp line further. वॉर्पिंग कसे दिसते हे ऐकण्यासाठी आपण कधीही स्पेस दाबू शकता आणि पुढे जाळे रेषावरील बिंदू बदलू शकता.
४३ ४३ If you wish to continue editing after completing your warp points, it is often advisable to click in the Track Control Panel of the audio track to select it (hold Shift and click to select multiple tracks) then Tracks > Mix > Mix and Render. This writes the resampled audio data directly to the project rather than resampling the speed-changed audio back to the current sample rate in real-time. This may therefore make project playback more responsive. जर आपण आपले रेप पॉइण्ट पूर्ण केल्यावर संपादन सुरू ठेवू इच्छित असाल तर ध्वनी गीतपट्टाच्या निवडक गीतपट्टा कंट्रोल पॅनेलमध्ये क्लिक करण्यासाठी (शिफ्ट दाबून एकाधिक गीतपट्टा निवडण्यासाठी क्लिक करा) नंतर गीतपट्टा> मिक्स> मिक्स आणि रेंडर क्लिक करावे. हे रीअल-वेळमधील वर्तमान नमुना दरापेक्षा वेगवान-बदललेला ध्वनी पुन्हा तयार करण्याऐवजी पुनर्प्रकाशाचा ध्वनी डेटा थेट प्रकल्पवर लिहितो. यामुळे प्रकल्प प्लेबॅक अधिक प्रतिसादात्मक होऊ शकेल.
४४ ४४ The quality setting of the resampling library used for playing or rendering Time Tracks is chosen automatically by Audacity. Therefore changing the converter quality settings in Quality Preferences will not affect Time Tracks. वेळ गीतपट्टा खेळण्यासाठी किंवा प्रस्तुत करण्यासाठी वापरल्या जाणार्‍या रीमॅम्पलिंग लायब्ररीची गुणवत्ता सेटिंग ओड्यासिटीद्वारे स्वयंचलितपणे निवडली जाते. म्हणून गुणवत्ता प्राधान्यांमधील कनव्हर्टर गुणवत्ता समायोजन बदलल्याने वेळ गीतपट्टावर परिणाम होणार नाही.
४५ ४५ To use the speed-changed audio in other applications, choose File > Export > Export Audio..., which opens the Export Audio Dialog. अन्य अनुप्रयोगांमध्ये वेग-बदललेला ध्वनी वापरण्यासाठी, धारिका> निर्यात> निर्यात ध्वनी ... निवडा, जे निर्यात ध्वनी संवाद उघडेल.
४६ ४६ Time Track is always active (and therefore modifies audio speed when playing or exporting) whenever it is visible in the project. This applies even after rendering the audio. Therefore after render (or as soon as you want to work with audio that you do not want to speed-change), use the [X] "Close" button to left of the Time Track to remove it. वेळ गीतपट्टा नेहमी सक्रिय असतो (आणि म्हणून प्ले करताना किंवा निर्यात करताना ध्वनी गती सुधारित करतो) जेव्हा तो प्रकल्पात दृश्यमान असेल. हे ध्वनी प्रस्तुत केल्यानंतरही लागू होते. म्हणूनच रेंडरनंतर (किंवा जसे की आपल्याला वेग-बदल नको असलेल्या ध्वनीसह कार्य करायचे आहे तितक्या लवकर), ते काढण्यासाठी वेळ गीतपट्टाच्या डावीकडील [एक्स] "बंद करा" बटण वापरा.
४७ ४७ Older computers may be incapable of playing audio in real-time that makes use of a Time Track. However you can always export the file as a WAV and then play the WAV file later. जुने संगणक रिअल-वेळमध्ये ध्वनी प्ले करण्यास अक्षम असू शकतात जे वेळ गीतपट्टाचा वापर करतात. तथापि आपण नेहमी डब्ल्यूएव्ही म्हणून धारिका निर्यात करू शकता आणि नंतर डब्ल्यूएव्ही धारिका नंतर प्ले करू शकता.
