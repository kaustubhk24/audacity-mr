२४ २४ Only labels in the uppermost label track are used for export. If more than one audio track is above the uppermost label track, the audio of all those tracks will be mixed into the exported files, unless you press the Mute button on some of the tracks to exclude them from the mix. केवळ सर्वात वरच्या नावपट्टी गीतपट्टामधील नावपट्टी निर्यात करण्यासाठी वापरली जातात. जर एकापेक्षा जास्त ध्वनी गीतपट्टा वरच्या नावपट्टी गीतपट्टाच्या वर असतील तर, त्या सर्व गीतपट्टाचा ध्वनी निर्यात केलेल्या धारिकांमध्ये मिसळला जाईल, जोपर्यंत आपण त्यापैकी काही गीतपट्टावर निःशब्द बटण दाबल्याशिवाय नाही.

२७ २७ • Region labels span each section of audio you want to make into a separate file. • प्रदेश नावपट्टीे आपण स्वतंत्र धारिका बनवू इच्छित असलेल्या ध्वनीच्या प्रत्येक भागाचे विभाजन करतात.

२९ २९ First file name: If you have checked "Include audio before first label", this option will be enabled. Type in the name you want to be used for the first exported audio file. If you have imported audio files, this text box contains the name of the first imported file, but you can still change the text. प्रथम धारिका नाव: आपण "प्रथम नावपट्टीच्या आधी ध्वनी समाविष्ट करा" तपासले असल्यास, हा पर्याय सक्षम केला जाईल. प्रथम निर्यात केलेल्या ध्वनी धारिकासाठी आपण वापरू इच्छित असलेल्या नावावर टाइप करा. आपण ध्वनी धारिका आयात केल्या असल्यास, या मजकूर बॉक्समध्ये प्रथम आयात केलेल्या धारिकाचे नाव आहे, परंतु आपण अद्याप मजकूर बदलू शकता.

31 31 Name files: धारिका नावे:
3२ 3२ Using Label/Track name: Each file will be named exactly as the text of each label if "Split files based on Labels" was selected, or exactly as the name of each track if "Split files based on Tracks" was selected. If any labels are empty (track names cannot be empty), Audacity will name each associated file as "untitled" and add a number as necessary to distinguish multiple "untitled" files. नावपट्टी / गीतपट्टा नाव वापरणे: "नावपट्टीवर आधारित स्प्लिट धारिका" निवडल्यास प्रत्येक धारिकाचे नाव नेमके असेल किंवा "गीतपट्टा्सवर आधारीत धारिका स्प्लिट" निवडल्यास प्रत्येक गीतपट्टाचे नाव नेमके असेल. कोणतीही नावपट्टी रिकामी असल्यास (गीतपट्टा नावे रिक्त असू शकत नाहीत), ओड्यासिटी प्रत्येक संबंधित धारिकाला "शीर्षक नसलेले" अशी नावे देईल आणि एकाधिक "शीर्षकहीन" धारिका वेगळे करण्यासाठी आवश्यक संख्या जोडा.
33 33 Numbering before Label/Track Name: Label or track names are used but a consecutive two-digit number and hyphen are prefixed to each file name. The numbering starts from "00-" if you chose "Include audio before first label", or from "01-" if you did not. This is a good choice if you are burning tracks to an Audio CD. नावपट्टी / गीतपट्टा नावाच्या आधी क्रमांकांकन: नावपट्टी किंवा गीतपट्टा नावे वापरली जातात परंतु प्रत्येक धारिकाच्या नावापुढे सलग दोन-अंकी क्रमांक आणि हायफन असतात. क्रमांकन "00-" पासून प्रारंभ होते जर आपण "प्रथम नावपट्टीपूर्वी ध्वनी समाविष्ट करा" किंवा आपण न केल्यास "01-" वरून प्रारंभ केले असेल. आपण ध्वनी सीडीवर गीतपट्टा जळत असल्यास ही एक चांगली निवड आहे.
3४ 3४ Examples for numbering before: If you had a label named "Reflection" marking the start of the first file, followed by a label marked "Rejoicing", then the exported files would be "01-Reflection" and "0२-Rejoicing"; If the labels were empty in the above case, the exported files would be "01-untitled" and "0२-untitled". आधी क्रमांक लावण्यासाठी उदाहरणे: आपल्याकडे प्रथम धारिकाच्या सुरूवातीस चिन्हांकित करणारे "प्रतिबिंब" नावाचे नावपट्टी असेल आणि त्यानंतर "आनंद" असे चिन्हांकित नावपट्टी असेल तर निर्यात केलेल्या धारिका "01-प्रतिबिंब" आणि "0२-आनंद" असतील; उपरोक्त प्रकरणात नावपट्टीे रिक्त असल्यास, निर्यात केलेल्या धारिका "01-अशीर्षकांकित" आणि "0२-शीर्षकविहीन" असतील.
3५ 3५ Numbering after File Name Prefix: Label or track names are ignored. Instead, each file is named using a standard prefix specified in "File Name Prefix" below followed by a hyphen and a consecutive two-digit number. The numbering starts from "-00" if you chose "Include audio before first label", or from "-01" if you did not. धारिका नाव उपसर्ग नंतर क्रमांकन: नावपट्टी किंवा गीतपट्टा नावे दुर्लक्षित केली जातात. त्याऐवजी, प्रत्येक धारिकाचे नाव खाली दिलेली "धारिका नेम उपसर्ग" मध्ये निर्दिष्ट मानक उपसर्ग वापरुन केले गेले, त्यानंतर हायफन आणि सलग दोन-अंकी क्रमांक. आपण "प्रथम नावपट्टीच्या आधी ध्वनी समाविष्ट करा" किंवा आपण न केल्यास "-01" वरून क्रमांकांकन "-00" पासून प्रारंभ होते.
3६ 3६ File Name Prefix: Specifies a file name prefix to be used which will be followed by consecutive numbering.
