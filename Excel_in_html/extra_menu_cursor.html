४ ४ The Extra > Cursor menu provides commands for moving either the editing cursor, playback cursor or an audio track or clip. अतिरिक्त> कर्सर मेनू संपादन कर्सर, प्लेबॅक कर्सर किंवा ध्वनी गीतपट्टा किंवा क्लिप एकतर हलविण्यासाठी आदेश प्रदान करते.

६ ६ Cursor Left  Left कर्सर डावे डावे
७ ७ When not playing audio, moves the editing cursor one screen pixel to left. When a Snap-To option is chosen, moves the cursor to the preceding unit of time as determined by the current selection format. If the key is held down, the cursor speed depends on the length of the tracks. When there is a selection, moves the cursor to the beginning of the selection, removes the selection, and scrolls the track to display the cursor (if needed). When playing audio, moves the playback cursor as described at "Cursor Short Jump Left" (below). जेव्हा ध्वनी प्ले होत नाही तेव्हा संपादन कर्सर एक स्क्रीन पिक्सेल डावीकडे हलवितो. जेव्हा स्नॅप-टू पर्याय निवडला जातो, तेव्हा वर्तमान निवडीच्या स्वरूपानुसार कर्सर त्यापूर्वीच्या पूर्वीच्या युनिटवर जाईल. की दाबून ठेवल्यास, कर्सर गती गीतपट्टाच्या लांबीवर अवलंबून असते. जेव्हा एखादी निवड असते, कर्सर निवडीच्या सुरूवातीस हलवते, निवड काढून टाकते आणि कर्सर प्रदर्शित करण्यासाठी गीतपट्टा स्क्रोल करतो (आवश्यक असल्यास). ध्वनी प्ले करताना, "कर्सर शॉर्ट जंप डावे" (खाली) मध्ये वर्णन केल्यानुसार प्लेबॅक कर्सर हलवा.
८ ८ Cursor Right  Right कर्सर राइट राइट
९ ९ When not playing audio, moves the editing cursor one screen pixel to right. When a Snap-To option is chosen, moves the cursor to the following unit of time as determined by the current selection format. If the key is held down, the cursor speed depends on the length of the tracks. When there is a selection, moves the cursor to the end of the selection, removes the selection, and scrolls the track to display the cursor (if needed). When playing audio, moves the playback cursor as described at "Cursor Short Jump Right" (below). जेव्हा ध्वनी प्ले होत नाही तेव्हा संपादन कर्सर एक स्क्रीन पिक्सेल उजवीकडे हलवितो. जेव्हा स्नॅप-टू पर्याय निवडला जातो, तेव्हा वर्तमान निवडीच्या स्वरूपानुसार कर्सर खालील वेळेच्या युनिटवर हलवेल. की दाबून ठेवल्यास, कर्सर गती गीतपट्टाच्या लांबीवर अवलंबून असते. जेव्हा निवड असते तेव्हा कर्सरला निवडीच्या शेवटी हलवते, निवड काढून टाकते आणि कर्सर प्रदर्शित करण्यासाठी गीतपट्टा स्क्रोल करतो (आवश्यक असल्यास). ध्वनी प्ले करताना, "कर्सर शॉर्ट जंप राइट" (खाली) मध्ये वर्णन केल्यानुसार प्लेबॅक कर्सर हलवा.
10 10 Cursor Short Jump Left  , कर्सर शॉर्ट जंप लेफ्ट,
11 11 When not playing audio, moves the editing cursor one second left by default. When playing audio, moves the playback cursor one second left by default. The default value can be changed by adjusting the "Short Period" under "Seek Time when playing" in Playback Preferences. जेव्हा ध्वनी प्ले होत नाही तेव्हा पूर्वनिर्धारितनुसार एक सेकंद बाकी संपादन कर्सर हलवेल. ध्वनी प्ले करताना, प्लेबॅक कर्सर पूर्वनिर्धारितनुसार एक सेकंद बाकी. पूर्वनिर्धारित मूल्य प्लेबॅक प्राधान्यांमध्ये "जेव्हा खेळण्याचा प्रयत्न करा" अंतर्गत "शॉर्ट पीरियड" समायोजित करुन बदलले जाऊ शकते.

13 13 When not playing audio, moves the editing cursor one second right by default. When playing audio, moves the playback cursor one second right by default. The default value can be changed by adjusting the "Short Period" under "Seek Time when playing" in Playback Preferences. जेव्हा ध्वनी प्ले होत नाही तेव्हा पूर्वनिर्धारितनुसार संपादन कर्सर एक सेकंद उजवीकडे हलविला जातो. ध्वनी प्ले करताना, प्लेबॅक कर्सर पूर्वनिर्धारितनुसार उजवीकडे एक सेकंद हलवेल. पूर्वनिर्धारित मूल्य प्लेबॅक प्राधान्यांमध्ये "जेव्हा खेळण्याचा प्रयत्न करा" अंतर्गत "शॉर्ट पीरियड" समायोजित करुन बदलले जाऊ शकते.

1८ 1८ When not playing audio and there is a selection region, all the above "Cursor" commands remove the region and place the editing cursor at the left or right boundary respectively of the former selection. जेव्हा ध्वनी प्ले होत नाही आणि तेथे निवड प्रदेश असतो तेव्हा वरील सर्व "कर्सर" आज्ञा्स प्रदेश हटवतात आणि संपादनाचा कर्सर आधीच्या निवडीच्या अनुक्रमे डावीकडे किंवा उजव्या सीमेवर ठेवतात.
1९ 1९ Clip Left डावीकडे क्लिप करा
20 20 Moves the currently focused audio track (or a separate clip in that track which contains the editing cursor or selection region) one screen pixel to left. सध्या फोकस केलेला ध्वनी गीतपट्टा (किंवा त्या गीतपट्टामधील एक वेगळी क्लिप ज्यात संपादन कर्सर किंवा निवड प्रदेश समाविष्ट आहे) एक स्क्रीन पिक्सेल डावीकडील.
21 21 Clip Right उजवीकडे क्लिप करा
22 22 Moves the currently focused audio track (or a separate clip in that track which contains the editing cursor or selection region) one screen pixel to right. सध्या स्क्रीनवर असलेला ध्वनी गीतपट्टा (किंवा त्या गीतपट्टामधील एक वेगळी क्लिप ज्यामध्ये संपादन कर्सर किंवा निवड प्रदेश आहे) एक स्क्रीन पिक्सेल उजवीकडील हलवेल.
