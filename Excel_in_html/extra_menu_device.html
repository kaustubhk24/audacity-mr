७ ७ Extra अतिरिक्त
८ ८ Displays the Select recording Device dialog for choosing the recording device, but only if the "Recording Device" dropdown menu in Device Toolbar has entries for devices. Otherwise, an recording error message will be displayed. ध्वनीमुद्रणउपकरण निवडण्यासाठी ध्वनीमुद्रणउपकरण निवडा संवाद संवाद प्रदर्शित करते, परंतु केवळ उपकरण साधनपट्टीमधील "ध्वनीमुद्रणउपकरण" ड्रॉपडाउन मेनूमध्ये उपकरणसाठी नोंदी असल्यास. अन्यथा, ध्वनीमुद्रणिंग त्रुटी संदेश दर्शविला जाईल.

1२ 1२ Change Audio Host  Shift + H ध्वनी होस्ट शिफ्ट + एच बदला
13 13 Extra अतिरिक्त
1४ 1४ Displays the Select Audio Host dialog for choosing the particular interface with which Audacity communicates with your chosen playback and recording devices. ओड्यासिटी आपल्या निवडलेल्या प्लेबॅक आणि ध्वनीमुद्रणउपकरणसह संप्रेषण करते त्या विशिष्ट संवादपटलची निवड करण्यासाठी ध्वनी होस्ट निवडा निवडा संवाद प्रदर्शित करते.
1५ 1५ Change Recording Channels  Shift + N ध्वनीमुद्रणवाहिनी बदला शिफ्ट + एन
1६ 1६ Extra अतिरिक्त
1७ 1७ Displays the Select Recording Channels dialog for choosing the number of channels to be recorded by the chosen recording device. निवडलेल्या ध्वनीमुद्रणउपकरणद्वारे ध्वनीमुद्रण करण्याच्या वाहिनीची संख्या निवडण्यासाठी ध्वनीमुद्रणवाहिनी निवडा निवडलेले संवाद प्रदर्शित करते.
