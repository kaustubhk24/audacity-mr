१ १ Tremolo ट्रेमोलो
२ २ From Audacity Development Manual ओड्यासिटी विकास माहितीपुस्तिकेकडून
३ ३ Jump to: navigation, search येथे जा: निर्देशक, शोध
४ ४ Tremolo modulates the volume of the selection at the depth and speed selected in the dialog. This is the same as the tremolo effect familiar to guitar and keyboard players. ट्रिमोलो संवादमधील निवडलेल्या खोली आणि गतीनुसार निवडीचे परिमाण बदलते. हे गिटार आणि कीबोर्ड प्लेयर्सना परिचित ट्रेमोलो इफेक्टसारखेच आहे.
५ ५ Accessed by: Effect > Tremolo... यावर प्रवेशः प्रभाव> ट्रेमोलो ...
६ ६ Waveform type वेव्हफॉर्म प्रकार
७ ७ Determines the "shape" of each tremolo (how it progresses between the lowest and highest volume as determined by the wetness level). Choice of sine, triangle, sawtooth, inverse sawtooth, square. प्रत्येक ट्रेमोलोचा "आकार" निर्धारित करतो (ओलावाच्या पातळीनुसार सर्वात कमी आणि सर्वोच्च व्हॉल्यूमच्या दरम्यान कसा प्रगती होतो). साइन ची निवड, त्रिकोण, सॉ टूथ, व्यस्त सॉ टूथ, स्क्वेअर.
८ ८ Starting phase (degrees) प्रारंभ चरण (अंश)
९ ९ Sets where to start the tremolo in the waveform cycle. The default (zero) starts the tremolo at the start of the cycle (as the waveform starts to rise from the lowest point). वेव्हफॉर्म सायकलमध्ये ट्रेमोलो कोठे सुरू करायचा ते सेट करते. पूर्वनिर्धारित (शून्य) चक्राच्या सुरूवातीस ट्रेमोलो सुरू होते (जसजसे वेव्हफॉर्म सर्वात कमी बिंदूपासून वर येण्यास सुरवात होते).
१० १० Wet level (percent) ओले पातळी (टक्के)
११ ११ Sets the depth of tremolo. ०% is no tremolo, १००% sweeps between zero and maximum volume based on the original amplitude level. ट्रेमोलोची खोली सेट करते. ०% हा ट्रेमोलो नाही, मूळ मोठेपणाच्या पातळीवर आधारित १००% शून्य आणि कमाल व्हॉल्यूम दरम्यान स्वीप करते.
१२ १२ Frequency (Hz) वारंवारता (हर्ट्ज)
१३ १३ Controls the speed of the oscillation; use higher frequencies for faster oscillation. दोलनची गती नियंत्रित करते; वेगवान दोलनसाठी उच्च वारंवारता वापरा.
१४ १४ Buttons बटणे
१५ १५ Clicking on the command buttons give the following results: आज्ञा बटणावर क्लिक केल्याने खालील परिणाम मिळतात:
१६ १६ Manage gives a dropdown menu enabling you to manage presets for the tool and to see some detail about the tool. For details see Manage presets व्यवस्थापित करा ड्रॉपडाउन मेनू आपल्याला साधनाचे पुर्वनिर्धारित सेट करण्यास आणि साधनबद्दल काही तपशील पाहण्यास सक्षम करते. तपशीलांसाठी पुर्वनिर्धारित्स व्यवस्थापित करा पहा
१७ १७ Preview plays a short preview of what the audio would sound like if the effect is applied with the current settings, without making actual changes to the audio. पूर्वावलोकन ध्वनीमध्ये वास्तविक बदल न करता, प्रभाव वर्तमान समायोजनसह लागू केल्यास ध्वनीला काय वाटेल हे एक लहान पूर्वावलोकन प्ले करते.
१८ १८ The length of preview is determined by your setting in Edit > Preferences > Playback, the default setting is ६ seconds. पूर्वावलोकनाची लांबी संपादन> प्राधान्ये> प्लेबॅक मधील आपल्या सेटिंगद्वारे निश्चित केली जाते, पूर्वनिर्धारित सेटिंग ६ सेकंद असते.
१९ १९ OK applies the effect to the selected audio with the current effect settings and closes the dialog ठीक आहे सध्याच्या प्रभाव समायोजनसह निवडलेल्या ध्वनीवर प्रभाव लागू करते आणि संवाद बंद करते
२० २० Cancel aborts the effect and leaves the audio unchanged, closing the dialog रद्द करा प्रभाव सोडतो आणि संवाद बंद केल्यामुळे ध्वनीमध्ये कोणताही बदल होत नाही
